## 1. 模型概述

Grok 4.3 是 xAI 最新发布的旗舰模型，于 2026 年 4 月 17 日发布。该模型是 xAI 迄今为止最智能、最快速的模型，专为 Grok 消费者服务场景设计。Grok 4.3 构建于 Grok 4.20/Grok 4.1 系列之上，增加了长上下文推理、多模态输入、原生视频理解和生成式文档输出等能力。

## 2. 核心功能

### 2.1 原生文档生成

Grok 4.3 支持直接从对话中生成可下载的格式化文档，包括：

- PDF 文档
- 电子表格（Spreadsheets）
- PowerPoint 幻灯片（Slide Decks）

这意味着用户不再需要手动复制粘贴文本，模型可以直接生成完整的输出文件，用户可以直接分享或提交。

### 2.2 原生视频理解

Grok 4.3 支持对话式视频理解，能够分析和理解视频输入内容，实现多模态交互。

### 2.3 语音 API

Grok 4.3 引入了两个独立的语音 API：

**Grok Speech-to-Text (STT) API**:

- 两种模式：批量模式（Batch）和实时模式（Real-time）
- 批量模式使用标准 REST 端点，返回带有时间戳和说话者信息的完整转录
- 实时模式使用 WebSocket 连接，在语音到达时推送部分转录
- 适用于实时字幕、会议笔记和语音代理界面

**Grok Text-to-Speech (TTS) API**:

- 将文本转换为语音
- 定价：$4.20/百万字符
- 比 OpenAI TTS（~$30/1M）低约 86%
- 比 ElevenLabs（~$50/1M）低约 92%

## 4. 定价信息

Grok 4.3 当前通过 SuperGrok Heavy 订阅提供，月费 $300/月。预计在 2026 年 5 月中下旬向更低订阅层级开放。

|------|------|--------------|
| Web Search | 搜索互联网和浏览网页 | $5 |

批量 API 允许异步处理大量请求，成本仅为标准定价的 20%-50%。

| | 实时 API | 批量 API |
|----------|---------|----------|

## 5. API 端点

| 端点类型 | URL | 描述 |
|----------|-----|------|
| Chat Completions | https://api.x.ai/v1/chat/completions | Grok 4 文本和多模态主要端点 |

API 兼容 OpenAI 和 Anthropic 的 SDK，迁移非常简单，只需生成 API 密钥并更改 URL。

## 6. 认证格式

所有请求需要 Bearer Token 认证：

```python
import requests

url = "https://api.x.ai/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "model": "grok-4",
    "messages": [
        {"role": "user", "content": "Your message here"}
    ],
    "max_tokens": 200
}
response = requests.post(url, headers=headers, json=data)
print(response.json())
```

## 7. 注意事项

1. **实时事件访问**: 未启用搜索工具时，Grok 无法获取当前事件或训练数据之外的信息。需要启用服务器端搜索工具（Web Search / X Search）来获取实时数据。

2. **角色顺序**: 聊天模型无角色顺序限制，可以在对话上下文中混合使用 `system`、`user` 或 `assistant` 角色。

3. **图像输入顺序**: 图像/文本输入顺序任意（例如，文本提示可以放在图像提示之前）。

## 8. 模型别名

- `<modelname>`: 指向最新稳定版本
- `<modelname>-latest`: 指向最新版本，适合希望访问最新功能的用户
- `<modelname>-<date>`: 直接引用特定模型发布版本，不会更新

## 9. 总结

Grok 4.3 是 xAI 的一次重大更新，引入了原生文档生成、视频理解和语音 API 等关键功能。该模型特别适合需要文档生成工作流程或语音 API 基准测试的开发者。$300/月的高级订阅门槛可能对部分开发者来说较高，但独立的语音 API 已经可用，值得进行基准测试。
