## 产品概述

CSS Studio 是一款运行在浏览器中的设计工具，可以连接到现有的 AI Agent（如 Claude），通过 MCP 协议将视觉编辑的更改实时同步到代码库。用户可以直接在网站上进行可视化设计，AI Agent 会自动编辑相应的代码。

**核心功能：**
- 文本编辑、样式调整、动画时间轴编辑器
- 通过 MCP 服务器与 AI Agent 通信
- 支持 Vite 等构建工具
- 可在开发者模式下编辑现有网站
- 价格：$99

## 主要反馈与建议

### 1. 营销展示问题（最多反馈）

**问题：**
- 落地页设计看起来像 AI 生成的，缺乏个性
- 视频演示不够明显，用户找不到
- 现场演示按钮位置不醒目（右上方），用户容易忽略

**建议：**
- 在标题下方添加 30 秒产品演示视频
- 添加明显的 "Try Live" 按钮引导用户进入编辑模式
- 考虑重新设计落地页，体现设计工具的专业性

### 2. 功能需求

**用户期望的功能：**
- **Diff 视图**：发布更改前显示将要修改的内容，让用户确认
- **Tailwind/CSS 框架集成**：自动识别并使用 Tailwind 类名或 CSS 变量，而非硬编码像素值
- **响应式断点视图**：同时查看多个屏幕尺寸的布局效果
- **撤销功能**：对元素设置 `display: none` 后可以恢复

### 3. 价格争议

部分用户质疑 $99 的定价，认为其他 AI 编码工具（如 Cursor/Windsurf）已包含类似功能。

### 4. 积极评价

- 概念创新，与现有 AI Agent 集成的方式很棒
- 与 Vite 项目无缝配合
- 体验流畅、界面美观
- 适合非技术人员进行小型设计修改

## 开发者响应

作者 SirHound 积极回应了社区反馈：
- 已在首页添加了视频演示
- 修复了价格显示问题（VAT 税收显示错误）
- 修复了博客日期显示问题

## 总结

CSS Studio 将可视化设计与 AI Agent 结合的概念具有创新性，解决了 AI 编码工具中设计调整困难的痛点。但用户普遍认为落地页设计和演示方式需要改进，以更直观地展示产品价值。
