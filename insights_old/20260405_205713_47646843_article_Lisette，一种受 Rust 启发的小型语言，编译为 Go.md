## 项目概述

Lisette 是一种受 Rust 启发的小型编程语言，可编译为 Go 代码。它结合了 Rust 的语法特性和 Go 的运行时优势。

## 核心特性

### 1. 语法特性（ Familiar）

- **与 Rust 相似的语法**：包括枚举、结构体、impl 块、模式匹配
- **表达式导向**：支持 if/else 表达式、代码块表达式
- **链式调用和 Lambda**：支持函数链式调用和闭包
- **接口和泛型**：支持接口定义和泛型函数
- **控制流**：支持 if let 和 let else 语法

### 2. 类型安全（Safe）

Lisette 在编译时捕获 Go 运行时问题：

- **模式匹配穷尽性检查**：确保所有枚举变体都被处理
- **不支持 nil**：使用 Option<T> 替代空值
- **Result 丢弃警告**：防止未处理的错误
- **私有类型检查**：防止私有类型暴露在公共 API 中
- **可变参数检查**：确保可变参数正确声明
- **结构体字段检查**：确保所有必需字段都被初始化

### 3. 人体工学（Ergonomic）

专为与 Go 生态系统互操作而设计：

- **管道操作符**：支持 |> 操作符进行函数链式调用
- **Try 块**：简化错误处理流程
- **并发支持**：通过 Channel 和 task 实现并发编程
- **Select 语句**：支持多通道选择
- **序列化属性**：支持 JSON 序列化配置（omitempty、string、skip 等）
- **Panic 恢复**：安全处理 panic
- **Defer 延迟**：支持 defer 语句

### 4. 透明性（Transparent）

Lisette 编译为可读的 Go 代码：

- 模式匹配编译为清晰的 if 语句
- Result 处理编译为显式的错误检查
- 代码输出结构化且易于理解

## 技术特点

- **代数数据类型**：支持枚举和结构体
- **模式匹配**：强大的模式匹配能力
- **默认不可变**：变量默认不可变，需使用 let mut
- **Hindley-Milner 类型系统**：强类型推断
- **Go 生态系统互操作**：无缝集成 Go 标准库
- **LSP 支持**：支持 VSCode、Neovim、Zed 编辑器

## 安装和使用

```bash
cargo install lisette
```

## 代码示例

```lisette
import "go:fmt"
import "go:os"

fn load_config(path: string) -> Result<Cfg, error> {
  let file = os.Open(path)?
  defer file.Close()
  let data = io.ReadAll(file)?
  parse_yaml(data)
}

fn main() {
  match load_config("app.yaml") {
    Ok(config) => start(config),
    Err(e) => fmt.Println("error:", e),
  }
}
```

## 总结

Lisette 是一个创新的实验性语言项目，它尝试将 Rust 的安全特性（模式匹配、代数数据类型、类型系统）带入 Go 生态系统。对于喜欢 Rust 语法但需要 Go 运行时优势的开发者来说，Lisette 提供了一个有趣的折中方案。
