## 项目概述

auto-identity-remove 是一个 macOS 上的自动化数据经纪人（data broker）退出运行器，能够从 **500+** 人搜索网站和数据库中移除用户的个人信息，按月定期执行，并支持 CAPTCHA 破解、人工操作引导和 iMessage 结果通知。

## 核心功能

### 1. 自动化流程
- 每月自动执行一次（通过 launchd 调度）
- 在每个数据经纪人网站上搜索用户姓名 + 州
- 找到用户个人列表（如需个人资料 URL）
- 自动填写并提交退出表单
- 通过 CapSolver（AI 驱动的 CAPTCHA 破解服务，成本约 $0.001/次）解决验证码
- 跳过 90 天内已退出的经纪人（避免重复提交）
- 通过 iMessage 发送结果摘要
- 自动打开需要手动操作的网站

### 2. 支持的经纪人规模

| 类别 | 数量 |
|------|------|
| 自动移除（内置策略） | 30+ |
| 自动检测（泛型运行器） | ~470 |
| 总计 | **500+** |

**内置自动移除（30+）**：Spokeo、WhitePages、FastPeopleSearch、TruePeopleSearch、BeenVerified、Radaris、Intelius、PeopleFinders、PeopleSmart、MyLife、Nuwber、FamilyTreeNow、CheckPeople、ThatsThem、USPhonebook、PublicDataUSA、SmartBackgroundChecks、SearchPeopleFree、PeopleSearchNow、InfoTracer、SocialCatfish、NationalPublicData、ClustrMaps、PrivateRecords、Acxiom、LexisNexis、ZoomInfo、Clearbit、Pipl 等。

**泛型运行器（~470）**：来自 The Markup 提供的 494 个退出 URL 列表和 BADBOOL 社区维护列表，自动尝试以下 4 种策略：
1. 点击"Do Not Sell My Personal Information"按钮
2. 通过 OneTrust / TrustArc / Osano 隐私管理器退出
3. 填写通用退出表单（邮箱、姓名、州）并提交
4. 查找并记录 DSAR / 数据请求链接供手动跟进

### 3. 状态跟踪机制
`state.json` 跟踪每个经纪人的退出状态，包括上次成功退出时间和总运行次数。状态结果包含：✅ 已移除、⏭ 已跳过（新鲜）、🔍 未列出、📋 需手动、❌ 错误。

### 4. 隐私保护
用户的个人信息（`config.json`）和状态文件（`state.json`）均通过 `.gitignore` 排除，不会离开用户本机。
