## 项目概述

**Project NOMAD** (Node for Offline Media, Archives, and Data) 是一个免费、开源的离线服务器，可以在任何计算机上安装。下载内容后即可永久离线使用——类似产品收费数百美元，而 Project NOMAD 完全免费。

## 核心功能

### 1. 离线知识库
- Wikipedia（维基百科）
- 医学参考资料
- 维修指南
- Project Gutenberg（古登堡计划）

### 2. 本地AI助手
- 使用 Ollama 驱动
- 完全离线运行大型语言模型
- 支持聊天、写作、分析、编程

### 3. 离线地图
- 使用 OpenStreetMap 数据
- 无需蜂窝网络即可导航和规划路线

### 4. 教育平台
- Khan Academy 课程
- 教育视频
- 互动课程
- 完整的 K-12 课程资源

## 目标用户

### 应急准备人群
当基础设施故障时，NOMAD 仍可工作——提供医学参考、生存指南和百科知识，无需互联网。

### 离网生活人群
在 cabin、房车或帆船上使用——随时携带完整图书馆、AI 助手和离线地图，实现真正的数字独立。

### 技术爱好者
运行本地 LLM、自托管知识库、完全掌控数据。

### 教育场景
Khan Academy、Wikipedia for Schools 等——为任何地方的家庭提供完整学习资源。

## 推荐配置

- **处理器**: AMD Ryzen 7 (Radeon显卡) 或 Intel i7+
- **内存**: 32 GB RAM
- **显卡**: AMD Radeon 780M+ 或 NVIDIA 独立GPU
- **存储**: 1 TB SSD
- **系统**: Ubuntu 22.04+ 或 Debian 12+

## 安装方式

两条命令即可在 Ubuntu/Debian 上安装：

```bash
curl -fsSL https://raw.githubusercontent.com/Crosstalk-Solutions/project-nomad/main/install/install_nomad.sh -o install_nomad.sh
sudo bash install_nomad.sh
```

## 总结

Project NOMAD 是一个功能强大的免费开源离线知识与AI服务器，集成 Wikipedia、本地LLM、离线地图和 Khan Academy 教育资源。与其他收费数百美元的竞品相比，NOMAD 完全免费且功能更全面，支持任意PC硬件（而非树莓派），非常适合应急准备、离网生活和技术爱好者使用。
