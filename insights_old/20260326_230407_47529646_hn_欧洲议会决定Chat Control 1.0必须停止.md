## 摘要

欧洲议会投票否决了Chat Control 1.0的延期提案，这是一项临时性的法规，允许对私人消息进行大规模扫描以打击儿童性虐待内容。

## 详细内容

### 什么是Chat Control 1.0？

Chat Control 1.0是一项临时性的法规，实际上是将Facebook等科技公司已经在做的事情合法化。自2021年以来，Chat Control 1.0一直在自愿的基础上运行。该法规原定于2026年4月4日到期，届时再次扫描欧洲公民的私人信息将再次变为非法行为。

### 投票结果

- **最终投票**: 36%赞成，49%反对 → 提案被否决
- **修正案投票**: 仅以一票之差（具体为赞成291票、反对290票）

### 各方立场

- **左翼政党**: 支持修正案，反对整体提案
- **EPP（欧洲人民党）**: 反对修正案和整体提案，因为他们希望进行无差别扫描而非针对性扫描
- **S&D（社会民主党）**: 两项都支持
- **德国 MEPs**: 压倒性反对
- **法国、匈牙利、波兰、爱尔兰**: 大多数支持该提案

### 后续展望

1. **Chat Control 2.0**: 永久性的"预防和打击儿童性虐待法规"的三方谈判仍在继续，欧盟各国政府继续坚持其"自愿"无差别扫描的要求

2. **年龄验证**: 下一个重大威胁是立法者将谈判是否强制要求Messenger和聊天服务以及应用商店实施年龄验证，这将需要用户提供身份证件或提交面部扫描

### 关键评论

- 有评论指出，Chat Control 1.0的真正目的可能不是为了 catch pedophiles，而是为了给警察更广泛的 surveillance 权力
- 有人担心这可能被欧盟一些更威权的政府滥用
- 也有人认为，即使这次失败了，Chat Control 2.0或后续版本仍将卷土重来
