## 基本信息

- **项目名称**：Webminal
- **运行时间**：15年（自2011年起）
- **服务器配置**：单台 CentOS Linux 服务器，8GB RAM
- **用户规模**：50万注册用户
- **收入模式**：零收入，纯免费运营

## 核心亮点

### 极简架构
- 单服务器运行，无 Kubernetes、无微服务、无自动扩展
- 15年来始终如一的单机架构

### 历经磨难依然存活

## 特色功能

### 1. Root Lab（根实验室）
- 使用 User Mode Linux 提供完整内核
- 每个用户获得：
  - 完整 Linux 内核（非容器，是真正的内核）
  - 4个虚拟块设备（各64MB），用于练习 LVM、RAID、fdisk
  - 256MB RAM
  - 写时复制（COW）overlay - 共享黄金镜像

### 2. Live Command Ticker（实时命令滚动条）
- 首页滚动的命令是真实的
- 基于 eBPF（execsnoop）实时追踪
- 已记录超过2800万条命令执行

### 3. 全新现代UI
- 干净、快速
- 无 Bootstrap 或 jQuery CDN 依赖
- 自托管字体，响应式设计

## 技术决策

### 为什么坚持使用 Shellinabox？
曾尝试替换为现代 WebSocket 终端，但几小时后出现黑屏和 Firefox 兼容性问题。Shellinabox 虽然古老、丑陋、缓慢，但它能穿透任何防火墙、代理和企业网络。团队选择回归：**有时候老东西才是正确的选择**。

### 为什么使用 User Mode Linux？
- Docker 无法提供真实的块设备
- 当学生输入 `fdisk /dev/sdb` 时，需要真正的块设备
- UML 可以做到
- COW overlay 使100个并发用户仅增加约2GB磁盘空间

## 成本与困境

- **零收入**：无广告、无追踪、无风险投资
- 服务器费用自掏腰包
- 曾多次想放弃：
  - 15年是很长的时间
  - 待业期间看着储蓄减少，服务器账单不断
  - 申请 YC 被拒绝
  - 尝试商业化（PayPal、Stripe、付费计划）均失败
  - 最需要 Webminal 的用户是付不起每月4美元的学生
