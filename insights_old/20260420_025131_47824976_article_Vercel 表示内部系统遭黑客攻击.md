## 事件概述

Vercel 是一个广受欢迎的云平台，用于开发和部署应用。该公司于周日披露其内部系统遭受入侵，**部分客户受到影响**。

## 关键信息

### 1. 受影响范围
- **“有限数量的客户”** 可能受到影响
- 公司已经直接联系受影响客户

### 2. 攻击者关联
- 有公开线索将此次入侵与 **ShinyHunters** 威胁组织联系起来
- 该组织通常通过社会工程和漏洞利用组合手段攻击目标
- 经常对受害公司提出资金要求，并在网络论坛出售窃取的数据和系统访问权限

### 3. Vercel 建议的防护措施
- 检查活动日志中是否存在可疑活动
- **轮换环境变量** 作为预防措施
- 使用敏感环境变量功能将 API 密钥等标记为敏感，Vercel 会以不可读格式存储

### 4. 公司回应
> “我们已经识别到涉及未经授权访问某些内部 Vercel 系统的安全事件。我们正在积极调查，并已聘请事件响应专家帮助调查和修复。我们已通知执法部门，并将随着调查进展更新此页面。”

## 待确认信息
- 具体哪些系统被入侵
- 受影响客户的具体数量

## 安全建议

1. 检查 Vercel 账户的活动日志
2. 立即轮换环境变量和 API 密钥
3. 启用敏感环境变量功能
4. 关注 Vercel 官方公告
