## 项目概述

Sheets 是一个终端电子表格工具，允许用户在命令行界面中查看、编辑和处理 CSV 文件。它提供了一个类 Vim 的文本用户界面（TUI），让熟悉 Vim 操作的用户能够高效地进行电子表格操作。

## 主要功能

### 1. 命令行界面

- 直接打开 CSV 文件：`sheets budget.csv`
- 从标准输入读取数据：`sheets <<< ID,Name,Age`
- 读取特定单元格：`sheets budget.csv B9`
- 读取单元格范围：`sheets budget.csv B1:B3`
- 修改单元格：`sheets budget.csv B7=10 B8=20`

### 2. 导航功能

- `h, j, k, l`：移动活动单元格
- `gg, G, 5G, gB9`：跳转到顶部、底部、指定行或指定单元格
- `0, ^, $`：跳转到行首、行中第一个非空列、行尾
- `H, M, L`：跳转到可视区域的顶部、中部、底部
- `ctrl+u, ctrl+d`：向上或向下移动半页
- `zt, zz, zb`：将当前行对齐到窗口顶部、中部或底部
- `/, ?`：向前或向后搜索
- `n, N`：重复上次搜索
- `ma, 'a`：设置标记并跳转
- `ctrl+o, ctrl+i`：在跳转历史中后退或前进

### 3. 编辑与选择功能

- `i, I, c`：编辑当前单元格、从开头编辑或清空单元格并编辑
- `ESC`：退出插入、可视或命令模式
- `enter, tab, shift+tab`：提交并向下、向右或向左移动
- `o, O`：在下方或上方插入一行并开始编辑
- `v, V`：开始单元格选择或行选择
- `y, yy`：复制当前单元格或整行
- `x, p`：剪切当前单元格或选区并粘贴
- `dd`：删除当前行
- `u, ctrl+r, U`：撤销和重做
- `.`：重复上次更改

### 4. 可视模式

- `=`：在可视模式下，在选区后插入公式 `=|(B1:B8)`

### 5. 命令模式

按 `:` 打开命令提示符，支持以下命令：
- `:w` - 保存
- `:w path.csv` - 另存为
- `:e path.csv` - 打开其他 CSV 文件
- `:q` 或 `:wq` - 退出
- `:goto B9` 或 `:B9` - 跳转到指定单元格

## 安装方式

```bash

```

或从 releases 页面下载二进制文件。

- 纯 Go 语言实现
- 轻量级终端应用
- 支持 CSV 格式
- Vim 风格的操作界面
- 支持公式计算（可视模式）

- 开发者日常处理 CSV 数据
- 需要在终端环境下快速查看和编辑表格数据

- 自动化脚本中的数据处理

## 总结

Sheets 是一个简洁实用的终端电子表格工具，特别适合需要在命令行环境下处理 CSV 文件的用户。其 Vim 风格的操作界面对于熟悉 Vim 的用户来说学习成本很低，能够显著提升数据处理效率。
