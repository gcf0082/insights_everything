## 核心内容

2026年4月27日，Microsoft 与 OpenAI 宣布结束双方的独家合作协议和收入分成安排。这一重大变化引发了广泛讨论，主要集中在以下几个方面：

### 协议主要变化

1. **收入分成终止**: Microsoft 不再向 OpenAI 支付收入分成
2. **独家地位取消**: OpenAI 现在可以在其他云服务商（如 AWS、Google Cloud）销售其产品
3. **技术授权延续**: Microsoft 保留对 OpenAI 技术的访问权至2032年
4. **OpenAI 付款义务**: OpenAI 需继续向 Microsoft 付款至2030年，并设有上限
5. **持股情况**: Microsoft 仍持有 OpenAI 27% 的股份

---

## 讨论焦点分析

### 1. AGI（通用人工智能）争议

这是讨论中最热门的话题。许多评论者质疑 OpenAI 对 AGI 的定义和承诺：

- **"宗教末世论"**: 有评论将 AGI 描述为一种"宗教末世论"，认为它更像是炒作而非科学事实
- **定义模糊**: AGI 被 OpenAI 定义为"在大多数经济价值工作上超越人类的自主系统"——这一定义被认为过于经济导向而非技术导向
- **历史对比**: 有用户指出，2002年 Microsoft 就曾表示 AGI 问题已"解决"，只是硬件跟不上，预计10年内实现——如今2026年了仍未实现

### 2. 投资逻辑质疑

评论者对 AI 投资的合理性提出质疑：

- 如果 AGI 10年内无法实现，这些投资是否合理？
- 有评论指出，这类似于 Uber 当年对自动驾驶汽车的炒作承诺
- 所谓的"AGI 聊天"主要面向散户投资者和播客主持人，而非机构投资者

### 3. 循环资金现象

有评论指出 Microsoft 和 OpenAI 之间存在"循环经济"问题：

- OpenAI 承诺购买 250 亿美元的 Azure 服务
- Microsoft 投资 132 亿美元持有 OpenAI 股份
- 这与 NVIDIA 和 OpenAI 之间数十亿美元的来回资金类似

### 4. 对 Google Cloud 的影响

讨论中的一个有趣观点是：

- Google Cloud 可能成为唯一能转售三大 AI 实验室（OpenAI、Anthropic、Google）模型的云服务商
- 这对 Gemini 可能不利，因为其只能在 Google Cloud 上使用

### 5. 微软持股变化

- Microsoft 最初持有 OpenAI 49% 股份
- 由于 OpenAI 发行新股，股份被稀释至 27%
- 有评论认为这是"稀释"而非"出售"

## 总结

这场讨论反映了科技界对 AI 行业现状的深层担忧：

1. **AGI 承诺的可信度**: 许多人认为 AGI 更像是一个营销概念而非可实现的目标
2. **商业模式可持续性**: 循环资金流动和昂贵的 AI 基础设施投资引发质疑
3. **合作关系变化**: Microsoft 从独家合作伙伴变为非独家云服务商，这一转变的意义值得观察
4. **行业竞争格局**: 非独家安排可能改变 AI 云计算市场的竞争态势
