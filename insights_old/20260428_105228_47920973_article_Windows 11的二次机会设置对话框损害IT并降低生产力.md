## 核心观点

Windows 11的"二次开箱体验"（SCOOBE）是一个具有欺骗性的功能，它在用户首次设置PC数月甚至数年后突然弹出，强迫用户订阅Microsoft的付费服务，如Xbox Game Pass Premium（每月14.99美元）、Office 365等。这一做法不仅令人厌恶，更对企业IT管理造成严重影响。

## 问题分析

### 1. SCOOBE的工作机制
- 可以在用户首次设置PC后数月或数年后触发
- 常在Windows更新后出现
- 用户可能在其PC生命周期内多次遇到
- 使用诱导性UI试图让用户订阅付费服务或更改系统设置

### 2. 推送的服务内容
- **推荐浏览器设置** - 诱导使用Microsoft Edge
- **手机连接** - 促使 Linking 手机与PC
- **Office订阅** - 向没有Office的用户推销Office 365
- **Xbox Game Pass Premium** - 每月$14.99（美国）
- **Windows使用技巧** - 浏览器中打开不必要的提示

### 3. 对企业IT的影响
- **支持成本增加**：员工误以为PC出现问题，开具支持工单
- **生产力损失**：每次弹出都是对工作流程的干扰
- **钓鱼风险**：员工可能被诱导安装IT部门不同意的软件
- **用户体验破坏**：损害用户对设备的信任

### 4. 真实案例
- **Unidata公司**：SCOOBE在数据收集项目活跃时弹出，导致一周内出现多个相同支持工单
- **Flowers N Baskets花店**：前台PC在婚宴客户咨询中途被SCOOBE劫持，推销Office订阅
- **小型营销机构**：每个员工的每分钟都至关重要，干扰影响显著

## 禁用SCOOBE的方法

### 个人用户
- 设置 → 系统 → 通知 → 额外设置
- 取消勾选："建议 ways to get the most out of Windows and finish setting up this device"

### IT管理员（组策略）
- 计算机配置 → 管理模板 → Windows组件 → 云内容
- 启用"关闭Microsoft消费者体验"
- 同时启用"不显示Windows提示"
- 检查任务计划程序，禁用"UserNotPresentOrFirstLogon"任务

## 结论

Microsoft将订阅利润置于已购买Windows授权的企业需求之上。这种行为是"垃圾化"（enshittification）的典型例证。企业在乎SCOOBE是因为切换操作系统成本高昂且不便，但对于小型组织影响更为严重。如果Apple在macOS更新中强迫用户订阅iCloud和App Store，可能会面临国会听证。
