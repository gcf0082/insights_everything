## 项目概述

Endless Toil 是一款创意十足的 AI 编程助手插件，中文可译为"无尽的苦役"。它的核心理念非常幽默：**让 AI 编程助手在阅读糟糕代码时发出人类呻吟声**，随着代码质量下降，声音的"痛苦程度"会不断升级。

---

## 核心功能

### 主要功能

- **实时监测代码质量**：Endless Toil 会在 AI 编程助手（如 Claude、Codex、Cursor）读取代码时进行实时分析
- **分级音频反馈**：根据代码的"糟糕程度"播放不同级别的人类呻吟和痛苦声音
- **声音分级体系**：包含 groan（呻吟）、wail（哀号）、abyss（深渊）等多个痛苦等级
- **跨平台支持**：支持 Codex Desktop、Codex CLI、Claude CLI、Cursor 等主流 AI 编程工具

### 集成方式

| 平台 | 集成方式 |
|------|---------|
| Codex Desktop | 打开插件页面搜索 Endless Toil 并添加 |
| Codex CLI | `codex plugin marketplace add ./` 后通过 /plugins 安装 |
| Claude CLI | `/plugin marketplace add ./` 后 `/plugin install endless-toil@endless-toil` |
| Cursor | 添加为本地插件市场，调用 `/endless-toil` 技能 |

## 使用方法

### 基本使用

1. 克隆仓库到本地
2. 根据使用的 AI 编程工具安装插件
3. 开启新线程或对话
4. 告诉 AI 使用 `Endless Toil`

### 测试声音

```bash
# 列出所有可用声音
python3 plugins/endless-toil/skills/endless-toil/scripts/test_sounds.py --list

# 测试特定声音
python3 plugins/endless-toil/skills/endless-toil/scripts/test_sounds.py groan wail abyss
```

## 特色亮点

### 幽默创意

这是一个**高度原创的幽默项目**，在严肃的 AI 编程工具中加入了一个"痛苦音效层"，用讽刺的方式表达对糟糕代码的无奈，非常契合程序员文化中"Cursed Code"（被诅咒的代码）的自嘲精神。

### 实用价值

虽然以娱乐为主，但如果 AI 助手在协作中能发出"痛苦呻吟"，可以：
- **提醒开发者**：代码可能需要重构
- **增加交互趣味性**：让 AI 编程过程更有趣
- **代码质量意识**：用声音强化"代码可读性"的重要性

## 总结

Endless Toil 是一个极具创意的开源项目，通过将"人类痛苦呻吟"与 AI 编程助手结合，为枯燥的代码审查过程带来了独特的幽默感。它展示了开源社区在 AI 辅助编程领域的创意探索，不仅是技术工具，更是一种程序员文化的延伸。虽然功能简单，但其幽默创意和对开发者体验的关注值得关注。
