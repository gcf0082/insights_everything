## 项目概述

build123d 是一个基于 Python 的参数化边界表示 (BREP) 建模框架，用于 2D 和 3D CAD 设计。它构建于 [Open Cascade](https://dev.opencascade.org/) 几何内核之上，提供了一个清晰、完全符合 Python 风格的接口，用于创建精确的模型，适用于 3D 打印、CNC 加工、激光切割和其他制造工艺。

## 核心特性

### 现代化设计理念

- **极简状态管理**：根据模式的不同，需要最少或无需内部状态
- **显式几何类**：提供明确定义的 1D、2D 和 3D 几何类及其操作
- **可扩展性**：通过子类化和函数组合进行扩展，无需猴子补丁
- **标准合规**：遵循 PEP 8 规范，包含丰富的类型提示

### 运算符驱动建模

- 使用代数运算符进行建模（如 `obj += sub_obj`）
- 支持 `Plane.XZ * Pos(X=5) * Rectangle(1, 1)` 形式的链式调用
- 实现代数化、可读性和可组合的设计逻辑

### 数据格式支持

支持导入导出多种数据格式：
- SVG 导入
- STEP 导入/导出
- STL 导出
- 支持与 FreeCAD、SolidWorks 等主流 CAD 工具互操作

## 两种建模模式

### 1. 代数模式 (Algebra Mode)

遵循无状态范式，每个对象由代数运算符显式跟踪和操作：

```python
line = Line((0, -3), (6, -3))
line += JernArc(line @ 1, line % 1, radius=3, arc_size=180)
sketch = make_hull(line.edges())
part = extrude(sketch, amount=2)
```

### 2. 构建器模式 (Builder Mode)

跟踪状态并以设计历史的方式组织，每个维度都是独立的：

```python
with BuildPart() as part_context:
    with BuildSketch() as sketch:
        with BuildLine() as line:
            l1 = Line((0, -3), (6, -3))
            l2 = JernArc(l1 @ 1, l1 % 1, radius=3, arc_size=180)
        make_face()
    extrude(amount=2)
```
