## 摘要

本文介绍了一系列实用的 Shell 技巧，旨在提高命令行操作效率。这些技巧分为两类：适用于几乎所有 POSIX 兼容 Shell 的通用技巧，以及仅适用于 Bash 或 Zsh 的增强功能。

---

## 通用技巧（适用于大多数 Shell）

### 1. 文本编辑快捷键

- **Ctrl + W**：删除光标前的整个单词
- **Ctrl + U**：删除光标到行首的所有内容
- **Ctrl + K**：删除光标到行尾的所有内容
- **Ctrl + Y**：粘贴之前剪切的内容
- **Ctrl + A**：跳转到行首
- **Ctrl + E**：跳转到行尾
- **Alt + B**：向后移动一个单词
- **Alt + F**：向前移动一个单词

### 2. 终端恢复

- **`reset`** 或 **`stty sane`**：当不小心 `cat` 了二进制文件导致终端显示乱码时，使用此命令恢复终端正常状态

### 3. 紧急退出

- **Ctrl + C**：立即取消当前命令
- **Ctrl + D**：发送 EOF 信号，空行时退出 Shell

### 4. 屏幕清理

- **Ctrl + L**：清除屏幕内容，与 `clear` 命令类似，但不会中断当前正在输入的命令

### 5. 目录切换

- **`cd -`**：切换到上一个目录
- **`pushd` 和 `popd`**：目录栈操作，方便在多个目录之间切换

### 6. 文件操作

- **`> file.txt`**：清空文件内容，同时保留文件权限和所有权

### 7. 上一条命令的参数

- **`$_`**：获取上一条命令的最后一个参数

### 8. 脚本安全选项

- **`set -e`**：命令出错时立即退出
- **`set -u`**：引用未定义的变量时报错

## Bash 和 Zsh 专用技巧

### 1. 历史搜索

- **Ctrl + R**：反向增量搜索历史命令

### 2. 上一条命令复用

- **`!!`**：展开为上一条完整命令，常用于 sudo 权限不足时（如 `sudo !!`）

### 3. 编辑器模式

- **Ctrl + X 然后 Ctrl + E**：将当前命令放入默认编辑器进行编辑
- **`fc`**：打开上一条命令进行编辑

### 4. 参数获取

- **Alt + .** 或 **ESC + .**：插入上一条命令的最后一个参数
- **`!$`**：获取上一条命令的最后一个参数（直接展开）

### 5. 大括号展开

- **`cp file{,.bak}`**：展开为 `cp file file.bak`
- **`mv file.{txt,md}`**：展开为 `mv file.txt file.md`
- **`mkdir -p project/{src,tests,docs}`**：同时创建多个目录

### 6. 进程替换

- **`diff <(sort file1) <(sort file2)`**：对命令输出进行对比，无需创建临时文件

### 7. 递归 Glob

- **`shopt -s globstar`**（Bash 启用后）：使用 `**/*.js` 递归查找文件

### 8. 后台任务

1. **Ctrl + Z**：暂停当前进程
2. **`bg`**：将暂停的进程放到后台继续运行
3. **`disown`**：将进程从当前 Shell 分离，即使关闭终端也能继续运行

### 9. 输出同时显示和记录

- **`command |& tee file.log`**：同时输出到屏幕和日志文件（包括 stderr）

## 总结

Shell 是一个强大的工具箱，本文介绍的技巧可以帮助你更高效地使用命令行。建议每次只学习一个技巧，强制自己在日常工作中使用一周，然后再学习下一个。长期坚持这些习惯将显著提升你的终端使用体验。
