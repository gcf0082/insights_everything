## 内容摘要

RedSun 是一个用于在 Windows 11/10 和 Server 上获取 SYSTEM 用户权限的工具，适配 2026年4月更新。同时，作者还提供了另一个工具 UnDefend，可以自动关闭 Windows Defender（支持被动和主动两种模式）。

## 热点讨论

### 1. Windows Defender 权限争议

评论者对 Windows Defender 拥有修改系统文件的特权表示担忧：
- Defender 可以读取系统文件进行分析
- 还可以重置/替换文件
- 这种"写"权限被视为安全隐患

传统上，Windows 上的杀毒软件都以 SYSTEM 身份运行（甚至有内核模式驱动），这带来了攻击面。卡巴斯基和迈克菲都曾出现过权限提升漏洞。

### 2. 杀毒软件的困境

- 杀毒软件需要高级权限才能有效检测和清除恶意软件
- 如果恶意软件利用权限提升漏洞，杀毒软件能做什么？
- 有评论提到 Windows 有 SeBackupPrivilege 给备份软件，为什么不能给杀毒软件类似的权限？
- 理想情况下，操作系统应该完成 SYSTEM 级别的操作，而扫描和行为分析应在沙箱化的低权限进程中运行

### 3. 怀旧讨论：Windows 辉煌历史

部分评论转向对 Windows 历史的怀旧：
- 提到 Windows XP SP2 是传奇版本
- 当时 IE 是最流行的浏览器
- MSN、Winamp、Nero、eMule 等软件时代的回忆
- 感慨现代聊天程序（如 Slack）内存占用高达 1GB，而 Miranda NG 只需 30MB

### 4. Linux vs Windows 安全对比

- 有评论提到 Linux 每周都有本地权限提升 CVE
- 回应称 Linux 只有在以 root 运行守护进程时才会有问题（这是错误的做法）
- 讨论了 Docker 默认配置的 UID 映射问题

### 5. 工具检测状态

- Windows Defender 已更新特征码，可以检测该工具
- 检测名称: Program:Win32/Wacapew.C!ml
- 但评论者质疑这是真正修复问题还是只是特殊处理
