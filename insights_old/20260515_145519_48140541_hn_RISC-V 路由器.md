## 内容摘要

Start9 公司推出了一款基于 RISC-V 处理器的路由器，运行 StartWRT（OpenWRT 的分支），配备现代 GUI，旨在从头重新定义路由器体验。

## 主要讨论观点

### 负面观点

1. **价格与规格不匹配**
   - 众筹目标 $250,000，最终用户价格约 $300
   - 只有 1 个 WAN 和 1 个 LAN 千兆端口，在 2026 年显得落后
   - 需要至少 2 个 SFP+ 端口才能满足未来需求
   - BananaPi BPI-F3 采用相同 CPU，售价仅约 $100

2. **软件分支问题**
   - 定制 OpenWRT 分支会增加长期维护负担
   - 建议与上游合作改进现有 GUI，而非创建分支
   - 小型初创公司过早定制软件栈风险高（参考众多失败众筹案例）

3. **功能限制**
   - 只有 2 个网口，无法满足需要备份 WAN 或 DMZ 的用户
   - 使用 VLAN 会增加额外网络设备成本和复杂度
   - 现代 GUI 虽然简洁，但可能无法满足高级用户的配置需求

4. **性能担忧**
   - RISC-V 处理器性能是否能稳定处理千兆网络流量+防火墙+路由
   - 缺少硬件加速单元，理论上可能成为瓶颈

5. **政策风险**
   - 使用中国制造的处理器和 WiFi 模块，可能受特朗普 FCC 外国制造路由器禁令影响

### 正面观点

1. **安全优势**
   - RISC-V 没有 Intel IME/AMT、AMD PSP 或 ARM TrustZone 等后门
   - 完全开源的引导堆栈（OpenSBI、U-Boot）和 Linux 内核
   - 减少对单一项目（OpenWRT）的依赖，降低 XZ Utils 类型的多 年渗透风险
   - 发布电路图，增加供应链透明度

2. **用户体验**
   - 现代 GUI 适合不熟悉 VLAN 或防火墙规则的非技术用户
   - VPN chaining 功能对追求隐私的用户很有价值

3. **硬件规格**
   - 8 核处理器（性能类似 Cortex-A55）
   - 4GB LPDDR4 + 16GB eMMC
   - 高于大多数现成路由器

## 竞争产品对比

| 产品 | 规格 | 价格 |
|------|------|------|
| Start9 RISC-V Router | 8核, 4GB RAM, 1WAN+1LAN | ~$300 |
| BananaPi BPI-F3 | 相同CPU | ~$100 |
| Turris Omnia NG | 2x SFP+, 4x 2.5GbE | 更成熟 |

## 结论

社区对这款产品持谨慎态度。主要争议点在于：价格过高、网口数量不足、以及 fork OpenWRT 的做法。虽然完全开源的理念受到认可，但产品能否成功取决于能否提供真实的性能基准测试数据，以及解决用户对长期维护的担忧。
