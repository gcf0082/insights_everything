## 内容摘要

### 论文核心发现

这篇论文展示了一个重大发现：使用**单个二元运算符**（称为EML）可以构建所有初等函数。EML定义为：

```
eml(x, y) = exp(x) - ln(y)
```

作者证明，通过这个单一的二元运算符和数字1，可以组合出完整科学计算器的所有功能，包括：
- 加法、减法、乘法、除法
- 指数函数 exp(x)
- 对数函数 ln(x)
- 三角函数（sin, cos, tan）
- 平方根 sqrt(x)
- 以及其他初等函数

### 关键公式

```
exp(x) = eml(x, 1)
ln(x) = eml(1, eml(eml(1, x), 1))
x + y = eml(ln(x), exp(y))
x - y = eml(ln(x), exp(-y))
```

## 社区讨论要点

### 1. 重大意义
评论者普遍认为这是"多年来最重大的发现之一"。作者将EML与数字电路中的NAND门进行类比——NAND是唯一需要优化的门来构建所有布尔逻辑，而EML可能成为连续数学中的类似基础原语。

### 2. 实际效率问题
有经验的开发者指出：
- 乘法需要深度8的树和41+叶子节点
- 简单的加法操作在EML中表达很困难
- 这类似于用NAND构建所有布尔逻辑——理论上可行但实际效率低下

### 3. LLM基准测试
有用户测试了多个大语言模型用EML表示"2x+y"的能力：
- ChatGPT(免费版)：一次成功
- Opus(付费版)：最初失败，提示后成功
- Gemini、DeepSeek：成功
- Kimi：输出过长

### 4. 与其他概念的关联
- 与Iota组合子（最小形式系统）类比
- 与Kolmogorov-Arnold表示的关联
- 与FRACTRAN编程语言的相似性
- 与Concatenative编程语言（如Rejoice）的关联

### 5. 潜在应用
- 使用EML树作为可训练电路
- 从数值数据精确恢复闭式初等函数
- 构建模拟科学计算器的可能性

### 6. 争议与批评
- 有人认为这是"有趣的 parlor trick"，而非重大发现
- 有人指出这实际上是用exp和ln的查表和级数展开实现的
- 与传统的Taylor级数相比效率问题
