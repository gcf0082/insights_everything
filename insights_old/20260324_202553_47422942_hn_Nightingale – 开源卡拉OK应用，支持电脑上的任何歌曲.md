## 内容摘要

Nightingale 是一款开源卡拉 OK 应用程序，可以将用户音乐文件夹中的任何歌曲转换为卡拉 OK 形式。

### 主要功能

- **人声分离**：使用 UVR Karaoke 模型或 Demucs 进行音轨分离
- **自动歌词**：通过 WhisperX 转录获取歌词，或从 LRCLIB 获取
- **歌词同步**：生成词级别同步的歌词
- **演唱评分**：带有玩家配置文件和排行榜的评分系统
- **视频支持**：支持视频文件
- **游戏手柄支持**：支持游戏手柄和电视友好型 UI 缩放
- **GPU 加速**：支持 NVIDIA (CUDA) 和 Apple Silicon (CoreML/MPS)
- **本地运行**：所有处理在本地完成，不上传任何数据，无需账户、订阅或遥测

### 技术栈

- 使用 Rust 和 Bevy 引擎构建
- 作为单个二进制文件分发，支持 Linux、macOS 和 Windows
- 首次启动时会自动设置独立的 Python 环境并下载所需的 ML 模型

## 社区反馈摘要

### 正面评价

- 用户对开源卡拉 OK 应用的出现表示兴奋
- 赞赏其对冷门、前卫和本地曲目的支持
- 对复古卡拉 OK 背景效果表示喜爱

### 问题与建议

- **歌词对齐问题**：在某些歌曲（如有合唱或伴唱的歌曲）对齐模型容易偏移
- **文字识别**：转录在某些语言（如日语）上表现不佳
- **功能建议**：
  - 添加跳过前进/后退功能以验证结果
  - 添加编辑歌词分析结果的选项
  - 考虑添加对 iGPU/NPU 加速的支持

### 技术争议

部分开发者对该应用的依赖管理方式提出质疑：
- 应用在运行时下载并安装 FFmpeg 和 Python，而不是检查系统已安装的版本
- 构建脚本尝试使用 Docker 容器
- 作者回应称这是为了降低用户使用门槛，实现"奶奶都能用"的安装体验
