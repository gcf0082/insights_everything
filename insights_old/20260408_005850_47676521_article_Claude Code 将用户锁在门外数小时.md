## 问题描述

Claude Code 在 Windows 平台上无法登录，使用 Google 账号登录时始终失败，错误信息为：

```
OAuth error: timeout of 15000ms exceeded
```

该问题完全阻止了用户使用 Claude Code。

## 预期行为

Claude Code 应成功完成 Google 登录流程，将用户登录到应用程序中，而不应出现超时错误。

## 错误信息

- **错误信息**: `OAuth error: timeout of 15000ms exceeded`

## 复现步骤

1. 在 Windows 上打开 Claude Code
2. 开始使用 Google 登录
3. 在浏览器中完成登录流程
4. 返回 Claude Code
5. Claude Code 显示错误：`OAuth error: timeout of 15000ms exceeded`
6. 多次重试登录
7. 每次都出现相同的错误

## 环境信息

| 字段 | 值 |
|------|-----|
| Claude Code 版本 | 2.1.92 |
| 最后工作的版本 | 2.1.92 |
| 平台 | Anthropic API |
| 操作系统 | Windows |
| 终端/Shell | WSL (Windows Subsystem for Linux) |

---

## 总结

这是一个 OAuth 登录超时问题，影响 Windows 平台上的 Claude Code 用户。用户在使用 Google 账号登录时持续遇到 15 秒超时错误，导致无法使用该应用程序。该问题在 2.1.92 版本中首次出现（或至少在该版本中被报告）。
