## 项目简介

这是一个 Windows 3.1 风格平铺背景图片的 BMP 格式存档项目。该仓库收集并整理了复古的 Windows 3.1 操作系统风格的背景图片，这些图片可以用于模拟或重现经典的 Windows 3.1 桌面外观。

项目包含一个 Python 脚本 (`crop.py`)，用于处理和裁剪背景图片，使其适合作为 Windows 3.1 的平铺背景使用。

## 功能特点

- 提供经典的 Windows 3.1 风格背景图片
- 图片格式为 BMP
- 包含裁剪工具脚本
- 可用于复古主题设计、模拟器或怀旧项目

## 使用场景

1. **复古主题开发**: 为现代应用创建 Windows 3.1 复古主题
2. **模拟器配置**: 为 Windows 3.1 模拟器设置背景
3. **怀旧项目**: 用于复古风格的艺术项目或网站设计
4. **历史研究**: 保存和展示早期操作系统的视觉元素
