## 执行摘要

Anthropic 发布了 Claude Mythos Preview（代号 Project Glasswing），这是一个具有卓越网络安全能力的新型通用语言模型。该模型能够自主发现并利用零日漏洞，涉及所有主流操作系统和 Web 浏览器。更令人震惊的是，模型可以在无人类干预的情况下完成从漏洞发现到完整漏洞利用的开发过程。

关键发现：
- **99%** 以上发现的漏洞尚未修补
- 在 Firefox 测试中，Opus 4.6 仅在 400 次尝试中成功 2 次；而 Mythos Preview 在同等条件下成功 **181 次**，另有 29 次达到寄存器控制
- 在 OSS-Fuzz 基准测试中，Mythos Preview 实现了 **10 次** 完全控制流劫持（Tier 5）
- 发现了一个 **27 年** 历史的 OpenBSD 漏洞
- 自主编写了链接 **4 个漏洞** 的浏览器漏洞利用，结合 JIT heap spray 逃逸沙箱

---

## 核心发现

### 1. 零日漏洞发现能力

Mythos Preview 能够自主发现主流操作系统和浏览器中的零日漏洞：

| 类别 | 发现 |
|------|------|
| OpenBSD | 27 年历史的 SACK 漏洞（已修补） |
| FFmpeg | 16 年历史的 H.264 解码器漏洞 |
| FreeBSD | 17 年历史的 NFS 远程代码执行漏洞（CVE-2026-4747） |
| Linux 内核 | 多个本地权限提升漏洞（包括 KASLR 绕过） |
| 内存安全 VMM | 客户机到主机的内存损坏漏洞 |
| 主流浏览器 | 每个浏览器都发现漏洞（包括 JIT 漏洞链） |
| 加密库 | TLS、AES-GCM、SSH 中的多个弱点 |

### 2. 漏洞利用开发能力

Mythos Preview 不仅能发现漏洞，还能自主开发完整漏洞利用：

1. **FreeBSD NFS 漏洞利用**
   - 远程代码执行，获取完整 root 权限
   - 需要构建 20  gadget 的 ROP 链，分散在 6 个 RPC 请求中
   - 完全自主完成，无需人工干预

2. **浏览器漏洞链**
   - 发现 4 个漏洞并链接
   - 编写复杂的 JIT heap spray
   - 逃逸渲染器和操作系统沙箱
   - 转换为跨域绕过和本地权限提升

3. **Linux 内核权限提升**
   - 成功链接 2-4 个漏洞实现完整 root 访问
   - 绕过 KASLR、读取内核数据结构、使用 use-after-free、heap spray 放置结构

### 3. 反向工程能力

模型能够对闭源软件进行逆向工程：
- 重建源代码并进行漏洞分析
- 发现远程 DoS 攻击、固件漏洞、本地权限提升漏洞链

### 4. 逻辑漏洞发现

Mythos Preview 不仅能发现内存损坏漏洞，还能发现：
- 多个完整的身份验证绕过（允许未授权用户获取管理员权限）
- 账户登录绕过（绕过密码和双因素认证）
- 远程拒绝服务攻击

## 评估方法

### 测试框架

1. **容器隔离环境**：模型在隔离容器中运行被测项目
2. **自动代理循环**：模型自主阅读代码、运行程序验证假设、输出漏洞报告
3. **文件优先级排序**：模型首先分析最可能包含漏洞的文件
4. **人工验证**：最终由人工确认漏洞有效性

### 负责任披露

- 使用协调漏洞披露流程
- 发送最高严重性漏洞给专业人工分类器验证
- 公开 SHA-3 承诺以提高透明度
- 90+45 天披露时间框架

## 防御建议

1. **短期**：与前沿实验室合作，在模型能力普及前开始安全评估
2. **长期**：重新审视依赖摩擦而非硬屏障的防御措施
3. **行业行动**：整个行业需要紧急协调应对

---

## 结论

Claude Mythos Preview 代表了网络安全能力范式的转变。模型不仅能发现漏洞，还能自主开发完整的漏洞利用链。这一能力既适用于零日漏洞，也适用于 N 日漏洞（已知但未修补的漏洞）。虽然最终语言模型将更有利于防御者，但过渡期可能充满动荡。Project Glasswing 的目标是让防御者在类似能力变得广泛可用之前，开始保护最重要的系统。
