## 事件概述

CPUID 官方网站遭受供应链攻击，下载链接被篡改指向恶意安装程序。受影响的软件包括：

- **CPU-Z**：流行的系统信息检测工具
- **HWMonitor**：硬件监控工具

## 事件时间线

- 攻击持续约 **6 小时**（2025年4月9日至4月10日 GMT 时区）
- 维护者 Sam 在发现后立即介入调查
- 发现最大的安全漏洞后，恢复链接并将网站设为只读状态
- 服务器上的文件经 VirusTotal 检测目前正常

## 技术细节

1. **攻击方式**：网站本身被入侵，下载链接被重定向到 Cloudflare R2 存储上的恶意可执行文件
2. **攻击者策略**：利用维护者Franck不在期间（OOO - out of office）发起攻击
3. **影响范围**：直接访问 cpuid.com 下载的用户

## 安全建议

1. **使用包管理器**：推荐使用 `winget install CPUID.CPU-Z` 或 `winget install CPUID.HWMonitor`
2. **验证签名**：检查软件数字签名
3. **避免直接下载**：尽量避免直接从官方网站下载，使用官方包管理器

## 相关事件

同一威胁组织此前还攻击了：
- **FileZilla**（上个月）：通过虚假域名发起攻击
- 攻击正从"诱导用户访问错误网站"演变为"让正确的网站提供错误的文件"

## 讨论热点

- 供应链攻击的防护措施
- 文件完整性检查工具（如 tripwire）
- Windows 数字签名机制
- 包管理器 vs 直接下载的安全性对比
