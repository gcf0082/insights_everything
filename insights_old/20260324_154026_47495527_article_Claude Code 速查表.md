## 内容摘要

这是一份关于 Claude Code CLI 工具的官方快捷键和功能参考指南，涵盖了从版本 2.1.81 的最新更新内容。

### 键盘快捷键

#### 常规操作

- `Ctrl+C`: 取消输入/生成
- `Ctrl+D`: 退出会话
- `Ctrl+L`: 清屏
- `Ctrl+R`: 反向搜索历史
- `Ctrl+G`: 在编辑器中打开提示
- `Ctrl+B`: 后台运行
- `Ctrl+T`: 切换任务列表
- `Ctrl+V`: 粘贴图片
- `Ctrl+F`: 终止后台代理（按两次）
- `Esc+Esc`: 撤销/回退

#### 模式切换

- `Shift+Tab`: 循环切换权限模式

#### 输入操作

- `\ + Enter`: 换行（快速）
- `Ctrl+J`: 换行（控制序列）

#### 前缀

- `/`: 斜杠命令
- `!`: 直接执行 bash
- `@`: 文件提及+自动补全

### MCP 服务器

#### 添加服务器

- `--transport http`: 远程 HTTP（推荐）
- `--transport stdio`: 本地 stdio
- `--transport sse`: 远程 SSE

#### 作用域

- `~/.claude.json`: 本地（每个项目）
- `.mcp.json`: 项目级（共享/VCS）
- `~/.claude.json`: 用户级

#### 管理命令

- `/mcp`: 交互式 UI 管理
- `claude mcp list`: 列出服务器
- `claude mcp serve`: CC 作为 MCP 服务器

### 斜杠命令

#### 会话命令

- `/clear`: 清空会话
- `/compact [focus]`: 压缩上下文
- `/resume`: 恢复/切换会话
- `/branch [name]`: 创建分支（/fork 别名保留）
- `/cost`: Token 使用统计
- `/context`: 可视化上下文（网格）
- `/diff`: 交互式差异查看器
- `/copy`: 复制上次响应
- `/export`: 导出会话

#### 配置命令

- `/config`: 打开设置
- `/model [model]`: 切换模型（←→ 选择 effort）
- `/fast [on|off]`: 切换快速模式
- `/vim`: 切换 Vim 模式
- `/theme`: 更改主题颜色
- `/permissions`: 查看/更新权限
- `/effort [level]`: 设置 effort 级别（低/中/高）**新增**
- `/color [color]`: 设置提示栏颜色

#### 工具命令

- `/init`: 创建 CLAUDE.md
- `/memory`: 编辑 CLAUDE.md 文件
- `/mcp`: 管理 MCP 服务器
- `/hooks`: 管理钩子
- `/skills`: 列出可用技能
- `/agents`: 管理代理
- `/chrome`: Chrome 集成
- `/reload-plugins`: 热重载插件

#### 特殊命令

- `/btw <question>`: 附带问题（无上下文开销）
- `/plan [desc]`: 计划模式（+ 自动启动）**新增**
- `/loop [interval]`: 计划重复任务
- `/voice`: 语音模式（20 种语言）
- `/doctor`: 诊断安装
- `/rc`: 启用远程控制
- `/pr-comments [PR]`: 获取 GitHub PR 评论
- `/stats`: 使用统计
- `/insights`: 分析会话报告
- `/desktop`: 在桌面应用继续
- `/remote-control`: 桥接终端到 claude.ai/code **新增**
- `/stickers`: 订购贴纸！

### 内存与文件

#### CLAUDE.md 位置

- `./CLAUDE.md`: 项目级（团队共享）
- `~/.claude/CLAUDE.md`: 个人级（所有项目）
- `/etc/claude-code/`: 托管级（组织级）

#### 规则与导入

- `.claude/rules/*.md`: 项目规则
- `~/.claude/rules/*.md`: 用户规则
- `paths: frontmatter`: 路径特定规则

#### 自动内存

- `~/.claude/projects/<proj>/memory/`
  - `MEMORY.md` + 主题文件，自动加载

### 工作流与技巧

#### 计划模式

- `Shift+Tab`: 常规 → 自动 → 计划
- `--permission-mode plan`: 以计划模式启动

#### 思考与 Effort

- `Alt+T`: 开启/关闭思考
- `"ultrathink"`: 本轮最大努力
- `Ctrl+O`: 查看思考详情（详细模式）
- `/effort`: ○ 低 · ◐ 中 · ● 高 **新增**

#### Git Worktrees

- `--worktree name`: 每个功能独立分支
- `isolation: worktree`: 代理在独立 worktree 中
- `sparsePaths`: 仅检出需要的目录 **新增**
- `/batch`: 自动创建 worktrees

#### 语音模式

- `/voice`: 启用按键说话
- `Space（按住）`: 录音，松开发送
- 20 种语言：英语、西班牙语、法语、德语、捷克语、波兰语…

#### 上下文管理

- `/context`: 使用情况 + 优化技巧
- `/compact [focus]`: 压缩并聚焦
- 自动压缩约 95% 容量
- 1M 上下文：Opus 4.6（Max/Team/Ent）
- **CLAUDE.md 在压缩中保留！**

#### 会话强力技巧

- `claude -c`: 继续上次会话
- `claude -r "name"`: 按名称恢复
- `/btw question`: 附带问题，无上下文成本
- `claude -p "query"`: 非交互式
- `--output-format json`: 结构化输出
- `--max-budget-usd 5`: 成本上限
- `cat file | claude -p`: 管道输入

#### 调度与远程

- `/loop 5m msg`: 重复任务
- `/rc`: 远程控制
- `--remote`: Web 会话

### 配置与环境

#### 配置文件

- `~/.claude/settings.json`: 用户设置
- `.claude/settings.json`: 项目（共享）
- `.claude/settings.local.json`: 仅本地
- `~/.claude.json`: OAuth、MCP、状态
- `.mcp.json`: 项目 MCP 服务器

#### 关键设置

- `modelOverrides`: 模型选择器 → 自定义 ID
- `autoMemoryDirectory`: 自定义内存目录
- `worktree.sparsePaths`: 稀疏检出目录 **新增**

#### 关键环境变量

- `ANTHROPIC_API_KEY`
- `ANTHROPIC_MODEL`
- `CLAUDE_CODE_EFFORT_LEVEL`: low/med/high
- `MAX_THINKING_TOKENS`: 0=关闭
- `ANTHROPIC_CUSTOM_MODEL_OPTION`: 自定义 /model 条目
- `CLAUDE_CODE_PLUGIN_SEED_DIR`: 多个插件种子目录

### 技能与代理

#### 内置技能

- `/simplify`: 代码审查（3 个并行代理）
- `/batch`: 大型并行更改（5-30 个 worktrees）
- `/debug [desc]`: 从调试日志排查问题
- `/loop [interval]`: 计划重复任务
- `/claude-api`: 加载 API + SDK 参考

#### 自定义技能位置

- `.claude/skills/<name>/`: 项目技能
- `~/.claude/skills/<name>/`: 个人技能

#### 技能 Frontmatter

- `description`: 自动调用触发
- `allowed-tools`: 跳过权限提示
- `model`: 覆盖技能模型 **新增**
- `effort`: 覆盖 effort 级别 **新增**
- `context: fork`: 子代理中运行
- `$ARGUMENTS`: 用户输入占位符
- `${CLAUDE_SKILL_DIR}`: 技能自己的目录
- !`cmd`: 动态上下文注入

#### 内置代理

- **Explore**: 快速只读（Haiku）
- **Plan**: 计划模式研究
- **General**: 完整工具，复杂任务
- **Bash**: 终端独立上下文

#### 代理 Frontmatter

- `permissionMode`: default/acceptEdits/dontAsk/plan
- `isolation: worktree`: 在独立 worktree 中运行
- `memory: user|project`: 持久化内存
- `background: true`: 后台任务
- `maxTurns`: 限制代理回合
- `SendMessage`: 恢复代理（替换 resume）**新增**

### CLI 与标志

#### 核心命令

- `claude`: 交互式
- `claude "q"`: 带提示
- `claude -p "q"`: 无头模式
- `claude -c`: 继续上次
- `claude -r "n"`: 按名称恢复
- `claude update`: 更新

#### 关键标志

- `--model`: 设置模型
- `-w`: Git worktree
- `-n / --name`: 会话名称
- `--add-dir`: 添加目录
- `--agent`: 使用代理
- `--allowedTools`: 预批准工具
- `--output-format`: json/stream
- `--json-schema`: 结构化
- `--max-turns`: 限制回合
- `--max-budget-usd`: 成本上限
- `--console`: 通过 Anthropic Console 认证
- `--verbose`: 详细模式
- `--bare`: 最小无头模式（无钩子/LSP）**新增**
- `--channels`: 权限中继 + MCP 推送 **新增**
- `--remote`: Web 会话
- `--chrome`: Chrome 集成

#### 权限模式

- `default`: 提示
- `acceptEdits`: 自动接受编辑
- `plan`: 只读
- `dontAsk`: 除非允许否则拒绝
- `bypassPermissions`: 跳过所有

#### 关键环境变量

- `ANTHROPIC_API_KEY`
- `ANTHROPIC_MODEL`
- `CLAUDE_CODE_EFFORT_LEVEL`: low/med/high
- `MAX_THINKING_TOKENS`: 0=关闭
- `CLAUDE_CODE_MAX_OUTPUT_TOKENS`: 默认 32K

## 最新更新 (2026年3月23日)

1. **`--bare` 标志**: 最小无头模式（无钩子/LSP/插件）
2. **`--channels`**: 权限中继与 MCP 推送消息（预览）
3. **`effort` frontmatter**: 技能与斜杠命令的 effort 设置
4. **`/fork` 重命名为 `/branch`**（保留别名）
5. **`SendMessage`**: 自动恢复停止的代理
6. **`/effort [level]`**: 设置 effort 级别（低/中/高）
7. **`worktree.sparsePaths`**: 稀疏检出目录
8. **`/plan [desc]`**: 计划模式（+ 自动启动）
9. **`/remote-control`**: 桥接终端到 claude.ai/code
