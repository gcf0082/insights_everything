## 摘要

本文讨论了 OpenAI 在二级市场的需求下降，而 Anthropic 的需求则持续攀升。投资者正从 OpenAI 转向 Anthropic，因为后者的估值较低（约 3800 亿美元 vs OpenAI 的 8520 亿美元）。

---

## 主要观点

### 1. 用户流失与产品问题

- 多位用户表示已取消 Claude Code 订阅，转向其他产品
- 抱怨包括：使用限制突然降低、缓存 bug 导致配额消耗等问题
- Anthropic 被指产品发布过于频繁，生态系统建设不完善

### 2. 竞争格局变化

- **Google AI Pro ($20/月)** 被认为性价比极高，可与家人共享
- **Codex** 在每日配额方面优于 Claude Code
- 用户现在会在多个提供商之间灵活切换

### 3. 投资视角

- OpenAI 估值过高（8520亿 vs Anthropic 的 3800亿）
- OpenAI 持续亏损，且面临人才流失
- 两者可能都被高估，但 Anthropic 相对"便宜"
- Sam Altman 的个人形象和信誉受到质疑

### 4. 商业模式担忧

- OpenAI 的"免费+广告"策略存在挑战：LLM 查询成本比 Google 搜索高约 100 倍
- 前沿模型实验室作为商业实体可能难以维持，因为开源模型几个月内就能追赶上来

### 5. 开发者态度

- 许多开发者并不忠诚于特定平台，而是哪家性价比高用哪家
- 企业采购周期长，难以跟上快速变化的市场

## 总结

这场讨论反映了 AI 市场正在发生的结构性变化：用户和投资者正从 OpenAI 向 Anthropic 流动，主要原因是估值差异、产品体验问题以及市场竞争加剧。同时，开源模型的崛起对闭源前沿模型提供商构成了长期威胁。
