## 项目概述

BrightBean Studio 是一个开源、可自托管的社交媒体管理平台，专为创作者、代理机构和中型企业构建。该平台支持跨 Facebook、Instagram、LinkedIn、TikTok、YouTube、Pinterest、Threads、Bluesky、Google Business Profile 和 Mastodon 等 10 多个平台进行内容策划、撰写、排期、审批、发布和监控，所有功能均可在单一的多工作区仪表板中完成。

该项目是 Sendible、SocialPilot 或 ContentStudio 的免费替代方案，无需按席位、渠道或工作区付费。每位用户都可使用所有功能，没有付费专区、功能门控或追加销售。

## 核心功能

1. **多工作区和团队** - 无限组织 → 工作区 → 成员，细粒度 RBAC 自定义角色、邀请，外部协作者专用客户端角色
2. **内容编辑器** - 富文本编辑器，支持平台级文案/媒体覆盖、版本历史、可复用模板、内容分类和标签、看板式创意面板
3. **日历和排期** - 可视化日历，支持每个账户的循环周发布槽和命名队列，自动将帖子分配到下一个可用槽
4. **发布引擎** - 直连第一方 API 集成（无聚合商）、自动重试、账户级速率限制跟踪、90 天发布审计日志
5. **审批工作流** - 可配置阶段（无/可选/内部/内部+客户端）、线程化内部和外部评论、提醒、完整审计跟踪
6. **统一社交收件箱** - 各平台评论、提及、DM 和评价统一汇集、情感分析、分配、线程回复、历史回填
7. **媒体库** - 组织和工作区范围的媒体库、嵌套文件夹、自动生成的平台优化变体、Alt 文本
8. **客户端门户** - 30 天无密码魔法链接访问，客户端可审批或拒绝帖子而无需创建账户
9. **通知** - 应用内、邮件和 webhook 投递，每个事件类型支持用户级偏好设置
10. **安全和运维** - 加密令牌和凭据存储、可选 2FA (TOTP)、Google/GitHub SSO、Sentry 支持、7 天可逆组织删除宽限期
11. **白标友好** - 每个工作区品牌化（徽标、颜色）和工作区默认标签、首发评论和发布模板

## 部署方式

### 一键部署

- **Heroku**: 使用部署按钮一键部署
- **Render**: 使用部署按钮一键部署
- **Railway**: 使用部署按钮一键部署

### Docker Compose (VPS 推荐)

在服务器上运行：

```bash
git clone https://github.com/brightbeanxyz/brightbean-studio.git
cd brightbean-studio
cp .env.example .env
# 编辑 .env 配置
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d
docker compose exec app python manage.py migrate
docker compose exec app python manage.py createsuperuser
```

这会启动 4 个容器：app (Gunicorn)、worker、PostgreSQL 和 Caddy (自动 HTTPS)。

### 本地开发 (Docker-free)

需要 Python 3.12+ 和 Node.js 20+，使用 SQLite 进行本地开发。
