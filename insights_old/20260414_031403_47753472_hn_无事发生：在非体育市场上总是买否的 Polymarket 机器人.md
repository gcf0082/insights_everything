## 项目概述

"Nothing Ever Happens" 是一个开源的 Polymarket 机器人，由 sterlingcrispin 开发，其核心策略是在非体育市场的所有预测市场上总是买入"No"。作者在 GitHub 上明确表示这是一个"meme"项目，没有任何风险管理系统。

**关键数据**:
- 73%的 Polymarket 市场最终以"No" resolve
- 机器人配置了0.65的买入价格上限
- 对新市场会在接近0.5的价格买入

## 社区讨论要点

### 1. 盈利能力分析

**traderj0e**:
- 回测显示这种策略有很好的回报(约100% APR)
- 但后来发现是因为预先知道事件何时resolve而"作弊"
- 需要大量交易量才能"可预测"
- 一次损失需要10-20次胜利才能回本

**pinkmuffinere**:
- 如果"NO"的平均成本低于0.73且73%的市场resolve为"NO"，那么"什么都不发生"策略是有利可图的
- 反之亦然

**验证**:
- traderj0e 尝试用100美元测试，一个月净亏损约5美元
- 没有足够的多元化

### 2. 数学上的争议

**gruez**:
- 逻辑不成立：不是每个赌注都有相同的赔付
- 例如：如果骰子是否掷出1的概率是83% resolve为"NO"，但只付1.1美元/美元赌注，仍然亏损

**fooker**:
- 这就像"在蒸汽压路机前捡 penny"
- 随机选择赌注即使遵循凯利准则也可能导致资金耗尽

### 3. 市场效率

**dheera**:
- 像这样的策略很容易成为正EV，直到足够多的人发现它
- 机制：发现某物统计上值0.70但能以0.60买入，0.70卖出，赚取0.10
- 一旦开源或公开，策略就会失效

**yyurgenson**:
- 必须承认更多的市场resolve为"NO"而不是"YES"，因为许多市场是关联的，只有一个赢家

### 4. 黑天鹅事件风险

**wormpilled**:
- 本质上是套利人类想象力
- 人们喜欢提出奇幻概念，因为它获得关注
- 市场越令人兴奋，越不可能发生
- 现实通常是无聊的

**krashidov**:
- 这就像在活跃的铁路上捡 penny
- 小赌注赢，大输被长尾事件完全抹去

### 5. 用户体验

**cordwainersmith**:
- 反向赌注很有趣
- 预测市场确实倾向于对戏剧性结果定价过高
- "总是买NO"并不像听起来那么愚蠢

**conductr**:
- 如果人们从这种市场获得类似的乐趣水平，那么与赌场没有区别
- 如果他们享受足够多，这可以作为一种更便宜的替代品

## 结论

这个项目更多是一个有趣的"meme"而非真正的盈利策略。社区讨论表明：
1. 73% resolve为NO的数据是真实的
2. 但盲目买入"NO"不一定盈利，因为买入价格和市场结构
3. 存在correlation，但数学上存在争议
4. 建议：找到正EV策略后，大量交易但不要开源
