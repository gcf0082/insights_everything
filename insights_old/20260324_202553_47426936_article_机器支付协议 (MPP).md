## 核心摘要

Stripe推出机器支付协议（Machine Payments Protocol，MPP），这是一个开放的互联网原生支付标准，使AI代理能够程序化地进行支付。MPP由Stripe和Tempo联合创建，为代理经济提供支付基础设施。

## 关键要点

### 1. 背景与痛点
- AI正从问答式聊天机器人演进为自主代理，能够制定计划、执行操作并评估结果
- 当前金融系统工具专为人类设计，代理难以使用
- 现有支付流程需要创建账户、导航定价页面、选择订阅套餐、输入支付详情等步骤，往往需要人工干预

### 2. MPP解决方案
- **开放标准**：机器支付协议（MPP）是一个开放标准
- **支持方式**：微交易、循环支付等
- **支付方式**：支持稳定币以及法币（信用卡、BNPL）
- **技术实现**：使用Shared Payment Tokens (SPTs)实现

### 3. 技术集成
Stripe用户可以通过几行代码使用PaymentIntents API接受MPP支付：

```javascript
const paymentIntent = await stripe.paymentIntents.create({
  amount: 1,
  currency: 'usd',
  payment_method_types: ['crypto'],
  payment_method_data: { type: 'crypto' },
  payment_method_options: {
    crypto: {
      mode: 'deposit',
      deposit_options: { networks: ['tempo'] }
    }
  },
  confirm: true
});
```

### 4. 已应用案例
- **Browserbase**：代理可启动无头浏览器，按会话付费
- **PostalForm**：代理支付打印和邮寄实体邮件
- **Prospect Butcher Co.**：代理为人类下单三明治，支持自取或配送至纽约市任何地点
- **Stripe Climate**：代理可程序化向环保项目捐款

### 5. 企业评价
Parallel Web Systems创始人Parag Agrawal表示：「Parallel为代理成为网页主要用户的世界而构建。我们只用几行代码就集成了Stripe机器支付，现在代理可以按API调用自主支付网络访问费用。」

## 战略意义

Stripe正在构建更广泛的代理金融基础设施，包括：
- Agentic Commerce Suite（代理商务套件）
- Agentic Commerce Protocol (ACP)
- MCP集成
- 对MPP和x402的支付支持

这标志着AI代理将成为互联网经济的重要组成部分，能够与企业和彼此进行交易。
