## 核心功能

**Email.md** 是一个用 Markdown 编写响应式邮件的开源工具。

### 主要特性

1. **Markdown 编写邮件** - 用户可以使用 Markdown 语法编写邮件内容，无需编写 HTML
2. **响应式设计** - 生成的邮件支持响应式布局，适配各种设备
3. **主题支持** - 支持自定义主题（如 dark 主题）
4. **组件化** - 提供多种内置组件：
   - `header` - 邮件头部
   - `callout` - 强调框（支持 center、compact 等参数）
   - `footer` - 邮件底部

### 产品页面

- **Templates** - 邮件模板库
- **Builder** - 在线邮件构建器
- **Docs** - 文档说明

### 技术栈

- 通过 npm 安装：`npm install emailmd`
- 开源协议：源码可在 GitHub 获取

---

## 总结

Email.md 是一个面向开发者的邮件编写工具，通过 Markdown 简化邮件开发流程，避免传统的 `HTMHELL` 问题。由 unMTA 公司开发和维护。
