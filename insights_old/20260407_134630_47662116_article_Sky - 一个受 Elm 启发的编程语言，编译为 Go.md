## 项目概述

Sky 是一种实验性编程语言，结合了 Go 的实用性和 Elm 的优雅性，创建了一个简单的全栈语言。它将代码编译为单个可移植的二进制文件。

### 核心理念

- **Go 特性**：快速编译、单一静态二进制经过实战检验的生态系统，涵盖数据库、HTTP 服务器、云 SDK 等
- **Elm 特性**：Hindley-Milner 类型推断、代数数据类型、穷尽模式匹配、纯函数、Elm 架构
- **Phoenix LiveView**：服务器驱动的 UI，包含 DOM 差异、会话管理和 SSE 订阅，无需客户端框架，无需 WebSocket

### 创作者背景

项目由独立开发者创建，作者有多年使用 Go、Elm、TypeScript、Python、Dart、Java 等语言的专业经验。作者希望通过结合 Go 的工具链（快速构建、单一二进制、真实并发、大规模生态系统）与 Elm 的开发体验（如果能编译，就能工作；重构无风险；架构可扩展）来创建一个理想的编程语言。

## 技术架构

### 语言特性

#### 类型系统

- 使用 Hindley-Milner 类型推断
- 支持类型类约束
- 支持代数数据类型（联合类型）
- 支持类型别名和记录
- 编译时类型检查，如果能编译，就能运行

#### 内置类型

| 类型 | 描述 |
|------|------|
| Int | 整数 |
| Float | 浮点数 |
| String | 文本 |
| Bool | 布尔值 |
| Char | 字符 |
| Unit | 空元组 |
| List a | 有序集合 |
| Maybe a | 可选值 |
| Result err ok | 成功/失败 |

#### 模块系统

- 每个文件声明一个带有暴露子句的模块
- 模块名称使用帕斯卡命名法（PascalCase）
- 支持层次结构（点分隔）
- 支持选择导入、限定别名、开放导入

#### 函数

- 所有函数都是柯里化的，支持部分应用
- 支持 lambda（匿名函数）
- 支持函数组合

#### 模式匹配

- 字面量模式
- 构造器模式
- 元组模式
- 列表模式
- As 模式
- 记录模式
- 嵌套模式
- 编译器检查穷尽性

### Sky.Live

Sky.Live 是一个服务器驱动的 UI 框架，灵感来自 Phoenix LiveView。编写标准 TEA 代码；编译器生成一个 Go HTTP 服务器，包含 DOM 差异、会话管理、SSE 订阅和一个小型（约3KB）JS 客户端。

#### 特性

- 无需 WebSocket - 使用 SSE 进行订阅，轮询作为无服务器后备
- 无服务器就绪 - 轮询后备可在 Lambda、Cloud Run、任何无状态环境中工作
- 可配置输入 - debounce（默认）或 blur
- 统一 Model/Msg - 一个 TEA 循环处理整个应用
- 会话存储 - 内存（默认）、sqlite、redis、postgresql
- 并发安全 - 每个会话锁 + 乐观并发

### Go 互操作（FFI）

Sky 可以导入任何 Go 包。编译器自动生成类型安全的、Task 包装的绑定，支持恐慌恢复。用户无需编写 FFI 代码。

#### 特性

- 自动生成绑定
- Task 包装
- 恐慌安全
- 指针安全
- 不透明结构模式（构建器）
- 回调桥接

### 包管理

Sky 内置了包管理器，支持 Sky 包和 Go 包。使用 `sky.toml` 作为项目清单。

#### 依赖管理

- `sky add` - 自动检测 Sky vs Go 包
- `sky remove` - 移除依赖
- `sky update` - 更新到最新版本
- 依赖存储：Sky 包在 `.skydeps/`、Go 包在 `.skycache/gomod/`

### 标准库

#### Sky.Core

自动通过 Prelude 导入的核心模块：

- `Sky.Core.Prelude` - Result、Maybe、identity、not 等
- `Sky.Core.Maybe` - withDefault、map、andThen
- `Sky.Core.Result` - withDefault、map、andThen、toMaybe
- `Sky.Core.List` - map、filter、foldl、foldr 等
- `Sky.Core.String` - split、join、contains 等
- `Sky.Core.Dict` - empty、singleton、insert 等
- `Sky.Core.Json` - JSON 编码/解码
- `Sky.Core.Task` - 任务和并发
- `Sky.Core.File` - 文件操作
- `Sky.Core.Process` - 进程操作

#### Std（应用框架）

- `Std.Log` - println
- `Std.Cmd` - 命令
- `Std.Sub` - 订阅
- `Std.Task` - 任务
- `Std.Program` - 程序
- `Std.Html` - 服务器端渲染
- `Std.Css` - CSS 支持

## 使用方法

### 快速开始

```bash
# 安装
curl -fsSL https://raw.githubusercontent.com/anzellai/sky/main/install.sh | sh

# 创建项目
sky init my-app
cd my-app
sky run
```

### CLI 命令

- `sky init [name]` - 创建新项目
- `sky build [file.sky]` - 编译为 Go 并构建二进制
- `sky run [file.sky]` - 构建并运行
- `sky check [file.sky]` - 类型检查
- `sky fmt <file-or-dir>` - 格式化代码
- `sky add <package>` - 添加依赖
- `sky remove <package>` - 移除依赖
- `sky update` - 更新依赖
- `sky upgrade` - 自我升级
- `sky lsp` - 启动 LSP 服务器

## 优点与特点

1. **单一二进制部署** - 编译输出单个 Go 二进制，无需复杂构建流程
2. **类型安全** - Hindley-Milner 类型推断，编译时检查
3. **无运行时依赖** - 编译为原生 Go 代码，无额外运行时
4. **服务器驱动 UI** - Sky.Live 提供现代服务器端渲染体验
5. **Go 互操作** - 可导入任何 Go 包并自动生成类型安全绑定
6. **自托管** - 编译器、CLI、格式化器和 LSP 都是用 Sky 本身编写的
7. **简洁的 TEA 架构** - 遵循 Elm 架构的纯粹函数式更新模式
