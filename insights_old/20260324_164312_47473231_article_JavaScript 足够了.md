# Gea 框架洞察报告

## 基本信息

- **洞察链接**: https://geajs.com/
- **洞察日期**: 2026-03-24
- **框架名称**: Gea
- **框架类型**: 轻量级响应式 UI 框架

## 框架概述

Gea 是一个编译时 UI 框架，将普通的 JavaScript 类和函数转换为精确的 DOM 更新。它不需要虚拟 DOM、hooks 或 signals，在构建时将普通代码转化为响应式代码。

## 核心特性

### 1. 零概念设计
- 无需学习新的原语
- stores 是类，组件是类或函数，计算属性是 getters
- 不需要 signals、hooks、依赖数组或编译器指令

### 2. 编译时响应式
- Vite 插件在构建时分析 JSX
- 确定哪些 DOM 节点依赖哪些状态
- 生成看不见的精确补丁更新

### 3. 无虚拟 DOM
- 更新直接到达变化的元素
- 无需 diffing 或 reconciliation
- 直接 DOM  mutations，构建时生成

### 4. 基于 Proxy 的 Stores
- 状态存储在普通的类中，由深度 Proxy 包装
- 直接修改属性
- 数组方法、嵌套对象都能正常工作

### 5. 完整工具包
- 内置状态管理和路由
- ~13 kb (含路由) / ~10 kb (不含)
- 零运行时依赖
- 内置客户端路由、35+ 无障碍 UI 组件、移动端原语、VS Code 扩展、项目脚手架

## 性能数据

## 代码示例

**Store (store.ts)**:
```typescript
import { Store } from '@geajs/core'

class CounterStore extends Store {
  count = 0
  
  increment() { this.count++ }
  decrement() { this.count-- }
}

export default new CounterStore()
```

**组件 (Counter.tsx)**:
```typescript
import { Component } from '@geajs/core'
import store from './store'

export default class Counter extends Component {
  template() {
    return (
      <div class="counter">
        <span>{store.count}</span>
        <button click={store.increment}>+</button>
        <button click={store.decrement}>-</button>
      </div>
    )
  }
}
```

## 功能列表

- **类组件**: 扩展 Component，添加本地响应式状态
- **函数组件**: 导出返回 JSX 的函数，零开销转换为类组件
- **客户端路由**: 内置 RouterView、Link、路由参数、编程式导航
- **事件委托**: 每个事件类型编译为单一委托监听器
- **细粒度数组**: push、splice、sort 生成粒度变更事件
- **Gea UI**: 35+ 基于 Zag.js 的无障碍组件
- **Gea Mobile**: ViewManager、侧边栏、标签视图、下拉刷新等
- **热模块替换**: Vite 插件支持 HMR

## 总结

Gea 是一个创新的编译时响应式 UI 框架，通过将普通 JavaScript 类和函数直接转换为高效的 DOM 更新，实现了极致的性能。它消除了学习曲线，无需新的概念或 API，对熟悉 JavaScript 的开发者非常友好。
