## 项目概述

Libretto 是一个用于构建健壮 Web 集成的工具包。它为编程代理提供了一个 Live Browser 和一个 Token 高效的 CLI，使其能够：

- 以最小的上下文开销检查实时页面
- 捕获网络流量以逆向工程网站 API
- 记录用户操作并将其重放为自动化脚本
- 交互式调试真实网站上损坏的工作流

该工具由 [Saffron Health](https://saffron.health) 团队构建，最初用于维护与常见医疗保健软件的浏览器集成，现已开源供其他团队使用。

## 核心功能

### 1. 实时页面检查
- 检查 Live Pages，最小化上下文开销
- 捕获 PNG + HTML 并使用 LLM 分析

### 2. 网络流量捕获
- 捕获浏览器网络流量
- 逆向工程网站 API
- 将浏览器自动化转换为直接网络请求

### 3. 用户操作录制与回放
- 记录用户在浏览器中的操作
- 将操作转换为 Playwright 脚本
- 支持交互式脚本构建

### 4. 调试功能
- 在任何点暂停工作流
- 检查实时页面
- 自主修复问题

## 使用场景

### 单次脚本生成
> 使用 Libretto Skill。打开 LinkedIn 并抓取前 10 条帖子内容、发布者、点赞数、前 25 条评论和前 25 条转发。

### 交互式脚本构建
> 我将在 eClinicalWorks EHR 中向您展示获取患者主保险 ID 的工作流。使用 Libretto Skill 将其转换为 Playwright 脚本，接受患者姓名和出生日期作为输入，返回保险 ID。

### 浏览器自动化转网络请求
> 将位于 ./integration.ts 的浏览器脚本转换为直接网络脚本。使用 Libretto Skill 进行安全分析。

### 修复损坏的集成
> 修复浏览器脚本中的选择器错误。使用 Libretto Skill 重现故障、检查实时页面并修复问题。

## 安装与配置

```bash
# 安装
npm install libretto

# 首次设置
npx libretto setup

# 检查状态
npx libretto status

# 配置 AI 模型
npx libretto ai configure <openai | anthropic | gemini | vertex>
```

## CLI 命令

```bash
npx libretto open <url>           # 启动浏览器并打开 URL
npx libretto snapshot --objective "..."  # 捕获 PNG + HTML 并使用 LLM 分析
npx libretto exec "<code>"        # 在打开的页面上执行 Playwright TypeScript
npx libretto close              # 关闭浏览器
```

## 项目结构

- `packages/libretto/src/cli/` — CLI 命令
- `packages/libretto/src/runtime/` — 浏览器运行时（网络、恢复、下载、提取）
- `packages/libretto/src/shared/` — 共享工具（配置、LLM 客户端、日志、状态）
- `packages/libretto/test/` — 测试文件
- `packages/libretto/skills/libretto/` — Libretto Skill 源码
