## 概述

Pardonned.com 是一个可搜索的美国总统赦免数据库，由开发者 vidluther 创建。该项目的灵感来源于 Liz Oyer 的赦免相关视频，作者希望通过该网站能够更方便地验证和查询所有赦免记录。

## 技术栈

- **Playwright**: 用于抓取美国司法部（DOJ）网站数据
- **SQLite**: 本地数据库存储
- **Astro 6**: 用于从 SQLite 数据库构建静态网站

所有代码均为开源，可在 GitHub 上获取。

## 讨论要点

### 1. 赦免制度的争议

讨论中对美国赦免制度提出了多项批评：
- **预先赦免（Preemptive Pardon）问题**：用户 siliconc0w 认为应该禁止预先赦免，因为赦免应该是针对特定已定罪犯罪的宽恕，而非给予广泛的豁免权
- **赦免权的滥用**：用户 vunderba 指出，赦免制度需要重大改革，建议实施：
  - 总统每届任期的赦免数量上限
  - 国会审议程序

### 2. 具体赦免案例讨论

- **拜登家族赦免**：用户 koolba 指出拜登家族的赦免范围涵盖11年以上（2014年1月至2025年1月），且没有具体细节描述
- **尼克松赦免**：用户 ceejayoz 将其与拜登家族赦免进行对比，指出尼克松赦免虽然没有那么长，但"更重要"（无暴力例外，赦免对象是前总统）
- **奥巴马赦免**：用户 hk1337 注意到Obama的赦免数量很高，但其中只有118条涉及大麻相关犯罪

### 3. 数据准确性问题

- 用户 millbj92 指出Trevor Milton的赦免中涉及超过5亿美元的赔偿金，但网站上可能未能完全体现
- 作者 vidluther 表示正在改进数据准确性，包括处理Obama赦免数据的显示问题

### 4. 赦免制度的历史观点

- 用户 shimman 认为赦免权是"君主制的残余"，在民主国家不应该存在
- 用户 fernmyth 认为赦免权是"一种替代政变和内战的方式"，为和平权力交接提供保障

## 总结

Pardonned.com 作为一个新的公民数据项目，使美国政府赦免记录变得可搜索，引发了关于赦免制度权力过大的广泛讨论。讨论反映了公众对赦免制度可能被滥用的担忧，特别是在近期政治环境下。
