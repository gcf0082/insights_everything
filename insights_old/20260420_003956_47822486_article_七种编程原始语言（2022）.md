# 洞察报告：七种编程原始语言(ur-languages)

**洞察链接**: https://madhadron.com/programming/seven_ur_languages.html

**发布日期**: 2022年

**作者**: madhadron

---

## 摘要

本文深入探讨了编程语言背后的七种"原始语言"(ur-languages)概念。作者认为，学习不同编程语言时，理解其背后的基础模式和思维方式比学习具体语法更为重要。这七种原始语言分别是：ALGOL、Lisp、ML、Self、Forth、APL和Prolog。每种原始语言代表了一种独特的编程思维范式，掌握这些范式能够帮助程序员更快速地学习新语言，并拓宽编程视野。

---

## 七种原始语言详解

### 1. ALGOL（命令式语言）

**特征**: 程序由赋值语句、条件判断和循环组成，组织成函数形式。许多语言增加了模块系统、新数据类型定义方式、多态性或异常和协程等控制流结构。

**代表语言**: C、C++、Python、Java、C#、Ruby、Pascal、JavaScript、Ada、Fortran

**历史背景**: 这是最古老的原始语言，可追溯到Ada Lovelace为Babbage的分析机编写程序。20世纪60年代，计算机科学学术界发展出结构化编程，最终导致ALGOL 60的出现，现代大多数命令式语言都源于此。

**重要观点**: 80年代，Self原始语言的概念被移植到这些语言中，形成了面向对象编程的类概念。自2010年以来，ML原始语言的概念也开始被引入。

---

### 2. Lisp

**特征**: Lisp由前缀表达式组成，括号括起来，例如`(+ 2 3)`。语言内置列表作为数据结构，代码本身就是列表的形式，这使得Lisp系统能够定义宏，允许程序员获取列表、修改它并传递给编译器。

**代表语言**: Common Lisp、Scheme、Clojure、Emacs Lisp、AutoLISP、Lush

**历史背景**: Lisp比Fortran年轻约一年，是目前仍在使用的第二古老语言。它的起源源于一个纯数学问题：如何编写一个能评估自身表达式的数学结构？John McCarthy在1958年给出了答案。

**重要观点**: Lisp的宏系统使其区别于其他语言。Common Lisp有loop语法，但它是作为宏定义的，而不是语言内置的。 Lisp常在编写大多数代码时表现得像其他原始语言（通常是ALGOL或ML），但其独特的宏系统允许程序员重新定义语言的语义。

---

### 3. ML（函数式语言）

**特征**: ML语言由函数是一等公民和Hindley-Milner类型的类型系统定义。所有迭代都通过递归完成。某些语言默认是惰性求值的（如Miranda和Haskell），直到真正需要时才求值。

**代表语言**: Standard ML、OCaml、Haskell、Miranda、Agda、Idris

**示例代码**:
```
sum [] = 0
sum (x:xs) = x + sum xs
```

**历史背景**: ML是Cambridge开发的定理证明程序的元语言（因此得名）。该语言后来流传到欧洲，在英国和法国特别流行。

**重要观点**: OCaml试图合并Self原始语言的概念。Agda和Idris混合了值和类型（称为依赖类型系统），1ML混合了模块和类型。

---

### 4. Self（面向对象语言）

**特征**: 程序由一组可以相互接收和发送消息的对象组成。所有行为都通过这种方式实现。你通过向现有对象发送消息来创建新对象。条件判断通过让一个变量引用true对象或false对象来实现。循环也是如此。

**代表语言**: Self、Smalltalk、JavaScript（源自Self的无类对象系统）

**历史背景**: Smalltalk是原始语言，于1970年代末在Xerox Parc开发。Strongtalk项目的研究成果成为Java的HotSpot即时编译器的基础。

**重要观点**: Self摒弃了类的概念，只使用对象。Common Lisp的对象系统泛化了根据接收消息的对象选择运行代码的思想。Erlang则将执行线程的概念从对象跳转到并行执行线程。

---

### 5. Forth（栈语言）

**特征**: Forth是Lisp的inverse，共享HP逆波兰 notation计算器的语法。有数据栈，写入数字如`42`会被推入栈中。简单算术看起来是反向的：`2 3 + 5 *`。Forth允许程序员拦截解析器并用自己的代码替换它，因此语法是完全可替换的。

**代表语言**: Forth、PostScript、Factor、Joy

**示例代码**:
```
: square dup * ;
3 square
```

**历史背景**: Forth最初于1970年编写用于控制射电望远镜，后来在嵌入式系统中广泛传播。PostScript于1980年代出现，作为描述文档到打印机的灵活方式。

**重要观点**: Forth程序通常定义小型语言，如Fortran子集或直接ASCII解析图表的方式。

---

### 6. APL（数组语言）

**特征**: 语言中的 everything 都是n维数组。操作符是一到两个符号长，实现这些数组上的高级操作。结果非常简洁，以至于符号序列成为操作的标签，而不是给另一个名称。

**代表语言**: APL、J、K

**示例代码**:
```
(+⌿÷≢) x
```
计算数组x的平均值。

**历史背景**: APL在1960年代由Kenneth Iverson创建，最初是一种数学符号。它在从事重计算的人群中一直享有小众追随者。其后代K在金融环境中非常流行。

**重要观点**: 数组上的高级操作已部分导出到许多环境中，如MATLAB、NumPy和R。

---

### 7. Prolog（逻辑语言）

**特征**: 程序由事实组成，要么是"基本"事实（如Bob是Ed的父亲），要么是非基本事实，通过使用变量（大写字母）定义如何从其他事实推导出一个事实。Prolog运行时获取这些事实和查询，并搜索查询的结果。

**代表语言**: Prolog、Mercury、Kanren

**示例代码**:
```
father(bob, ed).
father(bob, jane).
grandfather(X, Y) :- father(X, Z), father(Z, Y).
```

**历史背景**: 1970年代，法国的逻辑学家意识到可以用一阶逻辑来表达程序，并开始尝试实现。1980年代的日本第五代计算机项目大力押注于Prolog。

**重要观点**: Prolog程序本质上是搜索，它们像数据库查询一样被调优。Prolog往往出现在特定领域。Java的类型检查多年来是用Prolog实现的，Facebook的原始源代码搜索工具也是如此。

## 核心建议

### 每个程序员都应该：

1.  **首先**：精通ALGOL家族中的一门语言
2.  **其次**：学习Prolog家族的SQL。这是继ALGOL家族之后，在职业生涯中能给你带来最大收益的
3.  **然后**：扩展学习，每年学习一种不同原始语言家族的新语言

### 推荐的学习路径：

| 原始语言 | 推荐语言 |
|---------|---------|
| Lisp | PLT Racket |
| ML | Haskell |
| Self | Self |
| Prolog | Prolog |
| Forth | gForth |
| APL | K（通过ok） |

如果从事大量数值计算工作，可以更早学习K。如果从事大量嵌入式编程工作，可以更早学习gForth。

---

## 总结

理解这七种原始编程语言的概念对于程序员来说非常重要。当用一种原始语言的视角看起来完全不同的两件事，通过另一种原始语言的视角可能变成简单的比较。掌握这些基础范式能够帮助程序员更快速地学习新语言，并开辟新的编程思路。

正如作者所说："学习一种追溯到不熟悉原始语言需要significant时间和精力以及新的神经通路。"
