## 内容总结

### 概述

SolveSpace 是一个参数化 2D/3D CAD 软件。该页面介绍了其**实验性 Web 版本**。

### 主要内容

1. **实验性状态**: 这是一个实验性版本，虽然开发主要是作为桌面软件，但由于足够紧凑，编译为 WebAssembly 后可以在浏览器中运行。
2. **性能与限制**: 运行速度会有一定损失，且存在不少 Bug，但对于较小的模型通常可以使用。
3. **最新开发版本**: Web 版本基于最新的开发分支构建，用户可能会遇到桌面版本中不存在的 Bug。
4. **离线特性**: 该 Web 版本加载后无需网络依赖，用户可以自行构建并托管。
5. **反馈渠道**: 用户可以通过 GitHub 报告 Bug。

### 总结

SolveSpace 作为一款开源 CAD 软件，现在已经可以在浏览器中运行，虽然目前仍是实验版本，但对于轻量级使用场景具有一定的可行性。
