# 洞察报告：Coalton Mine IDE

## 基本信息

- **洞察链接**: https://coalton-lang.github.io/mine/
- **来源网站**: Coalton 官方文档
- **语言**: 英文
- **生成时间**: 2026-04-27

## 工具概述

Mine 是一个集成的开发环境（IDE），专为 Coalton 和 Common Lisp 设计，支持 Windows、macOS 和 Linux 三大操作系统。

## 核心功能

### 1. 双版本分发

- **mine-app**: 适用于 Windows 和 macOS，是一个完整的、无依赖的打包应用程序，无需安装即可直接使用
- **mine-core**: 适用于 Windows、macOS 和 Linux，是一个需要用户自行配置终端的命令行版本，需要终端支持 Unicode 字体和 Kitty 键盘协议

### 2. 语言支持

Mine 编辑器不局限于任何单一语言，同时支持：
- **Coalton**: 具有强类型系统的函数式编程语言
- **Common Lisp**: 动态类型的高级对象系统
- 开发者可以根据项目需求自由选择使用其中一种或混合使用

### 3. 集成 REPL

REPL（交互式解释器）完全集成于编辑器中，而非后期添加。从函数到整个项目，都可以"beam"到 REPL 中进行即时交互和测试。

### 4. 交互式调试器

当代码出现错误时，会弹出调试器显示错误信息、修正选项和堆栈跟踪，方便开发者快速定位和修复问题。

### 5. 内联诊断

Beam 代码时会标记错误和警告，直接在编辑器中显示。此外，优化提示也会高亮显示，指出代码可能存在的效率问题。

### 6. 类型提示和自动补全

编写 Coalton 代码时，当前光标所在函数的完整数据类型会立即显示，无需猜测每个函数的参数。如果不知道函数名称，按 Tab 键即可触发自动补全。

### 7. 结构化编辑教程

内置结构化编辑（如 ParEdit）教程，约 5 分钟即可学会，大大提升开发效率。结构化编辑是完全可选的，但能显著提高开发效率。

### 8. 本地代码编译

无需虚拟机或解释器，所有代码都被编译并优化为 CPU 的本地二进制代码，以获得最大性能。

## 总结

Mine 是一个功能完整的 Coalton 和 Common Lisp 集成开发环境，具有 REPL 集成、交互式调试、内联诊断、类型提示和自动补全等特性。它以应用和核心两种版本分发，支持三大主流操作系统，代码编译为本地二进制以获得最佳性能。
