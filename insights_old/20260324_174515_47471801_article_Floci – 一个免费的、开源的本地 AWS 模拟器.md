## 项目概述

Floci 是一个免费、开源的本地 AWS 模拟器。名称来源于"floccus"——一种看起来像爆米花的云形态。

该项目的诞生背景是 LocalStack 社区版于2026年3月停用，需要认证令牌并冻结安全更新，Floci 作为无附加条件的替代方案出现。

## 核心特性

### 1. 零门槛使用
- 无需 AWS 账户
- 无需认证令牌
- 只需 `docker compose up` 即可启动

### 2. 轻量级高性能
| 指标 | Floci | LocalStack社区版 |
|------|-------|-----------------|
| 启动时间 | ~24 ms | ~3.3 s |
| 空闲内存 | ~13 MiB | ~143 MiB |
| Docker镜像大小 | ~90 MB | ~1.0 GB |

### 3. 完整服务支持
- ✅ API Gateway v2 / HTTP API
- ✅ Cognito
- ✅ ElastiCache (Redis + IAM auth)
- ✅ RDS (PostgreSQL + MySQL + IAM auth)
- ✅ S3 Object Lock (COMPLIANCE / GOVERNANCE)
- ✅ DynamoDB Streams
- ✅ IAM (users, roles, policies, groups)
- ✅ STS (全部7个操作)
- ✅ Kinesis (streams, shards, fan-out)
- ✅ KMS (sign, verify, re-encrypt)

### 4. 测试覆盖
- 408/408 SDK 测试通过
- 20+ AWS 服务支持

## 快速开始

```yaml
# docker-compose.yml
services:
  floci:
    image: hectorvent/floci:latest
    ports:
      - "4566:4566"
    volumes:
      - ./data:/app/data
```

```bash
docker compose up
```

环境变量配置：
```bash
export AWS_ENDPOINT_URL=http://localhost:4566
export AWS_DEFAULT_REGION=us-east-1
export AWS_ACCESS_KEY_ID=test
export AWS_SECRET_ACCESS_KEY=test
```

## Docker 镜像标签

| 标签 | 描述 |
|------|------|
| `latest` | Native 镜像 - 亚秒级启动 (推荐) |
| `latest-jvm` | JVM 镜像 - 更广泛的平台兼容性 |
| `x.y.z` | 固定版本发布 |

## 配置选项

可通过环境变量 (`FLOCI_` 前缀) 进行配置：

| 变量 | 默认值 | 描述 |
|------|--------|------|
| `QUARKUS_HTTP_PORT` | 4566 | HTTP 端口 |
| `FLOCI_DEFAULT_REGION` | us-east-1 | 默认 AWS 区域 |
| `FLOCI_DEFAULT_ACCOUNT_ID` | 000000000000 | 默认 AWS 账户 ID |
| `FLOCI_STORAGE_MODE` | hybrid | 存储模式 (memory/persistent/hybrid/wal) |
| `FLOCI_STORAGE_PERSISTENT_PATH` | ./data | 数据目录 |

## SDK 集成示例

**Java (AWS SDK v2):**
```java
DynamoDbClient client = DynamoDbClient.builder()
    .endpointOverride(URI.create("http://localhost:4566"))
    .region(Region.US_EAST_1)
    .credentialsProvider(StaticCredentialsProvider.create(
        AwsBasicCredentials.create("test", "test")))
    .build();
```

**Python (boto3):**
```python
import boto3
client = boto3.client("s3",
    endpoint_url="http://localhost:4566",
    region_name="us-east-1",
    aws_access_key_id="test",
    aws_secret_access_key="test")
```

**Node.js (AWS SDK v3):**
```javascript
import { S3Client } from "@aws-sdk/client-s3";
const client = new S3Client({
    endpoint: "http://localhost:4566",
    region: "us-east-1",
    credentials: { accessKeyId: "test", secretAccessKey: "test" },
    forcePathStyle: true,
});
```

## 与 LocalStack 对比总结

Floci 相比 LocalStack 社区版的优势：

1. **无需认证** - 完全免费，无需任何认证令牌
2. **持续更新** - 继续提供安全更新
3. **资源占用低** - 启动更快、内存更少、镜像更小
4. **功能完整** - 支持更多 AWS 服务的完整功能
5. **开源协议** - MIT 许可证，无使用限制
