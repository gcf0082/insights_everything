## 核心变革摘要

**生效日期：2026年6月1日**

GitHub 宣布所有 Copilot 订阅计划将从此前的"高级请求单元(PRU)"计费模式转变为**基于使用量的计费模式**，即引入 GitHub AI Credits。

## 变革原因

1. **产品形态的根本性演变**：Copilot 已从单纯的编辑器内助手演变为具备自主能力的��能体平台，可运行长时间、多步骤的编码会话
2. **推理成本大幅增加**：代理式使用模式带来显著更高的计算和推理需求
3. **现有模式不可持续**：当前的按请求计费模式无法反映不同使用场景的真实成本差异
4. **确保服务可持续性**：更好地平衡定价与实际使用成本，维持长期服务可靠性

## 主要变化内容

### 1. 计费机制变化
- **旧模式**：按高级请求单元(PRU)计数
- **新模式**：按 token 消耗计费（包括输入、输出和缓存 token）

### 2. 基础价格保持不变
| 计划类型 | 价格 | AI Credits配额 |
|---------|------|---------------|
| Copilot Pro | $10/月 | $10/月 |
| Copilot Pro+ | $39/月 | $39/月 |
| Copilot Business | $19/用户/月 | $19/用户/月 |
| Copilot Enterprise | $39/用户/月 | $39/用户/月 |

### 3. 重要细节
- **代码补全和Next Edit建议**：所有计划均包含，不消耗 AI Credits
- **取消回退体验**：用户用完额度后将受可用 credits 和管理员预算控制限制，不再降级到低成本模型
- **Copilot代码审查**：将同时消耗 GitHub AI Credits 和 GitHub Actions 分钟数

## 对个人用户的影响

### 订阅用户
- Pro 和 Pro+ 月度订阅将获得与当前订阅价格等值的月度 AI Credits
- 年度订阅用户在计划到期前保持现有请求计费模式，6月1日后模型乘数将增加

### 年度订阅过渡
- 年度计划到期后：自动转为 Copilot Free，可选择升级到付费月度计划
- 可提前转为月度付费计划，剩余年度计划价值将按比例折算为 credits

## 对企业和组织的影响

### 过渡期优惠（6月-8月）
| 计划类型 | 月度AI Credits |
|---------|---------------|
| Copilot Business | $30 |
| Copilot Enterprise | $70 |

### 新增功能
- **资源池化**：组织内可共享 credits，消除资源闲置
- **预算控制**：管理员可在企业、成本中心和用户级别设置预算
- **灵活消费**：额度用尽后，组织可选择继续按公布费率消费或限制支出

## 预览账单

为帮助用户过渡，GitHub 将于5月初推出**预览账单**功能，用户可在 Billing Overview 页面查看预计成本。

## 关键要点

1. **价格不变**：所有计划的基础定价维持不变
2. **完全透明**：用户可全面掌控支出，追踪使用情况
3. **按需购买**：可随时购买额外 AI Credits
4. **平稳过渡**：预览账单和过渡期优惠帮助用户适应新模式
