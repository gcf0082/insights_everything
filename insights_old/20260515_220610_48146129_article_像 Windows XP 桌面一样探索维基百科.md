## 网站概述

Wikipedia File Explorer 是一个极具创意的互动式网站，它将 Wikipedia 的内容以 Windows XP 桌面界面的形式呈现，让用户能够像浏览本地文件夹一样探索 Wikipedia 的丰富资源。

## 主要功能

### 1. Wikipedia 浏览器
用户可以像操作 Windows XP 文件资源管理器一样浏览 Wikipedia 的所有分类。点击分类文件夹可以展开子分类，点击文章则会像打开文档一样呈现内容。整个 Wikipedia 的内容基本上都可以通过这种方式访问。

### 2. Media 媒体浏览器
这是 Wikimedia Commons 的分类浏览器。用户可以浏览和查看 Wikimedia Commons 上的图片资源。更实用的功能是：右键点击任意图片可以将它设置为桌面背景。

### 3. Geofile Explorer（开发中）
这是一个正在开发中的项目，旨在让用户能够像浏览本地文件夹一样探索地球。用户可以拖放图片上传到文件夹，或者右键写入文本笔记。

## 设计特点

- **复古美学**: 完全复刻了 Windows XP 的经典界面，包括任务栏、开始菜单、桌面图标等元素
- **怀旧体验**: 提供了极具情怀的用户体验，让熟悉 Windows XP 的用户感到无比亲切
- **交互细节**: 模拟了真实的桌面操作，如右键菜单、窗口拖拽、状态栏显示等
