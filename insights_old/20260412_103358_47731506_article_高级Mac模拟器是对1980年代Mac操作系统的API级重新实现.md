# 洞察报告：Advanced Mac Substitute

## 基本信息

- **洞察链接**: https://www.v68k.org/advanced-mac-substitute/
- **网站名称**: v68k - An m68k emulator
- **内容概述**: Advanced Mac Substitute 是一个1980年代Mac OS的API级重新实现项目

## 核心发现

### 项目简介

Advanced Mac Substitute (AMS) 是一个**API级别的重新实现**，而非传统模拟器。它在模拟68K处理器的同时，实际上**替换了整个操作系统**——直接启动应用程序，无需启动阶段。

### 技术架构

- **后端**: 包含68K模拟器，可在任何POSIX类系统上构建运行
- **前端**: 通用位图终端抽象，支持多种平台
  - SDL2（多平台）
  - macOS自定义实现
  - X11
  - Linux framebuffer (fbdev)

### 支持的功能

- 1位深图形
- 区域（Regions）
- 圆形和圆角矩形
- 线条
- 光标
- GrafPorts
- 文本
- 窗口
- 控件
- 菜单
- 对话框

### 可运行的应用示例

项目能够运行1984年的四款游戏：
1. **Amazing** - 迷宫追踪游戏
2. **Solitaire** - 纸牌游戏
3. **Missile** - Missile Command克隆
4. **IAGO** - 黑白棋游戏

还展示了其他应用演示：
- MacPaint（绘图应用）
- Lode Runner（动作游戏）
- The Fool's Errand（游戏）
- Nyanochrome Cat（动画演示）

### 源码位置

源代码托管在GitHub：https://github.com/jjuran/metamage_1/tree/master/68k

### 运行平台

- macOS / OS X
- X Window System
- Linux framebuffer控制台
- VNC客户端

## 总结

Advanced Mac Substitute是一个创新的复古计算项目，它不是简单模拟硬件，而是从API层面重新实现了1980年代Mac OS的功能，使得古老的68K Mac应用程序能够在现代系统上无需原始Apple ROM或系统软件即可运行。
