## 一、项目概述

**tvision** 是由开发者 **magiblot** 创建的一个现代版 Turbo Vision 2.0 移植项目。Turbo Vision 是 Borland 公司在 90 年代初开发的经典文本用户界面（TUI）框架，现在该项目已实现了跨平台支持和 Unicode 字符支持。

## 二、项目背景与目标

### 2.1 项目起源

该项目始于 2018 年底，最初的目标是：

1. 通过尽可能少地修改原始代码，使 Turbo Vision 能够在 Linux 上运行
2. 保持其在 DOS/Windows 上的可用性
3. 在源代码层面与旧的 Turbo Vision 应用程序保持最大程度的兼容性

### 2.2 发展历程

- **2018 年底**：项目启动
- **2020 年 5 月**：达到与原始版本功能接近持平的状态，代码开源
- **2020 年 7-8 月**：成功集成了完整的 Unicode 支持，并创建了新的文本编辑器 Turbo

## 三、核心特性

### 3.1 跨平台支持

| 平台 | 支持方式 | 特殊说明 |
|------|----------|----------|
| Linux | Ncurses (libncursesw) | 完整的鼠标和键盘支持 |
| Windows | Win32 Console API | 需要 Vista 或更高版本 |
| DOS | Borland C++ | 无 Unicode 支持 |

### 3.2 Unicode 支持

- 采用 UTF-8 编码，与现有 `char *` 数据类型兼容
- 支持双宽度字符（如中日韩统一表意文字）
- 支持组合字符（如梵文天城文）
- 兼容终端仿真器的字符宽度测量

### 3.3 颜色支持

- 从原始的 16 色扩展到 24 位颜色（1677 万色）
- 通过 `COLORTERM` 环境变量自动检测终端颜色能力

### 3.4 输入设备支持

- 鼠标滚轮（中键）
- 鼠标拖拽滚动
- X10 和 SGR 鼠标编码
- 键盘修饰键（Shift, Ctrl, Alt）
- Wayland 和 X11 剪贴板集成

## 四、架构特性

### 4.1 API 扩展

项目在保持向后兼容的同时添加了多个新特性：

1. **TStringView**: 类似于 `std::string_view`，用于处理字符串视图
2. **TSpan<T>**: 类似于 `std::span`，用于处理连续内存区域
3. **TDrawSurface 和 TSurfaceView**: 用于绘图的表面系统
4. **TKey**: 用于定义新的键组合

### 4.2 事件系统

- 鼠标滚轮事件 `evMouseWheel`
- 三击事件 `meTripleClick`
- 定时器支持 (`setTimer`, `killTimer`)
- 异步事件等待 (`waitForEvents`)

### 4.3 窗口系统

- 可调整大小的窗口
- 窗口拖拽（中键拖拽）
- 层叠（cascade）和平铺（tile）排列
- 任意屏幕大小支持（最高 32767 行或列）

## 五、应用场景

### 5.1 适用场景

Turbo Vision 适用于以下场景：

1. **终端应用开发**: 无需处理终端能力差异和直接 I/O
2. **跨平台工具**: 编写在 Linux 和 Windows 上无缝运行的应用程序
3. **复古应用移植**: 将已有的 DOS/Windows Turbo Vision 应用迁移到现代平台
4. **快速 UI 开发**: 利用现有的窗口、菜单、对话框等组件构建界面

### 5.2 实际应用案例

- **Turbo**: 概念验证的文本编辑器
- **tvterm**: 概念验证的终端仿真器
- **TMBASIC**: 用于创建控制台应用程序的编程语言

## 六、技术细节

### 6.1 构建要求

- **C++14** 或更高版本的编译器
- **Linux**: libncursesw, libgpm（可选，用于鼠标支持）
- **Windows**: MSVC 或 MinGW
- **运行时**: xsel/xclip (X11) 或 wl-clipboard (Wayland)

### 6.2 构建方式

支持多种构建系统：
- CMake（推荐）
- Vcpkg 包管理器
- Borland C++ Makefile（仅适用于 DOS/Windows）

### 6.3 环境变量

| 环境变量 | 描述 | 默认值 |
|----------|------|--------|
| TVISION_MAX_FPS | 最大刷新率 | 60 |
| COLORTERM | 终端颜色模式 | - |
| ESCDELAY | ESC 键延迟（毫秒） | 10 |
| TVISION_USE_STDIO | 使用标准 I/O | - |
