# 洞察报告：Mistral AI 发布 Mistral Medium 3.5 和 Vibe 远程代理

- **洞察链接**: https://mistral.ai/news/vibe-remote-agents-mistral-medium-3-5
- **来源**: Mistral AI 官方新闻
- **发布日期**: 2025年
- **报告生成日期**: 2026-04-30

---

## 核心要点

### 1. Mistral Medium 3.5 模型发布

Mistral Medium 3.5 是 Mistral 首款旗舰级合并模型，现处于公开预览阶段。该模型具有以下特点：

- **参数规模**: 128B 密集模型
- **上下文窗口**: 256k
- **功能整合**: 集指令遵循、推理和编码能力于单一权重集
- **自托管能力**: 最低仅需 4 张 GPU 即可运行
- **推理可配置**: 可根据请求配置推理工作量，既能快速回复聊天，也能处理复杂代理任务
- **视觉编码器**: 从零训练，支持可变图像尺寸和宽高比

### 2. 性能表现

Mistral Medium 3.5 在多项基准测试中表现优异：

- **SWE-Bench Verified**: 得分 77.6%，超越 Devstral 2 和 Qwen3.5 397B A17B
- **τ³-Telecom**: 得分 91.4，展现强大的代理能力

### 3. Vibe 远程代理

Mistral 宣布推出 Vibe 远程编码代理功能，核心特性包括：

- **云端运行**: 编码任务在云端运行，无需本地设备
- **并行处理**: 支持多个任务并行运行，不再成为每一步的瓶颈
- **启动方式**: 可通过 Mistral Vibe CLI 或 Le Chat 启动
- **本地会话迁移**: 正在进行的本地 CLI 会话可"传送"至云端继续运行
- **实时监控**: 可实时查看代理操作、文件差异、工具调用、进度状态

### 4. 集成能力

Vibe 与多种开发工具集成：

- **代码管理**: GitHub（代码和 Pull Request）
- **项目管理**: Linear 和 Jira（问题跟踪）
- **监控**: Sentry（事件监控）
- **通讯**: Slack 或 Teams（通知）

### 5. Le Chat 工作模式（预览）

全新 Work mode 功能特点：

- **跨工具工作流**: 一次性处理邮件、消息和日历
- **研究合成**: 深入研究主题，生成结构化简报或报告
- **收件箱分类**: 邮件分类、草拟回复
- **任务创建**: 可从团队讨论中创建 Jira 问题
- **持久会话**: 会话持续时间更长，代理可跨多轮持续工作
- **可见性**: 每个操作都可见，敏感任务需用户确认

### 6. 定价与可用性

- **API 价格**: 输入 $1.50/百万 tokens，输出 $7.50/百万 tokens
- **开源权重**: 在 Hugging Face 上以修改的 MIT 许可证发布
- **可用平台**: Mistral Vibe、Le Chat（Pro、Team、Enterprise 计划）
- **GPU 加速**: 通过 NVIDIA GPU 端点提供

---

## 总结

Mistral Medium 3.5 的发布标志着 AI 编码助手从本地向云端的重要转型。通过将编码代理迁移到云端运行，开发者可以并行启动多个任务，实现真正的异步工作流程。这一更新使得开发者能够专注于高价值的决策工作，而将重复性编码任务交给代理完成。Work mode 的引入进一步扩展了 Le Chat 的能力，使其能够处理复杂的多步骤任务，推动人机协作的新范式。
