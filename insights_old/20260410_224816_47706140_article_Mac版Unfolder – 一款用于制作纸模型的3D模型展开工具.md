## 一、产品概述

Unfolder是一款专为纸艺设计师打造的3D模型展开工具，可将3D模型快速展开为2D纸张模板，便于打印和制作纸雕模型。该软件仅支持Mac平台，售价29.99美元。

## 二、核心功能

### 2.1 自动展开

Unfolder采用智能展开算法，能在几秒钟内将3D模型生成2D展开图。该算法针对纸艺制作进行了高度优化，可最大程度减少后期编辑的工作量。

### 2.2 零件分割与合并

软件支持在2D视图或3D视图中通过点击边缘来轻松分割和合并零件，满足个性化需求。

### 2.3 折翼编辑

提供全面的折翼编辑功能，包括：

- 切换折翼类型
- 添加新折翼
- 删除折翼
- 合并折翼
- 调整折翼形状
- 自动优化折翼形状以避免碰撞

### 2.4 样式定制

可自定义以下元素的外观：

- 线条颜色
- 线条宽度
- 线条虚线样式
- 切割线、山脊折线、山谷折线可独立控制

### 2.5 导出功能

支持多种导出格式，满足不同用途需求：

| 格式 | 用途 |
|------|------|
| PDF | 打印（矢量格式） |
| DXF | CAD软件编辑 |
| SVG | 矢量图形编辑器编辑 |
| PNG | 打印或位图编辑器编辑 |

## 三、工作流程

典型使用流程如下：

1. **创建3D模型**：在任意支持OBJ格式的多边形建模软件中创建3D模型
2. **导入Unfolder**：将3D模型导入软件生成2D展开图
3. **编辑模板**：在Unfolder中进行折翼、分割等编辑
4. **导出**：导出模板进行打印或进一步编辑

## 六、技术规格

### 6.1 支持的导入格式

- OBJ格式（3D对象）

### 6.2 导入特性

- 支持材质和纹理
- 可与Blender和Cinema 4D一键同步3D对象

### 6.3 模型要求

- 建议使用低多边形模型，高面数模型展开的模板可能难以切割和粘贴
- 注意面法线方向，决定纸张的打印面
- 确保模型流形性正确

## 七、工具介绍

软件提供5种工具：

| 工具 | 功能 |
|------|------|
| 标尺工具 | 测量功能 |
| 折翼工具 | 切换、添加、删除和合并折翼 |
| 结合工具 | 分割和合并零件 |
| 子选择工具 | 选择边缘和折翼，调整折翼形状 |
| 选择工具 | 选择、移动和旋转零件 |

## 八、视图与导航

### 8.1 视图区域

软件界面包含四个区域：

- 工具栏：常用命令
- 3D视图：显示3D对象
- 2D视图：显示展开的2D零件
- 检查器：文档和工具设置

### 8.2 导航操作

- **旋转**：按住鼠标左键拖动
- **平移**：按住鼠标中键拖动，或按住空格键+鼠标左键
- **缩放**：按住鼠标右键拖动，或滚动鼠标滚轮
- **缩放至零件**：右键选择"缩放至零件"
- **缩放至全部**：右键选择"缩放至全部"

## 九、总结

Unfolder是一款功能专业的纸艺3D模型展开工具，其核心优势在于：

1. 智能化的自动展开算法，大幅减少手动编辑工作量
2. 全面的编辑功能，支持折翼、分割、合并等操作
3. 丰富的导出格式，满足打印、CNC切割等不同需求
4. 与主流3D软件（Blender、Cinema 4D）的良好集成

对于纸艺爱好者和专业纸艺设计师而言，Unfolder是Mac平台上值得考虑的创作工具。
