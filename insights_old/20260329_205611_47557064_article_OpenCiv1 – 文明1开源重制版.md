## 项目概述

OpenCiv1 是一个开源项目，旨在完全重写1991年由Sid Meier和Bruce Shelley设计的经典游戏《文明1》（Civilization 1）。该项目使用现代技术栈重建这款经典4X策略游戏，使其能够在当代操作系统上运行。

## 技术架构

### 核心技术栈
- **.NET 8** - 跨平台运行时
- **Avalonia UI** - 跨平台UI框架
- **C#** - 主要编程语言

### 支持平台
- Windows
- Linux
- MacOSX

### 技术实现特点
1. **虚拟CPU模拟器**：游戏的部分汇编代码通过虚拟CPU进行模拟
2. **代码重写**：将伪汇编语言翻译为原生C#代码
3. **基于原始版本**：游戏逻辑基于原始DOS文明1版本475.05的反汇编

## 编译指南

```bash
git clone https://github.com/rajko-horvat/OpenCiv1
cd OpenCiv1
dotnet build -c Debug
```

## 版权说明

- 代码并非完整可用的游戏副本
- 运行OpenCiv1必须合法拥有原始DOS文明1游戏
- 原始游戏的文件（.txt、.pic、.pal）因版权原因未包含在仓库中
- 重写的代码使用MIT许可证，与原始游戏版权无关

## 额外快捷键

- Alt + D - 启用/禁用调试模式
- Alt + P - 暂停/恢复游戏
- Alt + 1/2/3 - 显示/隐藏屏幕

## 总结

OpenCiv1是一个雄心勃勃的开源项目，旨在通过现代技术重制经典文明1游戏。项目采用跨平台技术栈（.NET 8 + Avalonia UI），在保持原版游戏规则和体验的同时，为经典游戏注入新的生命力。尽管需要原始游戏文件才能运行，但该项目为喜欢经典策略游戏的玩家提供了一个现代化的选择。
