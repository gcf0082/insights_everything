## 项目概述

Cal.diy 是一个社区驱动的完全开源预约调度平台，是 Cal.com 的社区版分支，移除了所有企业/商业代码。该项目专为希望完全控制调度基础设施的个人和自托管用户设计，无需任何商业依赖。

## 与 Cal.com 的区别

- **无企业功能** — 团队、组织、分析、工作流、SSO/SAML 等企业版专有功能已被移除
- **无需许可证密钥** — 开箱即用，无需 Cal.com 账户或许可证
- **100% 开源** — 整个代码库采用 MIT 许可证，无"开放核心"模式
- **社区维护** — 欢迎贡献，直接合并到本项目

## 技术栈

- Next.js — React 框架
- tRPC — 类型安全 API
- React.js — UI 库
- Tailwind CSS — 样式框架
- Prisma.io — ORM
- Daily.co — 视频通话

## 主要功能

1. **预约管理** — 创建和管理各类会议预约
2. **日历集成** — 支持 Google Calendar、Microsoft Calendar、Zoom 等
3. **可自定义的预订页面** — 灵活的主题和配置
4. **事件类型** — 多种事件类型支持
5. **团队可用性** — 基础团队功能

## 部署方式

### Docker 部署（推荐）

```bash
git clone --recursive https://github.com/calcom/cal.diy.git
cd cal.diy
cp .env.example .env
# 配置环境变量
openssl rand -base64 32  # 生成 NEXTAUTH_SECRET
openssl rand -base64 24  # 生成 CALENDSO_ENCRYPTION_KEY
docker compose up -d
```

### 本地开发

```bash
git clone https://github.com/calcom/cal.diy.git
cd cal.diy
yarn
cp .env.example .env
yarn dx
```
