## 内容概述

本次讨论围绕 lawsofux.com 这个收集了30条 UX 定律的网站展开。该网站以海报形式展示这些设计原则，涵盖如雅各布定律、菲特定律、希克定律等经典 UX 概念。

## 主要观点总结

### 正面评价

- 许多 UX 定律追溯到 Nielsen Norman Group 的研究数据
- 对于非专业设计师来说，这些"定律"可作为实用的检查清单
- 有用户尝试将 UX 定律用于 AI 辅助的 UI 审查，生成改进建议

### 批评意见

1. **内容深度不足**: 许多条目只是简单的定义，缺乏实际应用示例和具体指导
2. **网站本身违反 UX 原则**:
   - 过多的图片和动画占用了大量屏幕空间
   - 页面加载速度慢（Doherty Threshold 阈值超过 400ms）
   - 语言菜单交互问题
3. **命名不当**: 一些条目（如"认知偏误"）并非真正的"定律"
4. **缺乏交互性**: 无法切换到双窗格布局以便边浏览边参考

### 建议与讨论

- 添加"稳定性"定律：避免为了变化而变化
- 不要使用无意义的图标
- 以线性流程而非树状结构呈现信息
- 避免将观点包装为事实

## 相关资源推荐

- 《Designing Interfaces》 by Jennifer Tidwell - 被推荐为 UI 设计模式参考书
- shadcn、base UI 等组件库

## 结论

Laws of UX 作为一本 UX 设计原则的"字典"有一定参考价值，但作为实践指导工具仍显不足。有用户指出，该网站自身在 UX 方面存在诸多问题，这构成了一个有趣的讽刺。
