## 网站概述

claude.nagdy.me 是一个交互式学习平台，旨在帮助用户通过实践而非阅读来学习 Claude Code。该平台提供 11 个交互式模块，包含终端模拟器、配置构建器和测验，无需任何设置即可开始学习。

---

## 主要功能

### 学习模块
平台提供 11 个从入门到高级的学习模块：

1. **Slash Commands** (30分钟) - 初级
2. **Memory & CLAUDE.md** (45分钟) - 初级
3. **Project Setup** (45分钟) - 初级
4. **Commands Deep Dive** (30分钟) - 初级
5. **Skills** (1小时) - 中级
6. 以及更多模块...

### 核心特性

- **无需安装即可尝试**: 用户可以在浏览器中的终端模拟器中练习 slash commands、hooks 和 skills，无需设置或 API 密钥
- **构建真实配置**: 交互式表单可生成 CLAUDE.md、hooks 和插件配置，用户可以直接复制到项目中
- **验证理解**: 每个模块结束时都有测验，回答错误会获得解释，而不仅仅是答案

### 工具和资源

- **Playground**: 在专用全尺寸终端沙盒中练习命令
- **Config Builder**: 生成 CLAUDE.md、skills、agents、hooks、MCP 服务器和插件配置
- **Cheat Sheet**: 可打印的常用命令、快捷键和文件清单
- **Feature Index**: 按类型、级别、类别和模块搜索 Claude Code 完整功能列表
- **Resources**: 模板包、外部指南和社区链接
