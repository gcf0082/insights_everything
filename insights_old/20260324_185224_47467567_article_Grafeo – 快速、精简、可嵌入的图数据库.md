# Grafeo 洞察报告

## 基本信息

- **洞察链接**: https://grafeo.dev/
- **日期**: 2026-03-24
- **类型**: 图数据库技术调研

## 产品概述

Grafeo 是一个用 Rust 构建的高性能、轻量级、可嵌入的图数据库管理系统。该项目在 LDBC 社交网络基准测试中表现优异，无论是嵌入模式还是独立服务器模式，性能均为测试中最快的产品之一，且内存占用低于其他内存数据库。

## 核心特性

### 1. 高性能架构

Grafeo 核心引擎采用 Rust 开发，利用向量化执行、自适应分块和 SIMD 优化操作实现卓越性能。架构特点包括：

- **推送式执行引擎**：采用 morsel 驱动的并行处理
- **列式存储**：支持类型特定压缩
- **成本查询优化器**：具有基数估计能力
- **区域映射**：实现智能数据跳过
- **MVCC 事务**：快照隔离级别的 ACID 合规性

### 2. 双重数据模型支持

Grafeo 同时支持两种主流图数据模型：

**标签属性图（LPG）**：

- 节点和边支持标签与属性
- 丰富的属性数据类型
- 适用于社交网络、知识图谱、应用数据

**RDF（三元组）**：

- 主语-谓语-对象声明
- SPO/POS/OSP 索引优化查询
- W3C 标准兼容
- 适用于语义网、链接数据、本体论

### 3. 多语言查询支持

Grafeo 支持六种查询语言，开发者可根据项目需求和团队技能选择：

| 查询语言 | 数据模型 | 风格 |
|---------|---------|------|
| GQL（默认） | LPG | ISO 标准，声明式模式匹配 |
| Cypher | LPG | Neo4j 兼容，ASCII 艺术模式 |
| Gremlin | LPG | Apache TinkerPop，基于遍历 |
| GraphQL | LPG/RDF | 模式驱动，Web 开发者熟悉 |
| SPARQL | RDF | W3C RDF 查询标准 |
| SQL/PGQ | LPG | SQL:2023 GRAPH_TABLE |

### 4. 向量搜索能力

基于 HNSW 的相似性搜索，支持量化技术（标量、二进制、产品量化），可结合图遍历与语义相似性，适用于 AI 应用场景。

### 5. 部署灵活性

- **嵌入模式**：直接集成到应用程序，无需外部依赖
- **独立服务器**：提供 REST API 和 Web UI
- 适用于边缘设备到生产集群的各种场景

### 6. 多语言绑定

提供以下编程语言的原生绑定：

- Python（PyO3）
- Node.js/TypeScript（napi-rs）
- Go（CGO）
- C（FFI）
- C#（.NET 8 P/Invoke）
- Dart（dart:ffi）
- WebAssembly（wasm-bindgen）

### 7. 生态系统

- AI 集成：LangChain、LlamaIndex、MCP 支持
- 交互式笔记本小部件
- 基于 WebAssembly 的浏览器图可视化
- 独立服务器 Web UI
- 基准测试工具

## 安装方式

各语言安装命令：

```bash
# Python
uv add grafeo

# Node.js
npm install @grafeo-db/js

# Go
go get github.com/GrafeoDB/grafeo/crates/bindings/go

# Rust
cargo add grafeo

# C#
dotnet add package GrafeoDB

# Dart
# pubspec.yaml
dependencies:
  grafeo: ^0.5.21

# WebAssembly
npm install @grafeo-db/wasm
```

## 总结

Grafeo 是一个功能全面的现代图数据库，特别适合需要高性能、可嵌入性和多语言支持的应用程序。其 Rust 核心保证了内存安全和并发安全性，丰富的查询语言支持使其能够适配各种技术栈，AI 集成能力则为知识图谱和向量搜索场景提供了良好支持。
