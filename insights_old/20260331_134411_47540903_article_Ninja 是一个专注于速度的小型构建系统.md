## 项目概览

Ninja 是一个专注于速度的小型构建系统。它被设计为替代 Make，以更高效地处理大型代码库的构建。Ninja 由 Google 开发，最初用于 Chromium 项目的构建，现已被广泛应用于各种 C/C++ 项目中。

## 项目特点

### 1. 高性能
Ninja 的核心设计目标是速度。它通过以下方式实现高性能：
- 并行构建支持
- 增量构建优化
- 最小化解析开销

### 2. 简单性
- 配置文件简洁易读
- 专注于构建执行，不包含复杂的构建逻辑
- 易于集成到各种构建系统中

### 3. 广泛兼容性
- 支持 Linux、Mac 和 Windows 平台
- 可与 CMake、Meson 等构建系统配合使用

## 构建方式

Ninja 支持两种构建方式：

### Python 方式
```bash
./configure.py --bootstrap
./ninja all     # 构建 ninja_test 等辅助二进制文件
./ninja_test    # 运行单元测试套件
```

### CMake 方式
```bash
cmake -Bbuild-cmake -DBUILD_TESTING=OFF
cmake --build build-cmake
```
