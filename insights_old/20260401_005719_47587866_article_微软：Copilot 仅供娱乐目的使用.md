# 洞察报告：Microsoft Copilot 服务条款

## 基本信息

- **洞察链接**: https://www.microsoft.com/en-us/microsoft-copilot/for-individuals/termsofuse
- **报告日期**: 2026-04-01
- **信息来源**: Microsoft 官方网站

---

## 一、服务条款概述

Microsoft Copilot 服务条款（Terms of Use）于2025年10月24日生效。本条款规定了用户在使用 Microsoft Copilot 时需要遵守的规则和条件。

### 主要更新内容

- 澄清了条款适用范围
- 新增了 Copilot Actions、Copilot Labs 和购物体验的相关条款
- 修订了行为准则，明确了使用限制
- 重写并重组了条款，使其更加清晰简洁

---

## 二、条款适用范围

本条款适用于以下 Copilot 服务：

- 电脑或移动设备上的独立 Copilot 应用
- copilot.microsoft.com、copilot.com 和 copilot.ai 网站
- 通过其他 Microsoft 应用和网站与 Copilot 的对话
- 通过第三方应用和平台与 Copilot 的对话
- 其他链接到本条款的 Copilot 品牌应用和服务

**不适用**：Microsoft 365 Copilot 应用或服务（除非明确说明适用）

---

## 三、关键定义

| 术语 | 定义 |
|------|------|
| **您/您的** | 使用 Copilot 的个人 |
| **我们/我们的** | Microsoft 及其关联公司 |
| **提示 (Prompt)** | 您发送给 Copilot 的内容（文本、音频、图片、文件、语音或视频） |
| **回复 (Response)** | Copilot 发送给你的内容 |
| **创作 (Creations)** | Copilot 根据您的提示创建的原创内容或艺术作品 |
| **您的内容** | 您与 Copilot 对话中的提示和回复 |
| **操作 (Actions)** | Copilot 代表您自动执行的任务 |

---

## 四、使用年龄要求

- **最低年龄**：通常要求 13 岁，部分国家/地区要求 18 岁
- **未成年用户**：未满 18 岁或未登录使用的用户，部分功能可能被限制或关闭
- **禁止行为**：禁止使用机器人或爬虫程序访问 Copilot，仅限个人使用

---

## 五、使用须知

### AI 特性与局限性

1. **可能出错**：Copilot 是 AI 驱动的对话服务，可能产生错误信息
2. **信息来源不受控制**：Copilot 可能使用互联网上的信息，这些内容不受 Microsoft 控制
3. **答案可能不准确**：回复可能看起来令人信服，但实际不完整、不准确或不适当
4. **非唯一性**：您获得的回复可能与其他用户相同或相似

### 用户责任

- **自行判断**：在使用 Copilot 提供的信息做决定或采取行动前，请务必自行核实
- **报告问题**：如发现错误或不当内容，请使用报告或反馈功能
- **法律问题**：如对 Copilot 所说的内容有法律疑虑，请使用"报告疑虑"页面

---

## 六、行为准则

用户在使用 Copilot 时必须遵守以下规定：

### 禁止事项

1. **伤害自己或他人**：不得使用 Copilot 帮助骚扰、虐待、威胁或恐吓他人
2. **损害服务**：禁止使用机器人或爬虫、进行技术攻击、滥用、提示操纵、"越狱"等行为
3. **侵犯隐私**：禁止推断他人的敏感信息（如种族、政治观点、宗教信仰等）、不得进行面部识别、收集他人敏感个人数据
4. **欺骗他人**：禁止使用 Copilot 帮助误导或欺骗他人、创建虚假信息
5. **侵犯权利**：不得侵犯他人的知识产权或宣传权
6. **不当内容**：禁止创建或分享成人内容、暴力内容、仇恨言论、恐怖主义内容、自杀或暴力美化内容
7. **违法行为**：不得使用 Copilot 违法或帮助他人违法
8. **深度伪造**：未经许可，不得使用 Copilot 创建或编辑他人的图像、语音或视频（深度伪造）

Microsoft 保留以下权利：

1. **随时停止服务**：可随时限制、暂停或停止提供 Copilot 或任何功能
2. **限制访问**：可限制、暂停或永久撤销用户访问 Copilot 的权限
3. **违规处理**：可阻止、限制或删除违反条款的内容
4. **申诉机制**：如用户认为错误被限制访问，可提交申诉请求重新评估

## 八、购物体验

当通过 Copilot 购物时：

- **第三方商家**：产品由第三方商家销售和发货，非 Microsoft 本身
- **责任划分**：Microsoft 不对您与商家之间的纠纷负责
- **适用条款**：购买行为受商家条款和条件约束
- **个人信息**：Microsoft 会根据隐私声明收集、使用和共享您的个人信息

## 九、重要披露与警告

1. **广告**：Copilot 可能包含广告
2. **人工审查**：Copilot 可能包含人工审查数据的过程，不应分享您不希望被审查的信息
3. **无保证声明**：**Microsoft 不对 Copilot 做出任何保证或陈述**
4. **娱乐目的**：Copilot 仅供娱乐目的，可能出错，可能无法按预期工作
5. **自行承担风险**：不要依赖 Copilot 提供重要建议
6. **责任豁免**：如选择公开或与他人分享 Copilot 的回复，您将全权负责
7. **赔偿义务**：您同意赔偿 Microsoft 因您使用 Copilot 而产生的任何索赔、损失和费用

## 十、内容所有权

- **您的内容**：您不拥有您的内容，但您授予 Microsoft 使用您内容 的权限
- **使用范围**：Microsoft 可以复制、分发、传输、公开显示、公开表演、编辑、翻译和格式化您的内容
- **无需通知**：Microsoft 无需付费、征得许可或告知您使用情况
- **隐私控制**：Copilot 中的隐私选项让您可以控制部分使用
- **保证**：您保证拥有您内容的全部权利，不会侵犯他人权利

## 总结

Microsoft Copilot 服务条款详细规定了用户与 AI 助手交互时的权利和义务。关键要点包括：

1. 用户需年满13岁（部分情况需18岁）
2. Copilot 可能出错，不应作为重要决策的唯一依据
3. 禁止用于违法、伤害他人或侵犯隐私等目的
4. 用户对自己的内容负责，需赔偿 Microsoft 因使用导致的损失
5. Microsoft 可以随时限制或终止服务
6. 购物相关内容由第三方商家负责

**建议**：用户在使用 Copilot 前应仔细阅读并理解这些条款，确保合规使用。
