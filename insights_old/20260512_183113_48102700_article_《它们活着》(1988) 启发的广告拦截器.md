## 项目概述

**They Live Adblocker** 是一个基于 uBlock Origin Lite 的浏览器广告屏蔽扩展的个性化分支。该项目将原本隐藏广告的方式改为用白色贴片替换，并在贴片上显示1988年电影《They Live》中的经典口号，如 "OBEY"、"CONSUME"、"WATCH TV"、"SLEEP"、"SUBMIT"、"CONFORM"、"STAY ASLEEP"、"BUY"、"WORK"、"NO INDEPENDENT THOUGHT"、"DO NOT QUESTION AUTHORITY"。

## 核心功能

1. **广告替换而非隐藏**：将原本被隐藏的广告元素替换为带有随机口号的白色贴片
2. **口号随机显示**：每个被屏蔽的广告会从口号列表中随机选择一个显示
3. **支持多种浏览器**：Chromium、Chrome、Brave、Edge、Firefox
4. **无需权限**：基于 uBlock Origin Lite (uBOL)，可在无特别权限模式下运行

## 技术实现

### 工作原理
- uBO Lite 的 cosmetic filtering 通常通过注入 CSS (`display: none !important`) 来隐藏广告元素
- 该分支修改了注入逻辑，改为应用白色遮罩并添加 `::after` 覆盖层
- 使用 MutationObserver 监听 DOM 变化，处理延迟加载的广告

### 关键文件
- `platform/mv3/extension/js/scripting/they-live.js` - 新增文件，包含口号列表、CSS生成器、DOM标记逻辑
- `platform/mv3/extension/js/scripting/css-{specific,generic,procedural-api}.js` - 调用位置
- `platform/mv3/extension/js/scripting-manager.js` - 注册 they-live.js


## 使用注意事项

1. **过滤模式设置**：默认 Basic 模式在网络层阻止广告，不会产生 DOM 元素，因此看不到替换效果。需要切换到 "Optimal" 或 "Complete" 模式才能看到口号贴片。
2. **个人兴趣项目**：并非官方 uBlock Origin 产品
3. **布局影响**：强制显示之前隐藏的元素可能偶尔会改变页面布局
4. **自定义过滤规则**：用户自定义的 cosmetic filters 仍会正常隐藏，不显示口号
5. **网络阻止的广告**：大多数由 uBO Lite 网络层阻止的广告不会被替换，只有 cosmetic-filtered 的广告会被替换
