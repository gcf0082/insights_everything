## 业务概况

作者运营了一个3D打印副业业务约8个月（2024年夏至2025年冬），主要产品是定制纸牌展示架（Trading Card Stand），通过邻居的纸牌业务获得订单。

### 核心财务数据

| 项目 | 金额 |
|------|------|
| 总收入 | $3,666 |
| 总支出 | $3,352 |

| 完成的订单 | ~50个 |
| 打印时长 | ~3000小时 |
| 设计时薪 | ~$25/小时 |

## 关键教训

### 1. 定价严重不足

评论者指出，这些数字远低于合理水平：
- 材料定价应为成本的10倍
- 打印时间基础费应为$3/小时
- 设计时间收费应为$100/小时（1000个零件以下）

### 2. 无法规模化

- 所有工作（设计、打印、组装）都需要创始人亲自完成
- 属于典型的"生活方式企业"，无法雇人扩张
- 周末和夜晚时间被完全占用

### 3. 商标侵权风险

- 销售带有球队logo（如波士顿凯尔特人）的产品可能构成商标侵权
- 客户提供的图片不代表拥有其版权

### 4. 打印机问题

- 频繁故障和喷嘴堵塞
- 3000小时运行却需要经常更换零件，不符合常理

### 5. 市场定位问题

- 目标市场太小众：在线拍卖纸牌卖家
- 塑料制品本身附加值低

## 作者的反思

- 严重低估了设计复杂产品所需时间
- 后期曾考虑提价，但更愿意将时间用于其他爱好
- 目前仍进行3D打印，但专注于赠送性质的玩具和礼物

---

## 社区观点总结

1. **不应该规模化**: 如果做得开心且能cover开支，不一定要扩张
2. **定价是关键**: 必须覆盖人工成本，时薪不应低于McDonald's
3. **3D打印适合DIY**: 树脂打印找服务商更划算，线材打印可自营
4. **IP风险**: 体育团队和游戏公司（如Warhammer）会严厉维权
5. **可作为兴趣**: 作为side project是可以的，但作为主业需要更成熟的产品
