## 产品概述

Grafeo 是一款使用 Rust 构建的高性能、轻量级、可嵌入的图数据库管理系统。

---

## 核心特性

### 高性能
- 在 LDBC 社交网络基准测试中表现优异
- 无论是嵌入模式还是独立服务器模式均表现出色
- 内存占用低于其他内存数据库
- 采用向量化执行、自适应分块和 SIMD 优化技术

### 多语言查询支持
支持六种查询语言：
- **GQL** (默认) - ISO 标准声明式模式匹配
- **Cypher** - Neo4j 兼容的 ASCII 艺术模式
- **Gremlin** - Apache TinkerPop 基于遍历的查询
- **GraphQL** - 模式驱动，熟悉 Web 开发者的查询方式
- **SPARQL** - W3C RDF 查询标准
- **SQL/PGQ** - SQL:2023 GRAPH_TABLE 原生图查询

### 双数据模型支持
- **LPG (标签属性图)**: 包含标签和属性的节点与边，适合社交网络、知识图谱、应用数据
- **RDF (资源描述框架)**: 三元组 (主语-谓语-宾语) 声明，适合语义网、链接数据、本体

### 向量搜索
- 基于 HNSW 的相似性搜索
- 支持量化技术 (Scalar、Binary、Product)
- 可结合图遍历与语义相似性

### 部署方式灵活
- **嵌入式**: 直接嵌入应用，零外部依赖
- **独立服务器**: REST API + Web UI
- 适用于边缘设备到生产集群

### 核心技术
- Core 数据库引擎使用 Rust 编写，无必需的 C 依赖
- 可选分配器 (jemalloc/mimalloc) 和 TLS
- 内存安全设计，无fearless concurrency
- 完整 ACID 合规，基于 MVCC 的快照隔离

### 多语言绑定
- Python (PyO3)
- Node.js/TypeScript (napi-rs)
- Go (CGO)
- C (FFI)
- C# (.NET 8 P/Invoke)
- Dart (dart:ffi)
- WebAssembly (wasm-bindgen)

### 生态系统
- AI 集成 (LangChain、LlamaIndex、MCP)
- 交互式笔记本小部件
- 浏览器端基于 WebAssembly 的图可视化
- 独立服务器带 Web UI
- 基准测试工具

## 快速开始

### Python 示例
```python
import grafeo

db = grafeo.GrafeoDB()

db.execute("""
    INSERT (:Person {name: 'Alix', age: 30})
    INSERT (:Person {name: 'Gus', age: 25})
""")

db.execute("""
    MATCH (a:Person {name: 'Alix'}), (b:Person {name: 'Gus'})
    INSERT (a)-[:KNOWS {since: 2024}]->(b)
""")

result = db.execute("""
    MATCH (p:Person)-[:KNOWS]->(friend)
    RETURN p.name, friend.name
""")
```

## 架构亮点

- **推送式执行引擎**: 基于 Morsel 的并行执行
- **列式存储**: 类型特定压缩
- **基于成本的查询优化器**: 基数估计
- **MVCC 事务**: 快照隔离
- **区域映射**: 智能数据跳过

## 总结

Grafeo 是一个功能全面的现代图数据库解决方案，特别适合需要高性能、多语言支持和灵活部署选项的应用场景。其 Rust 原生实现确保了内存安全和并发性能，而丰富的多语言绑定使其易于集成到各种技术栈中。
