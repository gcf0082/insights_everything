## 硬件信息

### 外壳材质
- 铝材切割的顶盖、屏幕边框、键盘框架和腕托
- 丙烯酸树脂制作的侧面板和透明底板

### 屏幕问题
- 触控球在合上屏幕时可能挤压屏幕，造成屏幕出现轻微压痕

### 侧面板
- 可更换钢制侧面板（官方有售）
- 原有喷漆容易脱落，需要重新喷漆处理

## 支持的操作系统

1. **9front** - Plan 9 操作系统的分支
2. **Alpine Linux** - 完全可用
3. **Void Linux** - 有 SD 卡镜像（但在作者的机器上无法启动）
4. **Debian Linux** - 预装系统

## 音频问题解决

### 扬声器音量过低的解决方案

1. 重启系统通常可以解决音频问题
2. 无需重启的命令：
   ```bash
   echo 2-001a > /sys/bus/i2c/drivers/wm8960/bind
   ```

## LED 控制

关闭 WiFi LED：
```bash
echo 0 > /sys/class/leds/ath9k-phy0/brightness
```
