## 项目简介

JSON Formatter 是一个 Chrome 浏览器扩展，旨在帮助用户查看和探索 JSON API 响应。它能够让难以阅读的 JSON 数据变得易于阅读和理解。

## 核心功能

- **高性能**：即使在很长的 JSON 页面上也能快速运行
- **深色模式**：支持深色主题
- **语法高亮**：代码着色便于阅读
- **可折叠树状视图**：带缩进指引的树形结构
- **可点击 URL**：自动识别并可点击链接
- **极低性能开销**：对非 JSON 页面的性能影响小于 1 毫秒
- **通用兼容**：适用于任何有效的 JSON 页面
- **原始/解析切换**：提供按钮在原始 JSON 和解析后 JSON 之间切换
- **全局变量导出**：解析后的 JSON 导出为全局变量 `json`，可在控制台检查

## 技术限制（FAQ）

### 为什么大数字显示不准确？
这是 JavaScript 的限制。任何超过 `Number.MAX_SAFE_INTEGER` (2^53 - 1) 的数字会被调整向下。任何低于 `Number.MIN_SAFE_INTEGER` (-2^53 + 1) 的数字会被调整向上。如果 API 需要表示超出 JavaScript 安全范围的数字，应该将数字作为字符串引用。

### 为什么对象键顺序有时不正确？
JSON Formatter 显示的是解析后的对象/数组的表示形式。V8 引擎会优化将数字字符串键移动到顶部以提升性能。

## 安装方式

1. **从 Chrome Web Store 安装**：安装 JSON Formatter Classic
2. **从源码安装**：克隆仓库并运行 `bun install`，然后使用 `bun run build` 构建
