## 详细内容

### 关于书籍

**《Integrated by Design》** 是一本关于FreeBSD系统的综合性著作，由Vivian Voss撰写。本书涵盖从理念到实践的完整内容，共22章，371页。

**主要特点：**
- 5个月写作周期
- 3周最终校对
- 涵盖FreeBSD系统架构、设计哲学到实际应用

### 章节内容

根据配套代码仓库，该书包含以下章节主题：

| 章节 | 内容 |
|------|------|
| ch07 | 防火墙配置 (pf.conf templates) |
| ch16 | 从零开始配置服务器 (rc.conf) |
| ch17 | 第一个Jail配置 (jail.conf) |
| ch18 | Web服务栈 (Caddy + nginx + PostgreSQL) |
| ch19 | 生产环境Jails配置 (多租户模板、pf规则) |
| ch20 | 本地FreeBSD开发 (jferry + QEMU) |

### jferry 工具

第20章介绍了一个名为 **jferry** 的工具，允许用户使用QEMU在笔记本电脑上运行FreeBSD jails。该工具可以：
- 下载FreeBSD虚拟机镜像并启动
- 通过NFS挂载jail数据以便本地编辑文件
- 在服务器和本地之间推送/拉取jails

### 技术栈

根据作者网站信息，Vivian Voss的技术栈：
- 服务器：CASTD (~2 MB, Rust + Lua + SQLite)
- 模板：CAST — Content & Structure Templating
- 样式：castd.css (OKLCH, @layer, light-dark())
- 反向代理：Caddy
- 操作系统：FreeBSD
