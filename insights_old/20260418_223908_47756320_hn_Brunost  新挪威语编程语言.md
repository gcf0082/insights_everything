## 摘要

本文介绍了一门名为 **Brunost** 的新编程语言，其主要特点是使用挪威语 Nynorsk（尼诺斯克语）作为关键字。Brunost 是一门实验性语言，所有关键字都使用 Nynorsk 单词，例如：
- `fast` 表示常量（不可变）
- `endreleg` 表示变量（可变）
- `er` 表示等于
- `skriv` 表示打印

## 语言特性

### 变量声明示例

```nynorsk
fast fartsgrense er 80      // 常量：速度限制为80
endreleg fart er 70         // 变量：速度为70
skriv(fart)                // 打印速度
```

### 关键词演变

作者根据社区反馈不断调整关键词：
1. **初期版本**：使用 `endreleg` 表示可变，`fast` 表示不可变
2. **新版**：将 `endreleg` 改为 `låst`（锁定）和 `open`（开放），使字符数一致

```nynorsk
låst fart er 80             // 锁定：不可变
open fartsgrense er 110      // 开放：可变
```

### 社区建议的功能

- **语法检查**：要求名词使用正确的不定冠词（`en`、`ei`、`eit`），错误时编译失败
- **列表分隔符**：使用 `og`（和）作为最后分隔符，遵循挪威语语法
- **芬兰语版本**：建议开发类似概念的芬兰语编程语言

## 讨论热点

### 1. 语言设计的实用性

有人指出该语言不适合编程教学，因为它会让变量声明显得矛盾：
- `endreleg fart er 80` 表达的是"可变的变量是80"，这在语义上很奇怪
- 建议使用 `blir`（成为）替代 `er`（是）来表达变化

### 2. 挪威语言文字争议

讨论涉及挪威两种官方书面语：
- **Bokmål**：源于丹麦语，类似丹麦语
- **Nynorsk**：基于挪威方言，更接近古诺尔斯语

有用户指出 Nynorsk 在历史上曾被视为较"低劣"的语言，直到1980年代这种情况才改变。

### 3. 技术可行性

- 作者计划实现文件 I/O 和网络功能，用 Brunost 编写网站
- 有意者希望开发更专业的版本：编译型语言、类型安全、包管理器
