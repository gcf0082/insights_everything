## 项目概述

Kitten TTS 是一个开源的轻量级文本转语音（Text-to-Speech，TTS）库，基于 ONNX 构建。该项目提供了一系列参数规模从 15M 到 80M 的模型（磁盘占用 25-80 MB），能够在 CPU 上高效运行高质量语音合成，无需 GPU 支持。

> **状态**：开发者预览版 — API 可能在不同版本之间发生变化。

---

## 核心特性

1. **超轻量级** — 模型体积仅 25-80 MB，适合边缘设备部署
2. **CPU 优化** — 基于 ONNX 推理，无需 GPU 即可高效运行
3. **8 种内置声音** — Bella、Jasper、Luna、Bruno、Rosie、Hugo、Kiki、Leo
4. **可调语速** — 通过 `speed` 参数控制播放速率
5. **文本预处理** — 内置管道处理数字、货币、计量单位等
6. **24 kHz 输出** — 高品质音频标准采样率

## 快速开始

### 安装

```bash
pip install https://github.com/KittenML/KittenTTS/releases/download/0.8.1/kittentts-0.8.1-py3-none-any.whl
```

### 基本用法

```python
from kittentts import KittenTTS

model = KittenTTS("KittenML/kitten-tts-mini-0.8")
audio = model.generate("This high-quality TTS model runs without a GPU.", voice="Jasper")

import soundfile as sf
sf.write("output.wav", audio, 24000)
```

### 高级用法

```python
# 调整语速（默认：1.0）
audio = model.generate("Hello, world.", voice="Luna", speed=1.2)

# 直接保存到文件
model.generate_to_file("Hello, world.", "output.wav", voice="Bruno", speed=0.9)

# 查看可用声音
print(model.available_voices)
# ['Bella', 'Jasper', 'Luna', 'Bruno', 'Rosie', 'Hugo', 'Kiki', 'Leo']
```

## API 参考

### `KittenTTS(model_name, cache_dir=None)`

从 Hugging Face Hub 加载模型。

| 参数 | 类型 | 默认值 | 描述 |
|-----|------|--------|------|
| model_name | str | "KittenML/kitten-tts-nano-0.8" | Hugging Face 仓库 ID |
| cache_dir | str | None | 本地缓存目录 |

### `model.generate(text, voice, speed, clean_text)`

合成语音，返回 24 kHz 的 NumPy 音频数组。

| 参数 | 类型 | 默认值 | 描述 |
|-----|------|--------|------|
| text | str | 必填 | 要合成的输入文本 |
| voice | str | "expr-voice-5-m" | 声音名称 |
| speed | float | 1.0 | 语速倍增器 |
| clean_text | bool | False | 预处理文本（展开数字、货币等）|

### `model.generate_to_file(text, output_path, voice, speed, sample_rate, clean_text)`

合成语音并直接写入音频文件。

| 参数 | 类型 | 默认值 | 描述 |
|-----|------|--------|------|
| text | str | 必填 | 输入文本 |
| output_path | str | 必填 | 保存音频文件路径 |
| voice | str | "expr-voice-5-m" | 声音名称 |
| speed | float | 1.0 | 语速倍增器 |
| sample_rate | int | 24000 | 音频采样率（Hz）|
| clean_text | bool | True | 预处理文本 |
