## 项目概述

wacli 是一个基于 `whatsmeow` 库构建的 WhatsApp 命令行工具，专注于以下核心功能：

- **本地消息同步**：尽最大努力本地同步消息历史并进行持续捕获
- **快速离线搜索**：支持快速离线搜索消息
- **发送消息**：支持发送文本和文件消息
- **联系人及群组管理**：支持列出群组、管理群组成员

> 注意：这是一个第三方工具，使用 WhatsApp Web 协议，而非 WhatsApp 官方产品。

---

## 核心功能

### 1. 认证与同步
- `wacli auth`：交互式登录（显示二维码），然后立即执行初始数据同步
- `wacli sync`：非交互式同步循环（不再显示二维码；未认证时报错）

### 2. 消息管理
- **搜索消息**：`wacli messages search "关键词"`
- **消息列表**：支持显示反应、回复和媒体类型的显示文本
- **历史回填**：从主设备获取更早的消息（尽最大努力）

### 3. 发送功能
- 发送文本消息
- 发送文件（支持自定义显示文件名）
- 支持覆盖显示名称

### 4. 群组管理
- 列出群组
- 重命名群组
- 管理群成员

### 5. 媒体下载
- 支持下载消息中的媒体文件

## 安装方式

### 方式A：通过 Homebrew 安装
```bash
brew install steipete/tap/wacli
```

### 方式B：本地构建
```bash
go build -tags sqlite_fts5 -o ./dist/wacli ./cmd/wacli
```

## 环境变量配置

| 变量名 | 说明 |
|--------|------|
| `WACLI_DEVICE_LABEL` | 设置链接设备的标签（在 WhatsApp 中显示） |
| `WACLI_DEVICE_PLATFORM` | 覆盖链接设备平台（默认为 `CHROME`） |
| `--store DIR` | 自定义存储目录（默认为 `~/.wacli`） |
