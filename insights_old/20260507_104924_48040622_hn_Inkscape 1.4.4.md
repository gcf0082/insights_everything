## 摘要

Inkscape 1.4.4 版本发布，带来多项修复和改进。讨论涉及书写作工具问题、开源软件反馈文化、CMYK支持开发进度、SVG导出格式等话题。

## 热门评论摘要

### 1. 新功能贡献
用户 darknavi 贡献了一个小功能：允许用户设置默认保存文件名，方便日常使用。

### 2. 书写作(Calligraphy)工具问题
用户 pxoe 批评书写作工具相比 0.92 版本严重退化：
- 响应性差
- 分辨率更低
- 角度更尖锐
- 需要开启 Windows Ink
- 移除了设备面板，无法检查设备连接状态

官方回应称问题将在下一个主要版本中部分解决。

### 3. 开源软件反馈文化争议
围绕"免费软件是否应该接受批评"展开激烈讨论：
- 有观点认为免费软件不应被苛责
- 另一观点认为批评是用户权利，有助于改进
- 普遍认为补丁比反馈更有价值

### 4. CMYK 支持
CMYK 支持正在积极开发中，已进行约两年。

### 5. SVG 导出格式
用户反映 Inkscape 保存后会使手写 SVG 格式变乱，插入不必要的换行。推荐使用 SVGO 优化工具。

### 6. 积极评价
- 用户喜欢用 Inkscape 制作图标和简单 CAD 任务
- 游戏开发者使用 Inkscape 制作游戏素材
- 有丰富的扩展插件生态（Inkstitch、inkscape-silhouette 等）

### 7. 用户体验问题
- UI 在高分辨率显示器上显得过于笨重
- MacOS 版本曾有严重 bug，近年来有所改善

## 结论

Inkscape 1.4.4 是一个维护性版本，继续改进稳定性和用户体验。社区对该软件既有肯定也有批评，主要集中在书写作工具退化和 UI 体验问题上。作为开源软件，其发展依赖于社区贡献和反馈。
