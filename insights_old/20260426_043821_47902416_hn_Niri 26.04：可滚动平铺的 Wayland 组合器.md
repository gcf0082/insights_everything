## 项目简介

Niri 是一个可滚动平铺的 Wayland  compositor，灵感来自 PaperWM（GNOME 扩展）。它采用独特的"滚动式"窗口管理方式，窗口水平排列，用户可以通过滚动在窗口之间移动焦点。

## 主要功能特点

### 1. 滚动式窗口管理
- 窗口不再受限于物理显示器宽度
- 工作区数量不受快捷键数量限制
- 新窗口默认打开在当前窗口右侧，不会破坏现有布局

### 2. 垂直工作区
- 每个显示器有独立的工作区集合
- 工作区可以跨显示器移动
- 断开显示器后，工作区会自动迁移，重新连接后会自动回归

### 3. 键盘驱动的空间导航
- 使用 super+hjkl 或 super+方向键导航
- 支持 mod+c 居中当前窗口
- 现代化的 Alt-Tab 行为

## 相关项目

1. **OmniWM** (macOS) - 实现了 Niri 风格的滚动布局
2. **PaperWM** (GNOME) - Niri 的灵感来源
3. **MangoWM** (wlroots) - 资源消耗更少，布局更多
4. **Dank Material Shell** - 提供完整的桌面环境组件

## 结论

Niri 作为一款新兴的 Wayland 窗口管理器，以其创新的滚动式平铺理念获得了用户的高度认可。尽管缺少一些高级功能（如 scratchpad），但其出色的用户体验、稳定的 API 和优雅的设计使其成为 Linux 桌面用户的热门选择。特别适合使用超宽显示器和喜欢键盘驱动工作流程的用户。
