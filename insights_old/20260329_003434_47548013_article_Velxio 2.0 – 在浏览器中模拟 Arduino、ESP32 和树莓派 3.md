## 核心功能

### 代码编辑
- Monaco Editor — 完整的C++/Python编辑器，支持语法高亮、自动完成、最小地图和深色主题
- 多文件工作区 — 创建、重命名、删除和切换多个.ino/.h/.cpp/.py文件
- 通过arduino-cli后端编译Arduino代码
- 编译/运行/停止/重置工具栏按钮
- 可调整大小的编译控制台

### 模拟引擎
- **AVR8**: 真实ATmega328p/ATmega2560/ATmega32u4/ATtiny85模拟，完整GPIO、定时器、USART、ADC、SPI、I2C
- **RP2040**: 真实RP2040模拟 @ 133MHz，30个GPIO引脚、UART、ADC、I2C、S1、PWM
- **ESP32 Xtensa**: 通过QEMU实现真实Xtensa LX6/LX7双核模拟，支持WiFi、NeoPixel、LEDC/PWM
- **ESP32-C3 RISC-V**: 浏览器内RV32IMC模拟，无需后端，即时启动
- **Raspberry Pi 3**: 通过QEMU模拟BCM2837，可启动真实Raspberry Pi OS (Trixie)

### 串口监视器
- 实时串口输出
- 自动波特率检测
- 支持发送数据到RX引脚

### 元件系统
- 48+电子元件 (来自wokwi-elements)
- 带搜索、类别筛选的元件选择器
- 拖放定位
- 属性对话框

### 其他功能
- 库管理器 — 浏览和安装完整的Arduino库索引
- 认证与项目持久化 — 邮箱/密码和Google OAuth登录
- 内置示例项目

## 部署方式

### Docker (推荐)
```bash
docker run -d -p 3080:80 ghcr.io/davidmonterocrespo24/velxio:master
```

### Docker Compose
```bash
git clone https://github.com/davidmonterocrespo24/velxio.git
cd velxio
docker compose -f docker-compose.prod.yml up -d
```

### 手动安装
- 前端: Node.js 18+, npm install, npm run dev
- 后端: Python 3.12+, FastAPI, arduino-cli

## 总结

Velxio是一个功能强大的全本地、开源多开发板模拟器。它支持在浏览器中模拟19种不同的开发板，涵盖Arduino、ESP32、Raspberry Pi系列。核心技术包括avr8js、rp2040js、RiscVCore.ts以及QEMU模拟器。项目采用现代化的前后端分离架构，前端使用React 19 + TypeScript，后端使用FastAPI + Python。支持Docker一键部署，是一个非常适合嵌入式开发学习和原型设计的工具。
