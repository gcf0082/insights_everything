- **阅读时长**: 25分钟

## 核心观点

本文将Microsoft宣布的Windows 11"七点修复计划"比作"虐待关系中的道歉"——先对你拳打脚踢，然后带着鲜花出现说"我改变了"，周围的人却称赞他们"变好了"。作者认为，Microsoft花了四年时间向Windows 11植入广告、强制Copilot集成和臃肿软件，如今却将这些伤害作为恩惠来邀功。

## Windows 11的"罪行清单"

### 1. Copilot强制推广（2023年9月26日）

Microsoft在2023年9月26日将AI助手Copilot植入Windows 11，比正式的23H2版本更早。Copilot图标出现在开始菜单和系统托盘之间，无法移动、无法通过常规设置移除，并劫持了Win+C快捷键。在随后两年内，Copilot按钮扩散到截图工具、照片、记事本、小部件、文件资源管理器和系统设置中。Microsoft甚至计划强制在"符合条件的PC"开始菜单上安装Microsoft 365 Copilot应用。

### 2. 广告注入（2024年4月24日）

2024年4月24日，Microsoft推送更新KB5036980，在Windows 11开始菜单的"推荐"区域注入广告，标记为"Promoted"，推送Opera浏览器和密码管理器等无人需要的应用。广告还出现在锁屏、设置首页（推销Game Pass）、文件资源管理器（推销OneDrive）以及伪装成"提示"的通知中。

### 3. 强制Microsoft账户（2021-2025）

Windows 11发布时，家庭版就要求使用Microsoft账户。到2025年10月，Microsoft系统性地封堵了所有本地账户创建方法：包括`oobe\bypassnro`命令、BypassNRO注册表开关、`ms-cxh:localonly`技巧，甚至旧的虚假邮箱方法。Microsoft明确表示正在"移除在Windows安装体验中创建本地账户的已知机制"。

### 4. OneDrive自动同步（2024年）

2024年，Microsoft悄悄更改Windows 11安装程序，使OneDrive文件夹备份自动启用，无需用户同意，同步桌面、文档、图片、音乐和视频到微软云端。当用户发现并尝试关闭时，他们的文件从本地机器消失，因为OneDrive已将其移动到云端，将个人文件的所有权转移给微软云服务而不请自取。

### 5. Windows Recall监控功能（2024年5月）

Windows Recall是2024年5月宣布的AI功能，每隔几秒钟对屏幕截图并使其可搜索。安全研究员Kevin Beaumont演示称整个Recall数据库以明文形式存储在AppData文件夹中，任何恶意软件都可以提取。银行号码、社会安全号码、密码都保存在未加密的SQLite数据库中。英国信息专员办公室介入调查，Microsoft将其改为可选功能并添加加密，悄悄重新推出。

### 6. 电子垃圾危机：2.4亿台PC报废

Windows 11的硬件要求（需要TPM 2.0、特定CPU代、UEFI安全启动）使全球约20%的PC被标记为"过时"。Windows 10于2025年10月14日停止支持，继续获得安全更新的费用为：消费者30美元/年，企业设备61-244美元/年，且价格逐年翻倍。

### 7. Edge浏览器黑暗模式

Mozilla委托发布的独立报告《Over the Edge》记录了Microsoft的黑暗模式：确认羞耻（弹出框暗示不使用Edge是"购物方式愚蠢"）、在Google.com和Chrome应用商店注入伪装广告、默认浏览器设置劫持回Edge。尽管如此操纵，Edge仅占全球浏览器市场份额的5.35%。

### 8. 虚假遥测开关

在Windows 11家庭版和专业版上，无法完全禁用遥测。在非企业版上将`AllowTelemetry`设置为0会被静默覆盖回1。只有企业版和教育版才能真正关闭它。您付费的操作系统向Microsoft报告您的数据，而关闭它的设置在消费版上是一个谎言。

### 9. 反垄断历史

欧盟对Microsoft处以超过22亿欧元的罚款，其中仅因违反浏览器选择承诺就被罚款5.61亿欧元。Windows 7更新悄悄移除选择屏幕14个月，影响1500万用户。

## 结论

Microsoft的"swarming"努力针对的是最引人注目的UI烦扰——那些产生负面头条的问题。数据收集、供应商锁定、强制账户，这些都保留下来，因为这些才是真正的产品。在这一点上，**您就是产品**。

Microsoft花了四年时间故意降低人们支付139美元或更多钱购买的操作系统，而现在他们将自己造成的损害的移除当作礼物。广告本就不应该存在，Copilot按钮不应该被强制，任务栏不应该被削弱。而他们选择保留的东西——遥测、强制账户、数据收集——才是真正的产品。
