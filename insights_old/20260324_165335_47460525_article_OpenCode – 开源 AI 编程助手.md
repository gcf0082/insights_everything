## 一、OpenCode 概述

OpenCode 是一个开源的 AI 编程代理工具，可帮助用户在终端、IDE 或桌面应用中编写代码。用户可以免费使用内置模型，也可以连接任何模型提供商的模型，包括 Claude、GPT、Gemini 等。

## 二、核心功能特性

### 1. LSP 启用
自动加载适合 LLM 的正确语言服务器协议 (LSP)。

### 2. 多会话支持
可在同一项目上并行启动多个 AI 代理。

### 3. 分享链接
可分享任意会话的链接，便于参考或调试。

### 4. 集成现有 AI 订阅
- **GitHub Copilot**: 可通过 GitHub 登录使用 Copilot 账户
- **ChatGPT Plus/Pro**: 可通过 OpenAI 登录使用 ChatGPT Plus 或 Pro 账户

### 5. 支持任何模型
通过 Models.dev 支持 75+ LLM 提供商，包括本地模型。

### 6. 支持任何编辑器
提供终端界面、桌面应用和 IDE 扩展等多种使用方式。

## 四、隐私优先

OpenCode 高度重视用户隐私，不存储任何用户的代码或上下文数据，使其可在敏感隐私环境中运行。

## 五、产品系列

### 1. Zen
Zen 提供经过 OpenCode 测试和基准测试的精选 AI 模型，专门针对编程代理进行了优化。用户无需担心不同提供商之间的性能和质量问题，可以使用经过验证的模型。

### 2. Desktop App
桌面应用目前处于 beta 阶段，支持 macOS、Windows 和 Linux 系统。

## 六、安装方式

OpenCode 提供多种安装方式：

# 使用 bun

# 使用 brew
brew install opencode

# 使用 paru
