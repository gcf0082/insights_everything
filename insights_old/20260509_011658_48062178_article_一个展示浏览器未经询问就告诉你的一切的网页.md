## 概述

这是一个关于浏览器隐私的互动网页，演示了网页在用户访问时能够自动获取的各类浏览器数据。该项目是 "Since You Arrived" 系列的第四卷，旨在让用户直观地了解网站如何在未经许可的情况下收集访问者的设备信息。

> 核心设计理念：关闭 JavaScript 后，页面无法显示它收集了你的哪些数据——但数据仍然被收集了，只是没有告诉你。

---

## 核心功能与数据收集机制

### 1. IP 地理位置查询

- 通过 IP 地址获取用户的城市位置和互联网服务提供商信息
- 使用 ip-api.com（免费层，CC-BY-SA 许可证）
- 数据仅临时查询，不存储、不追踪，仅显示 IP 首尾段

### 2. 浏览器 API 数据采集

页面通过标准 JavaScript API 收集以下信息：

| 数据类型 | 采集方式 |
|----------|----------|
| 屏幕分辨率 | Screen API |
| 浏览器信息 | Navigator API |
| 语言设置 | Navigator API |
| GPU 信息 | WebGL API |
| CPU 核心数 | Navigator API |
| 电池状态 | Battery API |
| 安装字体 | 字体指纹检测 |
| 时区设置 | Date API |
| 颜色深度 | Screen API |

> 所有数据均通过 Mozilla 公开的 Web API 文档获取，无漏洞、无 hack，完全"合法"。

### 3. 字体指纹识别

通过测量渲染文字宽度来检测用户安装的字体。电子前沿基金会（EFF）的 "Cover Your Tracks" 工具证明了字体组合是浏览器追踪的最强信号之一。

### 4. Canvas 指纹识别

2014 年普林斯顿大学研究发现，5% 的顶级网站使用此技术——让浏览器绘制隐藏图像并读取像素作为标识符。本页未使用此技术，但用户访问的其他页面可能使用了。

### 5. 剪贴板 API

现代浏览器允许网页在用户点击时读取剪贴板内容（密码、地址、消息草稿）。该 API 可用，但本页未请求。

### 6. 电池状态追踪

学术研究表明，电池电量+放电时间的组合足以在 30 分钟内追踪用户，且无需 cookies。Firefox 于 2016 年移除该 API，但 Chrome 和 Edge 仍保留。

### 7. 登录状态检测（未使用）

页面可以检测用户登录了哪些网站（通过加载 favicon 并观察返回结果）。此技术无需权限且完全合法，但本页面未使用——不过你今天访问的其他页面可能用了。

### 8. 唯一条码生成

基于你的设备数据（GPU、字体、屏幕、语言、时区、操作系统、浏览器、颜色深度）生成独一无二的 16 条条形码。

## 数据发送与存储

### 发送的数据

仅向服务器发送两个匿名事件：
1. 你到达了页面
2. 你完成了阅读

- 无 cookies、无标识符、无 IP 存储
- 仅传输事件发生的事实，不含任何用户数据

### 本地存储

**零存储**：
- 无 cookies
- 无 localStorage
- 无 sessionStorage
- 无 IndexedDB
- 无 service worker cache

所有显示给你的数据都在你的浏览器中计算完成，从未离开你的设备。

## 与其他网站的对比

该页面强调：大多数网站不仅发送两个事件，还会向广告商、指纹追踪器、会话重放工具和标签管理器发送数百个额外的信标。
