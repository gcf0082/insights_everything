## 二、贝叶斯与频率派统计的对比

### 2.1 核心哲学差异

- **频率派**：参数是固定的（虽然未知），数据是随机生成的
- **贝叶斯派**：参数本身是随机变量，具有自己的概率分布

作者以掷骰子为例说明：假设游戏主持人背后藏着一种多面体骰子，你观察到的是掷出的数字。频率派认为骰子的面数n是固定的，只是未知；而贝叶斯派认为面数n本身服从某个分布P(n)，观测数据用于推断这个分布。

### 2.2 不确定性的处理方式

- **频率派**：不确定性是"马后炮"，通过置信区间（confidence interval）来表达，但这个区间含义是"长期来看这种方法构建的区间有多少比例会包含参数"
- **贝叶斯派**：参数本身就是一个分布，可信区间（credible interval）直接告诉你参数落在这个区间内的概率

### 2.3 贝叶斯定理

核心公式：
$$P(\theta|X) = \frac{P(X|\theta)P(\theta)}{P(X)}$$

- **似然函数** $P(X|\theta)$：在给定参数下观察到当前数据的概率
- **先验** $P(\theta)$：对参数分布的先验知识
- **后验** $P(\theta|X)$：观察数据后对参数的推断
- **归一化常数** $P(X)$：通常难以计算，但在MCMC方法中可以被绕过

## 三、实战示例：估算骰子面数

### 3.1 频率派方法

观察到数据：[1,1,2,3,3,4,4,4]，频率分别为0.250, 0.125, 0.250, 0.375。

最大似然估计：$P(X|n) = (1/n)^8$，当n=4时最大（因为不可能掷出4如果只有3面）。

置信区间[4,6]：通过计算所有使$P(X_{max} \leq 4|n) \geq \alpha/2$的n得到。

### 3.2 贝叶斯方法

使用均匀先验$P(n)=1/N$（非信息先验），后验为：
$$P(n|X) \propto n^{-k}$$

后验均值$\bar{n} \simeq 4.26$，可信区间[4,5]。

如果使用几何先验（融入领域知识：面数可能接近4），后验均值为$\bar{n}=4.10$，更接近真实值。

### 3.3 先验的影响

- **数据充足时**：似然函数主导，先验影响微弱（由Bernstein-von Mises定理保证）
- **数据稀疏时**：先验对后验影响显著

这正是贝叶斯统计的优势：用先验弥补数据不足，当数据足够时让数据自己说话。

## 四、实际应用案例：零售销售数据生成

### 4.1 问题背景

零售公司希望基于历史数据生成合成销售订单数据，其中一个挑战是如何地理分布这些数据。

### 4.2 频率派方法

对历史数据按类别聚类，估算每个聚类中邮政编码的条件概率分布$P(\text{postal code}|\text{category})$。

**问题**：
- 对新类别或新邮政编码不鲁棒
- 数据稀疏时估算的分布噪声很大

### 4.3 贝叶斯方法

- **似然**：使用Dirichlet-Multinomial分布建模历史销售分布
- **先验**：使用基于人口分布的先验（不稀疏，每个邮政编码都有机会被采样）
- **效果**：模型充分利用数据，同时优雅地处理新区域（先验作为后备）

## 五、数值方法：MCMC

### 5.1 Metropolis算法

核心思想：
1. 从参数空间某点$\theta_0$开始
2. 每步提议新点$\theta_{t+1}$，以概率$\min(1, \frac{P(\theta_{t+1}|X)}{P(\theta_t|X)})$接受
3. 由于只涉及后验比率，无需计算归一化常数$P(X)$

### 5.2 PyMC实现

```python
import pymc as pm

with pm.Model() as model:
    # 先验定义
    a = pm.Normal("a", mu=0, sigma=10)
    b = pm.Normal("b", mu=0, sigma=10)
    sigma = pm.HalfNormal("sigma", sigma=10)
    
    # 似然函数
    y_obs = pm.Normal("y_obs", mu=a * x + b, sigma=sigma, observed=y)
    
    # 采样后验分布
    trace = pm.sample(1000, tune=1000, chains=4)
```

### 5.3 处理异常值

贝叶斯方法可以轻松添加混合模型来处理异常值：
- 为每个数据点引入一个潜在参数$g_i \in [0,1]$
- $g_i \approx 1$时使用高斯似然，$g_i \approx 0$时使用大方差的高斯（背景噪声）
- 模型可以自动识别异常值

## 六、贝叶斯与正则化的联系

令人惊讶的是，常见的正则化技术实际上是贝叶斯方法的不同先验选择：

| 正则化方法 | 先验分布 | 对应贝叶斯估计 |
|-----------|---------|---------------|
| Ridge (L2) | 高斯先验 $N(0, \tau)$ | MAP估计 |
| Lasso (L1) | Laplace先验 | MAP估计 |

所以当你使用正则化线性回归时，你实际上在做一个贝叶斯推断！

---

## 七、总结

1. **核心优势**：贝叶斯统计更擅长处理不确定的数据，参数本身就是概率分布，不确定性已经包含其中
2. **先验的作用**：先验弥补数据不足，当数据充足时似然函数主导（Bernstein-von Mises定理）
3. **实际应用**：MCMC方法（如PyMC）使得复杂贝叶斯模型的构建变得简单
4. **与频率派的关系**：在数据充足时，两者会趋于一致
