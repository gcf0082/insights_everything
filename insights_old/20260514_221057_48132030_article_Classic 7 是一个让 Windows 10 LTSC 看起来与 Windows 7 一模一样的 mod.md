## 网站概述

Classic 7 是一个 Windows 10（IoT Enterprise LTSC 2021）修改项目，旨在将系统外观1:1还原为 Windows 7。该项目保留了 Windows 7 的经典特性，并添加了一些额外的功能。

## 核心特性

### 已还原的 Windows 7 功能

1. **Aero Glass 毛玻璃效果** - 完整的毛玻璃窗口效果
2. **.themepack 主题文件支持** - 支持 Windows 7 的主题包文件
3. **桌面小工具** - 所有经典的桌面小工具均可用
4. **Windows Media Center** - 完整功能的多媒体中心
5. **经典主题元素** - Windows 7 的标志性界面元素

### 未还原的功能

1. **窗口 3D 动画** - 包括打开、关闭、最小化、最大化的动画效果
2. **Flip 3D 切换** - 尚未有人成功移植此功能
3. **部分系统应用和小工具** - 由于 Windows 7 与 Windows 10 之间的兼容性问题，部分应用可能无法正常工作

## 特别说明

- Classic 7 提供了完整的 OOBE（开箱体验）界面还原，使用户仿佛回到了简化版的 PC 时代
- 该项目并非微软官方产品，而是粉丝自发开发的个人项目
