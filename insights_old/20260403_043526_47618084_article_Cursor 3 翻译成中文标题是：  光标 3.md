## 核心摘要

Cursor 3 是 Cursor 推出的全新统一工作空间，旨在为构建软件提供更高效、更清晰的 AI 辅助编程体验。该版本将代理（agents）置于界面核心位置，实现了多工作区布局、本地与云端代理的无缝切换，以及并行运行多个代理的能力。

---

## 主要更新内容

### 1. 统一代理工作区
新界面本质上是多工作区的，允许人类和代理跨不同代码仓库（repos）协同工作。

### 2. 并行运行多个代理
所有本地和云端代理都显示在侧边栏中，包括从移动端、网页端、桌面端、Slack、GitHub 和 Linear 启动的代理。云端代理会生成工作成果的演示和截图供用户验证。

### 3. 本地与云端代理无缝切换
- **云端到本地**：可将代理会话移至本地进行编辑和测试，Composer 2 非常适合快速迭代
- **本地到云端**：可将代理会话移至云端以便离线时继续运行，适合长时间运行的任务

### 4. 从提交到合并 PR 的完整流程
新的 diffs 视图提供更简洁的 UI，可更快地编辑和审查更改，支持暂存、提交和管理 PR

---

## 保留的经典功能

### 代码理解文件
可随时深入查看文件，并在编辑器中使用完整 LSP 跳转到定义

### 集成浏览器
Cursor 可使用内置浏览器打开、导航本地网站并进行提示

### Cursor Marketplace 插件
可浏览数百种插件，通过 MCP、skills、subagents 等扩展代理能力，一键安装

## 未来展望

Cursor 3 已具备基础组件——模型、产品和运行时——用于构建更自主的代理和更好的团队协作。团队将继续投资 IDE，直至代码库实现自动驾驶。随着更强大的编码模型出现，软件构建的界面将继续演变。
