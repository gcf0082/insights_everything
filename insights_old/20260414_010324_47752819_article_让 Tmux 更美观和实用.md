## 内容总结

本文是一篇关于如何自定义 tmux 配置的指南，作者 Ham Vocke 分享了他认为最有用的 tmux 自定义配置。

### 自定义基础

#### 1. 修改前缀键

将默认的 `C-b` 修改为 `C-a`，因为 `C-a` 更易于操作。配置如下：

```
unbind C-b
set-option -g prefix C-a
bind-key C-a send-prefix
```

#### 2. 直观的分屏命令

将垂直分屏从 `"` 改为 `|`，水平分屏从 `%` 改为 `-`，这样更符合分屏的视觉形状：

```
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %
```

#### 3. 快速重载配置

按 `r` 键快速重载配置文件：

```
bind r source-file ~/.tmux.conf
```

#### 4. 快速切换面板

使用 `Alt+方向键` 直接切换面板，无需前缀：

```
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D
```

#### 5. 鼠标模式

启用鼠标控制，可以点击选择窗口和面板，拖拽调整面板大小：

```
set -g mouse on
```

#### 6. 禁止自动重命名窗口

阻止 tmux 自动根据命令更新窗口名称，允许用户手动命名：

```
set-option -g allow-rename off
```

## 关键要点

1. **配置文件位置**: `~/.tmux.conf`（隐藏文件）
2. **最常用的自定义**: 修改前缀键、分屏快捷键、面板切换
3. **鼠标模式**: 有助于与不熟悉 tmux 的人协作
4. **颜色方案**: 使用终端默认颜色或 256 色范围（colour0-colour256）
