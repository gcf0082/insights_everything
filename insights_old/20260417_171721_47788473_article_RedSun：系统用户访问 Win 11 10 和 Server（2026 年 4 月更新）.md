## 项目描述

RedSun 是一个漏洞利用仓库，利用 Windows Defender 的一个有趣行为缺陷。当 Windows Defender 发现恶意文件具有云标签标记时，其行为令人啼笑皆非——这个本应保护系统的杀毒软件反而会将文件重新写入到原始位置，破坏了其自身的恶意文件检测机制。攻击者可以利用此行为覆盖系统文件并获取管理员权限。

## 技术原理

该漏洞利用了以下攻击场景：
1. 恶意文件被 Windows Defender 标记为恶意，并带有云标签
2. Windows Defender 的"修复"操作是将文件恢复到原始位置，而非删除
3. 攻击者利用此行为覆盖系统关键文件
4. 最终实现提权，获取管理员/系统权限

## 安全影响

这是一个典型且有趣的安全漏洞，展示了杀软在修复恶意文件时的逻辑缺陷。虽然杀软的本职工作是检测并移除恶意文件，但某些情况下反而会帮助攻击者完成攻击。
