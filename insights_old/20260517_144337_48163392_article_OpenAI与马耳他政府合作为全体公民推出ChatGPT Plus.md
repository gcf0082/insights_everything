# 洞察报告：OpenAI与马耳他ChatGPT Plus合作

## 基本信息

| 项目 | 内容 |
|------|------|
| **洞察链接** | https://openai.com/index/malta-chatgpt-plus-partnership/ |
| **合作双方** | OpenAI 与 马耳他政府 |
| **发布时间** | 2026年5月16日 |
| **合作类型** | 政府与企业AI普及合作 |
| **国家/地区** | 马耳他 |

## 洞察摘要

这是世界首创的全国性AI普及合作项目。OpenAI与马耳他政府宣布达成合作，将向所有马耳他公民提供免费的ChatGPT Plus一年访问权限。

## 合作详情

### 参与条件
- 公民需完成由马耳他大学开发的政府支持的AI素养课程
- 课程涵盖：AI是什么、AI的能力与局限性、如何在家庭和工作场所负责任地使用AI

### 实施安排
- **管理机构**：马耳他数字创新管理局（Malta Digital Innovation Authority）
- **第一阶段**：于2026年5月启动
- **扩展计划**：随着更多居民和海外公民完成课程，程序将逐步扩展

### 官方表态
马耳他经济、企业和战略项目部长Silvio Schembri表示：
> "马耳他是第一个启动这种规模合作的国家，因为我们拒绝让公民停留在数字时代之外。"

这一合作的目标是将AI"从一个陌生的概念转变为家庭、学生和工人的实际帮助"。

## 战略意义

这是OpenAI"为各国开放"（OpenAI for Countries）倡议的最新项目。该倡议通过与各国政府合作，推动从早期AI兴趣到全国性AI应用的转变。不同于标准化模式，该项目根据每个国家的优先领域进行定制，包括教育、劳动力培训和公共服务等领域。

## 相关背景

OpenAI此前已与多国政府开展类似合作：
- **爱沙尼亚**：2025年与爱沙尼亚政府合作，为所有中学生和教师提供ChatGPT Edu访问权限
- **希腊**：启动"OpenAI for Greece"项目，与希腊政府合作
