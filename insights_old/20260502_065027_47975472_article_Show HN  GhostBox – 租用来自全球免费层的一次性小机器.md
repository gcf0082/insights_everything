# Ghostbox/Ghostbox 洞察报告

## 基本信息

- **洞察链接**: https://www.ghost.charity/
- **网站名称**: Ghostbox (又称 Ghostbox)
- **服务类型**: 临时开发机器/Disposable little dev machines
- **洞察时间**: 2026-05-02

## 网站概述

Ghostbox 是一个提供临时开发机器的服务平台，允许用户借用临时的计算资源来完成工作，然后让机器自动消失。该服务主要面向开发者和编码代理，提供一种无需在本地机器上进行开发的方式。

## 核心功能

### 1. 临时机器租用
- 用户可以从 Global Free Tier（全球免费层）借用临时的开发机器
- 通过 SSH 连接进行远程操作
- 工作完成后机器会自动消失，无需手动清理

### 2. 开发场景
- .clone 代码库进行开发
- 运行复杂的构建流程
- 从任何地方打开 shell 会话
- 无需配置 DNS 即可暴露 Web 应用
- 为编码代理提供真实机器
- 保持连接稳定，即使隧道出现波动
- 工作完成后归还机器

### 3. 代理（Agent）支持
- 为编码代理提供真实的工作站环境
- 仅传递工作所需的 secrets
- 暴露预览地址而无需将笔记本暴露到互联网
- 集中管理日志、会话和清理工作
- 工作完成后让机器自动过期

### 4. Global Free Tier（全球免费层）
- GitHub Actions 只是临时机器的来源之一
- 网上存在许多临时计算的闲置资源
- Ghostbox 使这些资源感觉像一台小机器
- 无需平台管理
- 无需系统管理员
- 无服务器遗留

## Cycle of Ghosts（幽灵循环）

Ghostbox 将自己描述为"非官方的临时机器乡村俱乐部"：
- 所有能源都是借用的
- 有朝一日必须归还
- 每只幽灵都会回归云端
- 如果爱它，就让它自由

## 使用方法

首次使用时可以通过以下方式安装 CLI：
- 访问 [Downloads](/downloads.html) 页面
- 或运行命令：`curl -fsSL https://www.ghost.charity/install.sh | bash`
- 使用 `ghost up` 命令启动临时机器
