## 项目概述

NVIDIA NemoClaw 是一个开源参考栈，用于更安全地运行 [OpenClaw](https://openclaw.ai) 始终在线助手。它安装了 [NVIDIA OpenShell](https://github.com/NVIDIA/OpenShell) 运行时（NVIDIA Agent Toolkit 的一部分），为运行自主代理提供额外的安全保护。该项目还包括 NVIDIA Nemotron 等开源模型。

> **注意**：NemoClaw 是 Alpha 软件，非生产就绪。接口、API 和行为可能会随着迭代而变化。

---

## 核心功能

### 1. 安全的沙箱环境
- 使用 OpenShell 运行时创建隔离的沙箱环境
- 所有网络请求、文件访问和推理调用都由声明式策略管理
- 防止代理未经授权访问系统资源

### 2. 多层保护机制

| 保护层 | 功能 |
|--------|------|
| **网络** | 阻止未经授权的出站连接，可运行时热重载 |
| **文件系统** | 阻止在 `/sandbox` 和 `/tmp` 之外的读写操作 |
| **进程** | 阻止权限升级和危险系统调用 |
| **推理** | 将模型 API 调用重新路由到受控后端 |

### 3. 托管推理
- 推理请求永不直接从沙箱发出
- OpenShell 拦截每个调用并路由到 NVIDIA Endpoint 提供商
- 默认使用 `nvidia/nemotron-3-super-120b-a12b` 模型

### 4. 策略管理
- 提供声明式 YAML 策略文件
- 支持静态配置（持久化）和动态配置（会话级）
- 包含 PyPI、Docker Hub、Slack、Jira 等常用集成的预设策略

## 技术架构

### 组件说明

| 组件 | 职责 |
|------|------|
| **Plugin** | TypeScript CLI 命令，用于启动、连接、状态和日志 |
| **Blueprint** | Python 工件，编排沙箱创建、策略和推理设置 |
| **Sandbox** | 隔离的 OpenShell 容器，运行带策略的 OpenClaw |
| **Inference** | NVIDIA Endpoint 模型调用，通过 OpenShell 网关透明路由 |

### 架构流程

```
用户 → nemoclaw CLI → Blueprint → OpenShell → Sandbox (OpenClaw)
                                              ↓
                                    策略执行 (网络/文件系统/进程)
                                              ↓
                                    推理请求 → OpenShell Gateway → NVIDIA Endpoint
```

## 快速开始

### 前置要求

#### 硬件
| 资源 | 最低 | 推荐 |
|------|------|------|
| CPU | 4 vCPU | 4+ vCPU |
| RAM | 8 GB | 16 GB |
| 磁盘 | 20 GB 可用 | 40 GB 可用 |

#### 软件
- Linux: Ubuntu 22.04 LTS 或更高
- Node.js: 20 或更高
- npm: 10 或更高
- 容器运行时: Docker (推荐)
- OpenShell: 已安装

### 安装步骤

```bash
curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash
```

安装完成后，运行 onboard 向导创建沙箱：

```bash
nemoclaw onboard
```

### 使用代理

```bash
# 连接到沙箱
nemoclaw my-assistant connect

# 在沙箱内启动 OpenClaw TUI
openclaw tui

# 或使用 CLI 发送消息
openclaw agent --agent main --local -m "hello" --session-id test
```

### nemoclaw 命令（在主机上运行）

| 命令 | 描述 |
|------|------|
| `nemoclaw onboard` | 交互式设置向导：网关、提供商、沙箱 |
| `nemoclaw <name> connect` | 在沙箱内打开交互式 shell |
| `openshell term` | 启动 OpenShell TUI 进行监控和审批 |
| `nemoclaw start/stop/status` | 管理辅助服务 |

## 总结

NVIDIA NemoClaw 是一个专为安全运行 OpenClaw 代理而设计的参考栈。它通过 OpenShell 运行时提供多层保护，包括网络隔离、文件系统限制、进程控制和托管推理。对于希望在生产环境中安全部署 AI 代理的开发者和企业来说，NemoClaw 提供了一个可靠的解决方案。

该项目目前处于 Alpha 阶段，接口和 API 可能会发生变化，但凭借 NVIDIA 的支持和活跃的社区，它有望成为 AI 代理安全部署的重要工具。
