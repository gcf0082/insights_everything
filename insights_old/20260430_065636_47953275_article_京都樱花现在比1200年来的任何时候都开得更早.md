# 京都樱花盛开峰值报告

## 基本信息

- **数据来源**：https://jivx.com/kyoto-bloom
- **报告日期**：2026-04-30
- **分析对象**：京都樱花盛开峰值日期记录

## 报告概述

京都樱花的盛开峰值日期记录跨度超过一千年（812年至2026年），是目前世界上已知持续时间最长的自然现象记录之一。

## 关键数据

| 指标 | 数值 |
|------|------|
| 记录年限 | 1,215年（812–2026） |
| 观测总数 | 838次 |
| 2026年峰值日期 | 3月29日 |
| 历史最早峰值 | 3月25日（2023年） |
| 历史最晚峰值 | 5月4日（1323年） |
| 年度最大波动 | 27天（1556→1557年） |

## 趋势分析

### 小冰期表现
14世纪至19世纪期间，峰值日期呈现明显推迟趋势，反映了小冰期对京都樱花盛开的影响。

### 现代变化
约1900年起，峰值日期开始持续提前。到20世纪末，30年滚动均值已降至平安时代以来的最低水平。

### 2026年现状
2026年峰值出现在3月29日，比前现代时期平均值提前两周以上。

## 樱花词汇

日语为樱花各阶段赋予了精确词汇：

| 日语 | 罗马字 | 含义 |
|------|--------|------|
| 桜 | sakura | 樱花（树与花） |
| 开花 | kaika | 始开日（首批花蕾绽放） |
| 満開 | mankai | 盛开（一周左右的盛花期） |
| 花吹雪 | hanafubuki | 花吹雪（随风飘落的谢花期） |

## 数据来源

日本大阪府立大学 Yasuyuki Aono 编制的物候观测数据，基于皇室日记、寺院记录及现代气象数据，数据存档于 NOAA Paleoclimatology。数据来源：Our World in Data（CC-BY）。
