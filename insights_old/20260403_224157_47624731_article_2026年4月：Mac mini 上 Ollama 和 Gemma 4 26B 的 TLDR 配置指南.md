## 内容摘要

本洞察文章详细介绍了在 Apple Silicon Mac mini 上安装和配置 Ollama 及 Gemma 4 26B 模型的完整指南。

### 前置要求

- Mac mini (M1/M2/M3/M4/M5)
- 至少 24GB 统一内存（用于 Gemma 4 26B）
- macOS + Homebrew

### 核心步骤

1. **安装 Ollama**：通过 Homebrew cask 安装
2. **启动 Ollama**：`open -a Ollama`
3. **下载 Gemma 4 26B**：约 17GB
4. **测试模型**：验证 GPU 加速
5. **配置自动启动**：登录时自动启动 + 预加载模型
6. **保持模型常驻**：设置 `OLLAMA_KEEP_ALIVE="-1"`

### 新版本特性 (Ollama v0.19+)

- **MLX 后端**：Apple Silicon 自动使用 MLX 框架加速
- **NVFP4 支持**：NVIDIA 显存优化
- **改进的缓存**：对话间缓存复用，降低内存占用

### API 访问

本地 API：`http://localhost:11434`，支持 OpenAI 兼容格式。

## 实用命令速查

| 命令 | 描述 |
|------|------|
| `ollama list` | 列出已下载模型 |
| `ollama ps` | 显示运行中的模型及内存使用 |
| `ollama run gemma4:26b` | 交互式对话 |
| `ollama stop gemma4:26b` | 从内存卸载模型 |
| `ollama pull gemma4:26b` | 更新模型 |

## 注意事项
