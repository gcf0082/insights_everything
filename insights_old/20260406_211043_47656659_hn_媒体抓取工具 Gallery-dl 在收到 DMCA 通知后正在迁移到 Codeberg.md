## 事件概述

媒体下载工具 **gallery-dl** 在收到 DMCA 通知后，决定从 GitHub 迁移至 Codeberg。该工具的开发者 mikf 在 GitHub 上收到了 DMCA 投诉，涉及多个网站的提取器（extractors）被删除。

## 主要讨论观点

### 1. DMCA 投诉的合法性争议

- **ronsor**：这些项目并未违反版权法，也不是规避工具，发起 DMCA 投诉实际上是违法的。
- **direwolf20**：如果相信它是规避工具，那么就是合法的。"没有下载按钮"是否属于版权保护措施存在争议。
- **dns_snek**："没有下载按钮"与"主动实施措施防止下载"是不同的，反规避条款是一个需要消除的问题。
- **AnthonyMouse**：DMCA 第512条是关于侵权内容的删除通知，第1201条是关于反规避的独立条款，两者是分开的。

### 2. 法律诉讼的可行性

- **sammy2255**：没有人有勇气、时间和金钱来挑战它。
- **ddtaylor**：法律援助并不像人们想象的那么美好，律师通常有很小的公益配额。
- **awakeasleep**：EFF 对已经确立的法律不感兴趣，只会资源投入到能设立法律先例的案件上。
- **ohhman11**：绝对不需要花钱找律师来回应 DMCA 通知，除非你有恶意行为。

### 3. 关于盗版网站的争论

- **charcircuit**：链接到内容明显被盗的盗版网站似乎确实违反法律。
- **ddtaylor**：这应该通过正常的 DMCA 删除请求来处理非法托管的材料。
- **nondrool**：至少有些列出的网站已经应 fakku 的 DMCA 要求删除了内容，hentaifoundry 是用户发布自己艺术的网站，不是盗版网站。

### 4. 迁移至 Codeberg 的可行性

- **normie3000**：Codeberg 在 2020 年 YouTube-DL 事件中表现更支持。
- **landr0id**：建议使用 radicle.xyz 以获得真正的抗审查能力。
- **notThrowingAway**：Codeberg 遵循德国法律，不会很容易，因为德国法律比美国版权法更严格。
- **fweimer**：在欧盟特别是德国运行开放代码托管平台在法律上极具挑战性。

### 5. 对版权滥用的担忧

- **ddtaylor**：随着版权滥用和 AI 的增加，越来越多的人开始关注去中心化系统。
- **14**：考虑使用 BitTorrent 来托管 Git 仓库，因为去中心化一旦发布就很难停止。
- **lloeki**：推荐使用 radicle.xyz。
- **_imnothere**：批评 FAKKU 的行为是"令人厌恶的耻辱"。

### 6. 地缘政治考量

- **echelon**：德国和欧盟可能会向美国屈服，特别是如果 DMCA 请求或诉讼由足够大的玩家发起。如果 Codeberg 位于俄罗斯或美国的地缘政治对手手中，这些请求可能会被忽略。

## 总结

这一事件反映了开源社区与 DMCA 版权投诉之间的持续斗争。Gallery-dl 事件与 YouTube-DL、youtube-dl 等案例类似，引发了关于工具的合法性、版权法的适用范围以及去中心化托管解决方案的讨论。社区普遍认为 DMCA 滥用问题需要得到解决，但实际挑战这些通知的法律和财务障碍仍然很高。
