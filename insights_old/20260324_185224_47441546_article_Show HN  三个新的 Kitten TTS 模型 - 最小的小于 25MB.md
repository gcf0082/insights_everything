## 项目概述

KittenTTS 是一个开源的轻量级文本转语音（TTS）库，基于 ONNX 构建。该项目提供从 15M 到 80M 参数的模型（25-80 MB），可在 CPU 上运行高质量语音合成，无需 GPU。

## 核心特性

- **超轻量级**：模型大小从 25 MB（int8）到 80 MB，适合边缘部署
- **8 种内置音色**：Bella, Jasper, Luna, Bruno, Rosie, Hugo, Kiki, Leo
- **文本预处理**：内置处理数字、货币、计量单位等的管道
- **CPU 优化**：基于 ONNX 的推理，无需 GPU 即可高效运行
- **24 kHz 输出**：标准采样率的高质量音频

## 模型规格

| kitten-tts-micro | 40M | 41 MB | KittenML/kitten-tts-micro-0.8 |
| kitten-tts-nano | 15M | 56 MB | KittenML/kitten-tts-nano-0.8 |
| kitten-tts-nano (int8) | 15M | 25 MB | KittenML/kitten-tts-nano-0.8-int8 |

## 技术架构

1. **文本预处理**：数字、货币、缩写等转换为口语形式
2. **分块处理**：长文本拆分为 ≤400 字符的句子块
3. **语音合成**：espeak-ng 将文本转换为 IPA 音素
4. **分词**：IPA 字符映射为整数 token ID
5. **ONNX 推理**：模型接受 (input_ids, style, speed) 输入，输出音频
6. **音频拼接**：每个分块的音频连接为单一波形

## 使用示例

```python
from kittentts import KittenTTS

model = KittenTTS("KittenML/kitten-tts-mini-0.8")
audio = model.generate("This high-quality TTS model runs without a GPU.", voice="Jasper")

# 调整语速（默认：1.0）
audio = model.generate("Hello, world.", voice="Luna", speed=1.2)
```
