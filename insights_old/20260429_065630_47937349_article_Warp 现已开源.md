## 项目概述

Warp 是一款诞生于终端的智能开发环境（Agentic Development Environment）。用户可以使用 Warp 内置的编码代理，也可以携带自己的 CLI 代理（Claude Code、Codex、 Gemini CLI 等）。OpenAI 是该项目的创始赞助商，新的智能管理功能由 GPT 模型驱动。

## 核心特性

1. **智能代理集成** - 支持内置编码代理和第三方 CLI 代理
2. **跨平台支持** - 支持 macOS 和 Linux
3. **开源客户端** - 客户端代码完全开源
4. **双许可证模式** - UI 框架采用 MIT 许可证，核心代码采用 AGPL v3

## 技术架构

### 主要依赖

- **Tokio** - 异步运行时
- **NuShell** - Shell 环境
- **Fig Completion Specs** - 命令补全规范
- **Warp Server Framework** - Web 服务框架
- **Alacritty** - 终端模拟器
- **Hyper** - HTTP 库
- **FontKit** - 字体处理
- **Core-foundation** - macOS 核心框架
- **Smol** - 异步运行时
