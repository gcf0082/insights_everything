## 核心洞察

Apple Silicon（M5 MacBook Pro）在运行本地大语言模型推理时，成本比使用OpenRouter云服务高出约**3倍**。

---

## 详细分析

### 1. 电费成本

- **电费单价**：美国平均 residential 电价约 $0.173/kWh（2025年 EIA 数据），作者按 $0.20/kWh 计算
- **设备功耗**：M5 MacBook Pro 在推理负载下约 50-100 瓦
- **每小时电费**：约 $0.009-$0.018（取 $0.02）
- **每日电费**：约 **$0.48**（按 100% 推理负载计算）

### 2. 硬件成本

- **设备配置**：14寸 M5 Max MacBook Pro + 64GB RAM
- **采购价格**：$4,299
- **预期使用寿命**：3-10年（作者认为5年较为合理）
- **年均成本**：$860-$1,433
- **每小时硬件折旧成本**：$0.049-$0.164

### 3. 推理性能与Token成本

| 性能指标 | 数值 |
|----------|------|
| 推理速度 | 10-40 tokens/秒（Gemma 4 31b） |
| 每小时Token数 | 36,000-144,000 tokens |
| 每百万Token成本（本地） | $1.61-$4.79 |
| 每百万Token成本（OpenRouter） | ~$0.38-$0.50 |

### 4. 成本对比结论

| 场景 | 本地 vs 云端成本倍数 |
|------|---------------------|
| 乐观估计（50W + 40tokens/s + 10年） | 1x（与云端相当） |
| 保守估计（100W + 10tokens/s + 3年） | **10x** |
| **作者预期** | **约3x** |

---

## 关键结论

1. **本地推理成本更高**：Apple Silicon 推理成本约是 OpenRouter 的 3 倍，硬件折旧占主导因素

2. **速度是最大差距**：
   - 本地推理：10-20 tokens/秒
   - OpenRouter：60-70 tokens/秒（3-7倍更快）

3. **实际应用考量**：
   - 对于人类员工来说，工资成本是Token成本的约1000倍
   - 对于简单任务，云端服务在经济上更合理

4. **技术意义**：消费级设备能运行接近 Anthropic Sonnet 水平的模型仍是一件令人惊叹的事
