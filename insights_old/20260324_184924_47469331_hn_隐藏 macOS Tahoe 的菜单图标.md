# 洞察报告：隐藏 macOS Tahoe 菜单图标

## 基本信息

- **链接**: https://news.ycombinator.com/item?id=47469331
- **原文链接**: https://512pixels.net/2026/03/hide-macos-tahoes-menu-icons-with-this-one-simple-trick/
- **发布日期**: 2026年3月21日
- **来源**: 512 Pixels (Stephen Hackett)
- **HN评论数**: 28条

## 内容摘要

macOS Tahoe 引入了菜单图标功能，但这一设计选择受到了部分用户的批评。主要问题包括：

1. **视觉干扰**: 图标使菜单难以快速浏览
2. **图标质量参差不齐**: 许多图标选择缺乏意义，不同系统应用之间不一致

## 解决方案

Steve Troughton-Smith 发现了一个 Terminal 命令来禁用这些菜单图标：

```bash
defaults write -g NSMenuEnableActionImages -bool NO
```

执行该命令后需要重新启动应用程序生效。此设置会保留少数确实需要图标的场景，如窗口缩放/调整大小按钮的图标。

## 社区反应

HN 评论中普遍对 macOS Tahoe 的 UI 设计表示不满：
- Tahoe 看起来像2009年的 KDE 主题

- 许多人考虑转向 Linux
- 新设计负责人 Steve Lemay 据称是 Liquid Glass 的主要推动者

## 结论

作者认为 Apple 应该在 macOS 27 中撤销这一更改，或为不希望显示菜单图标的用户提供正式设置选项。
