# If America's So Rich, How'd It Get So Sad?

## Or: How the 2020s broke our brains

**作者**: Derek Thompson

**发布日期**: 2026年4月23日

**来源**: The Atlantic / Substack

---

# 1. 悲剧的二十年代

"美国长期以来是一个相当幸福的国家，"芝加哥大学经济学家萨姆·佩尔茨曼(Sam Peltzman)在2026年的一篇论文中写道。"现在已经不幸福了。"

佩尔茨曼分析了综合社会调查(General Social Survey)的数据，记录了新冠疫情后"美国人口自我报告幸福感突然、剧烈且史无前例的下降"，这种下降"主要持续"到2024年。他称之为国家情绪的"体制转变"。在经历了50年基本稳定的自我报告幸福感水平后，美国人的幸福感急剧下降。而且几乎没有恢复。

佩尔茨曼的分析并非孤立的声音；实际上有一系列悲观的情绪指标。本周，美联储的美国工人满意度衡量指标降至该调查自2014年开始以来的最低水平一周前，消费者信心降至密歇根大学经济调查70年历史上最低水平。同样，该指数在2020年左右大幅下降，就像一个登山者还在继续下行。美国人告诉民意调查者他们对经济的悲观程度超过了"大萧条"时期的低谷或1970年代痛苦的滞胀时期。

最后，美国在《世界幸福报告》中的排名也降至历史最低，这很大程度上是由于该国际调查中年轻人幸福感令人难以置信的快速下降。

如果你正在寻找一个对这一现象有同感的倾诉对象，千万别去找你的当地经济学家。那些通过就业或收入统计数据的视角看待世界的人对美国人的 blues 感到非常好奇。失业率在整个十年间基本上都低于5%，这基本上是你能要求的最好水平。整个十年，美国经济都显著增长，超过了欧元区和其他富裕国家，如日本和英国。美国人越来越富，按照大多数传统标准衡量是如此。越来越多的美国人正在"跻身中上层阶级"，而收入分配底层的工人工资增长比顶层更快。

因此，那些重视经济统计数据而非自我报告的人可能会这样总结情况：美国有弹性的经济是一个"事实"，而美国人的悲伤调查报告结果只是非理性的"感觉"。这是有道理的；所谓的"硬数据"(如失业率)和"软数据"(如调查)之间的差距确实很大，而且越来越大。但感觉是一种重要的事实。感觉不仅塑造消费者行为。它们塑造政治态度；态度影响投票；投票决定政策；政策塑造经济。要了解美国经济和整个美国的未来，不能对情绪采取傲慢的冷漠态度。

在情绪方面，我们有四个调查结论——四个"事实"，你甚至可以说，是美国人的忧郁——所有这些都指向一个不可避免的结论。这个 decade 恰恰与"繁荣"相反。我们陷入了悲剧的二十年代。

# 2. 谁谋杀了氛围?

佩尔茨曼论文中一个更显著的发现是，自2020年以来，幸福感的下降并不集中在年轻人、穷人或未婚人士——这三个群体通常更容易受到焦虑和悲伤的困扰。相反，幸福感的下降是一种跨群体的 10-15 点的全面衰减，几乎每个 demographics 都经历了。(在下图中，蓝色指的是 2020 年前的幸福感水平；粉色是 2020 年后的幸福感水平；黑色是下降，这在各群体之间非常一致。)

是什么重大的事情在過去六年不分年齡、意識形態、教育或性別地毀滅了美國人的幸福感?

罪魁禍首必須符合犯罪的時間。我們正在尋找的是2020年左右發生的事情(嗯，這似乎很明显)，然後沒有恢復(啊，這是困難的部分)。這個時間因素排��了一些其他表面上合理的嫌疑人。

這可能不是關於文化轉變，比如宗教的衰落。 文化保守派可能會試圖通過指出自由主義者中世俗個人主義的興起來解釋悲劇的二十年代，並指出宗教似乎是不幸福的良藥。但美國宗教無隸屬關係的興起是一個穩定的30年趨勢，而這種幸福感的下降始於2020年，當時世俗主義達到了最近的峰值。所以，這個解釋不行。

這也可能不是關於傳統的工資不平等。 左翼的一些人可能會傾向於認為美國的痛苦是對嚴重的階級不平等的合理和自動的社會反應。但自疫情以來，低收入工資增長異常強烈，正如經濟學家阿林·杜貝(Arin Dube)所指出的。中位數家庭收入現在比十年前更高。更重要的是，佩爾茨曼的分析發現，一些最大的幸福感下降似乎集中在富裕人群中，如老年人、白人和大學畢業生。所以，又一個嫌疑人不符合犯罪特徵。

這也可能不只是關於手機和社交媒體。 當涉及到美國人的焦慮和不幸福時，最明顯的嫌疑人智慧手機、社交媒體和美國新聞循環日益增長的負面情緒。正如我上個月在一篇長文中解釋的，我相當說服手機和社交媒體與——而且很可能，實際上正在——導致美國年輕人幸福感的下降。但年輕人日益增長的痛苦——通常正確地與手機和社交媒體使用增加有關——已經持續了約15年。我們在綜合社會調查和密歇根大學數據中看到的整體健康狀況的更突然崩潰指向了2020年左右發生的一次情緒break。所以，即使手機不是無罪的(我稍後會回來)，它們也不像是主要罪魁禍首。

如果文化墮落、物質不平等、手機和社交媒體似乎都不適合這個特定現象的形狀，我們必須繼續尋找2020年代是什麼打破了你的大腦。我能提供的最簡單解釋是：作為一種文化政治力量，2020年的疫情從未結束。

# 3. 永久疫情

## 疫情從未結束，第一部分：通貨膨脹的糟糕

如果不從2020年到來的危機開始，就無法假裝解釋2020年的幸福感崩潰，這個危機從未真正離開。新冠疫情不僅僅給地球帶來了冠狀病毒。 這種疾病的生物對手讓位於一系列經濟災難，從供應鏈中斷，到全球通貨膨脹，再到利率飆升。我們仍然生活在餘波的中間。

雖然官方 年度通貨膨脹率已經上漲，然後下跌，然後再次上漲，但典型家庭不會將價格變化體驗為每月更新12個月的平均值。他們在雜貨店、餐館或網上結賬頁面感受到的東西，更像是"他媽的，這是多少錢?"而那個激動的時刻最好理解為多年高於平均水平的通貨膨脹的累積。

這樣想：消費者價格在2007年夏天到2020年夏天之間上漲了25%，在2020年夏天到2025年夏天之間飆生了同樣的數量。在住房方面，2020年夏天到2025年夏天之間Case-Shiller美國國家房價指數上漲了50%，這與2004年到2020年之間房價上漲50%相同。在這兩種情況下，可以公平地說，美國人在21世紀經歷的通貨膨脹率大約是他们在2020年代習慣的三倍。所有他們購買的東西都感覺不斷滑出可承受的範圍，這對許多人來說 absolutely 是令人发疯的，無論經濟統計數據表明他們應該感覺到什麼。

經濟作家馬特·達林(Matt Darling)追踪了實際消費者情緒與"預測"情緒之間的關係，基於失業率、通貨膨脹率和利率。2020年左右，這種關係破裂了，消費者情緒直線下降 into 在下面的圖表中，深紫色線表示實際消費者情緒，而淺色虛線表示消費者情緒"應該"處於的位置。

人們很容易想：好吧，這只是說明通貨膨脹對最貧窮的美國人有多糟糕。但達林的分析中最有趣和令人困惑的一點是，實際上是最富有的三分之一家庭，其消費者情緒���降最顯著，相對於我們預期的位置。達林的解釋是聰明但令人沮喪：充分就業，特別是在高通脹時代，增加了所有涉及他人的事物的成本，並創造了一個脾氣暴躁的國家。以下是達林的觀點：

"我認為部分原因是許多中上和上層收入家庭習慣於能夠按需獲得低薪勞動力——用於兒童保育、餐飲服務、家庭醫療保健。中上和上層收入家庭對此感到沮喪，並認為這是整個經濟故事的組成部分；沒有意識到這種沮喪很大程度上是由低薪工人最終獲得了一些談判權力造成的。"

綜合來看：在過去40年裡，美國人已經習慣並珍惜無需考慮的可承受性。但在過去五年裡，各種事物的價格，包括住房，增長速度大約是美國人習慣的三倍；同時，充分就業給服務成本帶來了上行壓力。美國公眾的回應不僅是向民意調查者尖叫他們的痛苦，而且還衝向投票站，投票驅逐所有未能做什麼來應對2020年代"可負擔性"危機的現任者。美國人並不孤單：2024年是全球現任政黨的大屠殺，因為對高通脹的憤怒像疫情本身一樣席捲全球。

## 插曲：電話和英語國家

根據最新的《世界幸福報告》，近年來幸福感實際上在許多國家有所增加，包括中國、印度和越南。但幸福感在西方大部分地區有所下降，特別是在英語國家，如美國、加拿大、英國、愛爾蘭、澳大利亞和紐西蘭。[1](#footnote-1) 約翰·赫利威爾(John Helliwell)是英屬哥倫比亞大學的經濟學教授，也是《世界幸福報告》的合著者，他告訴我："如果你在尋找年輕人不幸福上升的特殊之處，它們主要是西方發達國家，而且大多是說英語的國家。"

英語國家可能會在過去十年經歷更大的幸福感下降，原因有以下幾點：(1) 一種個人主義文化，通常與與他人在一起的時間較少相關；(2) 高度的診斷膨脹，意味著焦慮、注意力缺陷多動症和其他精神健康障礙的擴展精神健康指南，從而機械地增加診斷的焦慮並提高對負面精神健康的認識；(3) 新聞生態系統和社交媒體中的高度負面情緒。

我認為這個插曲加強了兩個論點：首先，英語國家的精神健康存在獨特的問題；其次，高通脹率是悲劇二十年代現象的主要貢獻者，無論在美國還是整個西方。

## 疫情從未結束，第二部分：機構下降，個人主義上升

"疫情"一詞來自希臘語"pan"，意思是所有，"demos"，意思是人民。雖然詞源暗示整體性，但疫情的影響歷史上一直是破壞社會信任。在對西班牙流感的一次近期分析中，研究人員發現這種疾病對"個人行為有持久的影響，在社會信任方面"。因此，在他的論文中，經濟學家佩爾茨曼發現，整個2020年代對幾乎所有機構的信心都有所下降，包括聯邦政府、軍事、主要公司、教育和有組織的宗教。其他研究發現，對疾病控制與預防中心、高等教育、科學和醫學的信任直線下降。

不僅是美國人失去了對高高在上機構的信任。他們對彼此的信任遭受了更劇烈的下降。數十年來，綜合社會調查一直在問美國人同樣的基本問題："你是否認為大多數人在有機會的情況下會試圖佔你的便宜，或者他們會試圖公平？"在1970年代和1980年代，美國人壓倒性地認��其��人基本上是可信賴的。自2020年以來，對陌生人的信心直線下降，根據佩爾茨曼。說其他人是"公平"的受訪者比例下降得甚至比整體幸福感還多。

正如同美國人對機構和陌生人的信任下降，至少有一種狂喜的個人主義指標在其位置上上升了。美國人現在在自己身上花費前所未有的時間 along with不正常的大量時間在室內。這意味著他們與他人的接觸不成比例地受到調節，不是通過外部世界的真實生活體驗，而是通過屏幕上算法媒體的調節。正如紐約大學的心理學家杰伊·范·巴維爾(Jay Van Bavel)所指出的，網上對話 prioritize 並回報負面情緒和 group 敵意，這將那些本來可能在酒吧或辦公室享受(或忍受)彼此存在的人轉變為對立者。

我不認為機構信任的下降和孤獨的個人主義的上升應該為所有經歷它的人帶來不快樂。但信任、陪伴和社區是個人和國家危機時期的減震器。關於2020年代可以說的最後一件事是，它確實是一個接一個的危機。

## 疫情從未結束，第三部分：永久危機的十年

在他的2023年專欄"經濟很好。為什麼美國人情緒如此惡劣?"《華爾街郵報》專欄作家格雷格·伊普(Greg Ip) написа說，美國人的經濟悲觀類似於生物學上的"牽涉痛"現象。他寫道：

"就像你身體的一部分可能因為另一部分的傷害而疼痛，對經濟的悲觀可能反映對整個國家的不滿。近來有很多令人不滿的事情：加劇的政治和文化衝突和不容忍、疫情、邊境大規模槍擊、犯罪、烏克蘭战争， 現在是中東战争。"

確實，這個 decade 基本上就是一個垃圾箱著火，不是嗎？百年一遇的疫情產生了一代人一次 的通貨膨脹危机。烏克蘭、加沙、黎巴嫩、伊朗和波斯灣的战争接踵而至，一次又一次的戰爭。對氣候變化的存在恐懼讓位於對人工智能的存在恐懼。所有這一切都是在唐納德·特朗普(Donald Trump)像某種不祥的幽靈一樣盤旋在政治領域的時��發生的——對大約一半的國家來說代表著法西斯主義的迫近，而對另一半來說，意味著一位神圣的救世主來拯救傳統價值觀免受左翼惡魔的侵害。這一切都相當多。

在這個永久危機的十年裡，新聞變得異常嚴峻，我們有數據顯示究竟有多少。2024年布魯金斯学会對新聞情緒的分析發現，"在2018年至2020年期間，新聞 tones 比基本面預測的更負面，在2021年至2023年期間更負面，比預測的更負面。"今天的新聞比有記錄以來任何時期的新聞都更"令人驚訝的負面"。

新聞循環的歷史悲觀既是永久危機十年的反映，也是我們不斷處於危機邊緣的印象驅動者。作為全球健康緊急狀態，新冠疫情可能已經結束，但美國人在日常生活中接觸新聞時感受到的危機感並沒有消失。感染率下降了，但世界不斷充滿緊急狀態的感覺並沒有消失。

# 4. 悲劇的二十年代：裁決

因此，這是我能達到的最接近悲劇二十年代的統一理論。這個 decade 的美國悲傷是由永久且無情的經濟危機的事實和感覺锻造的，通過独特負面的新聞和媒體環境放大，並因孤獨的上升和受信任機構的中心性下降而加劇。通貨膨脹使今天的生活更難以承受，而對他人成就的環境意識在社交媒體上使明天的成功感覺更難以實現。對體制的信心崩潰使美國人異常迷失並對他們無法控制的機構感到不滿，而現代生活的選擇性自我隔離破壞了社區信任，因為我們越來越多地通過屏幕上有毒的超現實體驗他人的思想，而不是通過陌生意圖的embodied 現實，總的來說，和我們一樣好。

英語國家真的與過去幾年幸福感的下降有關嗎？赫利威爾提出了一個聰明的理論測試：看看魁北克，那裡80%以上的人口說法語。而在相鄰的安大略省，不到4%的人口說法語。那麼，魁北克人是否對英語國家的不快樂疫情有一定免疫力？儘管看起來很奇怪，答案似乎是肯定的："在用於《世界幸福報告》的蓋洛普數據中，30歲以下魁北克人的生活滿意度下降幅度是加拿大其他地區的一半，"赫利威爾告訴我。在對加拿大綜合社會調查的單獨分析中，該調查詢問受訪者他們的首選語言，英屬哥爾比亞大學和阿爾伯塔大學的研究人員發現，在家說法語的年輕人比在家說英語的年輕人幸福感下降幅度更小。

## 腳註

[1] 如果有什麼特別的與過去几年英語國家的幸福感下降相關？赫利威爾提出了一個聰明的理論測試：看看魁北克，那裡80%以上的人口說法語。 而在相鄰的安大略省，不到4%的人口說法語。那麼，魁北克人是否對英語國家的不快樂疫情有一定免疫力？奇怪的是，答案似乎是肯定的："在用於《世界幸福報告》的蓋洛普數據中，30歲以下魁北克人的生活滿意度下降幅度是加拿大其他地區的一半，"赫利威爾告訴我。在對加拿大綜合社會調查的單獨分析中，該調查詢問受訪者他們的首選語言，英屬哥爾比亞大學和阿爾伯塔大學的研究人員發現，在家說法語的年輕人比在家說英語的年輕人幸福感下降幅度更小。
