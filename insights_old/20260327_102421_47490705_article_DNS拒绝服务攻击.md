## 项目概述

DOOM Over DNS 是一个极具创意的项目，它通过 DNS TXT 记录来传输和运行完整的 DOOM 游戏。该项目利用 Cloudflare 的免费 DNS 服务，将 DOOM 游戏的全部数据分割存储在约 1,964 个 DNS TXT 记录中，运行时通过 PowerShell 脚本和公共 DNS 查询来获取数据并执行游戏。

## 核心特性

- **纯 DNS 传输**：游戏数据完全通过 DNS TXT 记录获取，无需传统的 HTTP 服务器
- **零磁盘存储**：WAD 文件从不写入磁盘，.NET 游戏引擎 DLL 直接加载到内存中
- **边缘缓存**：利用 Cloudflare 的全球边缘网络进行免费缓存
- **多区域分条**：支持多个 DNS 区域自动分发数据块
- **断点续传**：上传中断后可恢复传输

## 技术实现

1. **Start-DoomOverDNS.ps1**：客户端脚本，负责从 DNS 解析并加载游戏
2. **Publish-DoomOverDNS.ps1**：上传脚本，用于将游戏数据发布到 DNS 记录
3. **managed-doom**：基于 .NET 8 的游戏引擎修改版

## 使用要求

- PowerShell 7+
- Cloudflare 账户（用于 DNS 管理）
- DOOM1.WAD 文件（Shareware 版本）

## 总结

DOOM Over DNS 是一个展示了 DNS 协议创造性用法的实验性项目。尽管 DNS 并非设计用于文件存储，但该项目证明了通过巧妙的工程设计，可以利用公共 DNS 基础设施来传输和运行完整的游戏。这种技术虽然不适用于实际生产环境，但对于理解网络协议和创造性编程具有重要的教育意义。
