# 洞察报告：Mozilla Thunderbolt

## 基本信息

- **洞察链接**：https://www.thunderbolt.io/
- **发布日期**：2026年4月16日
- **来源**：Phoronix / Mozilla 官方发布
- **主题**：Mozilla 发布开源企业级 AI 客户端 Thunderbolt

---

## 洞察摘要

Mozilla 于2026年4月16日正式宣布推出 **Thunderbolt**，这是一个开源的企业级 AI 客户端，旨在为用户提供控制权和独立性。Thunderbolt 被设计为一个"主权 AI 客户端"（sovereign AI client）——一个开源的、可扩展的工作空间，用户可以通过聊天、搜索和研究等方式与 AI 交互，连接企业数据，并选择适合其需求的模型和工具。

## 核心功能

### 1. 模型灵活性
- 支持来自领先商业供应商的模型
- 支持开源模型
- 支持本地模型运行

### 2. 系统与数据集成
- 集成管道和开放协议
- 支持 deepset 的 Haystack 平台
- 支持 Model Context Protocol (MCP) 服务器
- 支持 Agent Client Protocol (ACP) 代理

### 3. 工作流自动化
- 生成每日简报
- 监控特定主题
- 编译报告
- 基于事件和调度触发操作

### 4. 跨平台支持
- Web 应用
- 原生应用：Windows、macOS、Linux、iOS、Android

### 5. 安全特性
- 支持自托管部署
- 可选端到端加密
- 设备级访问控制

## 技术细节

- **代码许可**：MPL 2.0（开源）
- **企业许可**：由 Mozilla 的 MZLA Technologies Corporation 提供
- **项目代码**：托管于 GitHub

## 市场意义

Thunderbolt 代表了 Mozilla 在 AI 领域的最新尝试，延续了其"为 AI 做就像为 Web 做的一样"的愿景。该项目旨在帮助组织和用户在不依赖云服务的情况下部署自托管的 AI 基础设施，体现了对数据隐私和用户控制权的重视。
