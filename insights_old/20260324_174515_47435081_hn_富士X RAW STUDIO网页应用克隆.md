## 项目概述

FilmKit 是一个富士 X RAW STUDIO 的网页版克隆应用，由开发者 notcodingtoday 创建。该项目允许用户在浏览器中通过 WebUSB 直接连接富士相机，利用相机自身的图像处理器进行 RAW 文件的转换和处理。

## 核心功能

1. **WebUSB 直连**：通过 WebUSB 协议与富士相机建立连接
2. **相机处理器转换**：利用相机自身的处理器进行图像转换，而非本地软件转换
3. **无需安装**：作为静态客户端应用，托管在 GitHub Pages 上
4. **配方调整**：允许用户调整胶片模拟配方（Film Simulation Recipes）
5. **PTP 协议**：通过图片传输协议（Picture Transfer Protocol）与相机通信

## 技术特点

- **工作原理**：FilmKit 发送 RAF（RAW）文件和转换参数给相机，相机处理后返回 JPEG
- **客户端应用**：纯前端静态页面，无需服务器端处理
- **跨平台支持**：支持多种操作系统，包括 Linux、macOS、Windows

## 用户反馈与讨论热点

### 正面评价

- 用户对项目表示赞赏，认为这是“令人惊叹的工作”
- 界面设计简约美观
- 为 Linux 用户提供了在官方应用无法运行时的替代方案
- 实测在 Fujifilm X-H2、X-T5、GFX 100S II 等机型上可用

### 技术讨论

1. **RAW vs raw 大小写争议**：有用户指出"RAW"不应全大写，因为并非缩写词，而是"raw"（原始数据）的通用写法。这一讨论引发了激烈争论，最终被认为是“已失败的战斗”。

2. **与官方软件比较**：该项目与官方 X RAW Studio 一样，利用相机处理器进行转换，因此无法完全复制富士的胶片模拟算法。第三方工具只能做到“近似”效果。

3. **兼容性问题**：
   - 旧款相机（如 X-T20、X-T30、X-E1）可能不支持
   - iOS 设备无法使用（Safari 不支持 WebUSB）
   - MacOS 上可能需要额外配置

4. **竞品推荐**：
   - **Capture One**：被评价为处理富士 RAW 色彩“最可靠”的第三方软件
   - **RawTherapee**：开源 RAW 处理工具，支持 X-Trans RAF 格式
   - **Fudge**：另一个富士胶片模拟应用，已被本项目列为参考

5. **iOS 限制**：由于苹果要求所有浏览器使用 WebKit 引擎，WebUSB 无法在 iOS 上使用。欧盟地区目前仍无非 WebKit 浏览器。

## 总结

FilmKit 为富士相机用户提供了一个便捷的网页端工具，可以在不安装官方软件的情况下利用相机自身处理器调整 RAW 文件。该项目解决了 Linux 用户无法使用官方 X RAW Studio 的痛点，同时也为其他平台用户提供了轻量级替代方案。尽管无法完全复制富士的封闭式胶片模拟算法，但项目本身仍然是富士用户社区的有价值工具。
