## 内容摘要

HopTab 是一款开源的 macOS 应用切换器和窗口平铺工具，旨在替代 macOS 原生的 Cmd+Tab 应用切换器并提供更强大的窗口管理功能。

### 主要功能

**新增功能：**
- 全局窗口平铺快捷键：Ctrl+Opt+Arrow（ halves ）、Ctrl+Opt+UIJK（ quarters ）、Ctrl+Opt+DFG（ thirds ），所有17个方向均可配置
- 窗口尺寸循环切换：按两次 Ctrl+Opt+Left 可从 1/2 切换到 1/3，再次按可切换到 2/3
- 撤销快照：Ctrl+Opt+Z 恢复上一个窗口位置
- 跨显示器移动：Ctrl+Opt+Cmd+Arrow 将窗口移动到下一个显示器
- 可配置窗口间距：0-20pt 间距，支持实时预览
- Profile 切换器显示应用图标：使用实际的应用图标而非通用头像

**已有功能：**
- 固定应用：通过 Option+Tab 在不同 Profile（如 Coding、Design 等工作流）之间切换，每个 Profile 有独立的热键
- 布局模板：50/50、IDE 60/40、三列、2×2 网格等
- 会话保存/恢复：保存每个 Profile 的窗口位置、大小、z-order
- 分配 Profile 到 macOS Spaces：切换桌面时自动切换
- 多窗口应用窗口选择器
- Profile 专属的便签功能

### 特点

- 免费、开源、无遥测
- 约 3MB 二进制文件
- 替代方案：Rectangle + AltTab + 手动拖拽窗口的组合

## 热门评论摘要

### 窗口管理是 Mac 用户痛点

从 Windows 切换到 macOS 的用户表示，窗口管理是最大的痛点。虽然 Rectangle Pro 有所帮助，但工作区集成仍然不够完善。

### AltTab 是必备工具

多位用户推荐 AltTab（免费开源），认为它是 macOS 上必备的工具：
- 切换到标签页时恢复最小化的应用
- 为缺少窗口的应用（如 Finder）创建新窗口

### 原生 macOS 功能改进

- 有用户指出 macOS 26 的 Spotlight 引入了原生剪贴板历史功能
- 有人提到可以通过系统设置将 Cmd+Tab 改为切换窗口而非应用（无需第三方应用）

### 对比评价

- **HopTab vs AltTab**：主要区别在于 HopTab 固定特定应用而非显示所有应用
- **HopTab vs Rectangle**：平铺功能与 Profile 和布局集成，保存会话后可次日恢复
- 用户还推荐了 Contexts、Maccy、Divvy 等类似工具

### 功能建议

- 有用户建议增加对多显示器配置的支持
- 有人希望增加自定义布局功能（特别是对于旋转显示器）
- 建议增加工作区层面的功能，将多个 Profile 组合在一起

### 潜在问题

- 部分用户反映菜单栏图标空间不足
- 视频演示需要旁白和更好的节奏来展示应用功能
