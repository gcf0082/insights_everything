# 洞察报告

**链接**: https://www.anthropic.com/news/higher-limits-spacex  
**来源**: Anthropic 官网新闻  
**发布日期**: 2026年5月6日  
**标题**: Higher usage limits for Claude and a compute deal with SpaceX  

---

## 核心内容总结

### 一、提高使用限制

Anthropic 宣布即日起实施三项重要变更，以提升高优先级用户的使用体验：

1. **Claude Code 五小时速率限制翻倍**：适用于 Pro、Max、Team 和基于席位的 Enterprise 计划
2. **移除 Claude Code 高峰时段限制降低**：适用于 Pro 和 Max 账户
3. **大幅提高 Claude Opus 模型的 API 速率限制**

### 二、与 SpaceX 的新计算合作

Anthropic 与 SpaceX 签署协议，获得 SpaceX Colossus 1 数据中心的全部计算容量：
- 新增超过 300 兆瓦的计算能力
- 超过 220,000 个 NVIDIA GPU
- 一个月内即可投入使用
- 将直接改善 Claude Pro 和 Claude Max 订阅用户的容量

此外，Anthropic 还表达了与 SpaceX 合作开发多个千兆瓦级轨道 AI 计算容量的兴趣。

### 三、其他计算合作项目

Anthropic 近期的计算合作伙伴包括：

- **亚马逊**：高达 5 吉瓦（GW）协议，到 2026 年底将新增近 1 GW 容量
- **谷歌和博通**：5 GW 协议，将于 2027 年开始上线
- **微软和英伟达**：战略合作伙伴关系，包含 300 亿美元的 Azure 容量
- **Fluidstack**：500 亿美元投资于美国 AI 基础设施

### 四、国际扩张计划

Anthropic 正在向国际扩展计算基础设施，以满足金融、医疗、政府等受监管行业的企业客户需求：
- 与亚马逊的合作包括在亚洲和欧洲增加推理能力
- 优先与民主国家合作，确保供应链安全
- 承诺覆盖美国数据中心导致的消费者电价上涨
- 探索将该承诺扩展到新司法管辖区

---

## 主要影响

1. **用户体验提升**：Claude Code 和 API 用户将获得更高的使用限制
2. **计算能力大幅增强**：通过 SpaceX 合作获得超过 300 MW 新容量
3. **AI 基础设施扩张**：多笔重大合作确保长期计算资源供应
4. **国际化布局**：满足企业客户的合规和数据驻留需求
