# Hacker News 洞察报告

## 基本信息

- **洞察链接**: https://news.ycombinator.com/item?id=47885596
- **标题**: Ubuntu 26.04
- **发布时间**: 2025年
- **评论数**: 118条
- **分数**: 206 points

---

## 核心洞察

### 1. GNOME 桌面环境争议

**中键粘贴功能被禁用**
- Ubuntu 26.04 (GNOME 50+) 默认禁用了中键粘贴功能，引发用户强烈不满
- 多位用户指出这是 GNOME 效仿 macOS 的决定，该功能自 X Window System 时代(1980年代)就已存在
- 部分用户(如 ThinkPad 用户)表示欢迎此更改，因为使用小红点滚动时容易误触
- 可通过 GNOME Tweak Tool 重新启用

**桌面环境选择讨论**
- 部分用户转向 KDE Plasma，认为其功能更完善
- 有用户认为 Cinnamon 是"功能极简主义"的良好选择
- XFCE 也获得好评，被描述为"简单不碍事"
- 讨论中涉及对 GNOME 开发团队决策风格的批评

### 2. Ubuntu 硬件支持改善

- 现代 Ubuntu 版本开箱即用，支持大多数计算机
- Wifi、蓝牙、GPU 驱动、打印机等外设支持显著改善
- 与20年前需要手动安装驱动的情况形成对比
- 但 Windows 仍是硬件厂商的主要目标平台

### 3. Snap 包管理器的争议

**用户对 Snap 的批评**
- 多位用户因 Snap 问题转向 Debian 或其他发行版
- 移除 Snap 后系统升级可能重新安装
- Firefox 等关键应用在 Snap 下频繁崩溃

**替代方案推荐**
- Linux Mint (Debian 版本) 被推荐为"去除 Canonical 糟粕"的替代品
- Debian 现在在 ISO 镜像中包含非自由驱动
- Fedora (配合 XFCE) 获得好评，被描述为稳定且包版本较新

### 4. Rust Coreutils 安全漏洞

- Ubuntu 正在用 Rust 重写 coreutils，但新实现发现多个 CVE 漏洞
- 发现 4 个高危漏洞，包括 chmod --preserve-root 绕过、mkfifo 权限问题等
- 讨论指出并非所有安全漏洞都与内存相关，API 使用错误同样危险
- 有人批评"Rewrite it in Rust"社区忽视_second-system effect_

### 5. Wayland 兼容性问题

- KDE Plasma 6.6 在 Wayland 下存在问题
- KeePass 自动输入功能不工作
- 多键盘布局切换功能受限
- 大多数用户仍使用 Xorg

### 6. Pop!_OS 与 Nvidia 驱动

- Pop!_OS 被评价为"最流畅的桌面体验"
- Cosmic 桌面环境仍在开发中，稳定性欠佳
- Ubuntu 24.04 搭载的 Nvidia 590 驱动是当时最新可用版本
- Debian 有自己的 Nvidia 驱动包

---

## 社区情绪总结

社区对 Ubuntu 26.04 总体持正面态度，但存在以下主要槽点：

1. **中键粘贴被禁用** - 最大争议点
2. **Snap 的强制性** - 导致部分用户流失
3. **GNOME 的"反用户"决策** - 引发桌面环境迁移讨论

积极方面：
- 硬件兼容性显著改善
- 安装体验更加友好
- 对于不需要深度定制的用户，Ubuntu 仍是可靠选择
