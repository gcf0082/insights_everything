## 摘要

本文介绍了 Avalonia 团队发布的 .NET MAUI 后端的首个预览版本，该后端允许开发者将 .NET MAUI 应用部署到 Linux 和 WebAssembly 平台。Avalonia 通过提供"绘制型"（drawn）UI 替代方案，实现跨平台一致性，让应用在所有平台上保持一致的外观和行为。

## 主要内容

### 1. 项目背景

自去年秋季以来，Avalonia 团队在将 Avalonia 强大的跨平台能力引入 .NET MAUI 方面取得了重大进展。这一新后端不仅为 .NET MAUI 带来了 Linux 和 WebAssembly 支持，还推进了 Avalonia 跨平台一致性的愿景。开发者可以选择使用原生 UI 或绘制型 UI，前者让应用与托管平台融合，后者确保应用在所有平台上具有相同的外观。

### 2. 四步快速入门

1. 创建 .NET MAUI 应用
2. 添加 `Avalonia.Controls.Maui.Desktop` NuGet 包
3. 添加 net11.0 目标框架
4. 在 `MauiBuilder` 中添加 `UseAvaloniaApp`

运行 net11.0 目标即可启动应用，无需创建 Avalonia 引导程序，团队已通过源代码生成器完成了这项工作。

### 3. 对 Avalonia 的改进

这个项目为 Avalonia 本身带来了重要改进。为缩小 .NET MAUI 与 Avalonia 可用控件集之间的差距，团队创建了新的导航 API 和控件，这些都将在 Avalonia 12 中引入。由于 .NET MAUI 处理器基于 Avalonia 原语构建，用户可以通过 Avalonia API 完全自定义这些控件。

### 4. 测试的应用程序

团队通过移植现有 .NET MAUI 应用来测试新库：

- **MauiPlanets** 和 **2048**：已发布的示例应用
- **.NET MAUI Control Gallery**：用于测试和演示服务和控件的工具
- **AlohaAI**：与 GitHub Copilot 合作开发的教育应用，关于大语言模型和机器学习
- **MyConference**：在 .NET MAUI 直播中开发的会议应用，展示"代理 AI"开发
- **WeatherTwentyOne**：.NET MAUI 示例应用，展示 WebView 功能
- **MapApp**：演示 SkiaSharp.Views.Maui 集成，包含地图视图

### 5. 技术特性

- **WebView**：使用新开源的 Avalonia WebView，支持 Linux 和 WebAssembly
- **GraphicsView**：基于 .NET MAUI GraphicsView 的控件可直接工作
- **SkiaSharp.Views.Maui**：已封装，允许依赖库与 Avalonia MAUI 互操作

### 6. 未来计划

- 开发基于 Avalonia 的 Maui.Essentials 自定义实现
- 实现与 WinUI 的互操作性，在其中托管 Avalonia 控件
- 为控件库作者建立简单模式，允许将控件扩展到绘制方法

## 总结

Avalonia MAUI Preview 1 标志着跨平台 UI 开发的重要里程碑。通过为 .NET MAUI 提供 Linux 和 WebAssembly 支持，开发者现在有了更多选择，可以根据需求在原生 UI 和绘制型 UI 之间切换。随着 .NET 11 的正式发布即将到来，Avalonia MAUI 有望成为跨平台应用开发的强大选择。
