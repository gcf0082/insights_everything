## 文章总结

本文是一篇关于在 OpenIndiana Hipster 2025.10 上设置 Sun Ray 服务器的详细教程。

### 主要内容

1. **在 Proxmox VM 中安装 OI**
   - 使用 OpenIndiana Hipster 2025.10 Live DVD
   - 推荐配置：q35 虚拟机，8GB 内存，4 核 CPU，60GB VirtIO 硬盘
   - 需要添加 VirtIO RNG 设备并启用 IOMMU

2. **安装 SRSS 软件包**
   - 首先更新系统并启用 hipster-encumbered 仓库
   - 安装 sunray-essential 包
   - 从 Oracle eDelivery 或 archive.org 下载 Sun Ray Server Software 5.4.0.0
   - 手动安装依赖：Java Runtime Environment 和 Apache Tomcat

3. **SRSS 补丁配置**
   - 安装 sunray/oi-adaptions 并运行 apply_patches
   - 手动修补：替换 ISC DHCP，修改 pkg 命令语法

4. **配置 SRSS**
   - 运行 utconfig 进行初始配置
   - 启用 kiosk 模式
   - 使用 utadm 和 utpolicy 允许客户端连接
   - 从 lightdm 切换到 gdm 显示管理器

5. **连接客户端**
   - 通过 DNS 条目配置或手动在 DTU 菜单中设置服务器 IP
   - 确保会话类型选择 MATE 而非 GNOME

6. **固件更新**
   - 从 SRSS 5.2 包中提取固件文件
   - 使用 utfwadm 配置 TFTP 根

7. **Web 管理**
   - 通过 http://<SERVER_IP>:1660 访问管理界面
   - 使用 admin 账户登录

### 关键要点

- SRSS 5.4.5.0 i386 版本难以获取
- 2025.10 版本不支持 hotdesk 会话
- 需要启用远程管理并使用 MATE 会话
- 推荐阅读第二部分获取移动会话支持
