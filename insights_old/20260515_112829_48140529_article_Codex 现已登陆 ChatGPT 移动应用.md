# 洞察报告：Work with Codex from anywhere

## 基本信息

- **洞察链接**: https://openai.com/index/work-with-codex-from-anywhere/
- **发布日期**: 2026年5月14日
- **来源**: OpenAI
- **主题**: Codex 移动端功能发布

---

## 内容总结

### 核心要点

OpenAI 宣布 Codex 正式登陆 ChatGPT 移动应用，用户现在可以通过手机随时随地与 Codex 进行协作。

### 关键功能

1. **移动端全功能体验**
   - Codex 现已集成到 ChatGPT 移动应用中
   - 可连接到任何运行 Codex 的机器（笔记本、Mac mini 或远程环境）
   - 实时加载活跃线程、审批、插件和项目上下文

2. **远程协作能力**
   - 不再局限于单一任务的远程控制
   - 可跨所有线程工作、审查输出、批准命令、更改模型或启动新任务
   - 文件、凭证、权限和本地设置保留在运行 Codex 的机器上
   - 实时同步更新到手机，包括截图、终端输出、差异、测试结果和审批

3. **安全架构**
   - 使用安全的 relay 层，使受信任的机器在设备间可访问
   - 无需将机器直接暴露在公共互联网
   - 保持活跃会话状态和上下文在任何地方同步

4. **适时介入**
   - 随着 Codex 处理更长时间的任务，及时指导变得尤为重要
   - 快速检查可以保持工作流程顺畅
   - 防止不必要的返工
   - 帮助 Codex 在正确的上下文中取得进展

### 市场数据

- 超过 **400万人** 每周使用 Codex
- 远程 SSH 功能现已正式向所有用户开放

---

## 分析

OpenAI 此次将 Codex 引入移动端，反映了 AI 编程助手在工作模式上的演进。随着代理承担更长时间运行的任务，开发者需要能够在任何设备上监控进度、进行审批和提供指导。移动端的集成填补了这一空白，使用户即使不在电脑前也能保持对 Codex 工作的掌控。

这种跨设备的无缝体验标志着 AI 辅助编程工具向更高灵活性和可用性方向发展。
