## 项目概述

**rkdebian** 是一个构建系统，专门为 **Doogee U10** Android平板电脑（采用Rockchip RK3562 SoC）生成完整可启动的Debian 12 Bookworm镜像。该项目从零开始逆向工程，无需BSP或厂商文档，利用AI工具（Claude、Codex、Antigravity）辅助开发，并以Firefly RK3562开源仓库为起点。

用户可以从SD卡启动Debian系统，移除SD卡后自动恢复运行原生Android系统，无需修改内部存储。

- 使用airockchip/rknn-llm运行时
- 目标平台: rk3562
- 量化方式: W8A8
- NPU核心数: 1

## 构建说明

### 构建依赖

```bash
sudo apt-get install \
  git make gcc-aarch64-linux-gnu \
  bc bison flex device-tree-compiler \
  genimage wget tar mtools \
  xz-utils \
  debootstrap qemu-user-static \
  e2fsprogs
```

### 构建命令

```bash
# 完整构建
./build.sh all

# 或带日志
set -o pipefail
./build.sh all 2>&1 | tee build.log
```

### 输出镜像

- `out/rk3562-debian.img.xz` - 压缩最终镜像（推荐）
- `output/update/update.img.xz` - Firefly兼容路径

## 目录结构

```
rkdebian/
├── build.sh              # 主构建入口点
├── build_rootfs.sh      # Debian rootfs构建器
├── genimage.cfg         # SD卡镜像分区布局
├── extlinux.conf        # 启动加载器配置

├── overlay/             # 自定义内核驱动、DTS、固件、服务和头文件
│   ├── arch/            # 设备树源文件
│   ├── drivers          # 内核外驱动
│   ├── firmware         # Wi-Fi固件
│   ├── include          # 内核头文件覆盖
│   ├── kernel-patches   # 内核补丁
│   ├── etc/             # 设备配置覆盖
│   └── *.sh/*.service   # 设备设置脚本和systemd单元
├── debs/                # 预构建的.deb包
├── mali/                # Mali GPU用户空间库
├── wifi/                # Wi-Fi固件和驱动
├── tools/               # 相机捕获和ISP诊断工具
├── docs/                # 设计规范和构建说明
├── src/                 # 克隆的源码
├── out/                 # 构建产物
└── output/update/       # 最终可刷入镜像
```

## 已知问题

1. 平板关机数小时后电池可能显示0%
   - 解决方案: `rk-battery-gauge-fix.service`可在启动时修复
   - 如果平板未完全关机，重启一次，下次启动电量应恢复正常

2. 前后摄像头预览/拍摄可用，但色彩略有偏差
   - 需要额外的ISP校准
