## 产品概述

Tilde 是一个用于安全运行 AI 智能体的平台，核心价值主张是"让 AI 智能体在生产环境中自由运行，而无需承担风险"。该平台将每一次智能体运行转化为可回滚的事务，提供版本化、可组合的文件系统，并确保所有出站调用都被检查和记录。

---

## 核心特性

### 1. 版本化可组合文件系统（Versioned Composable Filesystem）
- 提供真正的 POSIX 文件系统，支持任何工具和语言，无需 SDK
- 可将 GitHub 代码、S3 训练数据和 Google Drive 文档挂载为单一的 ~/sandbox
- 每个文件从第一次提交即开始版本化，任何智能体运行都可以即时回滚

### 2. 安全无服务器沙箱（Safe Serverless Sandboxes）
- 每次运行都是一个事务，在全新的隔离容器中执行
- 成功退出时，变更原子性提交；失败时，变更完全不影响现有系统
- 无需备份恢复、无需手动清理、无需管理基础设施

### 3. 网络隔离（Network Isolation）
- 默认阻止云元数据、私人网络和未授权主机
- 防止数据泄露、凭证滥用和提示注入呼叫
- 每个出站请求都基于策略进行检查和日志记录

### 4. 时间旅行与审计追踪（Time Travel & Audit Trail）
- 完整时间线浏览，支持检查差异和即时回滚任何提交
- 每次变更都绑定到产生该变更的人、进程或智能体

### 5. 智能体优先的 RBAC（Agent-first RBAC）
- 智能体作为一等公民，拥有自己的作用域权限
- 支持按智能体、仓库、动作进行允许、拒绝或需要人工批准的配置
- 使用简洁易读的 DSL 定义策略

## 使用方式

### CLI 使用
```bash
# 安装
curl -fsSL https://tilde.run/install | sh

# 在沙箱中运行智能体
tilde exec my-team/documents \
    --image python:3.12 \
    -- /sandbox/code/agent.py --input /sandbox/data/reports
```

### Python SDK 使用
```python
import tilde

repo = tilde.repository("my-team/documents")

with repo.shell(image="python:3.12") as sh:
    sh.run("pip install pandas")
    result = sh.run("python agent.py --input /sandbox/data")
```
