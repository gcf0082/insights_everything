## 产品概述

CrabTrap 是 Brex 开源的 LLM-as-a-judge HTTP 代理，旨在保护生产环境中的 AI 代理安全运行。

## 核心功能

### 1. 请求拦截
CrabTrap 作为 HTTP 代理运行，拦截 AI 代理发出的每一个请求，实时评估请求是否符合安全策略。

### 2. 策略评估
- 支持静态规则匹配
- 支持 LLM 判断（LLM-as-a-judge）
- 实时决策：允许或阻止请求

### 3. 快速部署
声称可在 30 秒内完成设置并开始使用。

## 技术特点

- **开源**: 可在 GitHub 上查看源码
- **实时保护**: 拦截并评估每一个请求
- **可视化决策**: 记录每个决策是静态规则匹配还是 LLM 判断结果
- **灵活配置**: 支持编辑和添加新规则

## 应用场景

CrabTrap 适用于需要保护 AI 代理安全的企业场景，确保 AI 代理在生产环境中按照预定策略执行操作，防止潜在的安全风险。
