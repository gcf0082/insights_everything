## 核心洞察摘要

### 1. 产品定位
Framework 正式发布 **Framework Laptop 13 Pro**，这是一款全新设计的可修复笔记本电脑，针对此前产品的多项不足进行了重大改进，并首次提供 Linux（Ubuntu）预装版本。

### 2. 主要升级亮点

| 升级项 | 具体改进 |
|--------|----------|
| **机身** | CNC 数控铝合金外壳（6000系列），更坚固，解决了此前铰链松动的问题 |
| **配色** | 新增 石墨黑（graphite）配色 |
| **重量** | 1.4kg（略重于前代的 1.3kg） |
| **电池** | 74Whr（提升 20%），续航可达 20 小时（4K Netflix 流媒体测试） |
| **处理器** | Intel Core Ultra Series 3（Panther Lake）或 AMD Ryzen AI 300 系列 |
| **显示** | 完全定制 IPS 屏幕，700 nits 亮度，1800:1 对比度，30-120Hz 可变刷新率，支持触控 |
| **内存** | LPCAMM2 可升级内存模块（16GB/32GB/64GB），速度最高 7467 MT/s |
| **扬声器** | 侧置扬声器，支持 Dolby Atmos |
| **触控板** | 新款 haptic 触控板，使用四个压电元件模拟点击 |
| **充电** | 100W GaN 充电器（升级自 60W） |
| **连接** | PCIe 5.0，Wi-Fi 7 |
| **显卡** | Intel B390 集成显卡（Arc B390/B370） |

### 3. 价格与上市

| 版本 | 价格 |
|------|------|
| DIY 版（Core Ultra 5，无 RAM/SSD/OS）| $1,199 |
| Ubuntu 预装版（已认证）| $1,499 |
| Core Ultra X7 配置| $1,599 |
| Core Ultra X9 "halo" 限量版| $1,799 |

**上市时间**：2026年6月开始发货，现已开放预购。

### 4. 重要亮点：Linux 支持

- 这是 Framework **首次**提供 Linux 预装笔记本
- 采用 **Ubuntu** 系统，并获得 Ubuntu 认证
- Ubuntu 版本销量超过 Windows 版本（超预期）

### 5.向后兼容性

- 新款主板可安装在旧款 Laptop 13 机身上
- 旧款主板也可安装到新 Pro 机身上
- 提供 "Bottom Cover Upgrade Kits" 升级套件
- 键盘/触控板/电池 trio 因新外壳 geometry 不兼容

## 洞察结论

Framework Laptop 13 Pro 是一次重大的进化，解决了此前产品的多个痛点（电池续航、屏幕质量、扬声器、铰链强度），同时保持了可升级维修的核心理念。首次提供 Ubuntu 预装版本标志着 Framework 对 Linux 社区的正式拥抱，销量超预期也验证了市场对此的积极响应。
