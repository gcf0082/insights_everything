## 什么是定时任务

定时任务（Scheduled Task）是一种使用 Anthropic 管理的基础设施自动运行提示词的重复性工作。即使关闭电脑，任务也会持续运行。

### 可自动化的示例工作

- 每天早上审查开放的拉取请求
- 夜间分析 CI 失败并汇总摘要
- PR 合并后同步文档
- 每周运行依赖审计

## 三种调度方式对比

|                      | Web 云端任务     | Desktop 桌面任务   | /loop CLI 循环任务 |
| :------------------- | :--------------- | :----------------- | :----------------- |
| 运行位置             | Anthropic 云端   | 你的机器           | 你的机器           |
| 需要机器开机         | 否               | 是                 | 是                 |
| 需要打开会话         | 否               | 否                 | 是                 |
| 重启后持久化         | 是               | 是                 | 否（会话作用域）   |
| 访问本地文件         | 否（全新克隆）   | 是                 | 是                 |
| MCP 服务器           | 每任务单独配置   | 配置文件和连接器   | 继承自会话         |
| 权限提示             | 否（自主运行）   | 每任务可配置       | 继承自会话         |
| 自定义调度           | 通过 CLI /schedule | 是                 | 是                 |
| 最小间隔             | 1 小时           | 1 分钟             | 1 分钟             |

**建议**: 需要可靠运行且无需机器时使用云端任务；需要访问本地文件和工具时使用桌面任务；会话期间快速轮询时使用 /loop。

## 创建定时任务

可以通过三种方式创建：
1. **Web**: 访问 claude.ai/code/scheduled，点击 "New scheduled task"
2. **Desktop**: 打开 Schedule 页面，点击 "New task"，选择 "New remote task"
3. **CLI**: 在任意会话中运行 /schedule

### Web 创建步骤

1. **打开创建表单**: 访问 claude.ai/code/scheduled，点击 "New scheduled task"
2. **命名并编写提示词**: 给任务起一个描述性名称，编写每次运行的提示词。提示词必须自包含，明确说明要做什么以及成功的样子
3. **选择仓库**: 添加一个或多个 GitHub 仓库。每次运行时仓库都会被克隆，默认从默认分支开始。Claude 会创建以 claude/ 为前缀的分支来存放更改
4. **选择环境**: 选择云环境，控制网络访问、环境变量和安装脚本。默认环境开箱即用
5. **选择调度频率**: 从预设选项中选择运行频率，默认每天本地时间 9:00 AM
6. **配置连接器**: 所有已连接的 MCP 连接器默认包含，移除不需要的
7. **创建任务**: 点击创建，任务出现在列表中，下次计划时间自动运行

### 频率选项

| 频率     | 描述                                              |
| :------- | :------------------------------------------------ |
| Hourly   | 每小时运行一次                                    |
| Daily    | 每天运行一次，默认 9:00 AM 本地时间               |
| Weekdays | 与 Daily 相同，但跳过周六日                      |
| Weekly   | 每周运行一次，在指定日期和时间                    |

### 仓库和分支权限

- 默认情况下，Claude 只能推送到以 claude/ 为前缀的分支
- 如需移除此限制，可启用 "Allow unrestricted branch pushes"

### 连接器

定时任务可以使用 MCP 连接器读写外部服务（如 Slack、Linear、Google Drive）。所有当前连接的连接器默认包含，可根据需要移除或添加。

### 环境

每个任务运行在云环境中，控制网络访问、环境变量和安装脚本。可配置环境以提供 API 密钥、安装依赖或限制网络范围。

## 管理定时任务

在 Scheduled 列表中点击任务可打开详情页，显示仓库、连接器、提示词、调度计划和历史运行记录。

### 查看和交互运行

点击任意运行可将其作为完整会话打开，可以看到 Claude 做了什么、审查更改、创建拉取请求或继续对话。

### 编辑和控制任务

从任务详情页可以：
- 点击 "Run now" 立即开始运行
- 使用切换按钮暂停或恢复调度
- 点击编辑图标更改名称、提示词、调度、仓库、环境或连接器
- 点击删除图标移除任务（任务创建的会话仍保留在会话列表中）

也可通过 CLI 管理：/schedule list 查看所有任务，/schedule update 更改任务，/schedule run 立即触发运行。
