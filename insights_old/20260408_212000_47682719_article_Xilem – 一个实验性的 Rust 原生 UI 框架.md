## 项目概述

Xilem 是一个实验性的 Rust 原生 UI 框架，采用响应式架构设计。它与 Masonry 共同构成了一个用于在 Rust 中编写 GUI 应用程序的高级架构体系。

### 核心组件

**Masonry** 是构建 Rust 原生 GUI 的基础 crate，提供保留型小部件树（retained widget tree），并在树上运行事件处理和更新传递。

**Xilem** 是一个高级响应式框架，灵感来自 React、SwiftUI 和 Elm。它允许用户创建轻量级视图树，并根据树的更改渲染应用程序。Xilem 拥有 Web 后端和 Masonry 后端。

### 技术栈

Xilem 和 Masonry 基于以下技术构建：

- **winit**: 用于窗口创建
- **Vello 和 wgpu**: 用于 2D 图形渲染
- **Parley 和 Fontique**: 用于文本排版
- **AccessKit**: 用于访问性 API 集成

## 特性与优势

1. **响应式设计**: 采用类似 React 和 SwiftUI 的响应式编程模型
2. **跨平台支持**: 支持桌面（通过 Masonry 后端）和 Web（通过 xilem_web）
3. **原生性能**: 作为 Rust 原生框架，提供高性能的原生体验
4. **现代化架构**: 借鉴 Elm 架构的 unidirectional 数据流
5. **完整的文本堆栈**: 集成 Parley 和 Fontique 处理复杂文本渲染
6. **无障碍支持**: 通过 AccessKit 支持 accessibility API

## 入门指南

### 运行示例

```bash
cargo run --example to_do_mvc
```

### 添加依赖

```bash
cargo add xilem
```

### Linux/BSD 环境依赖

需要安装 `pkg-config`、`clang`、以及 `wayland`、`libxkbcommon`、`libxcb`、`vulkan-loader` 的开发包。

**Fedora**:
```bash
sudo dnf install clang wayland-devel libxkbcommon-x11-devel libxcb-devel vulkan-loader-devel
```

**Debian/Ubuntu**:
```bash
sudo apt-get install clang libwayland-dev libxkbcommon-x11-dev libvulkan-dev
```
