## 详细内容

### CodexBar 介绍

CodexBar是一款macOS菜单栏应用，名为"Every AI coding limit in your menu bar"（每个AI编程限制尽在菜单栏），可追踪用户实际付费的AI编程服务的使用窗口、积分余额和重置倒计时。

### 主要功能

1. **30+提供商支持**：支持Codex、Claude、Cursor、OpenCode、Gemini、Copilot、DeepSeek等30多个AI编程服务
2. **使用窗口追踪**：提供每个提供商的会话、每周和每月使用窗口及重置倒计时
3. **积分与成本扫描**：显示积分余额和月度消费，并支持本地30天成本扫描（Codex和Claude）
4. **状态与事故监控**：提供商状态轮询在菜单中显示事故徽章
5. **CLI与小组件**：提供WidgetKit小组件和捆绑的codexbar CLI

### 技术特点

- 隐私保护：重复使用现有提供商会话（OAuth、设备流、API密钥、浏览器Cookie、本地文件），不存储密码
- macOS 14+通用应用
- 支持Homebrew安装
- 开源且免费

### 安装方式

- GitHub Releases下载
- Homebrew: `brew install --cask steipete/tap/codexbar`
