**标题:** Explore Wikipedia Like a Windows XP Desktop
**发布时间:** 5小时前
**得分:** 275 points
**评论数:** 66 comments

---

## 项目简介

这是一个将Wikipedia以Windows XP桌面风格呈现的网页应用，用户可以通过文件夹层级浏览 Wikipedia 的内容，仿佛在使用经典的 Windows XP 操作系统。

---

## 主要讨论观点

### 1. 怀旧与情怀
- 许多用户表示这勾起了对 Microsoft Network (MSN) 和 Windows 95/98 时代的回忆
- 有人提到这类似 Microsoft Encarta '97 的浏览体验
- 被形容为"真正的面向对象Web"的早期愿景

### 2. 界面设计赞赏
- 用户认为这种层级分类展示方式很好地映射了人脑组织知识的心理模型
- 大滚动条、可调整大小的窗口被认为是"被遗忘的艺术"
- 加载速度快，操作流畅

### 3. 知识组织方式的思考
- 有评论指出知识并不能完全对齐于嵌套层级结构
- 语言是传达知识的不完美媒介，人们以主观和高度个性化的方式存储知识
- 建议采用非结构化标签系统，而非严格层级分类

### 4. 改进建议
- 缺少搜索功能
- 包含省略号的文件夹名称点击后无法显示完整名称
- 建议添加开始菜单、Solitaire和Minesweeper游戏
- 有人希望可以切换成其他桌面模式（如Unix风格、书籍风格）

### 5. 技术实现
- 使用 XP.css 库作为基础
- 采用纯 JavaScript 实现
- 层级分类数据来源于 Wikipedia 的 Category 命名空间
- 为避免版权问题，图标和设计细节与真正的 Windows XP 略有不同

---

## 总结

这个项目以其独特的怀旧设计和创新的信息浏览方式获得了广泛好评。它不仅让Wikipedia的层级分类结构变得可视化，也让人们重新思考信息组织和呈现的方式。许多用户认为这种界面比传统搜索引擎更能帮助深入研究和学习知识领域。
