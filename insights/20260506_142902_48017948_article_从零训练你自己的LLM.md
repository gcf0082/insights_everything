## 项目概述

这是一个实践性的 workshops，旨在让用户从零开始编写完整的 GPT 训练流水线，深入理解每个组件的作用和原理。项目受到 Andrej Karpathy 的 nanoGPT 启发，将复杂的 GPT 模型简化到约 10M 参数规模，能够在笔记本电脑上于一小时内训练完成，非常适合在单次 workshops 中完成。

## 核心特性

### 训练目标
构建一个能在 MacBook 上运行并生成类似莎士比亚风格文本的 GPT 模型。

### 实现的组件
1. **Tokenizer（分词器）** - 将文本转换为模型可处理的数字
2. **Model architecture（模型架构）** - Transformer：embeddings、attention、feed-forward layers
3. **Training loop（训练循环）** - forward pass、loss、backprop、optimizer、learning rate scheduling
4. **Text generation（文本生成）** - 从训练好的模型采样生成文本

### 技术规格
- **训练设备**：自动支持 Apple Silicon GPU (MPS)、NVIDIA GPU (CUDA) 或 CPU
- **Python 版本**：3.12+
- **可选运行**：Google Colab

## 模型配置

| 配置 | 参数量 | n_layer | n_head | n_embd | 训练时间 (M3 Pro) |
|------|--------|---------|--------|--------|-------------------|
| Tiny | ~0.5M | 2 | 2 | 128 | ~5 min |
| Small | ~4M | 4 | 4 | 256 | ~20 min |
| **Medium（默认）** | **~10M** | **6** | **6** | **384** | **~45 min** |

所有配置使用字符级分词（vocab_size=65）和 block_size=256。

## 技术特点

### 分词策略
本项目使用**字符级分词**处理莎士比亚文本。理由是 BPE 分词（GPT-2 的 50k 词汇表）不适用于小数据集，因为大多数 token bigrams 过于罕见，模型难以从中学习模式。

### 架构流程

```
Input Text
    │
    ▼
┌─────────────────┐
│   Tokenizer     │  "hello" → [20, 43, 50, 50, 53]  (character-level)
└────────┬────────┘
         ▼
┌─────────────────┐
│  Token Embed +  │  token IDs → vectors (n_embd dimensions)
│  Position Embed │  + positional information
└────────┬────────┘
         ▼
┌─────────────────┐
│  Transformer    │  × n_layer
│  Block:         │
│  ┌────────────┐ │
│  │ LayerNorm  │ │
│  │ Self-Attn  │ │  n_head parallel attention heads
│  │ + Residual │ │
│  ├────────────┤ │
│  │ LayerNorm  │ │
│  │ MLP (FFN)  │ │  expand 4x, GELU, project back
│  │ + Residual │ │
│  └────────────┘ │
└────────┬────────┘
         ▼
┌─────────────────┐
│   LayerNorm     │
│   Linear → logits│  vocab_size outputs (probability over next token)
└─────────────────┘
```

## 总结

这是一个优秀的入门级 LLM 训练教程项目，适合想要深入理解 GPT 架构和训练流程的开发者。通过从零实现分词器、Transformer 模型、训练循环和文本生成模块，用户可以建立起对大语言模型核心原理的扎实理解。项目设计巧妙地将复杂概念分解为 6 个逐步递进的部分，配合约 10M 参数的中等规模模型，能在个人设备上快速完成训练，提供即时的实践反馈和成就感。
