## 内容摘要

Project Gutenberg的程序员JSeiko宣布网站在过去几个月进行了大量改进，并透露了更多更新即将来临。

## 核心要点

### 1. 网站改进进展
- **首页设计更新**：正在迭代中，用户反馈移动端存在滚动条问题
- **书籍详情页设计**：预计在未来1-2周上线
- **PDF格式支持**：计划今年内推出
- **OPDS 2.0支持**：即将推出，可通过在URL末尾添加.opds访问旧版本

### 2. 用户反馈的问题
| 问题 | 状态 |
|------|------|
| 移动端页面滚动条导致最后一行被截断 | 已收到反馈 |
| Android Chrome菜单无法关闭 | 团队成员将在本周末修复 |
| OCR错误处理 | 建议使用Distributed Proofreaders (pgdp.net) |
| 插图质量依赖扫描来源 | 依赖于Internet Archive、Hathitrust等来源 |

### 3. 项目历史背景
- 1971年由Michael Hart在伊利诺伊大学创立
- 最初数字化的是《美国独立宣言》
- 是ARPANET 15个节点之一

### 4. 社区建议与讨论

**关于电子书格式：**
- EPUB3已成为主流格式
- Plain Text版本仍然重要，便于辅助技术和无障碍阅读
- PDF支持正在开发中

**与商业平台的关系：**
- 讨论了Amazon Kindle和Barnes & Noble销售公版书的问题
- 建议商业平台从PG来源并资助项目
- 开源电子书阅读器的可能性被讨论

**相关资源：**
- Standard Ebooks: 使用PG作为来源但格式更精美
- Faded Page (加拿大)
- Runeberg (瑞典)
- Project Gutenberg ZIM文件可下载整个英语语料库(约60GB)

### 5. 团队信息
- JSeiko是贡献者，非全职员工
- gluejar (gluejar)是项目新任执行董事
- 代码仓库: https://github.com/gutenbergtools

## 有趣发现

1. **下载量异常**：前一天下载量最高的书籍是《混凝土施工：方法与成本》(23644次)，超过《白鲸》、《基督山伯爵》等热门书籍，怀疑是机器人流量。

2. **德国屏蔽问题**：PG曾长期在德国被地理封锁，现已解决。

3. **无障碍设计**：网站支持禁用样式，仍保持良好可访问性。

## 总结

Project Gutenberg作为互联网最古老的公益项目之一，正在持续现代化。团队积极回应用户反馈，计划在年内推出PDF支持并改进设计。社区对该项目充满敬意，认为它是"现代互联网设计哲学的对立面——安静、实用、易于访问、经久耐用"。
