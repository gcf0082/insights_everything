## 内容摘要

**Turbo Vision 2.0** 是一个经典的 TUI（终端用户界面）库的现代移植版本。Turbo Vision 最初随 Turbo Pascal 6 一起发布，后来有了 C++ 移植版本。

### 社区反馈亮点

1. **跨平台 wrapper 开发**
   - 有开发者正在为该仓库编写一个 wrapper，实现在 .Net 环境下（macOS 上）运行 Turbo Vision
   - 这位开发者表示这是"一种神奇的感觉"

2. **API 改进**
   - Wrapper 提供了更高级别的 API
   - 解决了原有"相当古老的调色板 API"问题
   - 正在添加布局功能

3. **开发中的特性**
   - 基于现代终端表面的调色板设计
   - 布局清理
   - 添加更多基本控件

4. **开发者背景**
   - 该开发者之前曾尝试 Terminal.Gui 库，但发现该库在 v2 过渡期间存在很多 bug
   - 认为 Claude 是一个关于 TUI 和库的很好教训——很多"不应该怎么做"的例子
   - 很高兴找到了这个现代化的 Turbo Vision 移植版本，并且已经支持 Unicode

---

## 技术要点

- **原始版本**: Turbo Pascal 6
- **C++ 移植版本**: 后续推出
- **现代更新**: 已支持 Unicode
- **项目类型**: TUI（终端用户界面）库

## 总结

Turbo Vision 2.0 的现代移植受到社区的积极响应。开发者们正在为其添加现代化特性，包括更好的调色板 API 和布局系统。有人正在构建跨平台 wrapper，使其能够在 .Net 环境下运行，这表明经典的 TUI 库在现代开发环境中仍具有生命力。
