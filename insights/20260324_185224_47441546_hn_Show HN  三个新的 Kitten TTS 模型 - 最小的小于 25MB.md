## 项目概述

**Kitten TTS** 是一个开源的超小型且富有表现力的文本转语音（TTS）模型系列，专为端侧应用设计。

### 发布内容

本次发布三个新模型：

- **80M 参数模型**：最高质量
- **40M 参数模型**：中等规格
- **14M 参数模型**：小于 25MB，在同类规模模型中达到新的表达力 SOTA

### 主要特性

- 支持英语文本转语音，包含 8 个语音（4 男 4 女）
- 模型量化为 int8 + fp16
- 使用 ONNX 运行时
- 可在 Raspberry Pi、低端智能手机、可穿戴设备、浏览器等设备运行
- **无需 GPU**
- 多语言模型即将推出

### 发展目标

端侧 AI 的瓶颈在于缺乏真正可用的超小型模型。团队目标是开源更多模型，使生产级语音代理和应用完全可以在端侧运行。

## 社区反馈摘要

### 积极反馈

- 模型质量在同等规模下表现出色
- 开发者响应迅速，积极修复问题
- 在 CPU 上运行效果令人印象深刻（Intel 9700 CPU 上约 1.5 倍实时）
- Bruno 语音获得较高评价

### 问题与改进建议

| 问题 | 详情 |
|------|------|
| **依赖问题** | 安装时拉取约 7GB 依赖（主要是 torch 和 NVIDIA CUDA 库），与"超小型"定位不符 |
| **语音风格** | 当前语音偏向卡通风格，用户希望添加更专业的语音 |
| **数字发音** | 数字和单位发音存在问题，需添加文本预处理 |
| **Python 环境** | 在 Python 3.14 上存在 spacy 相关 bug |

### 功能建议

- 添加更多专业语音
- 支持语音克隆（预计 5 月推出）
- 支持日语模型（预计 3 周内推出）
- 添加语调/重音控制
- 添加表达性标签支持（如 [sarcastically]、[laughs]、[pause]）

---

## 总结

Kitten TTS 是一个有潜力的端侧 TTS 解决方案，其 14M 参数模型在不到 25MB 的情况下达到了优秀的表达力。主要优势在于无需 GPU 即可运行，适合资源受限的设备。主要改进方向包括：减少安装依赖、优化语音专业度、添加多语言支持。该项目开发者非常活跃，社区反馈良好。
