- **作者联系方式**：bear@typewritten.org

## 内容概述

这是一个珍贵的计算技术历史档案，收录了从1983年至2007年间的各种经典操作系统和图形用户界面的截图。这些截图展示了个人电脑和工作站图形界面的演进历程，是了解早期计算机界面设计的重要资源。

## 主要内容分类

### 一、操作系统截图（1983-1999年）

#### 早期图形界面先驱（1983-1985年）

- **VisiCorp Visi On**（1983年）：最早的IBM PC图形界面之一
- **SunTools**（1984-1985年）：Sun工作站的桌面环境
- **HP Integral PC**（1985年）：HP-UX 5.0/RO系统

#### 桌面发布系统（1985-1987年）

- **GEM Desktop 1.2**（1985年）：Digital Research的图形环境，因苹果诉DRI"外观与感觉"案而成为历史
- **GEM Draw 1.0**（1985年）：早期绘图应用之一
- **Xerox Ventura Publisher 1.1**（1987年）：首批桌面出版软件之一

#### 工作站系统（1984-1990年）

- **DEC VAXstation Software（VWS）**（1987年）：VMS系统的图形界面
- **IRIS GL2**（1987年）：SGI IRIS工作站环境
- **SunView**（1988年）：Sun早期的窗口系统
- **NeWS**（1988年）：Sun的网络可扩展窗口系统
- **DEC UWS**（1988年）：ULTRIX工作系统软件

#### 苹果与Acorn系统（1987-1992年）

- **Arthur**（1987年）：Acorn Archimedes的早期桌面系统
- **RISC OS**（1988-1992年）：Acorn的后续操作系统
- **RISC iX**（1990年）：Acorn的Unix变体
- **A/UX**（1990-1993年）：苹果的Unix系统

#### PC操作系统（1988-1993年）

- **OS/2 Presentation Manager**（1988-1993年）：IBM与微软合作的操作系统的图形界面
- **Windows/286** 和 **Windows/386**（1988-1989年）：Windows 2.x版本
- **HP NewWave Office**（1988-1990年）：面向对象的Windows桌面环境
- **SCO OpenDesktop**（1989-1993年）：Unix桌面环境

#### IBM和Sun系统（1988-1994年）

- **AIX**（1988-1994年）：IBM的Unix系统，包含AIXwindows和CDE
- **Solaris/OpenWindows**（1989-1996年）：Sun的操作系统
- **IBM Academic Operating System（AOS）**（1990年）

#### 新兴操作系统（1990-1995年）

- **NeXTstep**（1989-1993年）：NeXT公司的操作系统，后来成为Mac OS X的基础
- **TI microExplorer**（1990年）：基于Lisp的工作站系统
- **Symbolics Genera**（1993年）：Symbolics Lisp机的操作系统
- **BeOS**（1995-1996年）：BeBox的操作系统
- **Copland**（1995-1996年）：苹果未能发布的Mac OS后续版本

#### Unix和工作站系统（1991-1995年）

- **Amiga System V Release 4**（1991年）
- **Apollo Desktop Environment**（1991年）
- **Geoworks Ensemble**（1991年）
- **OS-9/68k G-Windows**（1991年）
- **HP VUE**（1992年）：后来启发了CDE
- **Perihelion Helios**（1993年）：基于Transputer的工作站系统
- **Intergraph Environ V**（1995年）

#### Linux和早期开源系统（1994年）

- **Linux fvwm**（1994年）：早期Linux桌面环境，使用ATI Mach32显卡
- **Visix Looking Glass**（1994年）：基于X11的3D桌面环境

### 二、macOS演进（1997-2001年）

- **Rhapsody**（1997-1998年）：苹果收购NeXT后的过渡系统
- **Mac OS X Server 1.0**（1999年）
- **Mac OS X Developer Preview**（1999-2000年）：Aqua界面的早期版本
- **Mac OS X 10.0 "Cheetah"**（2001年）：现代macOS的起点
- **Mac OS X 10.1 "Puma"**（2001年）：首个重大更新版本

### 三、策展人的桌面演进（1996-2007年）

该部分展示了网站维护者个人使用过的桌面环境变迁：

- **1996年**：OS/2 Warp Connect 3.0
- **1997年**：Windows NT Workstation 4.0
- **1998年**：IRIX 6.5.3 和 NEXTSTEP 3.3
- **2003年**：Mac OS X 10.3
- **2005年**：Mac OS X 10.3.9

## 技术亮点

### 显示技术演变

- 从320×200的CGA分辨率到1600×1200的高分辨率
- 从16色到256色再到真彩色的色彩深度演进
- 显示适配器包括：CGA、EGA、VGA、8514/A、TIGA、Mach32等

### 窗口管理系统

- 从简单的平铺窗口到重叠窗口
- 出现了dock、任务栏等现代UI元素的雏形
- Motif、CDE等经典Unix桌面环境的演变

### 重要历史事件

- GEM因苹果诉DRI案而衰落（1988年）
- NeXTstep技术成为Mac OS X的基础（1997年）
- 从Rhapsody到Aqua的演变（1999-2000年）

## 结论

该网站是计算历史爱好者的宝贵资源，完整记录了1983年至2007年间图形用户界面的演进历程。通过这些截图，可以清晰地看到现代操作系统UI设计的源头和发展脉络，涵盖了从早期的字符界面到现代桌面系统的完整技术演进。
