## 项目概述

cell 是一个用 Rust 编写的终端电子表格编辑器，支持 Vim 键绑定。它是一个现代的、内存安全的电子表格工具，可以在终端中编辑 CSV、TSV 和原生 .cell 格式的文件。

## 核心功能

### 1. Vim 键绑定
- 完全支持 Vim 模式：Normal、Insert、Visual、Command
- 支持所有常用 motions 和 operators
- 支持计数前缀（如 `5j`、`10G`、`3dd`、`4yy`）
- 支持 marks 功能

### 2. 公式支持
- Excel 兼容语法：`=SUM(A1:A10)`、`=IF(A1>100, "high", "low")`
- 内置函数：SUM、AVERAGE、COUNT、MIN、MAX、IF
- 支持单元格引用和范围

### 3. 文件格式
- **CSV/TSV**：标准逗号/制表符分隔文件
- **.cell**：原生格式，保留公式，纯文本人类可读

### 4. 无头模式 (Headless CLI)
```bash
cell sales.cell --read A1              # 读取单个单元格
cell sales.cell --read B1:B10          # 读取范围
cell sales.cell --eval '=SUM(B1:B10)' # 计算公式
cell sales.cell --write A1 42          # 写入单元格
```

### 5. 架构设计
```
cell/
  crates/
    cell-sheet-core/    # 数据模型、公式引擎、文件I/O（无TUI依赖）
    cell-sheet-tui/     # Ratatui渲染、Vim模式、事件循环
```

## 与 sc-im 的对比

| 特性 | cell | sc-im |
|------|------|-------|
| 语言 | Rust + ratatui | C + ncurses |
| 编辑模型 | 真正 Vim 模态编辑 | Vim 灵感导航 |
| 公式语法 | Excel 兼容 (`=SUM`) | @ 前缀风格 (`@sum`) |
| 函数数量 | 较少 | 丰富 |
| 文件格式 | CSV, TSV, .cell | CSV, TSV, XLSX, Markdown |
| 格式化 | 尚不支持 | 支持粗体、斜体、颜色 |
| 脚本 | Headless CLI | Lua 脚本 |
| 图表 | 尚不支持 | GNUPlot 集成 |
| Windows | ✓ 支持 | 有限 |

## 选择 cell 的理由

- 你想要真正像 Vim 一样工作的编辑体验
- 你偏好 Excel 兼容的公式语法
- 你需要一个在 Windows 上开箱即用的二进制文件
- 你重视现代的、内存安全的代码库

## 技术栈

- **编程语言**：Rust
- **TUI 框架**：ratatui
- **发布方式**：crates.io、GitHub Releases

## 总结

cell 是一个有前景的终端电子表格编辑器，特别适合习惯 Vim 操作方式的用户。虽然功能相比成熟的 sc-im 还有差距，但其现代化的代码架构、Excel 兼容的公式语法和跨平台支持使其成为一个值得关注的替代选择。项目目前有 212 个星标和 6 个分支，社区参与度适中。
