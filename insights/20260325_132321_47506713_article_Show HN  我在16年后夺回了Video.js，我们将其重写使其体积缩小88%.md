# Video.js v10 Beta 洞察报告

## 基本信息

- **洞察链接**: https://videojs.org/blog/videojs-v10-beta-hello-world-again
- **发布日期**: 2026年3月10日
- **作者**: Steve Heffernan
- **标签**: Video.js, 开源, 视频播放器, Web开发

## 核心内容总结

### 1. 项目背景

Video.js v10 是一次彻底的重写，耗时约一年半。不仅是 Video.js 本身的重写，还包括 Plyr、Vidstack 和 Media Chrome 的协作开发，这些项目合计拥有 75,000 GitHub stars 和数十亿次月度视频播放量。

### 2. 主要改进

#### 2.1 体积大幅缩小
- 默认播放器体积减少 **88%**
- 移除自适应比特率（ABR）支持后，v10 默认播放器仍比 v8 小 **66%**
- 简单的 HLS 用例中，v10 使用 SPF 引擎仅是 v8 体积的 **19%**

#### 2.2 现代化的开发体验
- 新增一流（first-class）的 React、TypeScript 和 Tailwind 支持
- 支持深层次定制，采用熟悉的开发模式
- 新的 HTML 组件式 API：`video-player`、`video-skin`、`video` 等

#### 2.3 预设功能（Presets）
- **Video Preset**: 通用网站视频播放器
- **Audio Preset**: 播客/音频播放器
- **Background Video Preset**: 背景视频播放器（无控件、无音频）

#### 2.4 UI 定制改进
- 提供两个主题：默认主题（磨砂美感）和最简主题
- 使用无样式 UI 原语，便于深度定制
- 支持"弹出"（eject）获取完整源码

### 3. 新引擎 SPF（Streaming Processor Framework）

- 函数式组件组合的流处理框架
- 根据需求构建精简的流媒体引擎
- 简单 HLS 用例体积仅 12.1kB gzip，比 HLS.js-light 还小 **88%**

### 4. AI 集成

- 提供 llms.txt 优化 AI 导航
- 支持 Markdown 格式文档
- 构建 AI 技能集帮助开发

### 5. 发布计划

- **目标**: 2026年年中发布正式版
- **待完成**: 功能 parity、广告支持、迁移指南、更多预设

## 结论

Video.js v10 是一个重大升级，通过模块化架构实现了显著的体积缩小和现代化开发体验。它不仅解决了传统视频播放器的体积问题，还为 AI 辅助开发做好了准备。
