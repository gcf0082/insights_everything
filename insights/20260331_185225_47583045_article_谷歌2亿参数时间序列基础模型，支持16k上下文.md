## 项目概述

TimesFM 是 Google Research 开发的预训练时间序列基础模型，用于时间序列预测。该模型采用纯解码器（decoder-only）架构，能够在无需针对特定数据集进行训练的情况下，对各种时间序列进行预测。

## 主要特性

### TimesFM 2.5 新特性（2025年9月发布）

- **参数规模**: 2亿参数（相比2.0版本的5亿参数有所减少）
- **上下文长度**: 支持最高16,000个时间点的上下文（相比2.0的2,048大幅提升）
- **分位数预测**: 可选30亿参数的分位数头，支持最长1,000步的连续分位数预测
- **频率指示器**: 移除了频率指示器要求
- **新预测标志**: 增加多个新的预测配置选项

### 早前版本

- 版本1.0和2.0的代码已归档在 `v1` 子目录中
- 可通过 `pip install timesfm==1.3.0` 安装旧版本

## 技术架构

- **模型类型**: 纯解码器Transformer架构
- **预训练数据**: 包含来自Google内部的海量时间序列数据
- **推理后端**: 支持PyTorch和Flax两种后端

## 部署与使用

### 安装方式

```bash
git clone https://github.com/google-research/timesfm.git
cd timesfm
uv venv
source .venv/bin/activate
uv pip install -e .[torch]  # 或 [flax] 或 [xreg]
```

### 代码示例

```python
import torch
import numpy as np
import timesfm

model = timesfm.TimesFM_2p5_200M_torch.from_pretrained("google/timesfm-2.5-200m-pytorch")

point_forecast, quantile_forecast = model.forecast(
    horizon=12,
    inputs=[np.linspace(0, 1, 100), np.sin(np.linspace(0, 20, 67))],
)
```

## 总结

TimesFM代表了时间序列预测领域的重要进展，通过大规模预训练和Transformer架构，实现了跨领域的时间序列预测能力。2.5版本的发布进一步提升了模型的实用性和性能，降低了计算成本的同时扩展了上下文处理能力。
