## 一、讨论概述

本次讨论的核心主题是 DeepSeek v4 模型的发布及其性能表现。讨论涉及多个维度，包括模型的定价、与 Claude Opus 4.6 的性能对比、可用性、技术细节，以及围绕开源的哲学辩论。

定价方面，DeepSeek v4 Pro 版本每百万输出令牌仅需 3.48 美元，远低于同类性能的竞争产品，引发了关于 AI 模型定价逻辑的讨论。

## 二、定价信息

**Pro 版本**：
- OpenRouter：$1.74/百万输入，$3.48/百万输出
- 官方 API：类似定价

**Flash 版本**：
- $0.14/百万输入，$0.28/百万输出
- 缓存价格：$0.028/百万

**未来价格走势**：DeepSeek 声称下半年 Ascend 950 计算卡发布后，Pro 版本价格将显著下降。

---

## 三、性能对比

**与 Claude Opus 4.6 对比**：
- 根据 DeepSeek 官方公告：用户体验优于 Sonnet 4.5，交付质量接近 Opus 4.6 非思考模式，但与 Opus 4.6 思考模式仍有差距
- 用户实际测试反馈：略优于不启用思考功能的 Opus 4.6，但不如启用思考功能的 Opus 4.6

**基准测试数据**：
- 在 20 个指标上平均提升约 2 个百分点
- Claude 4.6 在长上下文问答和多次对话轮次中表现好 10 个百分点
- DS v4 在数学挑战 (IMOAnswerBench) 上好 14 个百分点
- DS v4 在基础问答 (SimpleQA-Verified) 上好 12 个百分点

**与 GPT 系列对比**：
- GPT 5.4 便宜但黑盒、慢
- GPT 模型是 Claude 模型在结对编程任务中的糟糕替代品

## 四、可用性和技术细节

**访问渠道**：
- OpenRouter：deepseek-v4-pro 和 deepseek-v4-flash
- DeepSeek 官方 API
- Hugging Face：发布模型权重

**技术规格**：
- 6710 亿参数（671B），370 亿活跃参数（37B active）
- 混合专家架构（MoE）
- 首次在 Hugging Face 上发布 1.6T Pro 基础模型

**自托管 GPU 需求**：
- 全精度运行：16-24 个 H100（预付 $400,000-600,000）
- 重量化（16位→8位）：8 个 H200
- 消费级：4 张 4090（约 $15,000）

**已知问题**：
- 通过 OpenRouter 在 Pi Coding agent 中测试时，无法正确使用读写工具
- 部分用户遇到 API 错误
- Anthropic 兼容端点尚未支持 V4

## 五、开源相关讨论

**核心争议**：
- DeepSeek 是否真正"开源"？还是仅"开源权重"(open weights)？
- 训练数据和训练脚本未发布

**社区观点**：
- "权重是源代码，训练数据是编译器"
- DeepSeek 以发布大量技术资源（代码、论文、33个 GitHub 仓库）而获得尊重
- 被认为是"世界上最开放的 AI 公司"

**实际价值**：
- 可下载、在本地系统运行、可微调
- 永远可用，不会有被" rug pulled"的风险

---

## 六、主要观点总结

1. **定价革命**：Pro 版本 $1.74/M 输入、$3.48/M 输出，被多次形容为"疯狂"，是 AI 模型 Commoditization 的里程碑

2. **性能定位**：达到前沿水平，略逊于 Opus 4.6 思考模式，优于非思考模式

3. **可用性**：已在 OpenRouter、DeepSeek 官方 API 和 Hugging Face 可用

4. **自托管门槛**：需要相当数量的 GPU 投资（$400,000-600,000），但技术上可行

5. **开源意义**：虽有争议，但 DeepSeek 的开放做法获得社区尊重
