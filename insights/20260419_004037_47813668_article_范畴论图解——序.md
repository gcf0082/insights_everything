## 内容概要

本文档总结了范畴理论图解中关于"序论"（Orders）的章节。序论是数学中描述对象之间顺序关系的理论，是范畴论的重要组成部分。

---

## 1. 序论的基本定义

### 什么是序

序（Order）是一种数学构造，由两个组件构成：

- **一个集合的元素**
- **元素之间的二元关系**（通常用箭头或 ≤ 表示）

这种二元关系必须服从某些特定定律，才能构成有效的序。

### 序与范畴的关系

序本质上是一种特殊的范畴：

- 每个对象至少有一条恒等态射（自反性）
- 如果存在 `a → b` 和 `b → c` 的态射，则自动存在 `a → c` 的态射（传递性）

这正是范畴论中合成和恒等映射的定义。因此，**所有预序都是范畴**。

---

## 2. 线性序（全序）

### 定义

线性序（Linear Order / Total Order）是每个元素都与任何其他元素可比较的序。

### 四大定律

| 定律 | 描述 | 形式化 |
|------|------|--------|
| **自反性** | 每个元素都大于或等于自身 | `a ≤ a` |
| **传递性** | 如果 `a ≤ b` 且 `b ≤ c`，则 `a ≤ c` | `a ≤ b ∧ b ≤ c → a ≤ c` |
| **反对称性** | 若 `a ≤ b` 且 `b ≤ a`，则 `a = b` | `a ≤ b ∧ b ≤ a → a = b` |
| **完全性**（totality） | 任意两元素都可比 | `a ≤ b ∨ b ≤ a` |

### 自然数的序

自然数在"大于等于"关系下形成典型的线性序。

**重要性质**：任何有限的线性序都与自然数的一个子集同构，即每个有限线性序都可以唯一地映射到 {1, 2, 3, ...}。

### 线性序的局限性

从范畴论视角看，线性序是"最无聊"的序——所有有限线性序（及其无穷版本）都同构于自然数，图表形态千篇一律。

---

## 3. 偏序（部分序）

### 定义

偏序（Partial Order）移除了完全性定律，保留其他三大定律：

- 自反性
- 传递性
- 反对称性

### 特点

偏序允许元素之间不可比较的情况。例如：

- 根据足球技能排序所有球员时，从未交过手的球员之间无法比较
- 包含多个"链"（线性有序子集），但链之间可能无法比较

### 链（Chains）

偏序中的线性有序子集称为**链**。例如：

- 链1：`m → g → f`（我自己 → 奶奶 → 奶奶的朋友）
- 链2：`d → o`

多个链可以部分连接，只要不是完全一对一连接（即末元素连接下一链的首元素），就仍保持偏序性质。

### 最大元与最小元

- **最大元**：大于所有其他元素的元素（存在时）
- **最小元**：小于所有其他元素的元素（存在时）

注意：可能有多个"最大的"元素，但它们互不可比，此时没有最大元。

### 并（Join）与交（Meet）

- **并（Join）**：两个元素的最小上界，即大于等于两者的最小元素
- **交（Meet）**：两个元素的最大下界，即小于等于两者的最大元素

在整数按整除关系形成的偏序中：

- 并 = 最小公倍数（LCM）
- 交 = 最大公约数（GCD）

### 哈塞图（Hasse Diagrams）

序的可视化图表，规则：

- 较大的元素总是位于上方
- 如果存在 `a → b` 的箭头，则 b 必须在 a 上方

## 4. 预序（Preorder）

### 定义

预序仅保留两条定律：

- 自反性：`a ≤ a`
- 传递性：`a ≤ b ∧ b ≤ c → a ≤ c`

移除了完全性和反对称性。

### 与偏序的关系

| 类型 | 自反性 | 传递性 | 反对称性 | 完全性 |
|------|--------|--------|----------|--------|
| 线性序 | ✓ | ✓ | ✓ | ✓ |
| 偏序 | ✓ | ✓ | ✓ | ✗ |
| 预序 | ✓ | ✓ | ✗ | ✗ |

### 预序的例子

用预序建模"谁击败了谁"：

- 直接击败：`a 击败 b`
- 间接击败：通过中间人传递（由传递性自动产生）

### 预序与等价关系

预序是偏序与等价关系的中间形态：

- 对称性 + 预序 → 等价关系
- 反对称性 + 预序 → 偏序

预序中所有双向可达的元素构成一个**等价类**。这些等价类之间的序形成偏序。

---

## 5. 格（Lattice）

### 定义

**格**是每个元素对都有并和交的偏序。

### 有界格

同时具有最大元和最小元的格称为**有界格**。所有有限格都是有界格。

### 分发格

满足以下条件的格是**分发格**：

```
x ∨ (y ∧ z) = (x ∨ y) ∧ (x ∨ z)
```

其中 ∨ 表示并，∧ 表示交。

---

## 6. 重要例子

### 颜色混合偏序

用颜色作为元素，"包含"关系作为序：

- 混合两种颜色得到新颜色 = 并（Join）
- 并运算对应颜色混合

### 整数整除偏序

按"整除"关系排序：

- 2 和 5 都在 10 之前（因为 2 × 5 = 10）
- Join = 最小公倍数
- Meet = 最大公约数

### 包含偏序

集合之间的包含关系构成偏序：

- Join = 并集
- Meet = 交集

### 数论与包含序的同构

整除偏序同构于素数（或素数幂）的包含序。

这由**算术基本定理**保证：每个整数可以唯一分解为素数的乘积。

---

## 7. Birkhoff 表示定理

**核心结论**：每个有限分发格都同构于其**并不可约元素**的包含序。

- **并不可约元素**：不能表示为其他两个元素的并的元素
- 这些"素元素"像素数一样，是构建整个结构的"基本砖块"

这揭示了序论与数论的深刻联系。

---

## 8. 序与范畴论的对应

### 薄范畴（Thin Category）

序实际上是一种特殊的范畴，称为**薄范畴**：

- 任意两对象之间最多只有一条态射
- 要么存在 `a ≤ b` 的态射，要么不存在

### 范畴运算的序对应

| 范畴概念 | 序论对应 |
|----------|----------|
| 余积（Coproduct） | 并（Join） |
| 积（Product） | 交（Meet） |

这意味着在预序范畴中，范畴余积正好对应序的并运算。

### 图表交换性

预序的一个重要性质：**所有图表都交换**。这是因为任意两点间至多一条态射，不存在多条不同路径的可能性。

---

## 9. 总结

本文档系统介绍了序论的核心概念：

1. **线性序**：完全可比，定律最严格，应用最广
2. **偏序**：允许不可比，更灵活的现实建模工具
3. **预序**：最基础的序结构，本质上是特殊的范畴
4. **格**：具有良好运算性质的偏序
5. **Birkhoff 定理**：连接序论与组合数学的桥梁

序论展示了数学中"顺序"概念的丰富性和深度，同时也揭示了序与范畴论的深刻联系——序本质上是只有一条态射的范畴，这一视角帮助我们理解更抽象的范畴论概念。
