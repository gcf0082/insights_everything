## 内容摘要

这是 Y Combinator W26 批次的一个创业项目 Launch：Kampala，由 Zatanna 公司开发。Kampala 是一个中间人(MITM)风格的代理工具，允许用户通过代理现有应用的工作流程来自动生成 API，无需编写脆弱的脚本。

## 核心功能

- **反向工程工作流**: 通过 MITM 代理方式拦截应用流量，自动生成可用的脚本或 API
- **会话复用**: 利用现有的会话令牌和反机器人 cookie，实现确定性自动化
- **智能 Agent**: 用户可通过自然语言提示让 Kampala 自动创建脚本/API

## 技术特点

- 支持 Deterministic 自动化（在复杂情况下可提供白手套服务）
- 不依赖截图发送给 LLM 的方式，而是直接逆向底层 API
- 尚不支持 SSL pinning

## 创始背景

- 创始人 Alex 和 Tarun 曾在牙科技术领域工作
- 为解决与多个第三方 Web 应用对接的需求，开发了这款工具
- 公司名 Zatanna 来自 DC 漫画角色
