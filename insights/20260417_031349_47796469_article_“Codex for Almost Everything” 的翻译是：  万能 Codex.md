## 产品概述

OpenAI Codex 是一款由 ChatGPT 驱动的 AI 编程助手，旨在帮助开发团队构建和交付软件。它能够端到端地完成从常规拉取请求到复杂重构、迁移等工程任务。

## 核心功能

### 1. 真实工程工作能力
- 构建功能、复杂重构、代码迁移
- 自动执行端到端任务
- 由 OpenAI 前沿编码模型驱动

### 2. 多智能体工作流
- 内置 worktrees 和云环境
- 支持跨项目并行智能体协作
- 将数周工作压缩至数天完成

### 3. 团队适配能力
- **Skills 功能**: 超越单纯写代码，理解代码、原型设计、文档编写
- 与团队标准对齐
- 可定制化工作流

### 4. 自动化后台运行
- **Automations**: 自动执行问题分类、告警监控、CI/CD 等常规工作
- 让工程师专注于构建

### 5. 提升团队代码质量
- 更完善的设计
- 全面的测试覆盖
- 高信号量的代码审查
- 在问题进入生产环境前捕获问题

## 使用方式

Codex 可在多个平台使用，均通过 ChatGPT 账户连接：

1. **Codex 桌面应用** - 完整的 IDE 体验
2. **编辑器** - VS Code、Cursor、Windsurf 等扩展
3. **终端** - Codex CLI
4. **云端** - Codex Cloud（运行在 OpenAI 基础设施上）

## 模型选择

Codex 提供多种模型选择：
- GPT-5.4、Mini、Codex 模型、Spark
- 支持低、中、高推理级别
- 根据成本和速度权衡选择

## 最佳实践

### 提示结构（四要素）
1. **目标** - 要完成什么
2. **上下文** - 相关的文件、文档、示例
3. **约束** - 标准、架构、安全要求
4. **验收标准** - 如何验证成功

### 使用 AGENTS.md
- 放置团队标准、代码规范、构建命令等持久指导
- 在 ~ /.codex 或项目级别配置

### 工作流建议
- 一个线程一个任务，保持 scoped 工作
- 复杂任务使用 Plan 模式
- 定期使用 /compact 压缩长线程
- 将重复指令放入 agents.md
- 先完善技能再自动化

### MCP（模型上下文协议）
- 连接外部工具和服务（Figma、数据库等）
- 支持 STDIO 和 Streamable HTTP 服务器

## 总结

OpenAI Codex 代表了 AI 辅助编程的重大进步，将曾经需要专业工程师数周完成的工作压缩至数天。它不仅是一个代码生成工具，更是一个全面的工程合作伙伴，能够理解代码、审查代码、自动化常规任务，并与团队工作流深度集成。
