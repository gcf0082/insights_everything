## 项目概述

CrossPoint Reader 是一款为 **Xteink X4** 电子纸阅读器开发的开源固件，基于 **PlatformIO** 构建，目标硬件为 **ESP32-C3** 微控制器。该项目旨在作为官方固件的开源替代方案，提供完全开放的阅读器体验。

**注意**：该项目与 Xteink 无关联，是社区项目。

## 主要特性

- EPUB 解析与渲染（支持 EPUB 2 和 EPUB 3）
- EPUB 内嵌图片支持
- 阅读位置保存
- 文件浏览器与文件选择器
  - 支持从根目录选择 EPUB
  - 支持嵌套文件夹
  - 带封面显示的 EPUB 选择器
- 自定义休眠屏幕（封面休眠屏幕）
- WiFi 图书上传
- WiFi OTA 无线更新
- KOReader Sync 跨设备阅读进度同步
- 可配置字体、布局和显示选项
  - 用户自定义字体
  - 完整 UTF-8 支持
- 屏幕旋转

## 多语言支持

支持多种语言的 EPUB 阅读，包括：英语、西班牙语、法语、德语、意大利语、葡萄牙语、俄语、乌克兰语、波兰语、瑞典语、挪威语等。

## 技术细节

### RAM 优化

ESP32-C3 可用 RAM 仅约 380KB，项目在设计上积极采用缓存策略以最小化 RAM 使用：

1. **数据缓存**：首次加载书籍章节时会缓存到 SD 卡，后续加载从缓存读取
2. **缓存结构**：`.crosspoint/epub_<hash>/` 目录存放书籍元数据、章节数据、封面图片和阅读进度

### 内部文件格式

- `progress.bin`：阅读进度（章节、页码等）
- `cover.bmp`：书籍封面图片
- `book.bin`：书籍元数据（标题、作者、目录等）
- `sections/`：所有章节数据存储目录

## 安装方式

### Web 安装（推荐）

1. 通过 USB-C 连接 Xteink X4 到电脑
2. 访问 [https://xteink.dve.al/](https://xteink.dve.al/) 点击刷入固件

### 命令行刷入

```bash
pip install esptool
esptool.py --chip esp32c3 --port /dev/ttyACM0 --baud 921600 write_flash 0x10000 firmware.bin
```

## 开发环境

- **语言**：Python 3.8+
- **调试**：使用 `scripts/debugging_monitor.py` 脚本
