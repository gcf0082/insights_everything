## 项目概述

LangAlpha 是一个专注于金融领域的AI代理平台，旨在帮助解读金融市场并支持投资决策。该项目被称为"氛围投资代理框架"（vibe investing agent harness），其核心理念是将投资研究视为一个迭代的贝叶斯过程——从论点开始，每日获取新数据，并据此更新投资信念。

项目受到软件工程领域的启发：代码库是持久化的，每次提交都在之前的工作基础上构建。LangAlpha 将这一理念引入投资研究，为代理提供一个持久的工作空间，使研究自然地累积和复合。

## 核心特性

### 1. 程序化工具调用（PTC）

LangAlpha 的核心创新在于程序化工具调用（PTC）。与传统AI代理通过一次性JSON工具调用将结果直接输入上下文窗口不同，PTC的运作方式是：代理编写并执行Python代码来处理财务数据，而不是将原始数据传递到LLM上下文窗口。

这大大减少了token浪费，同时能够执行那些否则会超出上下文限制的分析。代码在Daytona云沙箱中执行，通过生成的包装器模块与MCP服务器进行JSON-RPC通信。

### 2. 持久工作空间

每个工作空间映射到一个专用的沙箱，包含结构化目录和持久的内存文件（`agent.md`），使研究能够跨会话和线程累积。每个沙箱的工作目录结构：
- `work/<task>/`：每个任务的 working 区域（数据、图表、代码）
- `results/`：最终报告
- `data/`：共享数据集
- 根目录的 `agent.md`：跨线程持久内存文件

### 3. 财务数据生态系统

项目构建了一个多层次的数据提供者层级架构：
- **原生工具**：快速参考查询（公司概述、SEC文件、市场指数、搜索）
- **MCP服务器**：批量数据处理（价格数据、基本面数据、宏观经济学、期权数据）

数据提供者层级：
- 第1层：ginlix-data（托管代理）- 实时WebSocket价格馈送
- 第2层：FMP（Financial Modeling Prep）- 高质量基本面数据
- 第3层：Yahoo Finance（免费）- 价格历史、基本面数据

### 4. 财务研究技能

代理内置23个预构建的财务研究技能，分为以下类别：

| 类别 | 技能 |
|------|------|
| 估值与建模 | DCF模型、可比公司分析、三报表模型、模型更新、模型审计 |
| 股票研究 | 启动报告、财报预览、财报分析、论点跟踪器 |
| 市场情报 | 早间简报、催化剂日历、行业概览、竞争分析、创意生成 |
| 文档生成 | PDF、DOCX、PPTX、XLSX |
| 运营 | 投资演示QC、定时自动化、用户画像与投资组合 |

### 5. 代理群体架构

核心代理基于LangGraph构建，通过`Task()`工具生成并行异步子代理。子代理具有隔离的上下文窗口，防止长推理链中的漂移。代理群体支持：
- 并发执行多个子代理
- 主代理可以选择等待子代理结果或继续其他待处理工作
- 实时转向：向正在运行的子代理发送后续指令（`Task(action="update")`）
- 检查点恢复：从完成状态恢复子代理进行迭代优化

### 6. 中间件栈

代理包含24个可组合的中间件层，主要包括：
- **工具安全**：凭证泄漏检测、受保护路径、错误处理
- **上下文与技能**：agent.md注入、技能加载、多模态
- **协调**：人类在环中断、计划模式、转向、子代理调度
- **弹性**：自动摘要、重试与回退、提示缓存

### 7. 财务研究工作台

Web UI不仅仅是聊天界面，而是一个完整的研究工作台：
- 内联财务图表
- 内联HTML小部件（Chart.js图表、指标卡、数据表）
- 多格式文件查看器（PDF、Excel、CSV、HTML预览）
- TradingView图表
- 实时市场数据WebSocket
- 可分享对话

### 8. 自动化

代理可以从对话内部安排自己的任务：
- **时间触发**：标准cron表达式用于循环调度
- **价格触发**：当股票或主要指数达到目标价格或百分比变动时执行
- 条件和逻辑可组合，支持一次性和循环模式

### 9. 多提供商模型层

代理运行在提供商无关的模型层上，支持：
- **PTC模式**：深度多步投资研究
- **Flash模式**：快速对话响应和工作空间编排

支持的模型提供商：
- OpenAI（Codex OAuth）
- Anthropic（Claude Code OAuth）
- Moonshot（Kimi）
- Zhipu（GLM）
- MiniMax
- Qwen
- 本地模型（Ollama、LM Studio、vLLM）

### 10. 安全与工作空间保险库

- **静态加密**：使用PostgreSQL pgcrypto进行静态加密
- **凭证泄漏检测**：中间件扫描所有工具输出并编辑已知密钥值
- **沙箱代码执行**：每个工作空间在独立的Daytona云沙箱中运行
- **工作空间保险库**：每个工作空间内置秘密保险库，用于存储API密钥和凭证

## 技术架构

### 后端架构（FastAPI）

- `src/server/`：FastAPI应用、路由、处理器、模型、服务
- `src/ptc_agent/`：核心代理库——代理工厂、中间件栈、子代理、提示、沙箱/MCP集成
- `src/tools/`：LangChain工具（搜索、获取、市场数据、SEC文件）
- `src/llms/`：LLM包装器、token计数、定价、模型清单
- `src/config/`：设置、日志配置
- `src/data_client/`：财务数据协议抽象
- `src/utils/`：Redis缓存、共享工具

### 前端架构（React）

- React 19 + Vite 7 + TypeScript + Tailwind CSS 3 + shadcn/ui
- 状态管理：React Query
- 认证：OAuth

### 基础设施

- PostgreSQL：应用数据、LangGraph检查点存储
- Redis：SSE事件缓冲、API缓存、转向队列
- Daytona：云沙箱
- WebSocket：实时市场数据

## 快速开始

```bash
git clone https://github.com/ginlix-ai/langalpha.git
cd langalpha
make config   # 交互式向导——创建.env，配置LLM、数据源、沙箱、搜索
make up       # 启动PostgreSQL、Redis、后端、前端
```

访问：
- 前端：http://localhost:5173
- 后端API：http://localhost:8000（交互式文档在`/docs`）
- 验证：`curl http://localhost:8000/health`

## 总结

LangAlpha是一个功能全面的AI金融研究平台，其核心创新在于程序化工具调用（PTC）架构和持久工作空间设计。该项目将AI代理技术深度集成到投资研究流程中，支持多步骤财务分析、复杂数据处理和可视化。

项目的主要优势：
1. PTC架构大幅降低token消耗
2. 持久工作空间实现研究的累积
3. 23个预构建财务研究技能开箱即用
4. 多提供商模型层支持灵活性
5. 完整的安全机制保护敏感数据

适用于需要AI辅助进行投资研究的专业投资者和金融分析师。
