## 发布概述

scrcpy v4.0 是该项目的重大更新版本，距离上一版本 v3.3.4 带来众多新功能、性能优化和 bug 修复。此版本最核心的变化是从 SDL2 迁移到 SDL3，并引入了灵活的显示支持（Flex Display）等创新功能。

---

## 主要新功能

### 1. SDL3 迁移

本次更新将 scrcpy 从 SDL2 迁移至 SDL3，主要好处包括：
- 获得活跃的维护和持续的上游支持
- 利用最新的 bug 修复
- 启用新功能，例如调整窗口大小时保持纵横比锁定

### 2. Flex Display（灵活显示）

通过 `--flex-display` 或 `-x` 参数，可以创建可随客户端窗口动态调整大小的虚拟显示。

示例命令：
```bash
# 启动 Firefox 并使用灵活显示
scrcpy --new-display=/192 -x --start-app=org.mozilla.firefox --keep-active --no-vd-system-decorations

# 启动 Android 设置
scrcpy --new-display=1024x768/160 --start-app=com.android.settings --flex-display
```

### 3. 相机手电筒和缩放支持

新增相机控制功能：
- `MOD+t`：开启相机手电筒
- `MOD+Shift+t`：关闭相机手电筒
- `MOD+↑`：放大
- `MOD+↓`：缩小

命令行参数：
```bash
# 启动时开启手电筒
scrcpy --video-source=camera --camera-torch

# 设置缩放级别
scrcpy --video-source=camera --camera-zoom=1.5
```

### 4. 窗口纵横比保持

现在调整窗口大小时会自动保持内容纵横比，避免出现黑边。如需恢复旧行为，可使用 `--no-window-aspect-ratio-lock` 参数。

### 5. 保持设备活跃（Keep Active）

新增 `--keep-active` 参数，可定期向系统发送用户活动信号，防止设备因空闲而熄屏。与 `--stay-awake` 不同，此选项不会更改任何全局设置，且无论设备是否充电都能正常工作。

### 6. 背景颜色自定义

默认背景色从纯黑改为深灰色，并支持通过 `--background-color` 自定义：
```bash
scrcpy --background-color=#234567
scrcpy --background-color=234567
scrcpy --background-color=#567
```

### 7. 断开连接图标

当设备连接丢失时，窗口现在会显示断开连接图标 2 秒后再关闭，避免用户误以为 scrcpy 崩溃。

### 8. 新增快捷键

- `F11`：切换全屏模式
- `MOD+q`：退出 scrcpy

## Bug 修复

| 问题 | 修复内容 |
|------|----------|
| Meta Quest 闪烁 | 解决了固件升级后的闪烁问题 |
| 物理/逻辑尺寸混淆 | 修复了显示尺寸计算错误 |
| 根设备复制粘贴 | 修复了已 root 设备上的剪贴板问题 |
| 色彩空间转换 | 修复了颜色显示异常 |
| OPUS 静音高 CPU | 修复了播放静音时 CPU 占用过高的问题（因 denormal 数值导致） |
| 设备序列号空格 | 修复了序列号包含空格时的支持问题 |
| 屏幕保护程序 | 修复了无视频播放时屏幕保护程序意外禁用的bug |

## 技术升级

本次发布升级了多个关键依赖：
- ADB (platform-tools)：升级至 37.0.0
- FFmpeg：升级至 8.1.1
- SDL：升级至 3.4.8
- dav1d：升级至 1.5.3
