## 公司概述

Freestyle 是一个面向 AI 编码代理的沙箱基础设施平台，帮助开发者管理和运行大规模 AI 代理。

## 核心产品

### 1. Freestyle Sandboxes
专为运行数万个代理而设计的 VM 沙箱，具备以下特性：
- **即时启动**: VM 在 700ms 内完成配置
- **实时分叉**: 毫秒级克隆运行中的 VM
- **暂停与恢复**: 休眠 VM，恢复时精确还原状态

### 2. Freestyle Git
用于管理代理代码的 Git 仓库，支持：
- 任务里程碑追踪
- 事件流遥测
- 细粒度 Webhook 配置
- 与 GitHub 双向同步

### 3. Freestyle Deployments
从 Git 部署，支持 Freestyle Deploy 或克隆到 VM 中运行。

### 4. Freestyle Runs
代理执行运行管理。

## 技术特点

- **非容器化**: 完整的 Linux VM，具有真实 root 访问权限
- **嵌套虚拟化**: 支持 VM 内运行 VM、Docker 或任何虚拟化栈
- **用户隔离**: Linux 用户、systemd 服务和组；多用户隔离
- **完整网络栈**: 真实 root 访问的完整 Linux 网络堆栈

## 使用场景

1. **App Builder**: 类似 Lovable、Bolt、V0 的应用构建
2. **Background Agent**: 类似 Devin、Cursor Agent 的后台代理
3. **Review Bot**: 类似 Code Rabbit、Greptile 的代码审查
4. **AI Assistant**: 类似 OpenClaw、Claude 的 AI 助手

## 总结

Freestyle 提供了一个强大的代理规模化基础设施，通过即时启动的 VM 沙箱、实时分叉和暂停恢复功能，为 AI 编码代理提供了高效的运行环境。其完整 Linux VM 架构和嵌套虚拟化支持，使其区别于传统的容器化解决方案。
