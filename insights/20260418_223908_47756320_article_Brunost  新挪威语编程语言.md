## 洞察摘要

本文介绍了Brunost，一种有趣的新挪威语（Nynorsk）编程语言。Brunost得名于挪威著名的棕色山羊奶酪，是挪威文化中最具代表性的奶酪之一。作者创建这门语言的初衷是出于对纯挪威语编程语言的好奇心和娱乐精神。

## 核心内容

### 语言背景

挪威实际上有三种官方语言：Bokmål（书面挪威语）、Nynorsk（新挪威语）和 Sami（萨米语）。作者指出，Nynorsk 几乎可以说是一种纯书面语言——没有人真正说"纯正"的Nynorsk，它有许多方言，但没有一种是纯粹的Nynorsk。这促使作者产生了创建一个完全使用Nynorsk的编程语言的想法。

### Brunost 语言特点

1. **纯Nynorsk强制要求**：所有变量、参数和函数名必须使用Nynorsk。解释器内置Nynorsk词典，如果开发者使用了非Nynorsk词汇，会得到明确的错误信息。

2. **函数式语言**：Brunost是一种解释型函数式语言，类型约束非常宽松。

3. **Zig实现**：解释器使用Zig语言编写，性能出色。

4. **WebAssembly支持**：可以在浏览器中在线运行。

### 语法示例

**Hello World**:
```brunost
bruk terminal
terminal.skriv("Hei, verda!")
```

**变量声明**:
- 可变变量：`open fart er 80`
- 不可变变量：`låst fart er 80`

**条件语句**：使用 `viss` (if) 和 `ellers` (else)

**循环**：
- `forKvart` (foreach循环)
- `medan` (while循环)

**函数定义**：使用 `gjer` (做) 定义函数，`gjevTilbake` (归还) 返回值

**异常处理**：使用 `prøv` (尝试) 和 `fang` (捕获)

## 主要代码示例

### 斐波那契数列
```brunost
bruk terminal

gjer rekke(grense) {
    viss (grense < 2) gjer {
        gjevTilbake grense
    }
    gjevTilbake rekke(grense - 1) + rekke(grense - 2)
}

open teljar er 1
medan (teljar erSameEllerMindreEnn 15) gjer {
  terminal.skriv(rekke(teljar))
  teljar er teljar + 1
}
```

### FizzBuzz
```brunost
bruk terminal
bruk matte

open teljar er 1

medan (teljar erSameEllerMindreEnn 15) gjer {
    viss (matte.modulus(teljar, 15) erSameSom 0) gjer {
        terminal.skriv("FizzBuzz")
    } ellers viss (matte.modulus(teljar, 3) erSameSom 0) gjer {
        terminal.skriv("Fizz")
    } ellers viss (matte.modulus(teljar, 5) erSameSom 0) gjer {
        terminal.skriv("Buzz")
    } ellers {
        terminal.skriv(teljar)
    }
    teljar er teljar + 1
}
```

## 总结

Brunost是一个充满娱乐精神的编程语言项目，它将挪威新挪威语(Nynorsk)与编程结合在一起。虽然作者明确表示这不应用于实际生产环境，但它展示了语言设计的创造力和对文化传承的兴趣。该项目欢迎社区贡献，包括编辑器语法高亮、语言服务器扩展和标准库增强等功能。
