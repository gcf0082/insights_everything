# 洞察报告：CPU能预测多少分支？

## 基本信息

- **链接**: https://lemire.me/blog/2026/03/18/how-many-branches-can-your-cpu-predict/
- **作者**: Daniel Lemire
- **发布日期**: 2026年3月18日
- **网站**: Daniel Lemire's blog

## 核心洞察

现代处理器具备在单个核心上每个周期执行多条指令的能力。要在实际中实现这一目标，处理器需要对分支进行预测。作者指出，现代CPU的分支预测能力令人难以置信，但这使得基准测试变得困难，因为如果在小数据集上测试，可能会得到与实际数据不符的惊人结果。

## 测试方法

作者使用的基准测试函数如下：

```c
while (howmany != 0) {
    val = generate_random_value()
    if(val is odd) write to buffer
    decrement howmany
}
```

处理器尝试预测分支（if语句）。由于使用随机值，处理器应该每两次就会预测错误一次。然而，如果重复运行基准测试并始终使用相同的随机值，处理器会学习这些分支。

## 测试结果

在三款最新处理器上的测试结果：

| 处理器 | 完美预测分支数 |
|--------|---------------|
| AMD Zen 5 | 30,000 |
| Apple M4 | 10,000 |
| Intel Emerald Rapids | 5,000 |

## 结论

AMD Zen 5处理器表现最佳，能够完美预测30,000个分支；Apple M4位居第二，可预测10,000个分支；Intel Emerald Rapids排名最后，仅能预测5,000个分支。作者表示对Intel的表现感到失望，而AMD在这一基准测试中表现出色。
