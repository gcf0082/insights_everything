# 钠离子电动汽车电池突破：11分钟充电，450公里续航

**发布日期**: 2026年3月25日  
**作者**: Peter Johnson

---

## 核心要点

北汽集团（BAIC Group）宣布其钠离子电池技术取得重大突破完成了首款钠离子电池原型，并建立了方形电池的大规模生产工艺。

## 关键技术参数

| 指标 | 数值 |
|------|------|
| 能量密度 | >170 Wh/kg |
| 续航里程 | 450 km（CLTC工况） |
| 充电时间 | 约11分钟（4C超快充） |
| 工作温度范围 | -40°C 至 60°C |
| 低温保持率 | >92%（-20°C） |

## 行业背景

1. **量产进展**：2026年2月，宁德时代（CATL）和长安汽车推出了全球首款量钠离子电池车型——长安Nev A06

2. **能量密度对比**：CATL的"Naxtra"钠离子电池能量密度达175 Wh/kg，与磷酸铁锂（LFP）电池相当

3. **市场预测**：2025年全球钠离子电池出货量达9 GWh，同比增长150%；预计未来四年将突破1000 GWh

4. **成本优势**：钠资源储量丰富，价格敏感度低于锂，有望降低电池成本

## 结论

钠离子电池技术正在快速发展，凭借更宽的工作温度范围、更低的成本和快速充电能力，有望成为电动汽车领域的重要选择。中国企业在这一技术路线上处于领先地位。
