# 洞察报告：美国科技公司支持Sam Altman的World ID，尽管全球多国抵制

## 基本信息

- **洞察链接**: https://restofworld.org/2026/sam-altman-worldcoin-zoom-tinder-partnerships/
- **来源**: Rest of World
- **作者**: Ananya Bhattacharya
- **发布日期**: 2026年4月27日
- **主题**: World ID（曾用名Worldcoin）虹膜扫描ID项目在美国获得Tinder、Zoom、Docusign合作，尽管在多国被禁

## 核心发现

### 合作伙伴关系

2026年4月17日，World宣布与Tinder、Zoom和Docusign建立战略合作伙伴关系。这三家公司将使用World的数字身份验证用户，旨在减少深度伪造、诈骗和其他欺诈活动。

### 全球推广与争议

- **运营规模**: 该项目已在160个国家验证了超过1800万人
- **美国布局**: 2025年4月在美国6个城市设置了7000个Orb设备
- **数据收集方式**: 通过虹膜扫描创建"人类证明"，并曾以50美元加密货币奖励鼓励注册

### 各国监管行动时间线

| 时间 | 事件 |
|------|------|
| 2021年10月 | Worldcoin发布，由Alex Blania和Sam Altman联合创立 |
| 2022年4月 | MIT Technology Review调查曝光其在非洲和亚洲的不道德做法 |
| 2023年7月 | 在全球35个城市推出，提供50美元加密货币奖励 |
| 2023年8月 | 肯尼亚暂停生物识别验证 |
| 2024年3月 | 西班牙、葡萄牙先后禁止数据收集 |
| 2024年5月 | 香港数据监管机构命令停止运营 |
| 2025年1月 | 巴西因支付虹膜扫描费用违反数据法而禁止 |
| 2025年5月 | 印尼暂时中止数据收集 |
| 2025年10月 | 菲律宾命令立即停止运营 |
| 2025年11月 | 泰国关闭生物识别数据收集 |
| 2026年4月 | 宣布与Tinder、Zoom、Docusign合作 |

### 主要争议点

1. **隐私担忧**: 多个政府质疑生物信息收集的合法性
2. **伦理问题": 包括向弱势群体收集数据、未获得有效知情同意
3. **监管规避**: 斯诺登批评World"cataloguing eyeballs"（编目眼球）
4. **数据超范围**: MIT调查发现World收集了超越虹膜扫描的个人数据，包括心跳、呼吸和其他生命体征

### 美国市场优势

- 相比欧盟，美国对生物数据收集的法律保障较少
- 加密货币法规在不同州之间不那么严格和统一
- 允许更灵活的运营空间

## 结论

尽管World ID项目在亚洲、非洲、欧洲和拉丁美洲面临广泛抵制和调查，但通过与美国科技公司的合作，该项目正在企业市场找到新的增长点。公司声称在葡萄牙、西班牙和韩国进行了公民支持度调研，并发布了关于数字ID收入潜力的蓝图。
