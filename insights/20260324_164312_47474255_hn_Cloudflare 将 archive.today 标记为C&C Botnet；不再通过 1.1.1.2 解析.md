# 洞察报告

## 基本信息

- **洞察链接**: https://news.ycombinator.com/item?id=47474255
- **标题**: Cloudflare flags archive.today as "C&C/Botnet"; no longer resolves via 1.1.1.2
- **来源**: Hacker News
- **发布时间**: 2026年3月22日
- **得分**: 408+ points
- **评论数**: 274+

## 事件概述

Cloudflare 将 archive.today（以及相关域名 archive.is 和 archive.ph）标记为"C&C/Botnet"（命令控制与僵尸网络），导致通过 Cloudflare 的 1.1.1.2 DNS（"无恶意软件"模式）无法解析这些域名。

## 详细情况

1. **DNS 解析问题**
   - 使用 1.1.1.2 DNS 时无法解析 archive.today 域名
   - 使用普通 1.1.1.1 DNS 仍可正常解析
   - archive.today 当前被标记为：CIPA Filter、Reference、Command and Control & Botnet、DNS Tunneling

2. **历史争议**
   - 2019年起 Cloudflare 多次对 archive.is/ today 的解析做出调整
   - archive.is 曾被指控故意提供被篡改的存档页面
   - 2026年2月20日，英语 Wikipedia 宣布禁用 archive.today 链接，理由是 DDoS 攻击和存档内容被篡改的证据

3. **社区反应**
   - 用户对 Cloudflare 的标记行为看法不一
   - 有人指出 archive.today 在存档网页方面有良好声誉，且不像 archive.org 那样审查页面
   - 也有用户报告 archive.today 仍在对某些博客发起攻击
