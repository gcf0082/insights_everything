## 内容摘要

**Quarkdown** 是一个增强版的 Markdown 标记语言，扩展了标准 Markdown 的功能，使其更加强大。

### 核心讨论

1. **Markdown 的设计哲学**: 有评论者指出，Markdown 的核心价值在于其简洁性，可以在 VIM 中直接编辑，无需 GUI，所见即所得。增加过多功能会让它变得像"全键盘"一样复杂。

2. **与其他工具的比较**: 作者希望能对比以下工具：
   - MyST
   - Pandoc
   - Quarkdown
   - Quarto
   - Typst
   
   Quarto 和 Pandoc 使用 Pandoc Markdown，Quarkdown 和 Typst 提供类似 LaTeX 的可编程标记语言。

3. **Typst 的优势**: Typst 是现代版的 LaTeX，语法更简洁，但学术出版领域有 40 年的 LaTeX 历史需要克服。

4. **djot**: 被推荐为一个设计良好的 Markdown 超集。

5. **关于 Quarkdown 作者**: iamgioh 是 Quarkdown 的创始人和项目负责人最初作为大学研究项目开始。

### 正面观点

- Quarkdown 是对纯文本编辑的改进，与 LLM 结合效果更好
- 可用于创建文档、简历等富文本内容
- 支持 .row、.column、.grid 等布局函数

### 负面观点

- 增加功能违背了 Markdown 简洁的设计理念
- 最终会变得像 Microsoft Word
- 已有太多 Markdown 标准（xkcd 927 讽刺漫画）
- TeXmacs 等工具没有流行起来是因为需要 LaTeX 包

## 结论

Quarkdown 试图在保持 Markdown 简洁性的同时增加更多功能，但社区对其是否应该扩展 Markdown 存在分歧。Typst 作为更现代的 LaTeX 替代品目前更受关注，但学术出版领域的 LaTeX 生态系统仍是主要障碍。
