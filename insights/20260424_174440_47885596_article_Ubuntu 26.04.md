## 内容摘要

Ubuntu 26.04 LTS（代号"Resolute Raccoon"）已按计划正式发布。

### 主要新特性

- **TPM-backed full-disk encryption**：TPM支持的全盘加密
- **内存安全组件的扩展使用**：expanded use of memory-safe components
- **改进的应用权限控制**：improved application permission controls
- **Arm系统的Livepatch支持**：Livepatch support for Arm systems

### 支持周期

- **5年支持**：Ubuntu Desktop、Ubuntu Server、Ubuntu Cloud、Ubuntu WSL、Ubuntu Core
- **3年支持**：其他所有变体版本

### 官方 Flavor 版本

- Edubuntu
- Kubuntu
- Lubuntu
- Ubuntu Budgie
- Ubuntu Cinnamon
- Ubuntu Kylin
- Ubuntu Studio
- Ubuntu Unity
- Xubuntu

## 评论摘要

### MATE桌面版

用户询问MATE Desktop版本的维护状态，当前正在招募新的维护者。

用户在XUbuntu发布说明中发现OpenSSH相关bug，建议添加gcr-ssh-agent.socket到用户单元以解决SSH_AUTH_SOCK问题。
