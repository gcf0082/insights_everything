## 公司概述

Eden AI 是一个领先的 AI API 聚合平台，旨在为开发者提供统一的方式来访问多种顶级 AI 模型。该平台通过单一 API 接口整合了大语言模型（LLM）以及专业的 AI 模型，涵盖语音识别、视觉处理、OCR 光学字符识别、翻译等众多领域。这种聚合模式让用户无需管理多个供应商接口，即可轻松使用数百种 AI 能力。

## 核心功能

### 1. 统一 API 接口

Eden AI 提供一个统一的 API 来访问各类 AI 模型，包括大语言模型和专业化模型（OCR、语音转文字、文字转语音、视觉处理、翻译等）。开发者只需集成一次，即可无缝切换和使用不同的 AI 供应商，无需修改代码即可实现跨供应商的标准化集成。

### 2. 智能路由与故障转移

平台支持智能路由功能，允许用户根据自定义规则设置请求路由。当某个模型出现故障时，系统会自动切换到备用模型，确保生产环境的高可用性。这种内置的故障转移机制可以有效保护应用程序免受模型服务中断的影响。

### 3. 价格、性能与区域控制

用户可以根据成本、延迟或执行区域来选择模型，在流量和复杂度增长的同时保持完全的成本控制。这种灵活性使企业能够为不同需求选择最合适的模型，避免供应商锁定，并能够快速适应价格或性能的变化。

用户可以为每个需求选择最高效的模型，避免供应商锁定，并能够快速适应定价或性能的变化，从而优化整体支出。
