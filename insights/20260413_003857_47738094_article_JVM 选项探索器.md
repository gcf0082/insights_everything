## 站点概述

VM Options Explorer 是由 Chris Whocodes 维护的 JVM 参数参考网站，提供全面的 HotSpot VM 选项查询功能。该站点涵盖了从 OpenJDK 6 到 27 的所有版本，以及多种商业和开源 JDK 发行版。

## 支持的 JDK 版本

### OpenJDK
- 从 OpenJDK 6 到 OpenJDK 27 的完整版本列表
- 每个版本都有独立的选项参考页面

## 主要功能

1. **版本对比**: 提供各 JDK 版本之间的选项差异对比功能
2. **跨发行版比较**: 支持 OpenJDK 与其他发行版（如 Dragonwell、Corretto、Zulu 等）的选项对比
3. **选项搜索**: 包含详细的 JVM 选项参数说明、默认值、类型、引入版本等信息

## 相关工具

该站点还提供其他 Java 相关的探索工具：

- **VM Intrinsics Explorer**: JVM 内联函数参考
- **GC Explorer**: 垃圾回收器配置参考
- **JEP Search**: JDK 增强提案搜索
- **Full JEP**: 完整 JEP 列表

## 总结

VM Options Explorer 是 Java 开发者和性能调优工程师的重要参考资源，提供了最全面的 JVM 参数文档，覆盖了主流 JDK 发行版的所有版本。
