# OpenAI Codex 费率卡洞察报告

**洞察链接**：https://help.openai.com/en/articles/20001106-codex-rate-card  
**报告生成日期**：2026-04-06  
**数据来源**：OpenAI 帮助中心、Get AI Perks

---

## 一、概述

本报告基于 OpenAI Codex 费率卡（Codex rate card）撰写，详细介绍 Codex 在各订阅计划中的积分机制和使用限制。Codex 是 OpenAI 的 AI 编程助手，集成在 ChatGPT 订阅计划中，无需单独订阅。

**关键要点**：

- Codex 包含在 ChatGPT Plus、Pro、Business 和 Enterprise 计划中，无额外订阅费用
- 使用采用积分（Credit）制，不同任务类型消耗不同积分
- 模型选择影响积分消耗量：GPT-5.4 > GPT-5.3-Codex > GPT-5.1-Codex-Mini
- 积分按五小时滚动窗口刷新，非月度重置

---

## 二、积分消耗详情

### 2.1 按任务类型

| 模型 | 本地任务（每条消息） | 云端任务（每条消息） | 代码审查（每次 PR） |
|------|---------------------|---------------------|-------------------|
| GPT-5.4 | ~7 积分 | ~34 积分 | ~34 积分 |
| GPT-5.3-Codex | ~5 积分 | ~25 积分 | ~25 积分 |
| GPT-5.1-Codex-Mini | ~1 积分 | 不可用 | 不可用 |

**说明**：
- 本地任务：在 IDE 或 CLI 中执行，无需云端集成，消耗积分较少
- 云端任务：需要云端处理的任务，如自动化 PR 审查
- Mini 模型成本最低，但不支持云端任务

## 三、订阅计划对比

### 3.1 计划总览

| 计划 | 月费 | 本地消息（5小时窗口）| 云端任务 | 代码审查 |
|------|------|---------------------|---------|---------|
| Free/Go | 免费 | 极有限 | 不可用 | 不可用 |
| Plus | $20 | 33-168 | 不可用 | 不可用 |
| Pro | $200 | 223-1120 | 不可用 | 不可用 |
| Business | 自定义 | 更高限额 | 可用 | 可用 |
| Enterprise/Edu | 自定义 | 最高限额 | 可用 | 可用 |

### 3.2 计划详解

**ChatGPT Plus（$20/月）**
- 提供 GPT-4o 访问
- Codex 本地任务访问
- 不提供云端任务和代码审查功能
- 适合轻度编程使用者

**ChatGPT Pro（$200/月）**
- 提供更高的使用容量（约 Plus 的 6-7 倍）
- 仍不支持云端集成
- 适合需要频繁编码的专业开发者

**ChatGPT Business（自定义定价）**
- 按席位收费
- 包含 Slack 机器人、GitHub Actions 集成
- 支持自动化 PR 审查工作流
- 适合团队协作

**Enterprise/Edu（自定义定价）**
- 最高使用限额
- 全功能支持
- 适合大型组织

## 四、CLI 和 API 定价

### 4.1 Codex CLI
提供命令行访问，支持终端工作流：
- 支持与网络界面相同的积分消耗规则
- 上下文跨会话持久化
- 可并行启动多个 Agent

### 4.2 API 定价
截至 2026 年 3 月，OpenAI 未公布 Codex API 最终的每 Token 定价。行业分析预计将参考 GPT-4 Turbo 定价结构：
- 输入：约 $0.01-0.03 / 1K tokens
- 输出：约 $0.03-0.12 / 1K tokens

**建议**：月均 50-80 小时密集编码工作后，订阅制比按量付费更具成本效益。

## 五、使用限制与应对策略

### 5.1 限制机制
- 积分按五小时滚动窗口刷新
- 用尽后需等待刷新或升级计划
- 无超量费用，但会暂停服务

### 5.2 常见应对方法
1. 将相关任务批量处理，减少消息数
2. 对简单任务使用 GPT-5.1-Codex-Mini 节省积分
3. 在非高峰时段安排密集型重构任务
4. （不推荐）使用不同账户分别处理不同项目

## 七、结论与建议

1. **轻度用户**：ChatGPT Plus（$20/月）已包含 Codex 本地任务，零增量成本
2. **专业开发者**：Pro 计划（$200/月）提供更充足容量，但无云端集成
3. **团队用户**：Business 或 Enterprise 计划提供完整功能，包括 PR 自动化审查
4. **评估建议**：建议先订阅 Plus 两周，测量实际积分消耗后再决定是否升级

**注意事项**：当前处于限时促销期，积分限制翻倍。建议定期查看官方文档获取最新定价信息。
