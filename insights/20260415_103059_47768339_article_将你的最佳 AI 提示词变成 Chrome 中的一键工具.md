# 洞察报告：将最佳AI提示词转化为Chrome一键工具

**洞察链接**：https://blog.google/products-and-platforms/products/chrome/skills-in-chrome/

**发布日期**：2026年4月14日

**来源**：Google Chrome 官方博客

**作者**：Hafsah Ismail（Chrome 产品经理）

---

## 核心摘要

Google 宣布在 Chrome 浏览器中推出 **Skills（技能）** 功能，允许用户保存、重复使用和自定义 AI 提示词，将其转化为一键式工作流程。

---

## 详细洞察

### 1. 功能背景

人们在 Chrome 中使用 AI 来完成各种任务，如回答问题、比较信息、阐明概念。然而，重复执行相同的 AI 任务（例如在浏览不同页面时请求配方替换以使食谱变为素食）意味着需要重新输入相同的提示词。

### 2. Skills 功能介绍

**核心功能**：
- 将常用的 AI 提示词保存为 "Skill"
- 通过单击按钮即可重复运行保存的 Skill
- 支持在当前页面和选中的其他标签页上运行

**使用方法**：
- 在聊天历史中直接保存提示词为 Skill
- 在 Gemini in Chrome 中输入正斜杠（/）或点击加号（+）按钮选择已保存的 Skill
- 随时编辑已保存的 Skill 或创建新 Skill

### 3. 应用场景示例

- **健康与健身**：快速计算任何食谱的蛋白质含量
- **购物**：在多个标签页之间生成并排规格对比
- **生产力**：扫描长文档以提取重要信息

### 4. Skills 库

Google 还推出了一个预构建的 Skills 库，提供常用任务和工作流程的即用型技能。

**库中示例**：
- 分解在线查看产品的成分
- 通过将预算与收礼人兴趣交叉参考来选择完美礼物

用户可以添加这些 Skills 作为保存的技能并进行试用，也可以根据需要自定义编辑。

### 5. 安全与隐私

- Skills 构建在 Chrome 的安全和隐私基础之上
- 采用与 Gemini in Chrome 相同的保护措施
- 某些操作（如添加日历事件或发送电子邮件）需要确认
- 受益于 Chrome 的分层保护，包括自动红队测试和自动更新功能

### 6. 推出时间

- 从即日起向 Gemini in Chrome 用户推出（仅限英语-US）
- 适用于 Mac、Windows 和 ChromeOS
- 保存的 Skills 在任何登录的 Chrome 桌面设备上可用
- 可通过在 Gemini in Chrome 中输入正斜杠（/）然后点击指南针图标进行管理
