## 内容摘要

### 核心主题
Ensu 是 Ente 推出的离线 LLM（大型语言模型）应用，一个类似 ChatGPT 的应用，可以完全在设备上运行，保护隐私且零成本。

### 关键观点

**1. 为什么做这个**
- LLM 太过重要，不能留给大科技公司
- 存在前沿模型与本地运行模型之间的能力差距，但本地模型每天都在改进
- 一旦本地模型跨越某个能力阈值，它们将足以满足大多数用途，并带来完全的隐私和控制
- 这与 Ente Photos 的经历类似：最初人们认为不可能在设备上实现本地人脸识别、自然语言图像搜索等功能，但 Ente 最终做到了

**2. 做什么**
- 发布 Ensu：一个类似 ChatGPT 的应用，完全在设备上运行，保护隐私，零成本
- 即将支持通过 Ente 账户备份和同步聊天记录（可选的端到端加密）
- 应用开源，可在 iOS、Android、macOS、Linux 和 Windows 上使用
- 核心逻辑用 Rust 编写，支持图像附件

**3. 未来方向**
- 更专业的界面：类似永不过期的笔记，AI 提供建议、批评、提醒、背景、替代方案、观点和引用——"第二大脑"
- 更实用的方式：比如 Android 启动器，LLM 作为现有交互方式背后的实现细节
- 手机上的 AI 代理：无需设置、管理或手动备份，LLM 与用户一起成长，记住用户及其选择，管理任务，具有长期记忆和人格

### 产品定位
- 目前是 Ente Labs 项目
- 暂时不引入定价和稳定性相关问题
- 当前不如 ChatGPT 或 Claude Code 强大，但已经很有趣

## 主要结论

Ente 认为 LLM 是不能被忽视的技术，而大科技公司控制它们会带来隐私和操控风险。Ensu 代表了他们在本地 LLM 领域的首次尝试，旨在为用户提供一个隐私优先的 AI 对话体验。虽然目前能力有限，但 Ente 将其视为一段旅程的开始，未来的发展方向（包括同步功能）将根据用户反馈来确定。
