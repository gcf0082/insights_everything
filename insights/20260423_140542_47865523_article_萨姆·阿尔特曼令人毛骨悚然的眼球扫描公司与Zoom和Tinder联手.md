## 核心洞察

Sam Altman 的眼球扫描公司 World（前称 Worldcoin）宣布与 Tinder 和 Zoom 建立合作伙伴关系，将"人类证明"验证技术整合到这两个主流平台中。这一举措标志着 Altman 正在大规模推广其有争议的生物识别身份验证系统。

### Tinder 合作详情

- Tinder 已在日本进行试点项目，验证效果良好
- 将向全球用户推广 World ID 验证
- 用户需要在 World 的实体设备 Orb 处进行眼球扫描
- 验证成功后，用户获得"已验证人类"徽章
- 作为激励，验证用户可获得 5 次免费"提升"功能

### Zoom 合作详情

- Zoom 将集成 World ID Deep Face 技术
- 三重验证机制：
  1. 与 Orb 验证时的照片交叉比对
  2. 用户自行设备进行实时人脸检查
  3. 扫描现场视频帧进行验证
- 三项全部匹配后获得"已验证人类"徽章

### 背景信息

World 由 Sam Altman 于 2019 年联合创立，前身为 Worldcoin。该项目旨在创建全球 ID 系统和基于加密货币的货币体系，仅向验证过的人类开放。验证过程要求用户使用专用 Orb 设备扫描眼球，换取存储在手机上的独特数字 ID 代码。

## 专家观点

### 支持理由

- AI 技术快速发展背景下，深度伪造技术日益成熟，平台面临严重的虚假账户问题
- Tinder 早在去年就要求用户提交视频自拍照以确认身份
- World ID 整合是自然的安全升级

### 担忧理由

- 生物识别数据的收集和存储存在隐私风险
- 凭证可能被盗或转售
- 强制验证可能加剧
- 部分评论认为"一旦将眼球数据卖给极权主义者，他将毫不犹豫"

### 市场反应

- Worldcoin (WLD) 币价在消息发布当日下跌约 10%，至 0.286 美元

---

## 关键影响

1. **隐私争议**：Orb 设备扫描虹膜获取生物识别数据，引发隐私担忧
2. **商业扩张**：从加密货币钱包转向独立的身份验证应用
3. **平台整合**：Tinder、Zoom、Docusign 等主流平台开始采用
4. **AI 对抗**：作为对抗深度伪造和机器人的技术手段
