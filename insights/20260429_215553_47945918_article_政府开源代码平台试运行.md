# 荷兰政府开源代码平台软启动洞察报告

**洞察链接**: https://www.nldigitalgovernment.nl/news/soft-launch-for-government-open-source-code-platform/

**基本信息**:
- **标题**: 荷兰政府开源代码平台软启动 (Soft launch of open-source code platform for government)
- **发布日期**: 2026年4月27日
- **来源**: Digital Government (荷兰数字化政府)
- **主题**: 开源软件、数字化主权、政府协作

---

## 核心内容总结

### 1. 平台上线
荷兰政府代码平台 **code.overheid.nl** 已正式上线运营。该平台是政府范围内的代码发布和开发开源软件的统一平台，采用完全自托管方式，支持数字化主权。

### 2. 技术选型
- 当前试点版本使用 **Forgejo** —— 一个开源、欧洲 sovereign（自主可控）的 Git 平台，替代 GitHub 和 GitLab
- 目前并非所有政府组织都能使用该平台，处于试点阶段

### 3. 参与邀请
- 诚邀开发者贡献代码，目标是将平台发展为政府机构共享的 Git 平台
- 平台由内政部 (BZK) 开源项目办公室发起
- 合作方：DAWO (SSC-ICT)、Opensourcewerken、developer.overheid.nl
- 参与方式：发送邮件至 codeplatform@rijksoverheid.nl

### 4. 相关链接
- 平台地址：https://code.overheid.nl/
- 荷兰语博客：https://developer.overheid.nl/blog/2026/04/24/we-gaan-samen-code-overheid-bouwen

---

## 洞察分析

这是一个重要的里程碑，标志着荷兰政府在开源软件和数字化主权方面迈出了实质性步伐。通过自托管的 Forgejo 平台，政府能够更好地控制自己的代码资产，同时促进跨部门协作和代码复用。
