# 洞察报告：MXmap — 瑞士市政厅电子邮件提供商地图

## 基本信息

- **洞察链接**：https://mxmap.ch/
- **项目类型**：研究与数据可视化项目
- **数据状态**：⚠ 此数据可能已过时，正在进行进一步研究

## 项目概述

MXmap 是一个展示瑞士约2,100个市政区官方电子邮件提供商分布的地图项目，按管辖区域分组，基于公共DNS记录和其他公共网络信号。

## 核心功能

### 1. 什么是MXmap？
一个交互式地图，形象化呈现瑞士各市政区官方电子邮件的服务提供商情况。

### 2. 项目背景
- **数字主权问题**：美国提供商受美国 CLOUD Act 约束，允许美国当局请求存储的数据，无论数据实际托管位置
- 该项目使当前的提供商格局可见，促进对数字主权问题的理解

### 3. 工作原理
每个市政区的官方域通过以下11个信号进行检查：
- DNS 记录
- SMTP 横幅
- ASN 查找
- 公共 Microsoft API 端点

然后按提供商类型分类，并附带置信度评分。

## 数据说明

- DNS 记录指示邮件路由和授权发件人，不一定代表数据存储位置

## 总结

MXmap 是一个有价值的公共研究项目，帮助公众和决策者了解瑞士市政区电子邮件服务的提供商分布情况，进而理解数字主权相关的隐私和安全问题。该项目采用开源方式，鼓励社区参与和纠错。
