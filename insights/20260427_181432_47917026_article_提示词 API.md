## 1. API概述

Prompt API允许开发者直接在浏览器中向Gemini Nano发送自然语言请求。无需服务器端AI处理，所有推理都在本地设备上完成。

**典型应用场景**:
- AI驱动的搜索：根据网页内容回答问题
- 个性化新闻推送：动态分类文章并过滤
- 自定义内容过滤器：自动屏蔽敏感内容
- 日历事件创建：从网页提取事件信息
- 便捷联系人提取：从网站获取联系信息

## 2. 硬件要求

### 操作系统
- Windows 10或11
- macOS 13+（Ventura及更高版本）
- Linux
- ChromeOS（16389.0.0及更高版本，仅限Chromebook Plus设备）

### 存储与网络
- 至少22GB可用空间
- 网络连接（仅用于首次下载模型，之后可离线使用）
- 无计量限制的连接（Wi-Fi/以太网）

### 计算资源
- GPU：超过4GB VRAM
- CPU：16GB RAM及以上，4核或以上
- 注意：音频输入需要GPU

## 3. 核心使用流程

### 3.1 检查模型可用性

```javascript
const availability = await LanguageModel.availability({
  // 与prompt()或promptStreaming()相同的选项
});
```

### 3.2 创建会话

```javascript
const session = await LanguageModel.create({
  monitor(m) {
    m.addEventListener('downloadprogress', (e) => {
      console.log(`已下载 ${e.loaded * 100}%`);
    });
  },
});
```

### 3.3 发送提示并获取响应

```javascript
// 方式1：等待完整响应
const result = await session.prompt('写一首诗！');

// 方式2：流式响应
const stream = session.promptStreaming('写一首长诗！');
for await (const chunk of stream) {
  console.log(chunk);
}
```

## 4. 高级功能

### 4.1 多模态能力

支持的输入类型：
- **音频**: AudioBuffer, ArrayBufferView, ArrayBuffer, Blob
- **图像**: HTMLImageElement, SVGImageElement, HTMLVideoElement, HTMLCanvasElement, ImageBitmap, OffscreenCanvas, VideoFrame, ImageData

### 4.2 结构化输出

通过JSON Schema约束响应格式：

```javascript
const schema = { "type": "boolean" };
const result = await session.prompt(
  '这条消息是关于陶艺的吗？',
  { responseConstraint: schema }
);
```

### 4.3 会话管理

- **上下文窗口**：每个会话有最大token处理数量
- **上下文溢出**：超出时自动移除旧的对话记录（系统提示永不移除）
- **克隆会话**：使用`session.clone()`复制会话
- **终止会话**：使用`session.destroy()`释放资源

### 4.4 初始提示

```javascript
const session = await LanguageModel.create({
  initialPrompts: [
    { role: 'system', content: '你是一个有帮助的助手。' },
    { role: 'user', content: '意大利的首都是哪里？' },
    { role: 'assistant', content: '意大利的首都是罗马。' },
  ],
});
```

### 4.5 前缀约束

预填充assistant响应开头，引导模型输出格式：

```javascript
const characterSheet = await session.prompt([
  { role: 'user', content: '创建一个TOML角色表' },
  { role: 'assistant', content: '```toml\n', prefix: true },
]);
```

## 5. 性能与最佳实践

- 使用`prompt()`获取短结果
- 使用`promptStreaming()`流式显示长响应
- 复用会话以减少开销（创建会话可能耗时）
- 参考官方session management最佳实践
