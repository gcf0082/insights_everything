## 项目简介

VOID（Video Object and Interaction Deletion）是Netflix AI团队开源的视频对象和交互删除模型。该项目能够从视频中移除对象，同时消除它们对场景的影响——不仅仅是阴影和反射等次要效果，还包括物理交互（如移除人物后物体自然掉落）。

> **示例**：如果移除持有吉他的人，VOID也会移除人对吉他的影响——导致吉他自然掉落。

VOID基于CogVideoX构建，针对视频修复进行了微调，并使用交互感知的掩码调节技术。

## 核心功能

### 1. 对象移除与物理交互删除
- 移除视频中的指定对象
- 同时移除对象引起的物理交互（如物体掉落）
- 消除阴影和反射等次要效果

### 2. 双阶段推理管道
- **Pass 1**：基础修复模型
- **Pass 2**：扭曲噪声细化模型，提高时间一致性

### 3. 智能掩码生成
- 使用SAM2分割和VLM（Gemini）进行推理
- 生成四值语义区域掩码（quadmask）

### 4. 手动掩码编辑
- 包含GUI编辑器用于细化自动生成的掩码

### 模型结构

### 四值掩码编码
| 值 | 含义 |
|----|------|
| 0 | 主对象（待移除）|
| 63 | 主对象+受影响区域重叠 |
| 127 | 受影响区域（交互：掉落物体、移位物品等）|
| 255 | 背景（保留）|

## 使用方法

### 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 设置Gemini API密钥：`export GEMINI_API_KEY=your_key_here`
3. 下载预训练模型
4. 使用Colab笔记本运行推理

### 推理命令示例
```bash
python inference/cogvideox_fun/predict_v2v.py \
    --config config/quadmask_cogvideox.py \
    --config.data.data_rootdir="path/to/data_rootdir" \
    --config.experiment.run_seqs="my-video" \
    --config.experiment.save_path="path/to/output" \
    --config.video_model.model_name="path/to/CogVideoX-Fun-V1.5-5b-InP" \
    --config.video_model.transformer_path="path/to/void_pass1.safetensors"
```

- GPU：40GB+ VRAM（如A100）
- 支持多GPU训练（8× A100 80GB）
