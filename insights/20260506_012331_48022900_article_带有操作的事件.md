# GitHub Actions 事件洞察报告

## 基本信息

- **洞察链接**: https://www.githubstatus.com/incidents/1j40g94rn22j
- **事件ID**: 1j40g94rn22j
- **受影响服务**: GitHub Actions
- **事件状态**: 已恢复/监控中

## 事件概述

GitHub Actions 服务于 2026 年 5 月 5 日发生性能降级事件，主要影响美国东部区域的托管运行器。

| 5月5日 15:54 | 应用缓解措施：针对标准托管运行器的长队列时间和失败问题已应用缓解措施，持续监控恢复状态 |

## 影响范围

- **主要影响区域**: 美国东部 (East US)
- **影响服务**: GitHub Actions 托管运行器
- **影响比例**: 8%-10% 的运行受影响
- **受影响组件**: Standard Hosted Runners, Hosted Runners with Private Networking

## 缓解措施

1. 标准托管运行器已完全恢复
2. 私有网络托管运行器用户可切换到其他区域以缓解问题
3. 正在与计算提供商合作恢复美国东部区域的容量

## 当前状态

事件已基本解决，标准托管运行器已完全恢复，私有网络托管运行器仍在恢复中。
