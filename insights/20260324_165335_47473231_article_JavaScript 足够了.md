## 基本信息

**Gea** 是一个轻量级的编译时响应式 UI 框架，旨在让开发者使用普通 JavaScript 类和函数即可获得响应式能力，无需学习新的原语或概念。

- **GitHub**: https://github.com/dashersw/gea
- **npm**: @geajs/core
- **许可证**: MIT
- **创建者**: Armagan Amcalar

## 核心特点

### 1. 零概念设计
无需学习 signals、hooks、依赖数组或编译器指令。开发者只需要掌握 JavaScript 类和函数即可。

### 2. 编译时响应式
Vite 插件在构建时分析 JSX，确定哪些 DOM 节点依赖哪些状态，并连接精准的补丁——无感知地完成。

### 3. 无虚拟 DOM
更新直接作用于发生变化的元素，无需 diffing、reconciliation 或额外开销。在编译时直接生成 DOM 突变代码。

### 4. 基于代理的存储
状态存储在普通类中，由深度代理包装。可以直接修改属性——数组方法、嵌套对象都能正常工作。

### 5. 零运行时依赖
内置状态管理和路由功能，gzip 后约 13 kb（包含路由），约 10 kb（不含路由）。无需额外包。

## 代码示例

```typescript
// store.ts - 状态存储
import { Store } from '@geajs/core'

class CounterStore extends Store {
  count = 0
  
  increment() { this.count++ }
  decrement() { this.count-- }
}

export default new CounterStore()
```

```tsx
// Counter.tsx - 组件
import { Component } from '@geajs/core'
import store from './store'

export default class Counter extends Component {
  template() {
    return (
      <div class="counter">
        <span>{store.count}</span>
        <button click={store.increment}>+</button>
        <button click={store.decrement}>-</button>
      </div>
    )
  }
}
```

## 核心功能

- **类组件**: 扩展 Component，添加本地响应式状态，实现 template()
- **函数组件**: 导出返回 JSX 的函数，在构建时转换为类组件
- **客户端路由**: 内置 RouterView、Link、路由参数、通配符和编程式导航
- **事件委托**: 所有事件处理器编译为每种事件类型的单个委托监听器
- **细粒度数组**: push、splice、sort 等数组方法产生细粒度变更事件
- **Gea UI**: 35+ 基于 Zag.js 的可访问组件（对话框、菜单、选择器、标签页等）
- **Gea Mobile**: ViewManager、侧边栏、标签视图、下拉刷新、无限滚动、手势等
- **HMR 支持**: Vite 插件提供热模块替换
- **AI 辅助**: 附带 AI 技能，帮助 Cursor、Codex 等 AI 助手学习完整的 Gea API

## 总结

Gea 是一个创新的编译时 UI 框架，通过将普通 JavaScript 类和函数转化为响应式代码，实现了无虚拟 DOM 的精准 DOM 更新。其极小的包体积、零运行时依赖和卓越的性能使其成为现代 Web 应用开发的优秀选择。
