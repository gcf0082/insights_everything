# 荷兰中央银行选择Lidl云服务：欧洲云主权的里程碑

## 基本信息

- **洞察链接**: https://www.techzine.eu/news/infrastructure/140634/dutch-central-bank-chooses-lidl-for-european-cloud/
- **发布日期**: 2026年4月20日
- **来源**: Techzine Global
- **作者**: Berry Zwets
- **标签**: cloud sovereignty, De Nederlandsche Bank, DNB, European cloud, Lidl, Schwarz Digits, Stackit

## 核心洞察

荷兰中央银行（DNB）将于明天与Schwarz Digits（Lidl所属的Schwarz Group的IT部门）签署一份重大合同。DNB旨在减少对美国云公司的依赖，作为一个大型荷兰组织，它选择了一个欧洲合作伙伴。

## 关键要点

### 1. 战略动机
- DNB总监Steven Maijoor于2025年10月宣布打算“树立一个好榜样”，转向欧洲云服务
- 去年，DNB和荷兰金融市场管理局（AFM）警告荷兰金融业对外国IT服务提供商（尤其是美国公司）依赖过度
- 地缘政治紧张局势加剧了这种担忧

### 2. 欧洲云替代方案
- Schwarz Digits通过Stackit云平台定位为欧洲替代方案
- 所有数据均受欧洲法律管辖，这与受Cloud Act约束的美国提供商不同
- Schwarz Digits recently宣布投资110亿欧元在吕本劳建设大型数据中心

### 3. 现有客户案例
- Lidl和德国连锁超市Kaufland已在使用该云服务
- Deutsche Bahn也与Schwarz Group合作
- SAP和拜仁慕尼黑也是客户
- 荷兰政府最近也选择了Stackit

### 4. 面临的挑战
- 欧洲云服务在成熟度方面仍落后于美国竞争对手（美国云服务商有长达20年的开发优势）
- 石勒苏益格地区政府已遭遇从Microsoft迁移到开源环境的困难

### 5. 地缘政治背景
- 海牙国际刑事法院检察官被美国总统Trump切断Microsoft邮件账户
- ICC现也在转向非美国系统

## 行业影响

这一选择标志着欧洲云主权发展的重要转折点。作为荷兰金融业的监管机构，DNB的选择将为其他欧洲机构树立榜样，推动更多组织考虑欧洲本土云解决方案。
