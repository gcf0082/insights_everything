## 核心发现

### 研究背景

在数字硬件中，单个两输入门就能实现整个布尔逻辑。但在连续数学领域，长期以来没有类似的原始运算能够生成所有基本数学函数（如 sin、cos、sqrt、log 等）。本文打破了这一传统认知。

### 核心贡献

作者发现了一个单一的二元运算符 **EML (Exp-Minus-Log)**：

```
eml(x, y) = exp(x) - ln(y)
```

仅需这一个运算符以及常数 1，就能生成科学计算器所需的全部标准函数，包括：

- **常数**：e, π, i
- **算术运算**：加法、减法、乘法、除法、指数运算
- **超越函数**：指数函数、对数函数、三角函数等
- **代数函数**：平方根等

### 关键示例

- `exp(x) = eml(x, 1)`
- `ln(x) = eml(1, eml(eml(1, x), 1))`

### 方法论

该发现通过系统性穷举搜索获得，具有构造性地证明了其对标准科学计算器基础的适用性。

### EML形式化

在EML形式下，每个表达式都成为具有相同节点的二叉树，产生如下的简易语法：

```
S -> 1 | eml(S, S)
```

### 应用：基于梯度的符号回归

这种统一结构使得梯度-based符号回归成为可能——使用EML树作为可训练电路，配合标准优化器（如Adam），作者验证了：

- 在浅树深度（≤4）下，可以从数值数据精确恢复闭式 elementary 函数
- 同一架构也可以拟合任意数据
- 当生成法则是 elementary 函数时，可能恢复精确公式

## 总结

本文提出了一种极简且统一的数学运算框架，仅通过一个二元运算符 `exp(x) - ln(y)` 以及常数1，即可推导出科学计算器中的全部基本函数。这一发现不仅在理论上具有重要意义（证明了数学表达的极简性），而且在实践中为基于梯度的符号回归提供了新的可能架构。
