## 内容摘要

CadQuery 是一个开源 Python 库，用于构建 3D CAD 模型。该项目使用 Python 作为建模语言，基于 OpenCASCADE 内核（边界表示 BRep），允许用户通过编写代码来创建和操作三维模型。

### 核心功能与特点

1. **编程式 CAD**: 使用 Python 代码而非 GUI 进行建模，便于版本控制和参数化设计
2. **BRep 几何内核**: 与 OpenSCAD 的三角形网格不同 CadQuery 使用精确的曲面和边界表示
3. **可导出 STEP 文件**: 支持工业标准的 STEP 格式，便于与其他 CAD 软件互操作
4. **设计与查询机制**: 通过设计意图查询选择几何元素，比传统点选更加灵活

### 与 OpenSCAD 的对比

- OpenSCAD 使用三角形网格原生建模，而 CadQuery 使用精确的数学曲面表示
- CadQuery 的圆角、倒角等操作更加自然
- 导出时无需考虑分辨率问题，分辨率仅在导出时确定
- Python 语言可以访问丰富的 Python 生态系统

### 相关工具与替代品

- **build123d**: CadQuery 的姐妹库，采用不同风格的 Python API
- **Replicad**: 面向 Web 编辑器的库
- **FluidCAD**: 全功能 CAD 包代码与 UI 结合
- **modelrift.com**: 结合 OpenSCAD 和 AI 助手的工具

### 用户实践案例

1. **旋转计算手链**: 使用 build123d 制作乘法计算手链，利用对数刻度实现乘法运算
2. **张力敏感绞盘**: 使用代码式 CAD 设计
3. **Cosplay 头盔**: 从 Blender 导出 OBJ 后用代码处理厚度和切割
4. **电子元件库**: 为约1000个尺寸略有不同的电子元件生成3D模型

### 局限性

- AI 当前尚无法完全自动生成正确的 CadQuery 脚本，但 AI 自动补全可以提供帮助
- 用户希望有 GUI 工具能生成代码，而不仅仅是纯代码开发
- 需要记住所有几何选择逻辑，缺少可视化辅助

## 总结

CadQuery 为程序员提供了一种通过代码构建 3D 模型的方法，结合了传统参数化 CAD 的优势和编程的灵活性。随着 AI 代码助手的进步，这一工作流程正变得更加便捷。该项目在 3D 打印和需要参数化设计的场景中尤为有用。
