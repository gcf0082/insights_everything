## 项目概述

Agent Vault 是由 Infisical 开发的开源 HTTP 凭证代理和保险库，用于 AI 代理的安全凭证管理。该项目旨在解决 AI 代理在生产环境中安全访问凭证的问题，防止凭证通过提示注入被泄露。

GitHub 仓库: https://github.com/Infisical/agent-vault

## 核心特性

1. **代理访问模式** - 代理永不直接获取凭证，而是通过本地代理路由 HTTP 请求，凭证在网络层自动注入
2. **通用兼容** - 适用于任何支持 HTTP 的代理，包括自定义 Python/TypeScript 代理、沙盒进程、以及 Claude Code、Cursor、Codex 等编码代理
3. **静态加密** - 凭证使用 AES-256-GCM 加密，采用随机数据加密密钥（DEK），可选主密码通过 Argon2id 包装 DEK
4. **请求日志** - 记录每个代理请求的方法、主机、路径、状态、延迟和涉及的凭证密钥
5. **单一二进制** - 作为单个 Go 二进制文件提供，也可作为 Docker 容器部署

## 技术架构

- **本地转发代理** - 采用 MITM（中间人）架构，通过 HTTPS_PROXY 环境变量重定向流量
- **自建 CA** - 运行自己的证书颁发机构，需要在客户端信任存储中配置 CA 证书
- **TLS 终止** - 终止 TLS 连接以进行凭证代理，在建立新 TLS 连接上游之前检查流量
- **可扩展性** - 支持未来在此代理层添加防火墙类功能

## 安装使用

```bash
# 脚本安装 (macOS / Linux)
curl -fsSL https://get.agent-vault.dev | sh
agent-vault server -d

# Docker 部署
docker run -it -p 14321:14321 -p 14322:14322 -v agent-vault-data:/data infisical/agent-vault

# 本地代理运行
agent-vault run -- claude
agent-vault run -- codex
```
