## 报告摘要

国家安全局（NSA）正在使用Anthropic公司最先进的模型Mythos Preview，尽管国防部高层官员坚称该公司存在"供应链风险"。这一情况暴露了美国政府在人工智能政策上的内部矛盾。

## 关键发现

### 1. NSA与Pentagon的矛盾立场

- **NSA正在使用Mythos**：据两位知情人士透露，NSA正在使用Anthropic的Mythos Preview模型
- **更广泛使用**：一位消息人士称，该模型也在国防部内部更广泛地使用
- **矛盾之处**：NSA隶属于国防部，但其行动与国防部对Anthropic的立场相悖

### 2. Pentagon与Anthropic的纠纷时间线

| 时间 | 事件 |
|------|------|
| 2026年2月 | 国防部长Pete Hegseth推动Anthropic允许将Claude用于"所有合法用途" |
| 2026年2月 | 特朗普政府将Anthropic列为"供应链风险"，命令所有政府机构停止使用其服务 |
| 2026年3月 | Anthropic在两家联邦法院起诉国防部 |
| 2026年4月初 | Anthropic发布Mythos Preview |
| 2026年4月17日 | Anthropic CEO Dario Amodei与白宫幕僚长Susie Wiles和财政部长Scott Bessent会面 |
| 2026年4月19日 | Axios报道NSA正在使用Mythos |

### 3. Mythos的发布与访问

- **访问限制**：Anthropic将Mythos的访问权限限制在约40个组织
- **公开名单**：Anthropic仅公开宣布了12个合作伙伴
- **NSA身份**：据一位消息人士称，NSA属于未公开的机构之一
- **合作伙伴**：包括Amazon Web Services、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorgan Chase、Linux Foundation、Microsoft、NVIDIA和Palo Alto Networks

### 4. Mythos模型的能力

- **高能力模型**：Anthropic称其为"迄今为止最强大的编码和代理任务模型"
- **网络安全**：可扫描操作系统和软件中的漏洞
- **潜在风险**：官员们担心该模型可能被用于发动网络攻击

### 5. 各方回应

- **Anthropic和国防部**：拒绝置评
- **NSA和国家情报总监办公室**：未立即回应

## 事件意义

1. **政策矛盾**：政府网络安全需求似乎超过了与Anthropic的争端
2. **法律纠纷**：五角大楼正在法庭上争论使用Anthropic工具的必要性
3. **政府内部的分歧**：除国防部外的其他部门可能获得Anthropic的访问权限

## 结论

尽管Anthropic与Pentagon之间存在持续的法律纠纷和政策分歧，但NSA（作为国防部的一部分）已经开始使用Anthropic的Mythos Preview模型。这一情况凸显了美国政府在人工智能使用政策上的内部矛盾，以及国家安全需求与政策限制之间的张力。
