## 核心定位

Mojo 是一门新兴的编程语言，旨在结合 Python 的易用性和 C++ 的性能。其口号为"Write like Python, run like C++."，目标是为 AI 时代提供高性能编程能力，同时保持代码编写的友好性。

## 主要特性

### 1. 现代设计
- **Python 风格语法**：直观易懂的语法，学习曲线低
- **Rust 级别的内存安全**：通过所有权系统和借用检查确保内存安全
- **Zig 风格的编译时元编程**：强大的编译时求值和代码生成能力

### 2. AI 原生
- 专为现代 AI 系统所需的异构硬件（CPU、GPU、ASIC）设计
- 静态类型的编译型语言，适合智能体编程（agentic programming）
- 无需供应商特定的库即可编写 GPU 代码

### 3. Python 互操作性
- 可直接导入 Python 代码和库
- Mojo 代码可自然地打包成分布式 Python 包
- 支持渐进式优化：先用 Python 编写，然后用 Mojo 重写性能瓶颈部分

### 4. 编译时元编程
- 运行时和编译时代码使用同一语言
- 支持条件编译、硬件特定优化
- 通过编译时求值确保内存安全，消除运行时开销

### 5. GPU 编程
- 能够在同一语言中编写 CPU 和 GPU 代码
- 无需vendor lock-in
- 可直接编写高性能 GPU 内核

## 技术路线图

Mojo 于 2022 年底启动，分为四个阶段：

| 阶段 | 状态 | 描述 |
|------|------|------|
| Phase 0 | 完成 | 初始构建：核心解析器、内存类型、函数、结构体等基础 |
| Phase 1 | 进行中 | 高性能 CPU + GPU 编程 |
| Phase 2 | 规划中 | 系统应用程序编程，支持内存安全模型 |
| Phase 3 | 规划中 | 动态面向对象编程，支持 Python 的动态特性 |
