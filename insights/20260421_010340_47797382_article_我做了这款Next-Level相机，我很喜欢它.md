## 文章摘要

本文作者分享了自己动手打造一台"终极"相机的完整过程。这台相机的核心特点是拥有超大的光圈（约125mm，等效于f/1.0甚至更大的光圈），能够实现极浅的景深和惊人的背景虚化效果。

## 核心内容

### 1. 背景与原理

摄影中，光圈越大，景深越浅，背景虚化效果越强。然而，手机摄像头由于镜头尺寸限制，无法达到专业相机那种刀锐奶化的背景虚化效果。传统大光圈镜头（如Sigma 135mm f/1.4）的光圈直径约为96mm，但这类镜头变焦很大（等于135mm定焦），难以广角拍摄。

### 2. 核心挑战

- **物理限制**：大光圈 + 广角 = 不可能实现的组合，因为光线需要穿透相机金属机身才能到达传感器
- **现有方案**：Stanley Kubrick和Media Division曾尝试改造相机，但都未达到理想效果
- **成本问题**：专业级大光圈镜头价格极其昂贵

### 3. 解决方案：自己动手

#### 主要组件

1. **镜头**：Charles Beseler 18" Series III投影镜头
   - 估计光圈：约125mm
   - 焦距：457mm（13倍变焦）
   - 购买价格：约200欧元（含运费）

2. **传感器尺寸放大**：需要将传感器面积放大12倍
   - 原传感器：35mm x 24mm（全画幅）
   - 目标尺寸：420mm x 288mm（约42cm x 29cm）

3. **关键技术**：景深适配器（Depth-of-field adapter）
   - 原理：先通过大镜头在"假传感器"（半透明漫射屏）上成像
   - 然后用另一台相机拍摄这个"假传感器"上的图像

#### 漫射材料测试

| 材料 | 效果 |
|------|------|
| 纸 | 不行，完全吸光 |
|  Papier-mâché | 透光太少，纹理过重 |
| 烘焙纸 | 勉强可用，透光性一般 |
| IKEA PUCKELFLY | 可用，但有纹理 |
| 蜡层 | 理论最佳，但难以均匀涂抹 |
| 自制磨砂玻璃 | 效果最佳，但操作复杂 |
| **摄影漫射膜** | 最终选择：Lee Filters - 251 |

#### 其他组件

- **框架**：40x30cm画框
- **皮腔**：使用IKEA SCHOTTIS窗帘自制
- **聚光透镜**：菲涅尔透镜（Fresnel lens），40x30cm，从Aliexpress购买
- **支架**：金属滑轨 + 望远镜支架

### 4. 最终成果

作者将整个系统命名为**Lampone**，成功实现了：
- 在42cm x 29cm的"假传感器"上获得清晰图像
- 极浅景深的电影级画面效果
- 已用于短片拍摄

## 改进方向

1. **更长的皮腔和滑轨**：缩短最近对焦距离
2. **结构加固**：减少拍摄时的抖动
3. **更好的漫射材料**：消除画面纹理
4. **光线损失**：约3档光圈衰减，需要强力补光

## 结论

这篇文章展示了通过DIY方式突破商业相机限制的完整思路。尽管系统体积庞大（120cm长）、重量沉重，且需要良好的照明条件，但成功实现了商业方案无法达到的超大光圈效果。
