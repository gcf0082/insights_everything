## 概述

本文是作者在FPGA上运行Quake II游戏系列的第四部分，主要介绍了新一代电路板的设计与实现过程。新一代板载采用了更先进的BGA封装芯片，包括Efinix Ti60F256 FPGA和DDR3L内存，并实现了完整的片上系统（SoC）设计。

---

## 电路板设计

### 元器件选择

作者选择了以下核心元器件：

- **FPGA**：Efinix Ti60F256，256引脚BGA封装，0.8mm间距
- **内存**：IS8G16D3FFBG，1GB DDR3L芯片，96引脚BGA封装
- **TMDS serializer**：TFP410，用于避免之前版本的绿色伪影问题
- **SD卡**：支持UHS-1 SDR104模式，理论传输速度可达104 MB/s
- **实时时钟**：带电池，防止Linux启动时时间重置为1970年
- **ESP32模块**：作为WiFi适配器
- **电源管理**：增加第二个USB-C端口，可请求更高电压

### PCB设计

- 采用6层PCB设计（相比之前的两层板）
- 参考DDR3 PCB布局建议，进行严格的走线长度匹配
- 补偿不同层之间的信号传播速度差异
- 在FPGA下方焊接去耦电容，采用0603规格

## BGA焊接工艺

### 焊接流程

1. **表面贴装**：在焊盘上涂覆焊膏，放置芯片，加热焊接
2. **底部加热器**：使用 Uyue 946 恒温加热台，加热至221°C
3. **先焊正面**：底部加热器只能焊接正面元件
4. **后焊背面**：使用热风枪焊接背面元件，使用低熔点焊膏（183°C）

### 注意事项

- 焊膏熔点必须与芯片焊球熔点匹配
- 作者在练习板上进行了多次测试
- 第一次尝试即成功，仅在个别相邻引脚处有桥接，可轻松分离

## 片上系统（SoC）设计

### 架构组件

```
┌─────────────────────────────────────────┐
│           FPGA Ti60F256                 │
│  ┌─────────────────────────────────┐    │
│  │         VexiiRiscv              │    │
│  │     (RISC-V 处理器核心)          │    │
│  └─────────────────────────────────┘    │
│  ┌──────────┐ ┌──────────┐ ┌────────┐  │
│  │ DDR3     │ │ Video    │ │ DMA    │  │
│  │ Controller│ │ Controller│ │Controller│  │
│  └──────────┘ └──────────┘ └────────┘  │
│  ┌──────────┐ ┌──────────┐ ┌────────┐  │
│  │ UART     │ │ I2C      │ │ SPI    │  │
│  │ Controller│ │ Controller│ │Controllers│  │
│  └──────────┘ └──────────┘ └────────┘  │
│  ┌──────────┐ ┌──────────┐             │
│  │ USB      │ │ Audio    │             │
│  │ OHCI     │ │ Controller│             │
│  └──────────┘ └──────────┘             │
└─────────────────────────────────────────┘
```

### 使用的IP核心

- **VexiiRiscv**：RISC-V核心，使用SpinalHDL编写
- **SpinalHDL**：Tilelink互连、APB3桥接、USB OHCI控制器、PLIC
- **ZipCPU/sdspi**：SD卡控制器
- **Efinix DDR3 Soft Controller Core**：DDR3控制器

### 开发语言

作者逐步从Verilog转向SpinalHDL（基于Scala），认为SpinalHDL在处理硬件抽象和代码生成方面更为强大。

## 性能基准测试

| 指标 | 数值 |
|------|------|
| FPGA利用率 | 89% (53612/60800 XLR) |
| 内存利用率 | 95% (241/256 内存块) |
| CPU频率 | 207 MHz |
| Dhrystone | 511 DMIPS (2.46 DMIPS/MHz) |
| Coremark 1.0 | 783.08 |
| 内存带宽 | 1 GB |
| 浮点性能 | 207 MFLOPS (峰值) / 117 MFLOPS (实测) |
| SD卡读取 | 45.06 MB/s |
| CPU memset | 348 MB/s |
| DMA memset | 1130 MB/s (理论极限1.6 GB/s) |

### 性能对比

- Dhrystone性能：超越Pentium，接近Pentium II
- 内存容量：超过普通Pentium II PC
