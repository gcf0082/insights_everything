## 研究摘要

为了应对不断增长的人口，全球农业粮食系统需要更高效地将作物生产转化为可供人类食用的热量。本研究量化了50种作物在2010年至2020年期间的用途——包括食用、饲料、生物燃料和其他非食品用途——对可用热量的影响。研究发现，虽然总热量生产增加了23.9%，但可供人类食用的系统可用热量仅增加了16.6%。这种效率下降是由于用于牲畜饲料（31.2%）和非食品用途（36.2%）的热量增加所致。2020年，仅有一半（50.1%）的农地生产热量可供人们食用。

## 核心发现

### 1. 全球热量效率现状
- **总热量生产**: 2010-2020年间增加23.9%
- **可用热量**: 仅增加16.6%
- **损失热量**: 49.9%，相当于每年7.2×10^15千卡，足以支持72亿人

### 2. 热量用途变化（2010-2020）
| 用途类别 | 增长比例 |
|---------|---------|
| 直接食用 | +14.9% |
| 牲畜饲料 | +31.2% |
| 非食品用途 | +36.2% |
| 生物燃料 | +27.9% |

### 3. 饲料转化效率
牛肉生产是热量损失最大的来源：
- 消耗了36.0%的饲料热量
- 仅产出9.1%的动物源热量
- 饲料与食物转化比高达33:1

乳制品效率最高，转化比为2.5:1。

### 4. 地区差异
2020年热量生产前五国家/地区：
1. 美国
2. 中国
3. 印度
4. 巴西
5. 欧盟27国

这些地区占全球农地热量生产的57%。

### 5. 饮食改善潜力
如果48个高收入国家将牛肉消费降低至EAT-Lancet健康参考饮食定义的健康水平：
- 用鸡肉替代牛肉，可避免损失8.51×10^15千卡
- 足够满足8.51亿人的年热量需求
- 美国(37%)和巴西(21%)占全球潜力的58%

## 研究方法

### 分析范围
- 50种主要作物（占全球热量生产的97.6%）
- 三年平均值以减少波动（例：2020年为2019-2021年平均）

### 50种作物分类
| 作物类别 | 全球热量占比 |
|---------|-----------|
| 谷物 | 62.6% |
| 油料作物 | 19.2% |
| 根茎和块茎 | 5.2% |
| 糖料作物 | 5.0% |
| 水果 | 2.1% |
| 豆类 | 1.9% |
| 蔬菜和瓜类 | 1.5% |
| 其他 | 0.2% |

### 数据来源
- FAO供给利用账户(SUA)
- OECD-FAO数据浏览器
- FAOSTAT作物生产数据

## 环境与健康意义

### 牛肉的多重影响
1. **低效的土地利用**: 饲料转化效率最低
2. **高温室气体排放**: 对气候变化影响不成比例
3. **健康风险**: 心血管疾病的主要风险因素之一

### 政策建议
研究和结果表明，少数商品（尤其是牛肉和猪肉）和少量国家主导了当前农地利用的低效性。针对这些商品和国家的行动和政策可对改善粮食安全、健康和环境产生显著影响。

## 结论

本研究揭示了全球农业粮食系统效率的严峻现实：仅有一半的农地热量可供人类食用。通过减少低效商品（尤其是牛肉）的消费并转向更有效的蛋白质来源，可以显著提高全球粮食系统的效率，从而改善粮食安全、保护环境并促进人类健康。
