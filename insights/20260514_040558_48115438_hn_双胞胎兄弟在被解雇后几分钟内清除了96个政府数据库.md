## 事件概述

一对双胞胎兄弟在被解雇数分钟后，删除了96个政府数据库。这对兄弟是Muneeb Akhter和Sohaib Akhter，他们曾是Opexus公司的IT承包商，负责维护多个政府机构的数据库系统。

## 关键细节

1. **数据泄露规模**:
   - 在一小时内删除了约96个包含美国政府信息的数据库
   - 下载了1,805个EEOC（平等就业机会委员会）的文件
   - 获取了至少450人的联邦税务信息
   - 掌握了5,400个用户账户的用户名和密码

2. **密码安全问题**:
   - 密码以明文形式存储在数据库中
   - 其中一个兄弟曾向另一个兄弟索要投诉人的明文密码，用于未经授权访问其电子邮件账户

3. **破坏行为**:
   - 使用"DROP DATABASE dhsproddb"命令删除了国土安全部的数据库
   - 向AI工具询问如何清除SQL服务器和Windows Server 2012的系统日志

4. **后续发现**:
   - 警方在Sohaib家中发现了7支枪械和370发子弹
   - 鉴于其之前的犯罪记录，他本不应持有这些武器

## 社区讨论焦点

1. **密码存储方式**: 网友对明文存储密码的做法表示震惊，普遍认为应使用加盐哈希处理

2. **员工解雇流程**: 讨论是否应在解雇时立即终止所有访问权限，还是保留部分权限以便完成离职交接

3. **安全审计问题**: 质疑为何SOC2审计未发现这些明显的安全漏洞

4. **政府系统问题**: 惊讶于国土安全部使用Windows Server 2012（已接近支持期限）

5. **招聘审查质疑**: 质疑为何有犯罪前科的人能被雇佣从事敏感的政府IT工作
