## 项目概述

TRUST 是一个实验性的复古终端用户界面（TUI）集成开发环境，专门为 Rust 项目设计。该项目的灵感来源于经典的蓝屏 DOS 开发环境，如 Turbo Pascal 和 Borland C++，旨在让 Rust 编程体验回归到 1989 年的编程风格。

TRUST 不仅仅是一个怀旧项目，它具备完整的 IDE 功能，可以编辑文件、浏览 Rust 项目结构，并执行 Cargo 命令。项目状态标注为"experimental"（实验性），属于个人兴趣项目。

## 核心功能

### 1. 文件编辑
- 文本编辑器支持基本的编辑操作
- 脏缓冲区标记：未保存的文件在编辑器标题栏显示 `*` 号
- 保存文件：`F2` 或 `Ctrl+S`
- 打开文件：`F3` 或 `Ctrl+O`

### 2. 项目浏览
- 项目窗格显示目录结构
- 支持编辑的文件类型：`.rs`、`.toml`、`.lock`
- 自动过滤：忽略 `.git`、`target` 和常见的编辑器/构建目录
- 返回上级目录：`Backspace`

### 3. Cargo 命令集成
| 快捷键 | 命令 | 功能 |
|--------|------|------|
| `F5` / `Ctrl+R` | `cargo run` | 运行程序 |
| `F7` | `cargo check` | 检查代码 |
| `F8` / `Ctrl+T` | `cargo test` | 运行测试 |
| `F9` / `Ctrl+B` | `cargo build` | 构建项目 |

### 4. 菜单系统
- `F10` 打开菜单栏
- 左右箭头切换菜单
- 上下箭头浏览下拉选项
- 回车激活选中的菜单项
- `Esc` 关闭菜单
- `File > New`：创建新文件
- `Project > New project`：创建新 Cargo 项目
- `Window`：切换窗格焦点

### 5. 文本操作
- `Ctrl+C`：复制选中文本
- `Ctrl+V`：粘贴剪贴板内容
- `Ctrl+X`：剪切选中文本
- `Alt+X`：删除当前行
- `Alt+U`：复制当前行
- `Shift+导航`：选择文本
