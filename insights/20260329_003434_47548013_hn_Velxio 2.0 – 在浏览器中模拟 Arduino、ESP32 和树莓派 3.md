## 项目概述

Velxio 2.0 是一个免费、开源的嵌入式开发板模拟器，支持在浏览器中运行真实的编译代码。该项目完全本地化运行，无需云端依赖，无需学生账户，数据不会离开用户网络，可通过单个Docker容器自托管。

## 主要功能

- **支持19种嵌入式开发板**：Arduino、ESP32、Raspberry Pi、RISC-V等
- **5种CPU架构**：AVR8、Xtensa、RISC-V、ARM Cortex-M0+、ARM Cortex-A53
- **48+电子元件**：LED、按钮、舵机、显示屏（ILI9341 TFT）等
- **真实传感器模拟**：DHT22、HC-SR04、WS2812B NeoPixel
- **零配置**：无需安装工具链，无需熟悉SSH或终端

## v2.0新特性

| 特性 | 说明 |
|------|------|
| ESP32模拟 | 通过QEMU（lcgamboa分支）实现，支持真实flash镜像、ROM函数模拟、GPIO/ADC/定时器 |
| RISC-V核心 | ESP32-C3和CH32V003使用TypeScript编写的自定义RISC-V核心，完全在浏览器中运行 |
| Raspberry Pi 3 | 通过QEMU raspi3b模拟，可启动真实Pi OS并运行Python |
| 架构分离 | AVR、RP2040、RISC-V在客户端运行；ESP32 Xtensa和Pi 3在后端QEMU运行 |

## 技术架构

- **前端**：React + Vite
- **后端**：FastAPI
- **模拟器**：avr8js、rp2040js、自定义TypeScript核心
- **编译**：真实arduino-cli
- **部署**：Docker自托管

## 社区反馈亮点

1. **可访问性**：无需安装，无需配置，适合教学和初学者
2. **移动端支持**：在旧Android设备上也能良好运行
3. **示波器功能**：内置示波器可监控主板和引脚
4. **多板卡交互**：支持同一电路中多个异构开发板（如两个Arduino通过SPI连接）

## 与Wokwi的对比

- **相似之处**：使用wokwi-elements进行可视化SVG渲染
- **核心差异**：支持多异构开发板、完整模拟（包括ESP32 via QEMU和Raspberry Pi 3运行Linux）
