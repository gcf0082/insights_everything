## 重点项目

### 1. Sapphire - Ruby风格类型化编程语言
- **描述**: 受 Sorbet/Ruby 启发的类型化编程语言，带有渐进类型系统
- **地址**: https://github.com/sapphire-project/sapphire
- **特点**: 使用 Claude Code 开发，包含版本管理器 facet

### 2. Still Kicking - 老年人监护设备
- **描述**: 使用 mmWave 传感器和 ESP32 监测独居老人活动、跌倒和睡眠质量
- **地址**: https://moveometer.com
- **特点**: 无摄像头隐私保护，内置蜂窝数据模块

### 3. Jikimi - 隐私优先的家长控制平台
- **描述**: 基于本地 OCR 引擎(PaddleOCR)和 AI 情感分析的儿童网络保护工具
- **地址**: https://jikimi.app
- **特点**: 纯本地处理，数据不离开设备，支持 DNS 过滤和游戏时间控制

### 4. Tiled Words - 每日文字解谜游戏
- **描述**: 每日拼图益智游戏，用户可制作和分享谜题
- **地址**: https://tiledwords.com
- **成就**: 获得 Playlin Player's Choice 奖，每日数千玩家

### 5. Rail Raptor - 英国铁路离线路线规划
- **描述**: 完全离线运行的浏览器端英国铁路路线规划工具
- **地址**: https://railraptor.com
- **创新**: 使用 Raptor 算法，能发现标准运营商未提供的路线

### 6. crit - AI 编码代理输出审查工具
- **描述**: CLI 工具，用于像 GitHub PR 一样审查 AI 编码代理输出
- **地址**: https://github.com/tomasz-tomczyk/crit

### 7. x509dump - X.509 证书解码器
- **描述**: 将证书和 CSR 解码为结构化 JSON 格式
- **地址**: https://www.redkestrel.co.uk/products/x509dump

### 8. cardcast.gg - 远程万智牌游戏平台
- **地址**: https://cardcast.gg
- **特点**: 使用摄像头进行远程游戏，采用 transformer 模型进行视觉识别

### 9. Petrify - JVM 机器学习模型编译器
- **描述**: 将 ONNX 模型编译为 JVM 字节码，零依赖
- **地址**: https://github.com/exabrial/petrify

## 主题趋势

1. **隐私保护应用兴起**: 多个项目强调本地处理、离线运行、隐私优先
2. **AI 辅助开发普及**: 开发者大量使用 Claude Code、Cursor 等 AI 工具快速构建项目
3. **个人/家庭需求驱动**: 许多项目源于开发者个人或家庭需求(如老年人监护、家长控制、宠物卡牌管理)
4. **复古/怀旧项目**: 3D 蒸汽朋克时钟、Space Trader 游戏等
5. **开发者工具改进**: Git 工作流工具、证书解码器、AI 输出审查工具
