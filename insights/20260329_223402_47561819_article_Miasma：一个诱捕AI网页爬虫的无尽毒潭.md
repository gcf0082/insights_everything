## 项目简介

Miasma 是一个用于对抗AI网络爬虫的工具。它通过设置"毒药喷泉"（poison fountain），向爬取数据的AI发送被污染的训练数据，从而保护网站内容不被未经授权地用于AI模型训练。

## 核心功能

1. **毒药数据生成**：向AI爬虫返回被污染的训练数据
2. **自我引用链接**：在响应中包含多个自我引用的链接，使爬虫陷入无限循环
3. **高性能**：采用Rust开发，内存占用极低
4. **可配置**：支持自定义端口、主机、链接前缀、链接数量等配置

## 技术特点

- **语言**：Rust（高性能、低内存占用）
- **部署方式**：通过cargo安装或下载预编译二进制文件
- **依赖**：可从crates.io获取，已发布到 crates.io
- **CI/CD**：包含GitHub Actions工作流进行自动化检查

## 使用场景

Miasma 旨在帮助网站所有者对抗大规模的AI网络爬虫。通过以下方式部署：
1. 在网站中嵌入隐藏的爬虫陷阱链接
2. 配置反向代理（如Nginx）将爬虫流量重定向到Miasma
3. Miasma会向爬虫返回无限循环的毒药数据

## 配置选项

| 选项 | 默认值 | 说明 |
|------|--------|------|
| port | 9999 | 服务器绑定端口 |
| host | localhost | 服务器绑定地址 |
| max-in-flight | 500 | 最大并发请求数 |
| link-prefix | / | 自我引用链接前缀 |
| link-count | 5 | 每个响应包含的链接数 |
| force-gzip | false | 强制gzip压缩 |
| poison-source | rnsaffn.com/poison2/ | 毒药数据来源 |

## 安全说明

建议在robots.txt中保护友好的爬虫（如Googlebot、Bingbot等）避免误入陷阱。
