## 项目概述

Nightingale 是一款自包含的派对游戏，可将任何歌曲变成卡拉OK体验。该应用能够分离人声、转录歌词，并通过逐字同步和音高评分进行播放。

## 核心功能

### 1. 人声分离 (Stem Separation)
- 使用 UVR Karaoke 模型或 Demucs 进行人声分离
- 可调节主唱音量

### 2. 逐字级歌词 (Word-level Lyrics)
- 使用 WhisperX 转录并对齐每个词到音频
- 优先使用 LRCLIB 的现有歌词

### 3. 音高评分 (Pitch Scoring)
- 通过麦克风实时演唱并获得评分
- 星级评分和每首歌的排行榜追踪进度

### 4. 玩家档案 (Player Profiles)
- 支持多个配置文件
- 独立的分数历史记录
- 切换歌手时不会丢失任何人的记录

### 5. 视频文件支持 (Video File Support)
- 支持 .mp4 和 .mkv 文件
- 自动分离人声并播放原始视频作为背景

### 6. 动态背景 (Dynamic Backgrounds)
- GPU 着色器效果（极光、星云等）
- Pixabay 视频循环
- 视频文件的源视频

### 7. 游戏手柄支持 (Gamepad)
- 导航菜单、选择歌曲和控制播放
- 支持 D-pad、摇杆和面部按钮

### 8. 单二进制部署 (Single Binary)
- ffmpeg、Python、PyTorch 和 ML 模型在首次启动时自动安装
- 无需手动安装任何依赖

## 工作原理

### 第一步：分离
UVR Karaoke 或 Demucs 将音轨分离为人声和伴奏。视频文件会自动提取音频。

### 第二步：转录
首先在 LRCLIB 上查找同步歌词。如果未找到，则使用 WhisperX 转录人声并进行逐字对齐。

### 第三步：播放
伴奏播放时带有高亮歌词、音高评分、动态背景和游戏手柄支持。
