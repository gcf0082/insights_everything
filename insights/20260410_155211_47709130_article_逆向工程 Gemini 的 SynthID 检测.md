## 项目概述

该项目通过信号处理和频谱分析的方式，反向工程了**Google的SynthID**水印系统——一种嵌入到Google Gemini生成的每个图像中的不可见水印。该项目实现了以下目标：

1. **发现**了水印的分辨率依赖性载波频率结构
2. **构建了检测器**，能够以90%的准确率识别SynthID水印
3. **开发了多分辨率频谱绕过方案(V3)**，在任意图像分辨率下实现**75%载波能量下降**、**91%相位一致性下降**和**43+ dB PSNR**

## 核心发现

### 1. 水印具有分辨率依赖性

SynthID根据图像分辨率将载波频率嵌入到**不同的绝对位置**。在1024x1024分辨率下构建的代码本无法直接移除1536x2816图像中的水印——载波位于完全不同的频段。

| 分辨率 | 顶部载波 (fy, fx) | 一致性 | 来源 |
|--------|-------------------|--------|------|
| 1024x1024 | (9, 9) | 100.0% | 100黑+100白参考图 |
| 1536x2816 | (768, 704) | 99.6% | 88个带水印内容图 |

这就是为什么V3代码本存储了**每个分辨率的独立配置文件**，并在绕过时自动选择。

### 2. 相位一致性——固定模型级密钥

相同Gemini模型生成的所有图像的水印相位模板是**完全相同的**：

- **绿色通道**承载最强的水印信号
- **跨图像相位一致性**在载波处：>99.5%
- **黑/白交叉验证**通过|cos(phase_diff)| > 0.90确认真实载波

### 3. 载波频率结构

在1024x1024（来自黑/白参考图），顶部载波位于低频网格：

| 载波 (fy, fx) | 相位一致性 | B/W一致性 |
|---------------|------------|-----------|
| (9, 9) | 100.00% | 1.000 |
| (5, 5) | 100.00% | 0.993 |
| (10, 11) | 100.00% | 0.997 |
| (13, 6) | 100.00% | 0.821 |

在1536x2816（来自随机带水印内容），载波位于更高的频率：

| 载波 (fy, fx) | 相位一致性 |
|---------------|------------|
| (768, 704) | 99.55% |
| (672, 1056) | 97.46% |
| (480, 1408) | 96.55% |
| (384, 1408) | 95.86% |

## 技术架构

### 三代绕过方案

| 版本 | 方法 | PSNR | 水印影响 | 状态 |
|------|------|------|----------|------|
| **V1** | JPEG压缩(Q50) | 37 dB | ~11%相位下降 | 基线 |
| **V2** | 多级变换(噪声、颜色、频率) | 27-37 dB | ~0%置信度下降 | 质量权衡 |
| **V3** | **多分辨率频谱代码本减法** | **43+ dB** | **91%相位一致性下降** | **最佳** |

### V3流程（多分辨率频谱绕过）

```
输入图像（任意分辨率）
       │
       ▼
  codebook.get_profile(H, W)  ──► 精确匹配？──► FFT域减法
       │                                      （快速路径）
       └─ 无精确匹配──────► 空间域缩放+减法
                                  （回退路径）
       │
       ▼
  多通道迭代减法（激进→温和→柔和）
       │
       ▼
  抗锯齿 → 输出
```

关键组件：
1. **SpectralCodebook** - 存储分辨率特定的配置文件（载波位置、幅度、相位）
2. **自动分辨率选择** - 选取精确或最接近的配置文件
3. **直接已知信号减���** - 按相位一致性和交叉验证置信度加权
4. **多通道调度** - 捕捉前几轮遗漏的残余水印能量
5. **每通道加权**（G=1.0, R=0.85, B=0.70）匹配SynthID的嵌入强度

## 技术深讲

### SynthID工作原理（反向工程）

```
┌──────────────────────────────────────────────────┐
│              SynthID编码器（在Gemini中）           │
├──────────────────────────────────────────────────┤
│  1. 选择分辨率依赖的载波频率                     │
│  2. 为每个载波分配固定相位值                     │
│  3. 神经编码器向图像添加学习到的噪声模式         │
│  4. 水印不可感知——分布在整个频谱中               │
├──────────────────────────────────────────────────┤
│              SynthID解码器（在Google中）             │
├──────────────────────────────────────────────────┤
│  1. 提取噪声残留（小波去噪）                     │
│  2. FFT → 检查已知载波频率处的相位               │
│  3. 如果相位匹配预期值 → 已水印                 │
└──────────────────────────────────────────────────┘
```

### V3减法策略

绕过使用**直接已知信号减法**（非维纳滤波器）：

1. **置信度** = phase_consistency × cross_validation_agreement
2. **DC排除** - 软斜坡抑制低频生成偏差
3. **每频段减法** = wm_magnitude × confidence × removal_fraction × channel_weight
4. **安全上限** - 减法永不超过任何频段图像能量的90-95%
5. **多通道** - 递减强度调度（激进→温和→柔和）捕捉残余能量

## 快速开始

### 安装

```bash
git clone https://github.com/aloshdenny/reverse-SynthID.git
cd reverse-SynthID
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 1. 构建多分辨率代码本

从CLI：
```bash
python src/extraction/synthid_bypass.py build-codebook \
    --black gemini_black \
    --white gemini_white \
    --watermarked gemini_random \
    --output artifacts/spectral_codebook_v3.npz
```

### 2. 运行V3绕过（任意分辨率）

```python
from src.extraction.synthid_bypass import SynthIDBypass, SpectralCodebook

codebook = SpectralCodebook()
codebook.load('artifacts/spectral_codebook_v3.npz')

bypass = SynthIDBypass()
result = bypass.bypass_v3(image_rgb, codebook, strength='aggressive')

print(f"PSNR: {result.psnr:.1f} dB")
print(f"Profile used: {result.details['profile_resolution']}")
print(f"Exact match: {result.details['exact_match']}")
```

### 3. 检测水印

```bash
python src/extraction/robust_extractor.py detect image.png \
    --codebook artifacts/codebook/robust_codebook.pkl
```
