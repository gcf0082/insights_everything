## 执行摘要

Cloudflare 宣布将全面后量子安全的目标时间从原计划提前至**2029年**。这一重大调整源于近期量子计算领域的突破性进展，特别是 Google 宣布大幅改进破解椭圆曲线密码学的量子算法，以及 Oratomic 公布的研究表明破解 P-256 仅需约 10,000 个量子比特。

## 关键发现

### 1. 量子威胁时间线大幅提前

- **Oratomic 研究**: 破解 P-256（广泛用于保护互联网）仅需 10,000 个中性原子量子比特
- **Google 突破**: 大幅提升了破解 P-256 的算法效率
- **行业反应**: Google 将其迁移时间表加速至 2029 年
- **IBM 预测**: 不排除 2029 年就可能出现针对高价值目标的量子"登月攻击"

### 2. 三大技术前沿的独立进展

| 前沿 | 进展 |
|------|------|
| **硬件** | 中性原子和超导量子比特接近可扩展状态，多种技术路线并行推进 |
| **纠错** | 中性原子架构仅需 3-4 个物理量子比特即可实现 1 个逻辑量子比特 |
| **软件** | 量子算法破解效率显著提升，架构特定优化进一步降低资源需求 |

### 3. 关注重点从加密转向认证

- **历史焦点**: 后量子加密（缓解"现在收集，以后解密"攻击）
- **新优先级**: 后量子认证（防止量子攻击者伪造凭证冒充服务器）
- **原因**: 如果 Q-Day 临近，数据泄露严重，但认证被破解将是灾难性的

### 4. 高价值目标优先保护

- **长期密钥**: 根证书、API 密钥、代码签名证书等应优先升级
- **依赖链**: 第三方依赖（包括金融服务和公用事业）也需要评估
- **降级防护**: 必须禁用量子脆弱的密码学协议以防止降级攻击

## Cloudflare 路线图

| 时间 | 里程碑 |
|------|--------|
| 2014 | 提供免费通用 SSL 证书 |
| 2019 | 开始后量子迁移准备 |
| 2022 | 为所有网站和 API 启用后量子加密 |
| **2029** | **全面后量子安全（包括认证）** |

## 建议

### 对企业
- 将后量子支持作为采购的必备要求
- 评估关键供应商的量子安全准备状态
- 优先升级长期密钥

### 对监管机构和政府
- 指定牵头机构协调迁移工作
- 保持安全聚焦，推动使用现有国际标准
- 避免恐慌，自信引导迁移

### 对 Cloudflare 客户
- 无需采取缓解措施
- Cloudflare 将自动开启后量子安全（默认启用）
- 需关注浏览器、应用和源站的升级

## 结论

随着量子计算突破使 Q-Day 大幅提前，后量子认证已成为最紧迫的优先事项。迁移工作需要数年时间，而非数月。Cloudflare 承诺其所有后量子升级将对所有客户免费提供，使后量子安全成为互联网的新标准。
