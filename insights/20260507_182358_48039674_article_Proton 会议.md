## 产品概述

**Proton Meet** 是Proton推出的一款端到端加密视频会议服务，旨在为工作和生活中的私密对话提供隐私保护。与Zoom、Google Meet、Microsoft Teams等主流视频会议平台不同，Proton Meet 使用端到端加密技术，确保没有人能够窃听用户的通话内容，包括音频、视频、屏幕共享和聊天消息。

---

## 核心要点

### 1. 为什么需要隐私优先的视频会议？

- **数据隐私风险**：在线会议的元数据、聊天记录和通话本身都可能被拦截、泄露或暴露
- **AI数据利用**：科技巨头正在将音频、视频或聊天数据用于训练AI模型，私人对话片段可能泄露或重新出现在AI生成的内容中
- **法律合规挑战**：美国CLOUD Act等法律可以强制美国-owned的视频会议平台交出数据，即使服务器位于美国境外，这对受GDPR、CCPA等数据保护法律约束的组织构成严重合规挑战

### 2. 主要功能

| 功能 | 说明 |
|------|------|
| 端到端加密 | 所有通话使用Messaging Layer Security (MLS)协议，开源且经过独立审计 |
| 即时开会 | 无需登录，创建会议链接并一键邀请任何人，最多50人参加，最长1小时 |
| 屏幕共享与聊天 | 支持屏幕共享、实时聊天，聊天内容同样端到端加密 |
| 日历集成 | 可与Proton Calendar、Google Calendar、Microsoft Calendar集成 |
| 移动与桌面支持 | 支持桌面和移动设备，提供高清音视频 |

## 洞察总结

Proton Meet的核心价值主张是**隐私优先**——在AI时代和数据泄露频发的背景下，提供一个没有人（包括Proton本身）能够访问用户对话的视频会议解决方案。其端到端加密技术、开源协议、以及与Proton生态系统的深度集成，使其成为对隐私有高要求的个人和企业的理想选择。
