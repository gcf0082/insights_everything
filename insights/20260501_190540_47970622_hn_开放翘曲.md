## 项目概述

OpenWarp 是一个社区开发的终端 AI 工具，基于 Warp 终端构建，为其添加了 BYOP（自带提供商）能力。该项目允许用户在 Warp 终端中使用任意 AI 模型，提供商包括 OpenAI、Anthropic、DeepSeek、Qwen、Ollama、Groq 等主流模型。

## 核心功能

1. **自定义 AI 提供商**: 支持 6 种 API 协议原生路由，用户可以自由组合 Base URL、API Key 和模型，不再受限于特定的模型供应商。

2. **动态提示词模板**: 基于 minijinja 模板引擎，可根据当前工作目录、语言、角色实时渲染系统提示词。

3. **隐私优先**: 所有凭证仅保存在本地，默认不上传云端，不收集遥测数据。

4. **多语言界面**: 原生支持简体中文与英文 UI。

5. **完全开源**: 采用 AGPL-3.0 / MIT 双许可，代码全部公开。

## 技术架构

- **协议支持**: OpenAI / OpenAI Responses / Anthropic / Gemini / Ollama / DeepSeek
- **推理思考**: 支持 DeepSeek reasoning_content、Claude thinking、Gemini 多轮回传
- **许可证**: AGPL-3.0 / MIT
- **状态**: 早期开发阶段，尚未发布正式版本

## 背景信息

OpenWarp 是基于 Warp 开源代码的社区分支，与 Warp 官方公司无附属关系。该项目旨在为用户提供完全掌控 AI 模型的能力，保留 Warp 全部交互体验的同时，将 AI 层完全开放。
