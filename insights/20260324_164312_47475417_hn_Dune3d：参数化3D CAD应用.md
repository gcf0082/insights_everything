## 洞察摘要

Dune3d 是一个新兴的开源参数化 3D CAD 应用程序，基于 SolveSpace 开发。该项目由开发 Horizon EDA 的团队打造，核心定位是提供更友好的用户体验，同时保留参数化建模的核心能力。

### 核心特性

- 支持 STEP 文件导入
- 提供倒角（fillets）和倒圆角（chamfers）功能
- 基于 SolveSpace 的求解器
- 比 FreeCAD 更加轻量（Flatpak 安装包约 33MB vs FreeCAD 的 354MB）
- 采用 GTK4 框架开发
- 支持 Space 键快速命令访问

## 讨论热点

### 1. 开源 CAD 工具对比

评论中列举了多款开源 3D CAD 替代方案：

| 类别 | 工具 | 备注 |
|------|------|------|
| 代码建模 | CadQuery | Python 库 |
| 代码建模 | build123d | 基于 Python |
| 代码建模 | OpenSCAD | 支持"氛围编程" |
| GUI 应用 | FreeCAD | 功能强大但学习曲线陡峭 |
| GUI 应用 | SolveSpace | 工作流优秀但缺少倒角功能 |
| GUI 应用 | libfive | 使用 Guile Scheme |
| GUI 应用 | Truck | Rust 内核 |
| GUI 应用 | Brl-cad | 历史悠久 |
| 浏览器端 | JSCAD | 支持参数化定制 |

### 2. Dune3d 与 SolveSpace 的关系

Dune3d 使用 SolveSpace 的几何求解器，但进行了重要改进：
- 增加了 STEP 文件导入支持
- 实现了倒角和倒圆角功能
- 提供了更易上手的用户界面

### 3. FreeCAD 的用户体验争议

关于 FreeCAD 的可用性，社区存在明显分歧：

**负面评价：**
- 用户体验被认为是"最差的"
- 功能难以寻找
- 选择物体等基础操作不顺畅
- 需要稀疏的草图（每个草图只包含一个封闭图形）

**正面评价：**
- v1.1 版本有显著改进
- Part Design 工作台工作流程与 SolidWorks 类似
- 对于认真学习的用户已足够可靠

### 4. AI 辅助 CAD 编程

有用户分享了使用 LLM（如 Claude）配合 CadQuery 进行 3D 建模的经验，取得了不错的效果。

### 5. 安装与部署

- 源码编译需要多个依赖库
- 提供 Flatpak 安装方式（推荐）
- macOS 通过 Homebrew 可快速安装
- Windows 可下载预编译可执行文件
