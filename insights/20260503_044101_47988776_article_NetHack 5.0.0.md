# NetHack 5.0.0 发布说明

**洞察链接**: https://nethack.org/v500/release.html

**基本信息**:
- **游戏名称**: NetHack
- **版本**: 5.0.0
- **发布日期**: 2026年5月2日
- **发布团队**: NetHack DevTeam

---

## 概述

NetHack 5.0 是对地牢探索游戏 NetHack 的一次重大增强。NetHack 是 Rogue 和 Hack 的远距离后裔，也是 NetHack 3.6 的直接继承者。

作为 `.0` 版本，NetHack 5.0.0 可能存在一些bug。开发团队欢迎建设性建议、GitHub pull request 和 bug 报告。

---

## 主要特性与改进

### 架构改进

NetHack 5.0 进行了多项通用架构改进：

1. **C99 标准兼容**: 源代码符合 C99 标准
2. **跨平台编译支持**: 移除了在某一平台和操作系统上构建后无法在另一平台运行的障碍，实现了跨编译功能
3. **Lua 替代方案**: 将原来的 "yacc and lex" 基于的关卡编译器、地牢编译器和 makedefs 工具的文本处理功能替换为 Lua 文本替代方案，这些替代方案在游戏运行时加载和处理

### 其他信息

- 共有超过 3100 处修复和更改，具体可查看 `doc/fixes5-0-0.txt` 文件
- 旧的存档文件和骨骼文件将无法在 NetHack 5.0.0 中使用

## 验证与支持

### 校验和验证

从 nethack.org 下载的二进制文件的 sha256 校验和可在 Windows 平台上使用以下命令验证：

```bash
certUtil -hashfile nethack-500-win-x64.zip SHA256
certUtil -hashfile nethack-500-win-arm64.zip SHA256
```

### 文件路径查询

在大多数平台上，可使用以下命令查看 NetHack 可能使用的各种文件位置：

```bash
nethack --showpaths
```
