## 项目概述

korb 是一个用 Haskell 手写的命令行工具，允许用户以编程方式使用 REWE API 创建购物篮并下单到当地的 REWE 超市取货。该 CLI 主要设计供 AI 代理使用，所有输出均为 JSON 格式，便于自动化处理。

## 核心功能

1. **用户认证** - 通过 REWE PKCE 浏览器流程进行身份验证
2. **店铺管理** - 搜索和设置取货门店
3. **商品搜索** - 支持按名称或 EAN 条码搜索产品，可筛选有机、本地、素食等属性
4. **收藏夹管理** - 管理常用商品列表
5. **购物篮操作** - 添加、查看和管理购物篮商品
6. **下单取货** - 选择取货时间slots并完成结账
7. **订单管理** - 查看、获取和取消订单
8. **建议功能** - 根据购买频率建议商品以达到免取货费门槛
9. **电子收据** - 列出和下载数字收据 (eBons)

## 技术亮点

- **形式化验证**: 使用 Lean 4 对建议引擎进行数学证明，确保5个属性正确：建议商品具有正购买频率、以降序排列、来自有货商品、排除已在篮中商品、遵守数量限制
- **Differential Random Testing**: Haskell 测试套件中的 DRT 桥接器生成随机输入，运行两个实现并验证输出一致性
- **反向工程 API**: 包含 REWE 产品和结账 API 的 OpenAPI 规范

## 安装方式

- **推荐**: 从 GitHub Releases 下载预编译的 macOS/Linux 二进制文件
- **源码编译**: 需要 GHC 9.12+ 和 Cabal，以及 REWE mTLS 客户端证书

## 使用场景

作者描述的典型 E2E 流程：
1. 通过 Siri 快捷指令将商品添加到共享 markdown 文件
2. 每周告诉 AI 代理使用 korb 购买杂货
3. 代理使用 `korb orders history` 生成常用商品模板
4. 调整建议、搜索商品、添加到购物篮
5. 代理确认时间slot并完成结账
6. 清理购物列表
