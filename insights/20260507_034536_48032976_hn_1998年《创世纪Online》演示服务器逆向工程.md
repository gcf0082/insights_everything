## 核心内容总结

### 项目概述

经过10年的断断续续工作，作者发布了1998年Ultima Online演示服务器的完整逆向工程成果。从MSVC x86编译的约5000个函数被反汇编并翻译成可移植的C99代码，每个函数都经过逐条指令与二进制文件对比验证。

### 技术细节

- **UoDemo.exe**: 1998年10月发布的"Ultima Online: The Second Age"扩展包中包含的独立演示版本，捆绑了客户端和完整的服务器代码Windows移植版
- **编译环境**: Microsoft Visual C++ 5.0 (Visual Studio 97)，使用Pre-C++98 dialect的C++
- **逆向工具**: radare2
- **符号来源**: 通过实验性Linux版UO客户端1.25.37（自带C++符号）推断命名

### 主要发现

1. **类层次结构**: CEntity (0x10) → CResourceEntity (0x1C) → CItem (0x50) → CContainer (0x5C) → CMobile (0x37C) → CPlayer (0x458)

2. **修复内容**:
   - 稳定性问题（崩溃、溢出、未初始化变量等）
   - 游戏机制问题（技能提升、声望/恶意方向、生成密度等）
   - 恢复被禁用的生成系统和衰减系统

3. **生态系统恢复**: 著名的已退休生态学系统仍存在于代码中，已重新连接捕食者/猎物/食腐动物系统

4. **新增功能**: Meditation、Stealth、Remove Trap技能（1999年2月添加）

5. **客户端支持**: 扩展支持从1.25.30到5.0.9.1的所有客户端
