## 项目概述

**Charcuterie** 是一个视觉 Unicode 字符浏览器，用户可以通过绘制形状来搜索相似的 Unicode 字符。该工具使用 SigLIP 2 模型进行视觉嵌入，将字符光栅化后进行相似度匹配。

---

## 核心功能

1. **绘制搜索**：点击铅笔图标可以绘制形状来匹配字符
2. **下拉菜单浏览**：通过下拉菜单浏览字符
3. **文本搜索**：直接输入文本搜索
4. **radial glyph wave 动画**：视觉动画展示字符相似性
5. **音效反馈**：自然思维音效

## 主要反馈和见解

### 积极评价

- **优秀的视觉设计**：radial glyph wave 动画非常酷
- **实用的搜索功能**：可以 sketch 需要的字符然后缩小搜索范围
- **出色的嵌入技术**：能够通过手绘字符获得精确匹配
- **UX 直观易用**：已加入用户书签

### 问题和建议

- **空格搜索问题**：默认会 trim 输入，导致无法搜索空格 → 已修复
- **动画过多**：建议添加减少动画的选项 → 已添加开关
- **回退功能**：期望能够通过反转坐标来"返回"上一个字符
- **日文汉字支持不完整**：部分 CJK 字符无法搜索
- **名称记忆问题**：建议使用更易记的名称
- **移动端显示**：iPhone 上字符偏小

### 技术讨论

- 使用 SigLIP 2 进行视觉嵌入
- 将字符光栅化后进行嵌入匹配
- 完全运行在浏览器端

## 用户使用场景

1. 寻找特定样式的箭头字符
2. 在文件名中替换"禁用"字符
3. 探索相似图标
4. 查找 Unicode 字符
