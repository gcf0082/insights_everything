## 概述

这是 Hacker News 每月一次的 "What are you working on?" 主题帖，开发者们分享他们正在进行的项目。这是 HN 社区的传统活动，展示了开发者们的 side projects、副业项目和实验性想法。

## 项目分类汇总

### 1. 音乐与音频相关

- **Drum DSL**: 为鼓手设计的乐谱领域特定语言（DSL），支持将文本渲染为乐谱 PDF，并可作为游戏游玩（类似 DrumMania/DTXMania）
- **Underscore**: 基于 Web 的 SDK 和浏览器沙盒，用于使用 LLM 创建程序化音乐系统，基于 SuperCollider
- **Tiled Words**: 每日益智单词游戏，最近新增了用户登录功能

### 2. 开发工具与效率

- **Python 替代语言**: 一个全新的编程语言，基于独立的寄存器 VM，支持并发（类似 Erlang 的绿色进程），静态类型但借鉴 Python 语法
- **Hitomi**: 日志/数据分析工具，以恒定时间 O(1) 加载任意格式文件
- **nah.build**: 编程代理的权限层工具，可自动分类 Bash 调用为安全、模糊或危险
- **mdspec.dev**: Git 原生的 MD 文件同步管道，将 Markdown 文件同步到项目管理工具
- **Superlog**: Sentry 替代品，自动添加日志并修复发现的 bug

### 3. Web 应用与服务

- **cardcast.gg**: 通过网络摄像头远程玩 Magic: The Gathering 的平台，最近从传统计算机视觉迁移到基于 Transformer 的模型
- **Drawers**: macOS 应用，为每个项目提供独立的 dock、space 和窗口
- **Briefin**: 将任何主题转换为新闻简报的工具，已支持 Hacker News 个性化每日摘要
- **OrcaMark**: 为电子纸设备（如 Kindle）简化网页内容阅读的工具
- **Finbodhi**: 财务追踪应用，支持复式记账，本地优先

### 4. 游戏开发

- **Balls**: 已在 Steam 发布的链式反应游戏
- **21点算牌工具**: 教用户如何算牌以及识别可赢游戏的工具，采用类似 Duolingo 的掌握式学习方法

### 5. 硬件与物联网

- **杠铃传感器**: 基于 IMU 技术的杠铃速度和路径追踪传感器，用于力量训练
- **全息冲浪预报显示器**: 基于 ESP32-s3 和 "pepper's ghost" 幻象技术的全息显示设备

### 6. AI/机器学习

- **RVW**: 标准 Transformer 模型的改编版本，支持在线持续学习而不会发生灾难性遗忘
- **Trilogy**: 流式可组合 SQL 变体，正在探索 AI 原生的数据创作/探索体验

### 7. 其他项目

- **X/D Loom**: 为汽车爱好者创建汽车接线图的工具
- **The Flip Museum**: 芝加哥的弹珠机博物馆，即将开业
- **钩针数学**: 从数学角度探索钩针编织

## 趋势与观察

1. **LLM/AI 相关项目数量明显增加**: 许多项目都在探索如何将 AI/LLM 集成到现有工具中
2. **本地优先应用兴起**: 如 Finbodhi 等本地优先的数据应用受到关注
3. **开发者工具多样化**: 从权限管理到日志追踪，开发者正在构建更细分的工具
4. **跨平台与硬件结合**: 硬件项目（如传感器、全息设备）与软件结合的趋势明显
5. **个人项目商业化**: 多个项目已经从个人 side project 发展到公开测试或发布
