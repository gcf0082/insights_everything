## 洞察摘要

### 事件概述

Cloudflare 的安全系统错误地将 archive.today（一个流行的网页归档服务）标记为"C&C/Botnet"（命令控制服务器/僵尸网络），导致通过 Cloudflare 的 1.1.1.2 DNS 服务无法解析该域名。

### 背景信息

- **archive.today** 是一个广泛使用的网页存档服务，允许用户保存网页的快照以供将来参考
- **1.1.1.2** 是 Cloudflare 提供的公共 DNS 服务，主打隐私保护
- 这不是 archive.today 第一次与 Cloudflare 产生冲突，之前曾有用户报告该服务对其博客发动 DDoS 攻击

### 社区反应

该话题在 Hacker News 上引发热烈讨论，获得超过 408 点支持和 265 条评论。许多用户对 Cloudflare 的误报机制表示担忧，认为这可能会影响重要的互联网归档服务。
