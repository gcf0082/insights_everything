## 核心内容总结

### 一、为什么选择 Obsidian？

Obsidian 是一款强大的笔记软件，作者强烈推荐给任何希望进行知识管理的人。主要优势包括：

1. **本地 Markdown 文件存储**
   - Markdown 是开放格式，可在多种应用中使用
   - 保险库（Vault）仅由笔记和文件夹组成
   - 所有数据保存在本地设备上，文件归用户所有
   - 避免"应用陈腐化"（enshittification）风险

2. **强大的链接功能**
   - 支持使用 `[[ ]]` 创建笔记间的相互链接
   - 非常适合创建相互关联的内容

3. **高度可定制**
   - 用户可以根据自己的使用场景自定义
   - 拥有庞大的社区，可以找到或创建各种插件

4. **跨平台体验一致**
   - 桌面端和移动端功能一致，无功能限制

### 二、使用建议

- **保持简单**：探索可能性但设定界限，避免 FOMO 营销和焦虑
- **工具服务于目标**：Obsidian 应帮助你完成其他工作，而不是成为负担

### 三、作者的使用方式

1. **内容创作**：撰写博客文章和每月邮件通讯
2. **知识管理**：收集、总结信息（采用"自下而上"方法）
3. **项目管理**：管理个人项目和健康等生活方面
4. **媒体追踪**：收集书籍、视频游戏、YouTube 视频信息（Obsidian Bases 很好用）

**个人系统设计**：
- 避免自定义主题
- 尽量少用插件，优先使用核心插件
- 采用 Zettelkasten 和 Evergreen notes 理念
- 自下而上的组织方式

### 四、同步与备份

- 使用 Google Drive 同步（Windows 桌面端、Android 端）
- 定期备份到 GitHub

### 五、其他功能

- **Graph View（关系图谱）**：虽然社交媒体上常见，但日常使用中实用性有限
- **Canvas（画布）**：无限画布功能，可视化笔记，但使用频率不高

### 六、类似工具

作者同时使用的工具：
- **Notion**：协作内容/数据库（看板）
- **Logseq**：每日笔记/日记

其他值得关注的笔记应用：
- Capacities、Octarine、SilverBullet、SiYuan、AnyType、Affine

---

## 关键要点

1. Obsidian 的核心价值在于本地存储、Markdown 格式和文件所有权
2. 不必追求复杂的系统，简单的文件夹管理同样有效
3. 保持插件和主题的最小化，建立长期可持续的系统
4. 同步和备份策略很重要，建议使用云端同步+Git 备份
