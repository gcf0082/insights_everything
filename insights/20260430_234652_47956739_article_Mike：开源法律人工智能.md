## Mike 产品概述

Mike 是一个开源法律人工智能平台，旨在作为 Harvey 和 Legora 的开源替代方案。该平台提供法律事务所所需的所有功能，无需企业合同即可使用。

## 核心功能

### 1. 助手（Assistant）

一个聊天界面，可以：
- 读取文档内容
- 提供逐字引用
- 运行多步骤工作流
- 端到端起草和编辑合同
- 支持接入 Claude 或 Gemini API 密钥

### 2. 项目（Projects）

以事项（matter）为作用域的工作空间。用户可以上传信用协议、SPAs、租赁文件和尽职调查包；助手可以在所有对话和文档中保持完整的上下文。

### 3. 表格审查（Tabular Review）

电子表格风格的提取功能，可并行处理数百份文档。每个单元格都可验证地引用回原始页面的引文，无幻觉答案或失效链接。

### 4. 工作流（Workflows）

将经过验证的提示保存为可重用的工作流，包括：
- 信用协议摘要
- 控制权变更审查
- 符合 CP 检查清单
- 可创建全firm范围的模板，一键运行

## 开源优势

### 1. 零许可费用

- 无供应商锁定
- 无按席位定价（不会每年涨价）
- 仅需支付模型 API 费用（使用 Claude 或 Gemini）

### 2. 可自托管设计

- 部署在您自己的基础设施内
- 文档永不离开您的安全边界
- 合规、数据驻留和特权完全受您控制

### 3. 可分叉和扩展

- 添加特定实践的工作流
- 集成您的文档管理系统
- 连接您自己的引用引擎
- 代码库完全由您塑造

### 4. 代码审计

- 无黑箱操作
- 透明了解提示如何构建、引用如何解析、数据如何流动
- 代码开放、提示工程开放

## 部署方式

- **云端版本**：可免费注册使用
- **自托管版本**：可从 GitHub 仓库克隆部署

## 总结

Mike 是一个具有竞争力的开源法律 AI 解决方案，提供了与商业竞品（Harvey、Legora）相当的功能，同时具有以下优势：零许可成本、完全的数据控制、透明的开源代码库。对于希望避免供应商锁定、注重数据主权和成本控制的法律事务所来说，是一个值得考虑的替代方案。
