## 项目概述

remove-ai-watermarks 是一个用于从AI生成的图像中移除水印和元数据的命令行工具和Python库。该工具可以处理可见水印（如Google Gemini的Nano Banana闪光标志）和不可见水印（如SynthID、StableSignature、TreeRing），同时还能剥离触发社交平台"AI生成"标签的元数据。

## 核心功能

### 1. 可见水印移除
- 支持移除Gemini/Nano Banana的闪光标志
- 使用反向Alpha混合技术
- 离线、确定性方法，速度约0.05秒/图，无需GPU

### 2. 不可见水印移除
- 支持移除SynthID、StableSignature、TreeRing
- 基于扩散的再生技术
- 默认使用SDXL管道（2026年5月起）
- 可选的人性化模拟：胶片颗粒和色差

### 3. AI元数据剥离
- 移除EXIF标签（提示词、种子、模型哈希等）
- 移除XMP DigitalSourceType（触发Instagram、Facebook、X显示"AI生成"标签）
- 剥离PNG文本块（ComfyUI工作流、 AUTOMATIC1111参数）
- 移除C2PA内容凭证（Google Imagen、OpenAI DALL-E、Adobe Firefly）

### 4. 其他功能
- 智能人脸保护：自动检测并融合人脸，防止AI扭曲
- 批处理支持：可处理整个目录
- 三阶段NCC水印检测：带置信度评分

## 技术原理

### 可见水印移除
Google Gemini使用Alpha混合添加可见闪光标志：
```
watermarked = α × logo + (1 − α) × original
```

工具使用已知的Alpha图进行反向处理：
```
original = (watermarked − α × logo) / (1 − α)
```

三阶段NCC检测器动态找到水印位置和缩放。

### 不可见水印移除
使用扩散再生管道（默认SDXL）：
```
图像 → 调整到~1024px → 编码到潜在空间(VAE)
→ 添加受控噪声（前向扩散）
→ 去噪（反向扩散，约50步，强度0.05）
→ 解码回像素(VAE) → 上采样到原始分辨率
```

### 元数据剥离
解析各层（EXIF、XMP、PNG、C2PA），移除AI相关字段，保留标准元数据（作者、版权、标题）。

## 安装和使用

### 推荐安装方式
```bash
# 使用pipx
pipx install git+https://github.com/wiltodelta/remove-ai-watermarks.git

# 或使用uv
uv tool install git+https://github.com/wiltodelta/remove-ai-watermarks.git
```

### 基本用法
```bash
# 移除所有水印（可见+不可见+元数据）
remove-ai-watermarks all image.png -o clean.png

# 处理整个目录
remove-ai-watermarks batch ./images/ --mode all

# 仅移除可见水印（快速，离线）
remove-ai-watermarks visible image.png -o clean.png

# 仅移除不可见水印（需要GPU）
remove-ai-watermarks invisible image.png -o clean.png --humanize 4.0

# 检查/剥离AI元数据
remove-ai-watermarks metadata image.png --check
remove-ai-watermarks metadata image.png --remove
```

### 威胁模型
该工具针对已分发的AI图像，防御自动检测系统（社交平台"AI生成"标签、第三方分类器、内容策略过滤器）。它**不会**匿名化生成过程。

特别地，SynthID-Image v2嵌入136位有效载荷，可能编码用户/会话标识符。如果原始水印文件曾经通过提示符来源控制的系统，Google保留将原始文件链接到生成账户的能力。从持有的副本中剥离水印不会消除Google的服务器端记录。

### 适用场景
- 您自己生成了图像，希望将其作为自己的作品发布
- 您正在进行安全/鲁棒性评估
- 您正在保护艺术或历史记录以对抗误报"AI生成"标签

### 不适用场景
- 生成图像后期望通过移除水印来向Google匿名化
- 分发AI生成内容同时声称是人类创作

## 技术要求

- Python ≥ 3.10
- 可见水印移除/元数据剥离：仅CPU，无需GPU
- 不可见水印移除：推荐GPU（CUDA或MPS），CPU也可运行（较慢）
