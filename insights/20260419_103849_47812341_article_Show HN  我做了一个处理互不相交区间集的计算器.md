## 工具概述

Interval Calculator（区间计算器）是一个基于**区间并集运算**的计算器，而非传统的实数计算器。它是[区间并集算术](https://www.ime.usp.br/~montanhe/unions.pdf)的实现。

### 核心概念

- **区间 [a, b]**: 表示所有介于 a 和 b 之间（包含端点）的实数集合
- **区间并集 [a, b] U [c, d]**: 表示一组互不相交的区间集合

## 主要特性

### 1. 区间并集算术的优势

区间并集算术是传统区间算术的扩展，具有以下优势：

- **保持闭合性**: 运算结果仍然是区间并集
- **支持除以含零区间**: 可以安全地除以包含零的区间

示例：
```
➤ 2 / [-2, 1]
[-∞, -1] U [2, +∞]
```

### 2. 包含性质（Inclusion Property）

区间并集算术的一个重要特性是**包含性质**：如果从每个输入区间并集中任意选取一个实数，然后在实数域上计算相同的表达式，结果必定包含在输出区间并集中。这使得该计算器非常适合处理**不确定性**计算。

示例：
```
➤ 50 * (10 + [-1, 1])
[450, 550]
```

### 3. 全精度模式（Full Precision Mode）

- 使用 IEEE 754 双精度浮点数的向外舍入（outward rounding）
- 结果区间**保证包含**使用无限精度计算的真实值
- 可以解决著名的浮点数精度问题：`0.1 + 0.2 = [0.29999999999999993, 0.3000000000000001]`

## 支持的运算和语法

### 基本运算

| 运算 | 语法 | 示例 |
|------|------|------|
| 加法 | A + B | `[90, 100] + [-2, 2]` → `[88, 102]` |
| 减法 | A - B | `[14, 16] - [8, 12]` → `[2, 8]` |
| 乘法 | A * B | `[-5, 10] * [2, 4]` → `[-20, 40]` |
| 除法 | A / B | `[2, 4] / [-1, 2]` → `[-∞, -2] U [1, +∞]` |
| 幂运算 | A ^ B | `[2, 3] ^ [-2, 3]` → `[0.1111, 27]` |

### 支持的函数

- **数学函数**: `log`, `log2`, `log10`, `exp`, `sqrt`, `sin`, `cos`, `tan`, `asin`, `acos`, `atan`
- **辅助函数**: `lo(A)` - 下界, `hi(A)` - 上界, `hull(A)` - 凸包, `abs(A)` - 绝对值, `sqinv(A)` - 平方倒数
- **极值函数**: `min(A, B)`, `max(A, B)`
- **常数**: `pi`, `e`, `inf` / `∞`

### 语法说明

1. 可以使用方括号语法输入区间：`[1, 2]`
2. 也可以输入裸数字（视为宽度为零的区间）：`3.14` = `[3.14, 3.14]`
3. 支持区间嵌套：`[0, [0, 100]]` → `[0, 100]`
4. 使用 `U` 表示区间并集

## 精度模式对比

| 特性 | 全精度模式 | 普通模式 |
|------|------------|----------|
| 输入解释 | 解释为包含最近 IEEE 754 值的最小区间 | 解释为宽度为零的退化区间 |
| 输出显示 | 显示所有可用十进制位数 | 最多显示 4 位小数 |

## 应用场景

1. **不确定性计算**: 表示数值的区间范围
2. **浮点数精度问题**: 解决 `0.1 + 0.2 ≠ 0.3` 的问题
3. **科学计算**: 需要考虑测量误差的计算
4. **区间分析**: 用于数值分析和算法验证

## 总结

Interval Calculator 是一个强大的数学工具，它通过区间并集算术提供了一种处理不确定性和浮点数精度问题的优雅解决方案。其核心价值在于保证计算结果的包含性，使得即使在存在数值不确定性的情况下，也能获得可靠的结果区间。
