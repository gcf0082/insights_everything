## 项目概述

**legalize-es** 是一个创新的开源项目，将西班牙立法以Git仓库的形式存储和管理。该项目将每部法律转换为Markdown文件，每次法律改革作为一次Git提交记录，形成完整的版本历史。

## 核心特性

1. **大规模法律数据库**: 收录超过8,600部西班牙法律
2. **完整改革历史**: 记录自1960年以来的所有法律改革
3. **官方数据源**: 来自西班牙官方公报(BOE)的公开数据API
4. **版本控制**: 利用Git跟踪每部法律的演变
5. **标准化格式**: 采用Markdown格式存储，YAML frontmatter存储元数据

## 数据结构

每个法律文件包含以下元数据：
- titulo (标题)
- identificador (标识符)
- pais (国家)
- rango (法律级别)
- fecha_publicacion (发布日期)
- ultima_actualizacion (最后更新)
- estado (状态)
- fuente (来源链接)

## 法律类型

项目包含以下类型的西班牙立法：
- 西班牙宪法
- 有机法律
- 普通法律
- 皇家法令
- 皇家立法法令

## 使用示例

```bash
# 克隆仓库
git clone https://github.com/EnriqueLop/legalize-es.git

# 查看宪法第135条当前内容
grep -A 10 "Artículo 135" spain/BOE-A-1978-31229.md

# 查看该法律的修改历史
git log --oneline -- spain/BOE-A-1978-31229.md

# 查看2011年预算稳定改革的具体变更
git diff 6660bcf^..6660bcf -- spain/BOE-A-1978-31229.md
```
