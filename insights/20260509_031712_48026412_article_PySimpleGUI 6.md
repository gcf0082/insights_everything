## 项目概述

PySimpleGUI 是一个面向人类的 Python GUI 开发框架，于2018年启动，声称是"顶级 Python 应用程序开发环境"。该框架将 tkinter、Qt、WxPython 和 Remi 整合成一个简单、直观且有趣的开发体验，适用于业余爱好者和专业用户。

## 安装方式

```bash
# 从PyPI安装
python -m pip install PySimpleGUI

# 从GitHub直接安装
python -m pip install --upgrade https://github.com/PySimpleGUI/PySimpleGUI/zipball/master
```
