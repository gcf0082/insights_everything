---

## 核心洞察总结

### 一、2025年11月：转折点

2025年11月是LLM发展的关键转折点，尤其在编程领域。以下重要事件发生在11月：

1. **模型排名频繁更替**：所谓"最佳模型"的称号在Anthropic、OpenAI和Google之间交替了5次：
   - 11月初：Claude Sonnet 4.5（9月29日发布）被认为是最佳
   - 随后被GPT-5.1超越
   - 然后是Gemini 3
   - 接着是GPT-5.1 Codex Max
   - 最后Anthropic凭借Claude Opus 4.5重新夺回榜首

2. **编程代理真正成熟**：OpenAI和Anthropic通过**可验证奖励强化学习（RLVR）**显著提升了代码质量。编程代理从"经常能用"跨越到"基本可用"，可以日常使用而无需频繁修复错误。

3. **重要里程碑**：11月24日，一个名为"Warelay"的项目提交了第一个commit，这个项目后来演变为OpenClaw。

### 二、2025年12月至2026年1月：LLM狂热期

假期期间，许多开发者开始探索新模型和编程代理的能力。Simon Willison本人也经历了短暂的"LLM狂热"，启动了多个雄心勃勃的项目，其中一个是micro-javascript——一个用Python实现的JavaScript解释器。虽然演示很酷（JavaScript → Python → Pyodide → WebAssembly → JavaScript → 浏览器），但实际上并没有实际需求。

### 三、2026年2月：OpenClaw崛起

1. **OpenClaw爆发式增长**：Warelay项目在12月和1月经历多次更名（CLAWDIS → CLAWDBOT → Clawdbot → Moltbot → OpenClaw），到2月已成为备受关注的"个人AI助手"。

2. **"Claw"术语流行**：这类AI助手有了通用名称——"Claws"（包括NanoClaw、ZeroClaw等）。

3. **Mac Mini热销**：硅谷的Mac Mini开始售罄，因为人们购买它来运行自己的Claw。Drew Breunig戏称这是新的数字宠物，Mac Mini是Claw的完美"鱼缸"。

4. **Gemini 3.1 Pro表现出色**：该模型能够生成极其逼真的"鹈鹕骑自行车"图像，甚至包含细节如篮子里的鱼。

### 四、2026年4月：开源模型爆发

1. **Gemma 4发布**：Google发布了Gemma 4系列，这是来自美国公司的最强大的开源权重模型之一。

2. **GLM-5.1震撼发布**：中国AI实验室智谱AI发布了GLM-5.1，这是一个拥有7540亿参数、1.51TB的开源权重模型，性能非常出色，但需要强大的硬件才能运行。

3. **Qwen3.6-35B-A3B令人惊艳**：这个200亿参数的模型（20.9GB）可以在笔记本电脑上运行，在"鹈鹕骑自行车"测试中甚至比Claude Opus 4.7表现更好。

### 五、过去六个月的两大主题

1. **编程代理真正成熟**：从"可用"变为"好用"，可作为日常工具

2. **本地模型超出预期**：在笔记本电脑上运行的模型虽然比顶级模型弱，但开始展现惊人的能力

## 总结

过去六个月是LLM领域快速变革的时期。编程代理从实验性工具转变为实用的日常助手，而开源本地模型的能力也远超预期。中国AI实验室（智谱GLM、阿里巴巴Qwen）在开源领域取得了显著进展，与美国科技巨头形成了激烈竞争。
