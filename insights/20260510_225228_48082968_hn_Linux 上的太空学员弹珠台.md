## 主题概述

这篇文章介绍了一款在 Linux 上运行的 Space Cadet Pinball（太空学员弹珠台）游戏重现项目，作者通过反编译 exe 文件而非查看原始源代码的方式完成了这一重建。

## 关键洞察

### 1. 技术实现
- 作者通过反编译 exe 文件进行重建，完全没有参考原始源代码，实现了"盲重建"
- 这种方法类似于"净室"（clean room）开发模式，但讨论中对其合法性存有疑问
- 微软为 Pinball 提供了调试符号，简化了逆向工程过程
- 项目已移植到多个游戏主机平台，也有浏览器版本可玩

### 2. 法律与开源讨论
- 评论区讨论了使用 AI 重写反编译代码是否能构成"净室"开发——多数意见认为 AI 输出直接关联其读取的代码，无法满足净室要求
- 类似的反编译项目（如 GTA3/Vice City 的 re3）曾被起诉，最终庭外和解
- 文章提到了"源码 FLOSS 托管"的概念：软件闭源销售，停止售后自动变为开源
- 源码托管在大型企业软件合同中较为常见

### 3. 游戏历史背景
- Space Cadet Pinball 最初是 Full Tilt! Pinball 的一部分
- 最初作为 Microsoft Plus! 95 的组成部分，后被集成到 Windows NT 4.0、2000、Me 和 XP 中
- Windows Vista 起被移除，此后再未回归
- 原本差点被 DOOM 替代（称为 GLUEM 项目），但最终选择了弹珠台游戏

### 4. 物理化改造的挑战
- 有人提议将游戏制作为真实物理弹珠台，但面临技术难题
- 游戏中有一个球道穿过弹柱下方，而弹柱在真实设备中需要向下延伸较长的空间
- 评论中提出了多种解决方案，如使用提升式演奏台、让不同的球从出口出来等
- 有人提到曾向 Stern 公司提出这个想法，但那已是 21 年前的事

### 5. 游戏体验对比
- 部分用户认为 Space Cadet 相比其他弹珠游戏（如 Pinball Dreams、Pro Pinball）图形和音效较差
- 也有人认为 Space Cadet 设计出色，游戏手感好，元素多样性高
- 怀念过去系统中内置小游戏的传统，如今智能手机和操作系统已很少这样做了
- 技能型玩家确实会使用倾斜键来微调球的走向

### 6. 平台分发
- 项目可通过 Flatpak、Snap 等方式在 Linux 上安装
- 浏览器版本: https://pinball.alula.me/
- GitHub 仓库: https://github.com/k4zmu2a/SpaceCadetPinball
