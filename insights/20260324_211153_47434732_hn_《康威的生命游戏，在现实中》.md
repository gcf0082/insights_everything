## 内容摘要

这是由Michal Zalewski（lcamtuf）发布的一个极具创意的硬件项目：将康威生命游戏（Conway's Game of Life）带入现实生活。

### 项目概述

作者构建了一个17×17的物理矩阵（289个按钮），每个按钮代表生命游戏中的一个细胞。这些按钮可以按下表示"活细胞"，松开表示"死细胞"。整个系统运行康威生命游戏的标准规则：
- 活细胞如果少于2个邻居则"饿死"
- 活细胞如果有4个或更多邻居则"过度拥挤而死"
- 死细胞如果有正好3个邻居则"复活"

### 技术规格

| 组件 | 详情 |
|------|------|
| **按钮型号** | NKK JB15LPF-JF |
| **矩阵大小** | 17×17 (289个按钮) |
| **微控制器** | Microchip AVR128DA64 |
| **控制方式** | 10kΩ电位器控制速度 (0-10Hz) |
| **成本** | 约1000美元（仅按钮） |
| **外壳** | 木质外壳 |

## 核心洞察

### 1. 硬件设计亮点

- **矩阵扫描驱动**：使用扫描方式驱动LED和检测输入，需要添加MOSFET和晶体管
- **平滑的交互体验**：输入暂停和恢复模拟之间的去抖动处理非常流畅
- **AVR-Dx系列被低估**：评论指出AVR-Dx系列微控制器非常适合这类项目，无需外部元件即可工作

### 2. 成本讨论

这是讨论的热点之一。评论者提出了多种降低成本的方案：

- **电容式感应开关矩阵 + SMD LED**：可以大幅降低成本
- **Cherry MX机械开关 + 0805 LED**：成本会低得多
- **3D打印键帽**：可以进一步降低成本

但作者回应称，主要考虑因素是时间成本，而非单纯材料成本。项目即使不计算迭代设计键帽和制造300个键帽的时间，已经非常耗时。

### 3. 创意价值

这个项目体现了"创客精神"的典型特征：
- 没有"实际用途"但极具创意
- 物理交互带来的满足感远超纯软件实现
- 完美的执行和工艺
