## 项目概述

SUPER ZSNES 是经典 SNES 模拟器 ZSNES 的完全重制版。由 ZSNES 的两位原始开发者联手重新打造，这款采用 GPU 加速的模拟器旨在为用户带来既熟悉又新鲜的体验，同时突破传统模拟器的功能边界。

## 核心功能

### 1. 核心技术升级

- **CPU 和音频核心**: 比原版 ZSNES 更加精确
- **GPU 加速 PPU 核心**: 支持高分辨率 Mode 7 特效和特定游戏的增强功能
- **经典 UI 风格**: 保留飘雪特效，全面升级为高清画质并优化用户体验

### 2. 模拟器功能

- 快进快退功能
- 存档状态保存
- 自动保存历史
- 书签保存
- 金手指作弊码
- 快速加载功能

### 3. 开发理念

- 采用经典开发方式，拒绝" vibe coding"（即兴编程）
- 持续优化和改进

## 超级增强引擎

超级增强引擎（Super Enhancement Engine）是该模拟器的核心特色，允许开发者逐个为游戏添加增强功能。目前已支持 7 款热门游戏，增强功能将随开发进度不断增加。

### 增强功能类型

| 功能 | 说明 |
|------|------|
| 高分辨率 | 非简单的自动拉伸，而是使用内部绘图程序手动绘制高分辨率细节 |
| 纹理/法线贴图 | 为背景添加细节，提升高分辨率视觉效果 |
| 倍频 | 针对频繁出现卡顿的游戏进行超频处理 |
| 宽屏模式 | 在游戏支持的情况下启用宽屏显示 |
| 无损音频替换 | 精选无损音频样本替换原有的高压缩音频 |
| 3D 支持 | 仅支持透视风格 Mode 7，用 3D 高度映射数据替换瓦片 |

**注意**: 增强数据包不含任何 ROM 或受版权保护的内容。用户需自行提供 ROM，请勿向开发者索要 ROM。
